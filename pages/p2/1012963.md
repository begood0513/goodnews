 <p>
  【看中国2022年7月29日讯】截至目前，中共“
  <ok>
   孙力军政治团伙
  </ok>
  ”六大成员已全部受审待判。中共中央政法委秘书长
  <ok>
   陈一新
  </ok>
  近日撰文，将该团伙定性为三大“典型”，并放出重话，强调“政治建设最急迫的问题”，是“彻底肃清孙力军政治团伙
  <ok>
   流毒
  </ok>
  影响”。一场新的政治风暴，正向中共
  <ok>
   政法系统
  </ok>
  袭来。
  <br>
   <br>
    7月28日，落马近11个月的中共政法系“大老虎”
    <span href="https://zh.m.wikipedia.org/zh-hans/%E5%82%85%E6%94%BF%E5%8D%8E">
     傅政华
    </span>
    在长春中级法院受审，他被控受贿等罪，据称敛财1.17亿余元。
    <br>
     <br>
      目前中共官方口中的“孙力军政治团伙”涉及的其他5个“政法虎”孙力军、王立科、龚道安、邓恢林、刘新云均已受审待判，孙力军本月初受审，他被控受贿数额最高，达6.46亿元。
      <br>
       <br>
        此前一天，7月27日，中共中央党校机关报《学习时报》，在其头版发表了一篇署名文章，题为《把捍卫“两个确立”做到“两个维护”作为政法战线政治建设的根本要求》，作者是中共中央政法委秘书长陈一新。他和中共公安部部长王小洪、司法部部长唐一军并称为习近平在中共政法系统的三大亲信。
        <br>
         <br>
          陈一新在文中强调，“政治建设最急迫的问题”，是“彻底肃清孙力军政治团伙流毒影响”。毫无疑问，中共政法系统将面临再一次强力整肃。
          <br>
           <br/>
           令人关注的是，陈一新连续7次强调“肃清”，并将“孙力军政治团伙”定性为三大“典型”，即“对党不忠诚不老实、‘七个有之’的典型，政治问题和经济问题交织、极度腐化堕落的典型，大搞特权、恣意妄为、顶风违纪违法的典型”。
           <br/>
           <br/>
           排名第一的“不忠诚不老实”罪名，显然是指对习近平“不忠诚不老实”。
           <br/>
           <br/>
           而这三大“典型”的定性，似乎也预示着“孙力军政治团伙”成员将面临重判。
           <br/>
           <br/>
           陈一新在文中大谈“肃毒”（肃清流毒）的“紧要性”、“复杂性”、“彻底性”，强调“彻底肃清孙力军政治团伙流毒影响，是确保‘刀把子’牢牢掌握在党和人民手中的必然要求”，“务必将彻底肃清流毒影响作为一项紧迫而严肃的政治任务来抓”。
           <br/>
           <br/>
           需要说明的是，中共口口声声说的“人民”从来和真正的人民没有半点关系，这里的“党和人民”其实指的是习近平。这句话等于说是要确保中共政法系这个“刀把子”掌握在习近平手中。
           <br/>
           <br/>
           不过，这里的潜台词是“刀把子”过去其实并未掌握在习的手中，而是像外界所观察的那样，掌握在江系人马之手。孙力军等人“不忠诚不老实”的罪名，也证明了这一点。他们忠诚的不是习近平，而是江泽民、曾庆红、孟建柱等人。
           <br/>
           <br/>
           陈一新在这篇文章中，不断放出重话。他把孙力军政治团伙称之为“毒瘤”，“流毒影响到一些地方和部门政法干警”，要求务必“正视“毒症”之重、“毒灶”之广”。
           <br/>
           <br/>
           陈一新还要求“线索核查要到位，对中央通报中提及的问题、有关单位回溯揭批中发现的问题线索，必须深挖彻查。‘人、事、案’排查处理要到位，全面排查与孙力军政治团伙密切接触交往的重点人员，严肃查处违纪违法行为”。
           <br/>
           <br/>
           陈一新这番公开表态，发出了强烈的政治信号，意味着中共政法系统还将有一些“老虎”面临落马。
