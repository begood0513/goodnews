 <p>
  【看中国2022年7月18日讯】日本前首相
  <ok>
   <span href="https://www.secretchina.com/news/gb/tag/安倍晋三" target="_blank">
    安倍晋三
   </span>
  </ok>
  7月8日
  <ok>
   遇刺身亡
  </ok>
  ，举世震惊。在全球民众哀悼这一悲剧的同时，中国大陆网络上涌出大量幸灾乐祸、庆祝的留言。然而，还有一些中国人与之相反，她们对于安倍
  <ok>
   去世
  </ok>
  表示哀悼。
  <br>
   <br>
    7月9日，大陆著名表演艺术家吕丽萍，在社交平台上公开发文悼念安倍晋三，引发网友热议。虽然有一些小粉红对这条微博冷嘲热讽，但是更有十几万网友为其点赞。
    <br>
     <br>
      这些网友善良的举动和小粉红粗鄙、幸灾乐祸的语言形成了鲜明的对比。
      <br>
       <br>
        吕丽萍分享了一段安倍晋三弹钢琴的视频。
        <span id="hideid" name="hideid" style="color:red;display:none;">
         <span href="https://www.secretchina.com">
         </span>
        </span>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  她还贴上一段话，“安倍晋三：‘在这世上，每个人都活得这样辛苦，我曾经羡慕过的人，我曾经向往过的人，我曾经爱过的人，我曾经恨过的人。最后我才知道，他们每一个人，其实都和我一样，活得千辛万苦’”。
  <br>
   <br>
    虽然有网友说，这并非安倍所写，但是吕丽萍的哀悼之意，已经表露无疑。
    <br>
     <br/>
     现在的一些年轻人虽然对吕丽萍不一定很熟悉，但是在中国大陆，吕丽萍曾经因为主演过《编辑部的故事》和《激情燃烧的岁月》等众多的影视作品而家喻户晓，她还拿过金鹰奖的视后和金马奖的影后。
     <br/>
     <br/>
     尤其是她在《编辑部的故事》中饰演的”戈玲”这个角色，犀利幽默，获得无数中国人的喜爱。
     <br/>
     <br/>
     7月5日，吕丽萍还晒出了一段烟花视频，当天也是美国的“国庆节”。吕丽萍还在帖文中引用了美国前总统约翰逊·肯尼迪的一段话，来庆祝美国节日。
     <br/>
     <br/>
     吕丽萍发帖地址显示美国。看来，她之所以能够坦然吐露心声，也是因为身处美国这个相对言论自由的环境。
     <br/>
     <br/>
     为吕丽萍而感到高兴！
     <br/>
     <br/>
     当然，和她在一起的还有她的丈夫孙海英，他也是一位著名的表演艺术家。
     <br/>
     <br/>
     中国人有句俗话，“不是一家人，不进一家门”，从吕丽萍和孙海英看来，确实很有道理。
     <br/>
     <br/>
     孙海英因心直口快被网民称为“孙大炮”。在文革极左环境中长大的他，多次发帖批评文革、
     <span href="https://www.secretchina.com/news/gb/tag/毛泽东" target="_blank">
      毛泽东
     </span>
     、质疑共产党的领导。因为正直敢言，孙海英被《环球时报》、《解放军报》以及《红旗》杂志等中共喉舌点名批评，​​​​称其为美国扳倒中国第三种武器的典型代表，并因此被大陆影视圈长达封杀六年之久。
     <br/>
     <br/>
     对于一名演员、一位表演艺术家来说，中共的全面封杀无疑是对孙海英艺术生命的一种扼杀。
     <br/>
     <br/>
     此前不久，孙海英在沉寂数年之后，再度接受媒体采访。这一次，孙海英在接受自由亚洲电台专访时依然保持其”大炮“风格。
     <br/>
     <br/>
     在采访中，记者就徐州八个孩子母亲的事件，向孙海英提问说，”有一部电影叫《盲山》被很多人关注，这部和被拐妇女有关的电影一度也禁片。很多类似的电影都被认为是丑化中国社会。您对中国影视圈这种所谓家丑不可外扬，必须主题正确的这样一个局限性，您有什么看法呢？”
     <br/>
     <br/>
     “孙大炮”发声了：我说过了，你只要政体不自由不开放的话，你就回到闭关锁国了。你闭关锁国你艺术上就不可能说真话，你怎么能允许说真话呢？你自己有黑暗的地方，你多了你。农村生活那个状态了，都猪狗不如了，你还假装吹什么牛啊？当然这跟政治有关系了，但是我说的是实话啊。你艺术作品反映出来的东西，你就不敢贴近生活，如实反映现状了，你怎么敢呢？说出来就。。。像我说真话几句就给你踢一边去了，踢了六年，谁还敢说话？你现在（看看）微博上微信上有吗？没一个敢说真话的了。那你当然。。。是不是这八年？说说，这个现象是不是从八年前开始的？你开始闭关锁国开始的？当然了。你外表很光鲜，里边所有事你都不管，不顾，反正猪狗不如，养猪一样。对人民养猪一样怎么行！
     <br/>
     <br/>
     这“孙大炮”，火力依然很猛！
     <br/>
     <br/>
     记者再度就热门问题向孙海英进行采访，她说“长津湖之水门桥这个电影，上映一周的票房就是25.9亿人民币。您如何看待有很多中国民众对爱国剧的热爱？您怎么看待这个现象呢？
     <br/>
     <br/>
     孙海英：文革样板戏《沙家浜》浜到什么程度？《红灯记》记了多少年？十年，你有它收入高吗？5分钱一张票，多少人看？包场满场，你算起来比你多少倍？这国家体制当时就这样，就觉得什么都不能干。你现在不跟它一样吗？有什么可夸耀的？假造一个故事，胡编一个东西，觉得好。我还不能说，说多了我又怎么的了。那是什么东西？还自己要夸耀呢你。你回头看看《激情燃烧的岁月》里边，援朝和回来部队的整个建制、以及对文革的态度，你就知道了。真正的百分之几十是贴近现实的表达？那不会胡说八道夸耀有些。。。你这叫什么？敢说真实历史的吗？回顾那战役，你敢吗？因为你这个时候跟样板戏没什么区别，十个样板戏，你就推出了一部或者两部（爱国电影），不也是十部吗？全是一群二傻子。我不是贬低他们，是因为我们有一群人愿意这样做，那你就这样做。我将来也可以演一些这样的人，演得比你还像信吗？现在这种状态时局，整个的状态国家这种形态不好，因为太封闭，正好走下坡。（我）再说一遍，艺术上是这样的，正好往下走，走到最低了可以说。所以谁也别去说现在哪个东西好多少个亿，那什么东西多少个亿？戏剧用钱衡量？人物作品用钱衡量，那是东西吗？卖菜的？卖肉啊？哪有这个事情对吧？它是（表现）人，你必须把人表达得那么准确，多少代以后看着还好才行，才准确。
     <br/>
     <br/>
     下面我们再来聊聊中国大陆现在最热的焦点之一。看看老百姓真实的感受和孙海英说的有没有不同。
     <br/>
     <br/>
     一位网友说，“如果暴风雨不可避免，那就让我们这一代一次经历完吧”，听上去充满悲凉。
    </br>
   </br>
  </br>
 </p>
 <p>
  自2021年下半年中国楼市进入寒冬以来，各大房企不断传来爆雷新闻。
  <br/>
  <br/>
  伴随着各大房企爆雷的是，各大城市大量楼盘纷纷停工烂尾，引发一波又一波的业主维权浪潮。
  <br/>
  对于业主来说，掏空六个钱包买的房子一旦烂尾，就等于失去一切。在这种情况下，还要让他们继续为烂尾楼偿还二、三十年房贷，简直岂有此理。
  <br/>
  <br/>
  本来在预售制度下，如果开发商、银行以及政府监管部门都严格遵守相关法律法规，对于那些已经售罄的楼盘，是绝对不可能烂尾的。
  <br/>
  <br/>
  正是由于开发商和银行胡作非为、肆意挪用监管资金，监管部门监管不力，甚至存在贪腐现象，才导致楼盘烂尾。而在中国大陆现行制度下，楼盘烂尾的后果由购房者全部承担，开发商和银行却逍遥法外，无人追责。
  <br/>
  <br/>
  因此，中国大陆多个地方近日开始刮起了一阵停贷风潮。很多被逼无奈的烂尾楼业主豁出去征信不要了，纷纷向银行发出集体停贷告知函，要求强制停贷。
  <br/>
  <br/>
  据网络消息，短短几天的时间，从刚开始的10几个楼盘，到30几个再到50几个，13日已达到百起，后来达到了270多家。业主选择集体停贷，迅速演变成全国性的现象。这个雪球估计会越滚越大，再加上银行业此起彼伏的爆雷，恐怕是已经吹响大风暴来临的前奏了。
  <br/>
  <br/>
  7月13日，中共官媒《证券时报》头版发评论文章：谨防烂尾楼盘停供风险扩散。
  <br/>
  <br/>
  文章开始就说，“近期有烂尾楼盘业主集体决定停止还贷，引得不少烂尾楼盘业主效仿。如果这股风潮蔓延开来，有可能对楼市造成负面冲击，对楼盘销售更为不利，也不利于金融系统稳定。”
  <br/>
  <br/>
  看出来了吗？这是为老百姓担心吗？根本不是，中共政府考虑的依然是楼盘销售、金融稳定、政权稳定。而不是为这些业主着想。
  <br/>
  <br/>
  就像是
  <span href="https://www.secretchina.com/news/gb/tag/孙海英" target="_blank">
   孙海英
  </span>
  所说的那样，“外表很光鲜，里边所有事你都不管，不顾，反正猪狗不如”。
  <br/>
  <br/>
  不过，老子曰：民不畏死、奈何以死惧之？越来越多的人在绝望中，敢于说出自己的心声了。
  <br/>
  <br/>
  网络上，人们的议论异乎寻常的热烈。
  <br/>
  <br/>
  7月13日，一位网友说，“这只是开始，大家准备好了吗？”，26654网友点赞，表示认同。
  <br/>
  <br/>
  还有人说，“天天征信征信的，现在还有一点威慑力吗？上个厕所不洗手都上征信算了”。这条留言获得2万多人支持。
  <br/>
  <br/>
  还有人说，“原来坑死了这么多人，他们要是不发声估计就这样家破人亡了。真是把人不当人” ，有1万多网友点赞，表示支持这种说法。
  <br/>
  <br/>
  一位网友总结说，“人生三大幸事:买房没有买到烂尾楼，出去吃饭没有挨打，钱存银行没有丢”。这番充满辛酸的讽刺，获得 7千多个点赞。
  <br/>
  <br/>
  而另一位网友意味深长的说，“一个人停贷是zf（政府）给我们上征信，一百万人停贷是我们给zf（政府）上征信”。
  <br/>
  <br/>
  有网友介绍，“下半年开始是房地产公司还债的高峰段”，也有人算了一笔账“全国三百多个地级市州，按每个平均50个问题楼盘，这就是1.5W+”。
  <br/>
  <br/>
  不过，也有网友提醒说，“法不责众，但还有一句，枪打出头鸟。总有人会被打掉。弱势群体没有办法”。
  <br/>
  <br/>
  另一位则回复到，“这是断供，不是出头鸟那么简单，你抓一个人，别人就能心甘情愿一辈子供一套烂尾房了？太天真了吧，这涉及到自己下半辈子的利益，没有人能那么容易屈服的”。
  <br/>
  <br/>
  一位网友发表了不同看法，他说“不还钱影响的是你自己的征信，有啥用，银行可不会惯着你”
  <br/>
  <br/>
  其它网友则说，“命都没有了你跟我说征信”。
  <br/>
  <br/>
  其中，最令人伤感的是，一位网友表示，“如果暴风雨不可避免，那就让我们这一代一次经历完吧。”
  <br/>
  <br/>
  老子曰：人不畏死，奈何以死惧之，此乃大威至也！
