 <p>
  【看中国2022年6月15日讯】5月16日至6月7日，中共《人民日报》以“钟声”名义，连续推出8篇系列文章，主题是所谓的“
  <ok>
   美国
  </ok>
  当深刻检讨自身的
  <ok>
   <span href="https://www.secretchina.com/news/gb/tag/人权" target="_blank">
    人权
   </span>
  </ok>
  赤字”，内容包括所谓的“枪声击碎美式人权幻象、贫富分化加剧社会不公、粗暴对待非法移民暴露伪善面目、
  <ok>
   种族主义
  </ok>
  贯穿美国政体、私营监狱酿造人权悲剧、‘黑监狱’滥施酷刑”等等。
  <br>
   <br>
    中共这些宣传文章，痛批美国的人权，称这是美国治理失灵的集中体现。
    <br>
     <br>
      众所周知，人民日报的“钟声”系列国际评论，是“中国之声”的谐音缩写，于2008年推出，评论主题围绕所谓的重要国际问题和涉中问题。
      <br>
       <br>
        就在中共不断炮轰美国人权、社会治安，让人感觉美国简直是社会问题充斥各个角落，人民生活在水深火热之中。
        <br>
         <br>
          仅仅三天之后，6月10日凌晨，多名男子在唐山烧烤店调戏不成、暴力殴打四名女子，震惊整个了中国，消息很快在世界范围内扩散开来。
          <br>
           <br/>
           中共连发8篇评论文章攻击、抹黑美国，美国方面没啥感觉，自己这边却炸锅了！
           <br/>
           <br/>
           现场影片曝光后，这几名男子的野蛮行径引发众怒，相关讨论贴文在很短时间内突破数千万条。 微博上有人写说，“2022年怎么还会发生这种事情？ 给他们判刑，别让任何一人逃脱。”
           <br/>
           <br/>
           还有人讽刺到，“老人倒了不敢扶、码扫不出来没人帮，女人给三个壮汉打得头破血流没几个人去拉，学生打急救对面给说’我感觉你没事啊’，楼里邻居阳性了喊的是’捉羊’..这就是伟大复兴的中国梦么？我梦里也没这么离奇。”
           <br/>
           <br/>
           一些网友们在网络留言中，意外暴露出中国大陆一些惊人内幕，让人深感大陆的治安问题，以及很多人没有挺身而出，其实与中共政府有关。
           <br/>
           <br/>
           一位网友说，““就是真遇到这样的事出不出手，为什么，本人的一个小故事:我听到有人喊抓小偷，我和朋友追上去，小偷已经开始打人了，我们去制止，结果肯定是小偷反抗，我们武力拿下小偷并报警，结果我和朋友被关了一晚上，可气的是和小偷一起关押，警察劝我们和解，第二天体检，被拘留了7天，罪名殴打他人。这就是社会，谁出手，我看到这样的事扭头就走，我不是心狠，是心凉，兄弟们顶我上去，绝对属实。”，也有人补充说，“许多事例表明，被打还手都算互殴！”
           <br/>
           <br/>
           他们的留言，获得了众多网友的点赞支持。

            <span href="https://www.secretchina.com">

 <p>
  还有人说，“不要说米国（美国）怎么不安全，就看看唐山吧！！！”，这句话虽短，在当下却击中要害。
  <br/>
  <br/>
  对于唐山打人事件的整个过程和细节，连日来各大媒体已经进行了详细报道，我们这里就不再重复，今天主要从另外一个角度来谈谈这个事件。
  <br/>
  <br/>
  首先是中共喉舌们的集体反应。打人事件发生后不久，中共喉舌机器一起开动，对打人事件进行强烈的谴责。这显然与其一贯维稳做法大相径庭。
  <br/>
  <br/>
  比如，中共央视新闻指，黑恶势力是人民群众深恶痛绝的社会毒瘤，如果不彻底铲除，就会严重损害人民群众的安全感。
  <br/>
  <br/>
  《人民日报》官方微博，也高调宣称“
  <span href="https://zh.m.wikipedia.org/zh-hans/2022%E5%B9%B4%E5%94%90%E5%B1%B1%E7%83%A7%E7%83%A4%E5%BA%97%E6%89%93%E4%BA%BA%E4%BA%8B%E4%BB%B6">
   唐山这起群殴女子事件
  </span>
  ，令人震惊”，“病床上的她们需要一个公道，全社会需要一个交代”。还说要“让施暴者付出沉重代价，人们拭目以待！ ​​​”
  <br/>
  <br/>
  全国妇联机关报《中国妇女报》官方微博也对此事件接连发布两条评论。
  <br/>
  <br/>
  以严厉口吻呼吁“不严惩不足以平民愤，不严惩不足以抚伤痕，不严惩不足以护尊严，不严惩不足以儆效尤，不严惩不足以树法威”。
  <br/>
  <br/>
  还说“对于此类严重侵犯妇女权益的恶性案件，只能而且必须零容忍”
  <br/>
  <br/>
  6月12日，《人民日报》客户端再度发文强调《坚决依法严惩施暴者，坚决谴责对无辜者的“网暴”》；同时也开始引导舆论，说“对唐山烧烤店打人事件的讨论，不能失焦，更不能跑偏。”
  <br/>
  <br/>
  显然，这是中宣部的统一安排。中共喉舌媒体似乎一下子充满了正义感。
  <br/>
  <br/>
  但是对比一下此前曾经轰动全网的徐州”铁链女“事件， 这些中共喉舌媒体却集体噤声！
  <br/>
  <br/>
  当时，中共对“铁链女”的新闻和话题采取了全力全方位的封杀，曝光此事的多名关键人物陆续遭到抓捕、威胁。其中曾前往丰县声援“铁链女”的网友乌衣，两次被徐州警方抓捕，至今无音讯。中国知名导演王圣强及大陆维权律师卢廷阁、媒体人赵兰健等人，也都因持续关注事件遭到徐州公安跨省约谈及打压。
  <br/>
  <br/>
  中国是世界上网络控制最严厉的国家。人权观察在2020年世界人权报告中称，近些年，中共在网络控制和舆论控制方面大幅度增加了资源和科技投入，“把人工智能、生物识别和大数据变成了监控工具，整顿14亿人民的思想与行为。”
  <br/>
  <br/>
  中国几乎所有的网站和社媒体都置于网信办和中宣部的严密监控之中，监管的精准度可以到达每一条新闻、每一条视频、甚至网民发出的每一条帖子。只要是不符合这些监管机构的有关规定，无论是谁，轻则予以封杀，重则被取消账户甚至有牢狱之灾。
  <br/>
  <br/>
  因此，中宣部在对待”铁链女“和”唐山打人事件”上的不同态度，绝非表面上那么简单。
  <br/>
  转移视线 严控社会
  <br/>
  <br/>
  一些对于中共宣传手法比较熟悉的网友，意识到这是中共当局开始在有意带风向，把大众的注意力从另外的事情上引开。
  <br/>
  <br/>
  这个说法显然很有道理，因为最近一段时间，中国大陆多个地方出现了更为恶劣、更为惊人的暴力案件，但是在网络上很快就被屏蔽掉了，只有这个唐山打人事件，不仅没有被屏蔽，而且中共众多喉舌还一起发声，这绝非偶然。
  <br/>
  <br/>
  从接下来的这些报道，或许可以看出一点端倪。
  <br/>
  <br/>
  6月12日下午，唐山市召开大会，宣布开展夏季社会治安整治“雷霆风暴”专项行动。
  <br/>
  同时，大陆多地警方密集开展夏季夜巡。
  <br/>
  <br/>
  据保定晚报发布消息称，据河北省公安厅统一部署，河北省开展“夏季夜巡专项行动”，保定市、涿州市公安局等都发布了“夏季夜巡专项行动”的现场画面，视频中出现了多处巡查烧烤摊、烧烤店的场景。
  <br/>
  据官媒报道，河南、宁夏、甘肃多个地方也开展类似行动。
  <br/>
  <br/>
  就在唐山打人事件发生的头一天，6月9日，中央政法委召开工作会议。中央政法委秘书长、平安中国办主任陈一新强调，要为所谓的”二十大胜利召开创造安全稳定的政治社会环境”。
  <br/>
  <br/>
  显然，中共官方的宣传报道，也让这些行动变得似乎更加“合情合理“，在二十大来临之际，中共对社会的控制只会越发紧张，而不会有丝毫放松。
  <br/>
  <br/>
  《人民日报》8篇所谓“重磅评论”，没有轰垮美国，却让中共的治安情况在世界面前现了原形。
  <br/>
  <br/>
  有人翻出了中共共青团中央在2017年的一则微博，上面写道“ 深夜出门撸串？不好意思，美国给不了你这种安全感”
  <br/>
  <br/>
  这是一幅多么具有讽刺意味的画面！
