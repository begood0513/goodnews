   驻蓉部队
  </ok>
  大校以上领导干部和建制团主官。中共新华社公布的照片显示，最近被疯传“被军委纪委架走”的习近平军中心腹---中央军委办公厅主任
  <ok>
   钟绍军
  </ok>
  参加了这次接见。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  中共军方喉舌《解放军报》10日在头版上以整版篇幅刊出新华社报道。
 </p>
 <p>
  6月3日，有人在推特上放出消息，称“
  <span href="https://zh.m.wikipedia.org/zh-hans/%E4%B9%A0%E8%BF%91%E5%B9%B3">
   习近平
  </span>
  的铁杆心腹、中央军委办公厅主任钟绍军（将军）,在一次会上直接被军委纪委架走（不是普通的带走）。钟绍军深度卷入疫苗和核酸的暴利活动，获利惊人，被查！”
  <br>
   <br>
    由于钟绍军的特殊身份，该消息被视为“极具爆炸性”，并引发极大关注 。
    <br>
     <br>
      而中共新华社最新公布的照片，令该传闻再次被证实为网络谣言。
      <br>
       <br>
        自5月份开始，有关习近平及其亲信的各种政治传闻、政治谣言，突然在网络上密集出现，而且花样翻新。
        <br>
         <br>
          一些分析人士认为，这些谣言的背后是习近平党内最大政敌---江泽民、曾庆红派系操纵。其目的是让中共官场、军队人心动荡，不仅可以削弱对习近平的支持度，而且能够利用中共党内、军队各种反习势力，最终在二十大上搬倒习近平。
          <br>
           <br/>
           这一次他们瞄准的是习近平的重要军中心腹---钟绍军。
           <br/>
           <br/>
           钟绍军现任中央军委办公厅主任，兼中央军委主席办公室主任，中央军委改革和编制办公室主任，中央军委深化国防和军队改革领导小组办公室主任，中国解放军选举委员会委员，中将军衔。
           <br/>
           <br/>
           钟绍军被外界视为习近平“七大智囊”之一，且被认为是最神秘、低调的一个。 有分析指出，中共二十大前后，钟绍军有望晋升上将。
           <br/>
           <br/>
           钟绍军曾经在帮助习近平整肃军中江曾势力的过程中扮演重要角色。香港《前哨》杂志2015年9月号曾披露，习近平军中反腐，扳倒“国妖”的徐才厚等大老虎的关键证据，正是来自一早埋伏在军委的钟绍军。
           <br/>
           <br/>
           香港《明报》于2021年9月22日的评论文章指，2006年4月，得知时任浙江省委书记习近平身边缺一名秘书时，已是省组织部副部长的钟绍军，据说毛遂自荐，不惜降级，借调去做习的贴身秘书，从此搭上仕途的顺风车。他后来跟习到上海，再随习进京，历任中办副局级秘书、正局级秘书、中办调研室政治组组长，并一直任习的贴身大秘。
           <br/>
           <br/>
           近年，凡习近平出席的涉军场合，钟绍军都陪伴在侧，担任习的“监军”角色，掌管军委中枢机要。
           <br/>
           <br/>
           习近平此次到四川考察，钟绍军随同现身官方报道，也证实了这一点。
