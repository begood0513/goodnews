 <p>
  【看中国2022年6月25日讯】据中共新华社25日发布最新消息，
  <ok>
   <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
    习近平
   </span>
  </ok>
  将出席庆祝
  <ok>
   香港
  </ok>
  回归25周年大会暨香港特别行政区第六届政府就职典礼。
  <br>
   <br>
    今年“七一”，习近平会否到香港参加庆祝活动曾引发外界很大关注。
    <br>
     <br>
      江泽民、曾庆红派系一直对此极为关切，其控制下的网媒“香港01”于24日报导，连接大陆的香港西九高铁站内，近日出现大批带有“中央广播电视总台”字样的摄影器材。有政界人士估计，“国家领导人”可能坐高铁来港。
      <br>
       <br>
        报导引述中国人大港区代表田北辰说，如果“国家领导人”访港，只会当天来回，并有可能先到深圳或广州，翌日早上搭乘高铁来港。
        <br>
         另据报导，香港科学园有员工近日收到“重要人物”参观通知。
         <br>
          <br>
           中共全国人大港区代表叶国谦几天前也曾公开表示，相信北京领导人“七一”当天会来香港，参加庆祝活动。但官媒香港电台24日报导，叶国谦在当天早上的电台节目上改口，称目前难以确定北京领导人会否会来香港。
           <br>
            <br/>
            他说，今年是中共收回香港主权25周年，北京很重视，相信习近平就此会发表“重要讲话”。但目前港府官员有染疫风险，习近平改为网上发表讲话的机会很高。
            <br/>
            <br/>
            习近平和夫人彭丽媛上一次七一访港，是2017年林郑政府上任之时，当时6月29日至7月1日访港3天，除了出席庆回归大会和就职礼，还到石岗军营检阅驻港中共军队，并到访八乡少年警讯中心。 彭丽媛则探访幼儿园、安老院，并以广东话向老人们问好。
            <br/>
            <br/>
            习近平访港，是他自疫情爆发两年多来，首次踏出中国大陆土地。 他上一次出访，还是2020年1月17日到缅甸进行国事访问。
            <br/>
            <br/>
            习近平此次访港，或欲借此破除外界种种政治传闻、政治谣言。
            <br/>
            <br/>
            最近一段时间，海内外对习近平不利的各种政治传闻、政治谣言蜂起，声势浩大。在这些如潮的政治谣言下，现在一些人开始打上了问号。
            <br/>
            <br/>
            这些传闻、谣言内容花样百出，从习近平所谓的“禅让”、到习近平患上了各种绝症、到习近平被软禁在中南海、到习近平的军中大秘钟绍军被抓。
            <br/>
            <br/>
            还包括“习近平已被边控”！
            <br/>
            <br/>
            对于这些政治传闻、谣言，众多海外分析人士认为这是江泽民、曾庆红派系发动的一场针对习近平的“超限战”，其幕后真正的操盘手是江系的二号人物曾庆红。
            <br/>
            <br/>
            在中共内斗如此激烈的情况下，香港警方在中南海的要求下，表示要“总动员出动”。
            <br/>
            <br/>
            香港警务处处长萧泽颐6月14日出席区议会会议后透露，警方去年初已开始筹划今年“七一”的安保工作，加强在线线下的情报搜集，为确保“七一”当日万无一失，警方将“总动员出动”，包括特动部队、后勤人员及辅警，亦会加派便装人员留意可疑人物或物品。
