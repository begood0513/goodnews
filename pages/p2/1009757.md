 <p>
  【看中国2022年6月21日讯】中共中央办公厅近日印发规定，对中共
  <ok>
   高官亲属经商
  </ok>
  提出“禁业要求”。最新报道显示，
  <ok>
   习近平
  </ok>
  在“禁业要求”中增加两大情形。
  <br>
   <br>
    中共新华社19日报道，中办近日印发《领导干部配偶、子女及其配偶经商办企业管理规定》 。
    <br>
     <br>
      《规定》的适用对象，“主要是党政机关、群团组织、企事业单位厅局级及相当职务层次以上领导干部”；“经商办企业情形，主要是投资开办企业、担任私营企业或外资企业等高级职务、私募股权基金投资及从业、从事有偿社会中介和法律服务等行为”。
      <br>
       <br>
        据称，“《规定》对不同层级、不同类别领导干部配偶、子女及其配偶经商办企业分别提出了禁业要求，领导干部职务层次越高要求越严，综合部门严于其他部门。”
        <br>
         <br>
          21日，中共《人民日报》在其《规范领导干部配偶、子女及其配偶经商办企业行为工作综述》中披露，“在试点期间3类经商办企业情形的基础上，落实党中央要求，...又逐步增加了‘有偿社会中介和法律服务’、‘私募股权基金投资及从业行为’2类情形。”
          <br>
           <br>
            也就是说，是在习近平的要求下，增加了以上两类情形，这颇耐人寻味。
            <br>
             <br>
              因为，提到“私募股权基金”，就不能不提到中共前党魁江泽民家族长孙江志成及其私募基金博裕资本。
              <br>
               <br>
                2014年路透社在其特别报道《私募股权基金行业的太子党》中，第一个就指向了
                <span href="https://zh.m.wikipedia.org/zh-hans/%E6%B1%9F%E6%B3%BD%E6%B0%91">
                 江泽民
                </span>
                的孙子江志成。
                <br/>
                <br/>
                江志成利用其家族背景，身体力行江泽民的“闷声发大财”，借助私募资金通吃中国国内的优良资产。利用博裕资本将江家拥有的政治关系转化为巨额的商业财富。
                <br/>
                <br/>
                路透社报道，由于中共对中国经济和社会各方面的全面控制，让“太子党”有机会借助他们的政治关系聚敛财富。而中（共）国规范利益冲突的法律薄弱，牵扯到政治精英的商业交易在官方媒体也受到严格审查。
                <br/>
                <br/>
                在金融、能源、国内安全、电信、媒体等商业领域，“太子党”都扮演着核心角色。通常透明度不高的私募股权基金已成为“太子党”的天然避风港。
                <br/>
                <br/>
                在中国私募股权基金领域，路透当时确认的15家公司要么由“太子党”创办，要么是聘用“太子党”担任高职。自1999年以来，这些私募股权基金已经募集至少175亿美元投资资金。这是年的数据。
                <br/>
                <br/>
                几位有限合伙人告诉路透，他们的公司依据太子党拥有的政治关系及其将这种关系转化为商业交易的能力，对太子党进行评估。这些投资者表示，在他们的名单上，江志成和博裕资本的排名位列前茅。
                <br/>
                <br/>
                <ok>
                 江志成与江世干
                </ok>
                <br/>
                <br/>
                博裕资本成立于2010年，它于2011年收购日上免税行的控股股权，给投资者留下了深刻印象。他们认为，这项交易证明江志成能够涉足受严格控制的国资垄断行业，并将这些资产转化成利润丰厚的投资。
                <br/>
                <br/>
                江志成的朋友和商业伙伴称，创立、打造并将日上免税行股份出售给博裕资本的是美籍华人江世干（Fred Kiang），江世干与江家关系密切。
                <br/>
                <br/>
                江泽民1989年出任中共中央总书记，几乎与此同时，1990年江世干开始在上海拓展生意。
                <br/>
                <br/>
                江世干1999年创立日上免税行。“巧合”的是，当年正值江泽民领导的中央政府向国际竞标者开放上海浦东国际机场的免税店运营。
                <br/>
                <br/>
                2000年，中共国务院批准一项措施，将上海以外的所有免税店业务控制权从地方政府移交给国有的中国免税品(集团)有限责任公司。外资企业不能在华成立免税店合资企业或直接拥有免税店业务。
                <br/>
                但是，中共官媒《第一财经》2009年的一则报道称，尽管限制外资拥有国内免税店，国务院特批了日上免税行在华运营免税店。
                <br/>
                <br/>
                路透报道，博裕资本对日上免税行的投资在三年时间里赚了三倍。在私募股权基金行业，五年内赚一倍就算成功了。
                <br/>
                <br/>
                因此，路透社发出质疑，江世干为何要以似乎是折扣价格卖掉生意兴隆的日上免税行40%的股权，成为围绕这宗交易最令人困惑的问题。
                <br/>
                江志成和马云
                <br/>
                <br/>
                2012年，阿里巴巴创始人马云与江志成会面。博裕资本当时加入了中国投资有限责任公司（CIC,简称中投）牵头的财团，为马云从雅虎手中购回约20%股权所需的71亿美元募集部分资金。中投牵头的财团藉此获取了阿里巴巴5.6%的股权。阿里巴巴当时的估值约为380亿美元。
                <span id="hideid" name="hideid" style="color:red;display:none;">
                 <span href="https://www.secretchina.com">
                 </span>
                </span>
               </br>
              </br>
             </br>
            </br>
           </br>
          </br>
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  分析师们估计阿里巴巴2014年的市值至少达到了1,400亿美元，这就意味着，作为财团成员之一的博裕资本在18个月内取得了约两倍半的收益。
  <br/>
  <br/>
  两名了解交易的消息人士表示，江志成还让博裕资本投资约5,000万美元入股国有的信达资产管理公司。信达成立于1999年，负责承接国有银行的不良贷款。信达在IPO之前向各家银行和私募基金等战略投资者出售了总计16亿美元的股份。
  <br/>
  <br/>
  这些战略投资者包括全国社会保障基金、瑞银UBSN.VX、中信资本，渣打银行STAN.L以及凯雷集团CG.O和博裕资本两家私募股权基金。
  <br/>
  <br/>
  路透社这篇2014年的报道中，其中有些内容现在看来非常富有远见。
  <br/>
  <br/>
  报道说，太子党特权在中国不一定是永久的，即便江志成也一样。2012年，原重庆市委书记薄熙来的倒台更是给很多私募股权投资者强化了这一观念。
  <br/>
  <br/>
  报道认为，随着中共国家主席
  <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
   习近平
  </span>
  掀起的反腐败运动以及美国证券交易委员会对华尔街在华聘用活动的调查，这种担心也与日俱增。
  <br/>
  <br/>
  不过，江泽民的孙子江志成也用脚回答了这个问题。
  <br/>
  <br/>
  2021年2月份，华尔街日报引据知情人士透露，博裕资本(Boyu Capital)的两名联合创始人已搬至新加坡，该公司的部分业务也已从香港总部迁至当地。
  <br/>
  <br/>
  2022年6月21日，《人民日报》报道，习近平在《领导干部配偶、子女及其配偶经商办企业管理规定》要求加入两类情形，其中之一是“私募股权基金投资及从业行为”。
