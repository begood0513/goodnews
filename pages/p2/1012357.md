 <p>
  【看中国2022年7月22日讯】在中共
  <ok>
   二十大
  </ok>
  之前，中共党内的
  <ok>
   权力斗争
  </ok>
  越演越烈。最近一段时间，中共各大喉舌全部开动马力，以“
  <ok>
   反腐败
  </ok>
  ”为名进行舆论造势 。
  <br>
   <br>
    随即， 在短短一周之内，肖建华、孙力军、傅政华案件全部出现新的进展，外界认为此前的中共宣传系统造势，似乎与这三大案件有关。
    <br>
     <br>
      然而， 有迹象显示，一切似乎远远没有结束。
      <br>
       一切并未结束
       <br>
        <br>
         13日，中纪委国家监委网站发表文章，题为《法规建设这十年丨严明政治纪律和政治规矩》，大谈所谓“政治纪律和政治规矩”，同时将傅政华和孙力军称之为“十九大以来查处的严重违反政治纪律和政治规矩的典型”，这无形中承认，虽然傅政华和孙力军分别以不同罪名被起诉、审判，其实根本的原因是“政治问题”，换句话说，都是政治案件。这似乎验证了外界的种种政治传闻。
         <br>
          <br>
           中纪委在列举了傅政华和孙力军通报中的大量罪名之后，强调“等待二人的将是法律的制裁”。
           <br>
            <br>
             随后，中纪委文章出现罕见信号。文章引用中共《关于新形势下党内政治生活的若干准则》中内容，目标对准中共的高级干部，“特别是中央委员会、中央政治局、中央政治局常务委员会的组成人员”。
             <br/>
             <br/>
             更引用《中国共产党党内监督条例》，强调监督的重点对象是中共的领导机关和领导干部特别是主要领导干部，还强调中共中央委员会成员必须严格遵守中共的“政治纪律和政治规矩“，发现其他成员有违反党章、破坏中共党纪、危害团结统一的行为”应当坚决抵制”，并及时报告。
             <br/>
             <br/>
             中纪委公开在文章中点名“中央委员会、中央政治局、中央政治局常务委员会”，不仅非常罕见，而且不排除意有所指。一般来讲，如果没有相应的针对性，中共是不会公开曝出其内斗信号的。
             <br/>
             <br/>
             中纪委之所以要发表这样的文章，是因为这些文章的读者群，基本都是中共各级党员、干部，他们要定期对这些中共喉舌文章进行所谓的“学习”，从某种意义上讲，这也是一种传达政治信息的方式。而且，中共党内的很多中高层官员，也是通过这些喉舌文章中暗藏的信息，来对高层的权力斗争进行分析，以免关键时候站错了队，表错了态。
             <br/>
             <br/>
             中纪委在这篇文章中，还列出多名已经落马判刑的中共高官。
             <br/>
             <br/>
             比如，列出被查处的“周永康、薄熙来、孙政才、令计划等严重违纪违法案件”，以及查办的“孙力军、邓恢林、龚道安、王立科、刘新云、傅政华等人的政治团伙案”，还有查处“王富玉、周江勇等政治问题和经济问题交织的腐败案件，防止党内形成利益集团，斩断权力与资本勾连的纽带“。
             <br/>
             <br/>
             其中一个关键的地方在于，中纪委在谈到孙力军“政治团伙案”时，使用了“等人”一词。这不仅意味着该团伙将会再度增加新的名单，而且也为将来斩落孙力军背后的更高层政治人物，预留了空间。
             <br/>
             可以这么说，最近一个月以来，也就是从6月17日中共政治局进行集体学习至今，中共各大喉舌围绕着所谓的”反腐“、”自我革命”，不断进行舆论造势，这个阵势之大，时间持续之长是极不寻常、极为罕见的。
             <br/>
             <br/>
             就在中纪委发表这篇文章的第二天，《人民日报》、中纪委国家监委再度发表文章 ，而且还是特殊署名的重要评论文章。人民日报文章中，还再度点名“铁帽子王”。
             <br/>
             再提“铁帽子王”
             <br/>
             <br/>
             7曰14日，中共《人民日报》在其第四版，以署名“仲音”开始刊发系列评论文章《坚决反对和惩治腐败》。同一时间，中纪委国家监委网站，以署名“钟纪言”发表评论文章《提高一体推进“三不腐”能力和水平全面打赢反腐败斗争攻坚战持久战》。“钟纪言”是中纪委国家监委言论的谐音简称。也就是说，这篇文章代表中纪委的态度，绝非一般的评论文章。
             <br/>
             <br/>
             中共两大喉舌，在同一天分别以“仲音”和“钟纪言”就反腐这一题材发表重要评论文章，说明
             <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
              习近平
             </span>
             在一周之内，连续推动傅政华、孙力军、肖建华三大案件出现新的进展之后，并不打算告一段落。
             <br/>
             <br/>
             其中，在人民日报“仲音”评论文章中，再提“没有免罪的‘丹书铁券’，也没有‘铁帽子王’”。而在中纪委评论文章中，则明确说明“反腐败斗争首先要从政治上看”。这呼应了习近平在6月17中共中央政治局学习时所指的，反腐败是一场“重大政治斗争“。
             <br/>
             <br/>
             在“钟纪言”署名的评论文章中，要求坚决清除“坚决防止权力和资本相互勾连、干部成为资本代言人代理人，坚决防止资本集团攫取政治权力“。其中“干部成为资本代言人代理人”这一说法似乎首次出现。
             <br/>
             <br/>
             今年1月28日，人民日报海外版旗下账号“侠客岛”发文《中央纪委首提“斩断权力与资本勾连”你看懂了吗？》。
             <br/>
             <br/>
             文章称华融原党委书记赖小民是一个“典型案例”。指出赖小民在任期间，曾向多家不达标企业违规放贷、输血数亿元。
             <br/>
             <br/>
             还有原中共610办公室副主任彭波，在担任网信办专职副主任等职务期间，“靠网吃网”“大搞权钱交易”。
             <br/>
             <br/>
             侠客岛文章还指出，“权力与资本勾连”还表现为市场内幕交易、操纵股票价格、恶意炒作期货、借内部消息拉拢腐蚀利害关系人等，严重干扰市场预期。比如，反腐专题片《零容忍》中提到公安部原副部长孙力军，在操纵证券市场方面“情节特别严重”。
             <br/>
             <br/>
             其中，被称为“典型案例”的赖小民已经被除以死刑。傅政华、孙力军也被中共官方定性为十九大以来查处的严重违反政治纪律和政治规矩的典型。
             <br/>
             <br/>
             中共如此大阵仗的进行舆论造势，究竟是准备对孙力军、傅政华、肖建华等人进行重判，还是要动更大的“老虎”，令人猜测。
             <br/>
             <br/>
             而这个“铁帽子王”，既可以说是泛泛而指，也可以说具体指向曾庆红。
             <br/>
             <br/>
             2015年6月11日，原中共政治局常委周永康被判处无期徒刑。
             <br/>
             <br/>
             就在当年的1月份，中纪委官网发表文章《腐败没有“铁帽子王” 反腐败绝不封顶设限》；2月又发表文章《大清“裸官”庆亲王的作风问题》， 庆亲王也称“铁帽子庆亲王”，是晚清政府册封的一个世袭制王爵，庆亲王曾手握大权，其人为官贪婪奢侈，留有骂名。曾庆红与庆亲王同有一个“庆”字，外界普遍认为，这是习近平当局在暗指曾庆红。
             <br/>
             <br/>
             在中共党内的政治斗争中，习近平最大的政敌对手，依然是江泽民、曾庆红派系。 习近平自从上任以来，就一直在于江泽民派系进行权力争夺。
