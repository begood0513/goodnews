   中共和塔利班的“承诺”
  </ok>
  <br>
   <br>
    《澎湃新闻》报道称，阿富汗塔利班最高领导人阿洪扎达7月1日出席了阿富汗大支尔格会议（亦称大国民会议）并发表讲话，呼吁海外阿富汗商人回国投资。
    <br>
     <br>
      在阿富汗当地媒体播出的讲话录音中，阿洪扎达说，由于外国援助并不能重建阿富汗的经济，希望在海外的阿富汗商人能回国投资建设。
      <br>
       <br>
        阿洪扎达表示，阿富汗对任何人都没有恶意，也不会伤害任何邻国。他还在讲话中呼吁执行伊斯兰教法并维护社会公平正义。
        <br>
         <br>
          此前不久，当地时间6月22日凌晨，阿富汗东部发生强烈地震。
          <br>
           <br>
            阿富汗临时政府外交部发言人阿卜杜勒-卡哈尔·巴尔基（Abdul Qahar Balkhi）指出,“经济上无力在其所需要的范围内帮助人民”。
            <br/>
            <br/>
            塔利班最高领导人“对任何人都没有恶意，也不会伤害任何邻国”，听上去和中共的各种承诺极为相似。
            <br/>
            <br/>
            邓小平1982年在中英谈判期间曾以“马照跑、舞照跳、股照炒”为比喻，承诺1997之后会让香港的生活方式与资本主义制度维持50年不变。
            <br/>
            <br/>
            <span href="https://zh.m.wikipedia.org/zh-hans/%E4%B9%A0%E8%BF%91%E5%B9%B3">
             习近平
            </span>
            7月1日在香港，出席新一届香港政府的宣誓仪式，之后发表演说，称“一国两制”是个好制度，没有理由改变，“必须长期坚持”。
            <br/>
            <br/>
            事实上，邓小平所说的”50年不变“，没过一半的时间，香港的所谓”一国两制“早已面目皆非。

 <p>
  <ok>
   不敢回国&amp;夺路而逃
  </ok>
  <br/>
  <br/>
  2021年8月31日，阿富汗塔利班文化委员会成员萨曼加尼（Enamullah Samangani）承诺“阿富汗未来将发展经济，改善就业。有数据显示90%的阿富汗人生活在贫困线下，塔利班将解救阿富汗人民。”他还呼吁海外的阿富汗人，回国一同建设国家。
  <br/>
  <br/>
  现在看来，这些海外的阿富汗人并没有人相信塔利班的承诺。
  <br/>
  <br/>
  这边是阿富汗人害怕塔利班，不敢回到故乡。而另一边，在中共的高压统治下，大量资金撤离、富人也拼命夺路而逃。
  <br/>
  <br/>
  香港方面，根据
  <span href="https://www.secretchina.com/news/gb/tag/投资" target="_blank">
   投资
  </span>
  咨询公司Henley&amp;Partners公布的报告（2022年全球最多高收入阶层流入与流出国家或地区报告），估计今年香港将有大约3000名，百万美元等级高资产人士，离开香港前往他国，流出人数之高，仅次于中国和俄罗斯，原因不外乎，高资产人士对香港的前景失去信心。
  <br/>
  <br/>
  大陆方面，据媒体报道，中国债券市场连续四个月出现资金外流，这可能是近年来最大规模的一次外资撤离。
  <br/>
  <br/>
  根据中国中央国债登记结算有限责任公司（CCDC）和上海清算所的数据，国际投资者5月共减少了约162亿美元（1，094亿元）的债券头寸，与4月减持情况趋同。
  <br/>
  <br/>
  CCDC的数据显示，截至5月底，外国人在中国债券的持仓总额相当于5，428亿美元，是一年来的最低水平。
  <br/>
  <br/>
  这波撤离潮最早从2月开始，因为俄罗斯入侵乌克兰，引发了外国投资者对投资中国的地缘政治风险的担忧。
  <br/>
  <br/>
  到4月，因为中美货币政策背道而驰，中国政府债券收益率在十多年来首次低于美国国债的收益率，使得中国债券对外国投资者的吸引力进一步减弱。
  <br/>
  <br/>
  最新数据显示，5月外国投资者又减持了160亿美元的中国债券，使2月以来的总流出量达到约610亿美元。
  <br/>
  <br/>
  在中共“动态清零”政策影响下，很多大陆人开始研究如何永远离开这里。
  <br/>
  <br/>
  美国智库外交关系委员会（CFR）的一项研究发现，4月3日，在中共政府重申“动态清零”政策的同一天，微信上“移民”的搜索量增加了440%。加拿大是最受欢迎的目的地，腾讯报告称，在3月28日至4月3日的一周内，“移居加拿大的条件”一词搜索量增加了2846%。过去一个月，移民咨询公司的咨询数量也猛增。
  <br/>
  <br/>
  《金融时报》6月27日报导，专家们表示，清零持续的时间越长，中共领导层与中国社会的“社会契约”，尤其是中共迄今设法与城市中产阶级之间的“社会契约”越有长期磨损的风险。而城市中产阶级是中共当局拉拢的对象。
  <br/>
  <br/>
  面对塔利班和中共政府的“承诺”，两国的人民都用脚投出了真实的一票。
