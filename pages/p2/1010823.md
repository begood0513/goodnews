  <br>
   <br>
    今年的3月28日，共军驻港部队发言人吴韬上校发布消息，
    <span href="https://zh.m.wikipedia.org/zh-hans/%E8%B5%96%E5%A6%82%E9%91%AB">
     赖如鑫
    </span>
    任驻港部队政委。众所周知，香港长期是曾庆红的地盘。
    <br>
     <br>
      公开资料显示，赖如鑫曾任南海舰队政治部干部处处长、...南海舰队某综合保障基地政委等，2018年出任海军研究院政委，同年12月晋升海军少将军衔，后转任北部战区政治工作部副主任。
      <br>
       <br>
        2003年，中共成立中央港澳工作协调小组，时任中共政治局常委、国家副主席曾庆红任组长。从此，曾庆红全面介入香港事务。
        <br>
         <br>
          曾庆红的心腹廖晖在1997年至2010年由时任中共党魁江泽民提拔，担任港澳办公室主任13年。是中共掌管港澳工作最长时间的官员。在港澳事务决策方面，廖晖、前中共组织部长安子文之子安民，以及曾庆红胞弟曾庆淮三人构成了曾庆红控制香港的三大管道。唐家璇、廖晖等都直接对曾庆红汇报。可以说，香港长期在曾庆红的掌控之中，至今依然有一些港媒不断反习捧江，显露出明显的江泽民、曾庆红派系色彩。
          <br>
           <br>
            赖如鑫任驻港部队政委后，几天后，江西省军区司令换人。
            <br>
             <br/>
             据江西省退役军人事务厅网站消息显示，4月2日上午，江西省军区政委鲍泽敏，省军区司令员张弓等人到南昌市烈士陵园，开展祭扫活动。上述消息显示，张弓已任江西省军区司令员。
             <br/>
             <br/>
             澎湃新闻报道显示，张弓曾任北海舰队航空兵参谋长、东海舰队副参谋长等职务，2019年6月晋升海军少将军衔。
             <br/>
             <br/>
             此前担任江西省军区司令员的是吴亚非，他从2017年8月起任江西省军区司令员，于2021年11月不再担任江西省委常委，“戎装常委”由江西省军区政委鲍泽敏担任 。
             <br/>
             <br/>
             2021年7月23日，据《江西日报》报道显示，原北京卫戍区副政委鲍泽敏已经调任江西省军区政委。鲍泽敏现年56岁，曾任北京卫戍区副政委兼纪委书记，2018年12月晋升少将。
             <br/>
             <br/>
             到目前为止，在二十大之前，对于江西省---这个曾庆红的老家，习近平已经将其军区司令、政委全部换人。
             <br/>
             <br/>
             而武警方面，武警江西总队政委徐云飞2021年也在《解放军报》向习近平表忠心。中共十九大后，武警政工部组织局原局长徐云飞接替陈杭，出任武警省江西总队政委。
             <br/>
             曾经有陆媒分析认为，由武警总部“空降”担任省级武警总队主官，这种人事安排并不多见。
             <br/>
             <br/>
             习近平对军队的两大人事任命，分别剑指曾庆红的老家江西省和其经营多年的香港，似乎不是偶然的。
             <br/>
             <br/>
             习近平的第三个动作是，中央港澳工作领导小组副组长换人。
             <br/>
             <br/>
             6月24日下午，中共全国人大常委会宣布多个任免名单。其中公安部长赵克志被免职，王小洪接任。
             <br/>
             <br/>
             就在当天，中共中央政法委官网“中国长安网”就更新“领导・机构”栏目信息，王小洪已任中央政法委副书记，与赵克志并列，但在赵克志之后。
             <br/>
             <br/>
             王小洪被公认是习近平的亲信，20世纪90年代习任福州市委书记时，王小洪就负责习的安全保卫。
             <br/>
             去年11月，王小洪升任中共公安部党委书记，外界就猜测他很快会接任部长，只是半年过后一直没有消息。
             <br/>
             <br/>
             次日，两家香港媒体突然同时对外放风。
             <br/>
             <br/>
             香港《明报》6月25日发文标题是：“习旧部王小洪任公安部长 接替赵克志兼任港澳小组副组长”，《星岛日报》同天的发文标题是《王小洪任公安部长兼港澳事务副组长》。后者更直接说，“香港2015年修例风波之后，公安部长同时担任中央港澳工作领导小组副组长……”
             <br/>
             <br/>
             前面提到，有一些港媒不断反习捧江，显露出明显的江泽民、曾庆红派系色彩。我们发现《明报》、《星岛日报》以及《香港01》、《南华早报》，似乎都在扮演这样的角色。
             <br/>
             <br/>
             习近平在短短的几个月中，不断调兵遣将，涉及到的一个焦点人物就是曾庆红。而就是2017年，习近平上一次访港前夕， 红二代罗宇披露中共高层内幕。
             <br/>
             <br/>
             <ok>
              红二代曝惊人内幕

             <br/>

             罗宇说，“从习近平上台到现在，习王反贪腐目标就是江泽民、曾庆红，因为他们是最大的贪腐者。在这期间江曾有好几次用很强烈的语气跟习说：‘你想干什么呀，你想把共产党的丑事都名扬天下啊！’他们跟习王说了好几次，但习王没有理睬。”
             <br/>
             <br/>
             罗宇解释：“原因是他首先得保住自己，才能做后面他想做的事情。他得破除掉江曾策划的阴谋——把他推翻让薄熙来上台。所以这是习近平上台做的第一件事，他做的相当成功，胡温已经把薄熙来拉下马了，他再把周永康拉下马，他这是保命。”
             <br/>
             <br/>
             罗宇说，“习近平上台5年来一直在肃清江曾的势力，到了今天他们的势力也肃清的差不多了。但是习近平遇到难逾越的阻力：江曾的势力就是贪腐的势力，贪腐的势力就是全党的势力。所以到了现在，两边都没有退路了。”
             <br/>
             <br/>
             从今年以来习近平对驻港部队政委、江西省军区司令、港澳工作领导小组副组长这一系列动作来看，依然没有跑出罗宇所说的这个范围。

             <br/>
             时至今日，习近平和江曾派系的权力斗争更是呈现白热化状态。

  据报道，习近平于30日下午抵港，参加七一香港回归25周年庆祝大会暨香港新一届政府就职。习近平夫人彭丽媛也会随行。港警29日起已在各重点区域布防驻守巡逻。
  <br/>
  <br/>
  据称，为确保安全，港警已将今次的保安级别定为最高，除西九站要塞外，亦会沿线遥距护航及高度布防，包括沿途有人驻守及巡逻，确保万无一失。
  <br/>
  <br/>
  有别过往乘坐专机，习近平夫妇今次选择取道陆路往返，据悉两人亦将坐高铁当天往返，不在香港过夜。6月30日晚参加礼宾府特设晚宴后搭高铁返回深圳，7月1日早上再赴港出席港府新班子就职典礼。这显然是出于安全考虑。
  <br/>
  <br/>
  从深层意义上说，习近平更主要的目的要防范的是他的政治对手，利用这次访港对其下手，这一点，从一条新闻可以得到验证。
  <br/>
  <br/>
  中国媒体报导，至少从两个城市包括从福州和厦门乘铁路前往深圳方向的旅客，自本周日（26日）至7月2日须接受“二次安检”，始可上车。

   百足之虫，死而不僵

  6月29日消息，香港政府通知至少七家媒体的十名已经报名的记者，他们将不能参加“七一”新一届港府就职典礼和香港回归25周年的纪念活动。

  香港记协说，受到影响的媒体包括“明报”、“香港01”、“南华早报”、“Now新闻台”、法新社、路透社等。当然还有一些香港媒体未获准进行采访活动，其中的一个理由是“保安理由”。

  让我们来谈谈“明报”、“香港01”、“南华早报”这几家媒体。

  <span href="https://www.secretchina.com/news/gb/tag/江泽民" target="_blank">
   江泽民
  </span>
  、曾庆红派系背景的中共大外宣“多维网”，突然宣布停止运作。这被外界普遍认为是在内斗中，遭到习近平整肃，直接导致这个江曾派系的重要喉舌媒体被关闭。
  <br/>
  <br/>
  这个《多维网》此前最关心的就是习近平的出访问题。
  <br/>
  <br/>
  2021年7月6日，该网站发文《一年半未迈出国门，港媒披露习近平疫后是否首访意大利》，
  <br/>
  <br/>
  9月10日，再度发文《习近平600天未外访创纪录，美媒称会影响与西方国家关系》。
  <br/>
  <br/>
  看看这个标题，“习近平你都600天没出国了，再不出国都会影响和西方国家的关系了”。江泽民、曾庆红派系似乎比习近平还着急。可以说是一直想挤兑习近平离开中国大陆。
  <br/>
  <br/>
  有分析认为，习近平不出国显然和二十大有关，其中最关键的就是他可能担心一旦离开中国，中共党内的大批反习势力会联起手来，发生政变！
  <br/>
  <br/>
  这样的事情确实发生过，比如说，1964年10月12日，苏共中央主席团在领导人赫鲁晓夫缺席的情况下，召开了一场撤销赫鲁晓夫职务的会议，并取得一致意见，此时赫鲁晓夫还在黑海之滨，安然度假。
  <br/>
  <br/>
  10月12日晚上，勃列日涅夫胆战心惊的拨通了赫鲁晓夫的电话，请他立即赶回莫斯科，参加一个关于农业问题的主席团特别会议，结果一下飞机赫鲁晓夫就被软禁起来，在克格勃的护送下，直达克里姆林宫。最终赫鲁晓夫被逼迫下台！
  <br/>
  <br/>

  <br/>
  而这个拼命挤兑习近平出国的《多维网》，他的老板就是香港商人于品海。

  <br/>

  <br/>
  <br/>

  <br/>
  至于说明报，也曾经被发现与几个江系媒体如《多维网》、《星岛日报》配合的非常有默契，多次步调一致地发表打击习近平的文章。
  <br/>
  <br/>
  《新纪元周刊》也曾经报导，“《明报》背后的势力属于江泽民团伙，曾庆红在背后发挥作用。《明报》集团报系旗下的媒体，在全球范围内一直替中共江派在非常敏感的问题上发挥作用。”
  <br/>
  <br/>
  报道还说，“曾庆红是江泽民时代的二号实权人物，号称江的‘大内总管’。随着江泽民身体越来越弱，曾庆红已成为江泽民集团的实际掌门人”。
