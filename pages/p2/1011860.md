  7月13日在国新办新闻发布会上的一番话，瞬间引爆大陆网友怒火，人们在网上骂声不绝，超过56万人在网上留言，发表看法。
  <br>
   <br>
    新闻发布会上，彭博社记者提问说:您是否对当前在河南等地区出现的储户兑款存在困难的情况有什么看法？这是否会造成因为地方政府的偿债能力而引发的整个金融和银行系统的风险？对于河南等地所发生的事情您有什么看法？谢谢。
    <br>
     <br>
      孙天琦回答，“整体看，我国金融风险收敛，总体可控，99%银行业资产处在安全边界内，2021年第四季度央行评级结果显示，在安全边界内的银行占银行业参评机构的93%，资产规模占参评规模的99%；有316家高风险银行业金融机构，但其资产规模仅占银行业参评机构的1%，绝大多数中小银行的央行评级均处于安全边界内”。
      <br>
       <br>
        孙天琦还说，“请各位媒体朋友在报道的时候，既要围绕高风险机构作报道、讲故事，也要围绕4000多家好的机构作报道、讲故事。既要报道坏机构、坏事情，也要报道好机构、好事情...”
        <br>
         <br>
          中共各大官媒随后纷纷以《我国绝大多数中小银行的央行评级处于安全边界内》为标题，对孙天琦表态进行报道，进行舆论维稳、安抚人心。然而，中国大陆的网民完全不再相信中共官方的说辞，在网络上纷纷留言，对其言论进行强烈质疑。留言数字很快超过56万，可谓一石激起千层浪。
          <br/>
          <br/>
          排名第一的留言只有五个大字，“我信你个鬼”。有62672名网友点赞表示支持。超过了总留言人数的10分之一。
          <br/>
          <br/>
          还有一则留言说“突然一阵恶心”，这条留言也获得了56897名网友的支持。
          <br/>
          <br/>
          还有一则留言只有一个日期，“7.10”，除此之外，一个字也没有说。
          <br/>
          <br/>
          7月10日，数千名的村镇银行储户在中国人民银行郑州中心支行门前进行维权活动，遭到中共警方围困、以及大批不明身份便衣的暴力驱赶、殴打，一些储户因此受伤。
          <br/>
          <br/>
          超过2万名网友对“7.10”这条留言点赞，人们对中共的怒火跃然纸上。
          <br/>
          <br/>
          还有几条超过万人点赞的留言是，
          <br/>
          <br/>
          “***告诉人民，什么叫绝大部分银行？ 银行都是归银监会管，什么叫绝大部分银行？？”
          <br/>
          <br/>
          “马云：中国金融没有系统”
          <br/>
          <br/>

          <br/>
          <br/>
          “安全不是用嘴说的，把储户的存款都赔了。比谁说都强，唐山事件，再加上储户事件。民心已经流失...”
