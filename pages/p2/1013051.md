 <p>
  【看中国2022年7月30日讯】备受瞩目的美国国会众议院议长
  <ok>
   南希∙佩洛西
  </ok>
  （Nancy Pelosi）的亚洲之行于星期五启程，中共《人民日报》在其最新的重要评论文章中，突然降低了调门，开始大谈“加强沟通避免误判”。
  <ok>
   胡锡进
  </ok>
  此前叫嚣击落“佩洛西的飞机和美国战斗机”，似乎在给
  <ok>
   习近平
  </ok>
  有意“挖坑”。
  <br>
   <br>
    7月30，中共《人民日报》以”钟声”笔名发表评论文章，题为《处理中美关系必须加强沟通避免误判》。
    <br>
     <br>
      “钟声”署名是所谓“中国之声”的谐音缩写，是中共《人民日报》于2008年推出，围绕所谓“重要的国际问题和涉中问题”发表的评论，代表中共官方的声音。
      <br>
       <br>
        仅从标题就可以看出，中共官方开始有意降低了调门。
        <br>
         <br>
          文章开始部分，一贯以战狼姿态示人的中共，突然开始大谈“世界和平稳定和发展繁荣”，还强调什么“保持对话沟通，加强互利合作，有效管控分歧...”
          <br>
           <br/>
           中共还似乎迫不及待的表示，在诸多方面，“中美加强协调合作的必要性和紧迫性正变得更加突出”。
           <br/>
           <br/>
           中共还就习近平7月28日同拜登通电话，声称“两国元首都强调了加强沟通的重要性，也就后续继续沟通合作达成了共识”。显然，中共方面此番表态和不久前中共外交部以及军方的表态大相径庭，已经从此前的强硬表态上大步后退。
           <br/>
           <br/>
           文章里虽然也重复了
           <span href="https://zh.m.wikipedia.org/zh-hans/%E4%B9%A0%E8%BF%91%E5%B9%B3">
            习近平
           </span>
           在通话中所说的“玩火必自焚”，但是完全没有提及佩洛西访台一事。
           <br/>
           <br/>
           对于习近平在通话中所放出的这句“玩火必自焚”，一位白宫资深官员在通话当天下午的简报中语气淡定地回应，去年11月就听过习近平类似的说法，言外之意，中共只不过是“老调重弹”。
           <br/>
           <br/>
           美国方面，备受瞩目的美国国会众议院议长南希∙佩洛西（Nancy Pelosi）的亚洲之行于星期五启程。
           <br/>
           <br/>
           美国全国广播公司（NBC）引述两位消息人士的话说，佩洛西率领的美国国会代表团星期五离开华盛顿，只是在启程之时是否前往台湾还不确定。
           <br/>
           <br/>
           两位消息人士中的一位星期四下午看到了这次访问的行程，此行目的地包括日本、韩国、马来西亚和新加坡，而台湾一站则标明“暂定”。
           <br/>
           <br/>
           佩洛西在周五早些时候的新闻会上拒绝证实她是否将会访问台湾。她说，“我从不谈论我的出访，因为正如你们所知，这是一个安全问题。”但她随后又补充说，“如果我们去了那些国家的话，我是非常兴奋的，你们会在路上听到的。”
           <br/>
           <br/>
           中共《人民日报》以“钟声”发表的这篇国际评论，凸显出胡锡进在关键时刻，有意无意的在给习近平添乱。在中共二十大临近，内部政治斗争复杂、激烈的情况下，中共党内的反习势力，一直希望习近平和美国发生正面冲突，而台湾问题就是一个最理想的引爆点。
           <br/>
           <br/>
           胡锡进7月29日在社交媒体推特上叫嚣：“如果美国战机护送佩洛西的飞机进入台湾，那就是入侵。中共军方（PLA）有权强行驱逐佩洛西的飞机和美国战斗机，包括鸣枪示警和进行战术性的阻击运动。如果无效，则将其击落。”
           <br/>
           <br/>
           不过，胡锡进的叫嚣似乎没有太大作用。
           <br/>
           <br/>
           白宫新闻秘书卡里娜‧让-皮埃尔说，胡锡进说的是一种“假设”。她表示，白宫为出访官员提供建议和指南，但“我们不会就佩洛西的行程发表言论”。
           <br/>
           <br/>
           一位不愿透露姓名的美国军方官员说，军方目前为止没有向台湾附近地区增派军舰或飞机。
           <br/>
           <br/>
           美国在亚太地区已有大量的军事存在，包括在南中国海周边。美国的一艘航母目前正在南中国海巡弋。
