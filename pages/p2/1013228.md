 <p>
  【看中国2022年8月1日讯】近日，
  <ok>
   佩洛西访台
  </ok>
  成为全球关注的焦点，人们同时也目不转睛地关注着中共方面，分析、研判其是否真的会做出疯狂举动。
  <br>
   <br>
    随着美国方面的坚决表态，中共方面的态度开始出现微妙松动，这一点从其各大喉舌的表现可以看出端倪。
    <br>
     <br>
      7月26日，“九二共识”30周年座谈会召开后，中共喉舌纷纷开始大谈“九二共识”。在7月30日中共《人民日报》以”钟声“署名发表评论文章后，中共喉舌媒体纷纷进行转发，就连胡锡进的腔调也变了。
      <br>
       <br>
        7月26日，中共“九二共识”30周年座谈会在北京举行，全国政协主席汪洋出席并讲话。
        <br>
         <br>
          外界注意到，汪洋在讲话中完全不提佩洛西访台一事，而是施展中共统战、欺骗手法，大谈“九二共识”、“和平统一”，和中共外交部、军方发言人表态完全不同。汪洋在整篇讲话中一共提及“和平”两字17次，其中提及“和平统一”6次，有分析认为，这体现出
          <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
           习近平
          </span>
          当局对佩洛西访台的真正思路。
          <br>
           <br>
            7月27日，中共新华社发表《新华时评：坚定不移坚持“九二共识” 奋力推进祖国统一进程》，
            <br/>
            <br/>
            7月中共新华社发表《署名文章：“九二共识”的思想内涵与历史意义》，作者是中国社会科学院台湾研究所所长杨明杰，还是大谈“九二共识”。
            <br/>
            <br/>
            显然，这些信号显示出中共更倾向于继续使用“统战、欺骗”手法，而非马上大打出手，更不会对佩洛西和美军动武。
            <br/>
            <br/>
            7月30日，中共《人民日报》发表署名为“钟声”的所谓“重要评论文章”，题为《处理中美关系必须加强沟通避免误判》，突然开始大谈“世界和平稳定和发展繁荣”，还强调什么“保持对话沟通，加强互利合作，有效管控分歧...”
            <br/>
            <br/>
            这篇文章很快引发中共各大官媒注意，纷纷以《人民日报钟声：处理中美关系必须加强沟通避免误判》等标题进行转发。
            <br/>
            <br/>
            随后，7月31日，中共民族主义小报《环球时报》前总编辑，网络有“胡叼盘”之称的胡锡进，一改往日战狼姿态，没有再提击落佩洛西战机，似乎开始准备找台阶下了。
            <br/>
            <br/>
            他在微博中说佩洛西，”她有可能想做一个姿态：台湾不是我的访问目的地之一，我即使去也是中停，或者是用私人身份去。以此降低她窜访台湾的挑衅意义，但她只要降落台湾，中方就不会接受。现在我们不能大意，必须继续大声警告她：不许去台湾，去了必有严重后果。”
            <br/>
            <br/>
            胡锡进这番言论，再一次成了众多网友的笑料，
            <br/>
            <br/>
            一位网友嘲讽道，“祖国完全统一在老胡嘴里怎么变成“严重后果”了呢”
            <br/>
            <br/>
            还有人分析，”那为什么要多次警告她？真正有这个能力的警告她一次就够了“
            <br/>
            <br/>
            另一位也表示赞同，“这么强烈的反对, 正说明没有万事俱备”
            <br/>
            <br/>
            还有人说，“真想打就别喊那么响，让人看着没底气一样”
            <br/>
            <br/>
            一位网友直截了当的说，”你老胡少说两句吧 都特么成笑话了“
            <br/>
            <br/>
            有网友分享了自己的观察，“现在官宣的政策还没改变，依然还是和平统一，一国两制。”
            <br/>
            <br/>
            有人提问说，“大家有没有发现一件怪事，平日吵的最凶的台湾，现在出奇的安静”
            <br/>
            <br/>
            一位网友回复说，“因为你在墙里，喂你吃啥你吃啥...”
            <br/>
            <br/>
            也有网友发表了类似的看法，”有墙看不到台湾说什么”
            <br/>
            <br/>
            有人嘲讽道，”不符合**主义核心价值观的消息，应该都过滤掉了“
            <br/>
            <br/>
            8月1日，一位网友在胡锡进最新的微博下方留言，“别自欺欺人了，她去是肯定会去的，不知你脸到时疼不疼。”，也有人说，“胡总说最狠的话，把自己脸打的最疼”。
            <br/>
            <br/>
            其实，中共更是这样，说最狠的话，把自己的脸打的最疼！
