 <p>
  【看中国2022年7月31日讯】7月26日和27日两天，中共高层在一个神秘的地点召开了一次神秘的
  <ok>
   会议
  </ok>
  。会议的标题是《省部级主要领导干部“学习
  <ok>
   <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
    习近平
   </span>
  </ok>
  总书记重要
  <ok>
   讲话精神
  </ok>
  ，迎接党的
  <ok>
   二十大
  </ok>
  ”专题
  <ok>
   研讨班
  </ok>
  》。（简称为2022研讨班）。
  <br>
   <br>
    而五年前，2017年召开的另一次“专题研讨班”，我们在接下来简称为2017研讨班。
    <br>
     <br>
      中共央视用了超过16分钟对这次会议进行报道。显然这次会议对于中共而言非常重要。
      <br>
       <br>
        从中共央视播出的会议画面可以清楚看到，除了台上的7名政治局常委以及王岐山手中有纸本资料之外，台下所有与会官员桌上只有名牌、茶杯，完全没有纸笔或其他可用以记录的工具。
        <br>
         <br>
          这说明会议有非常保密的内容，因此只准听、不准记录，会议内容不允许外传。至于说央视播出的内容，那都是经过筛选的，真正的要害问题，根本不会出现。
          <br/>
          <br/>
          与之相比，今年1月份，习近平在省部级领导干部的一次研讨班上讲话，从央视给出的现场画面看，这些与会官员们都有做记录，和其它的会议没有什么区别。
          <br/>
          <br/>
          另外，今年1月份这次会议地点选在了中共的中央党校（国家行政学院），而最新召开的这次2022研讨班则是选在“京西宾馆”。
          <br/>
          <br/>
          从地点的不同，也能反映这次研讨班的特殊之处。
          <br/>
          <br/>
          我们先来谈谈这个神秘的“京西宾馆”。
          <br/>
          <br/>
          京西宾馆
          <br/>
          <br/>
          了解中共内幕的朋友都知道，“京西宾馆”一贯被视为“外表低调、氛围神秘”，它的名字和中共历史上一些重大事件联系在一起。比如“文革”中的各种荒诞剧、以及后来的审判四人帮、中共十一届三中全会等等等等,文革与改革在京西宾馆里轮番上演着...
          <br/>
          <br/>
          在这里给大家讲一个文革中发生的故事，1967年，许世友躲在京西宾馆，外面来了六七百名南京造反派要抓他。许世友一看大事不好，马上组织人将京西宾馆的开水壶全部集中在七楼（也有的说是十二楼）当做“水雷弹”。
          <br/>
          <br/>
          许世友下令让工作人员关闭电梯，准备造反派敢往上冲，就用开水壶往下浇开水。
          <br/>
          <br/>
          而且，许世友自身还带了枪，准备做最坏的打算，当时的气氛紧张到了极点。最后，周恩来怕把事闹大，只好下令：“京西宾馆从1967年2月7日零点起实行军事接管，未经北京卫戍区允许任何人不许进入。”
          <br/>
          <br/>
          前中共副总参谋长张黎上将曾分管京西宾馆, 在给2015年出版的《京西宾馆那些年那些事》一书做序时，他介绍了京西宾馆的一些内幕。
          <br/>
          <br/>
          他说京西宾馆是北京“最难进”的宾馆：安全保密措施最严格; 对人员出入检查严格，没有会议证，任何身份证、工作证都不准通行，只有会议保卫人员通知，会务组或宾馆工作人员到门口迎领才能进入; 它还是北京唯一不直接接待散客、外宾的宾馆。
          <br/>
          <br/>
          不过，这个号称中共最安全的开会地点曾爆出丑闻，当时的宾馆总经理、解放军总参大校刘存水与前中办主任令计划勾结，利用宾馆对政要监控窃听，用于私人目的;直至令倒台后这事才被揭露。
          <br/>
          <br/>
          《京西宾馆那些年那些事》中还透露，京西宾馆拥有自己专门的电话管理队伍——属于“军委一号台”的通讯女兵，据说每年两会期间，京西宾馆承接代表团接待任务，因此这些女兵能辨别十几种方言、并且可以听音辨人。
          <br/>
          <br/>
          书中还透露京西宾馆的一些奢华细节：餐厅提供的自制酸奶，奶源由总参管理保障部自己的副食品生产基地提供，酸奶口感独特，是宾馆一绝; 京西宾馆自制的各式小菜也享有盛誉，据说某个香港特首夫人品尝了霉豆腐后赞不绝口，临走时特意购买了100罐带回香港馈送亲友; 宾馆的家具是桃木的，卫生间的镜子都是自动加热，房间内每幅画都是各地成名画家专门创作的真迹。
          <br/>
          <br/>
          这就是口口声声说“全心全意为人民服务”的中共官员们享受的待遇。
          <br/>
          <br/>
          而在另一篇名为《京西宾馆，最低调的政治地标》的文章中介绍，京西宾馆最具特色的是会场倒水、续水、送毛巾服务。 女服务员列队进入会场，步调一致、速度相同、整体移动、操作轻盈。
          <br/>
          <br/>
          这段描述，让人想起了北朝鲜的女服务员们，估计北朝鲜是跟中共学来的“先进经验”。
          <br/>
          <br/>
          中共的重要会议多在京西宾馆召开，包括中共十一届三中全会等等。
          <br/>
          <br/>
          近年来钓鱼台国宾馆、北京大会堂等中共党政领导人重要会议举办地，都已对外开放，可京西宾馆仍未对外开放，可以说是中共最“神秘”的宾馆。
          <br/>
          五年前的另一次神秘会议
          <br/>
          <br/>
          五年前，也是在7月的26日、27日，我们前面说的2017研讨班在京举行，中共《人民日报》官方微信号“侠客岛”以夸张的“大事终于来了”形容，吹捧此会议为“十九大前最重要的一次高层会议”。
          <br/>
          <br/>
          我们查看中共官方的报道，发现2017年这次会议，中共官员们面前同样无纸无笔，不允许记录，据媒体报道，该会议的完整内容，至今依然没有公开。
          <br/>
          <br/>
          以此类推，这个2022研讨班也可称之为是二十大召开前“最重要的一次高层会议”。
          <br/>
          <br/>
          中共“侠客岛”的分析文章指出，在开会之前，把所有中共的这些高级官员召集起来开会，其实就是四个字“统一思想”。
          <br/>
          <br/>
          至于习近平要在这个关键的时间点怎样“统一思想”，那是不言而喻的。
          <br/>
          <br/>
          我们注意到，在2022年研讨班讲话中，习近平有一段话，是五年前所没有的，那就是“将科学谋划未来5年乃至更长时期党和国家事业发展的目标任务和大政方针”，结合外界一直以来的分析，这段话的内涵，显然与习近平长期执政紧密相关。
          <br/>
          <br/>
          可以说，这是习近平在二十大上寻求连任，寻求长期执政的重要信号。
          <br/>
          <br/>
          至于有的媒体注意到，习近平在这里还说到“明确宣示举什么旗、走什么路...”,在2017年研讨班上也有同样表述，属于中共的陈词滥调。
          <br/>
          <br/>
          这里还有一个关键的观察点。在2017年研讨班召开之前两天，中共的一个大老虎刚刚落马！
          <br/>
          <br/>
          2017年研讨班是26日举行开幕式，24日，原中央政治局委员、重庆市委书记孙政才被宣布接受调查 。
          <br/>
          <br/>
          孙政才是中共十八大后，第一个在任上落马的中共政治局委员，也是中纪委处理最快的省委书记，从被宣布调查到“双开”，仅耗费了2个月零5天，同样的过程他的前任薄熙来用时5个月零18天。
          <br/>
          <br/>
          由于孙政才曾被视为中共最高权力接班人之一，因此他的落马震动中共政坛。2017年研讨班就是在这种情况下召开的。
          <br/>
          <br/>
          外界一直有分析认为，在今年的二十大之前，将会有中共政治局委员层级的高层人物落马。
          <br/>
          <br/>
          5月3日，法广刊登加拿大维多利亚大学政治和历史系教授、中共前总书记赵紫阳的智囊团成员吴国光的专访。
          <br/>
          <br/>
          吴国光表示，习近平在二十大的目标很明确，就是进入第三任期。再进一步，习近平想在二十大上组织一个让他满意的政治局常委会和政治局。
          <br/>
          <br/>
          由于中共在经济、外交等方面遭遇困境，现在传出很多对习近平不满的声音，尤其是对他最近几年的施政以及他寻求连任等。
          <br/>
          <br/>
          吴国光说，如果体制内的中高层官员也有这个不满，对习近平来讲就相当具有挑战性了，他必然会进行反击。
          <br/>
          <br/>
          吴国光预测，“今年夏天，最迟七八月份以前，会发生习对高层政治人物的政治清洗。”
          <br/>
          <br/>
          他分析说：习近平要借此来进一步威慑党内精英，使各种可能挑战他的力量不敢有所动作，甚至不敢发出声音。同时，习近平也借此机会清除他不喜欢的人，空出位子来，提拔他想要的人等等。
          <br/>
          <br/>
          最近孙力军、肖建华、傅政华三大案件，在一周之内全部出现新的进展，同时《人民日报》《中纪委》等还在释放反腐信号，种种迹象显示，或许还会有更大的老虎落马，或许很快就会发生。
