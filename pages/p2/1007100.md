  【看中国2022年5月22日讯】经济日报报业集团主办的《证券日报》5月21日报道“
  <ok>
   花样年

  系卖掉超过45.6亿元资产， 自救之路仍困难重重”。“花样年”从2009年在香港上市到现在“债务缠身”，从当年的“风光无限”到现在的“至暗时刻”，还不到13年时间。
  <br>
   <br>
    <ok>
     失去融资输血通道
    </ok>
    <br>
     <br>
      “眼下，受债务违约影响，“花样年”这一上市平台相当于已经失去融资输血通道，所以公司仍要加快旗下资产变现，充盈现金流保交付和化解债务危机。”有不愿具名人士向《证券日报》记者表示，想快速变现，一般是要付出一定代价的，比如资产要折价才能达成交易。
      <br>
       <br>
        之前，花样年已经先后转让两个项目公司部分股权，有业内人士评估，这两笔交易预计亏损合计为6100万元。
        <br>
         <br>
          5月19日晚间，“花样年”宣布订立协议，向中交地产出售中交花创51%股权，总交易金额7.6亿元。“花样年”在公告中表示，出售事项为公司正推行以缓解流动资金问题的措施之一。
          <br>
           <br/>
           据悉，2021年9月28日，“花样年”旗下物管上市平台彩生活将核心资产邻里乐控股的全部股权出让给碧桂园服务，总代价不超过33亿元。此外，“花样年”自去年10月份公开债务违约之后，已经出让多笔资产，加上上述资产出售事项，合计出让资产金额超过12.6亿元。由此可知，2021年9月份以来，“花样年”系转让资产额度已超过45.6亿元。
           <br/>
           <br/>
           <ok>
            停牌至今
           </ok>
           <br/>
           <br/>
           从当下“花样年”公布的财务数据来看，并未披露2022年需偿还的债务规模和资金缺口。目前，因未如期交出经过审计的2021年全年业绩，“花样年”及其物业管理上市平台“彩生活”已从4月1日起停牌，至今未复牌。
           <br/>
           <br/>
           一个公司长期停牌，或者长期不对外发布公告信息，会加重外界对公司风险等级的评估，对公司项目销售、融资等将造成负面影响。
           <br/>
           <br/>
           据“花样年”方面披露的官方数据，截至2021年12月6日，“花样年”海外债规模超过40亿美元，约260亿元人民币。
           <br/>
           <br/>
           据花样年投资者关系官微显示，截至2021年12月18日，“18花样年”和“19花样02”债券获得展期，条件均为80%利息展期一年，本金展期两年。“20花样02”债券80%利息展期一年。三笔债券本金累计展期16.79亿元，利息累计展期1.95亿元，总计18.74亿元。
           <br/>
           <br/>
           3月28日晚间，“花样年”与“彩生活”发布联合公告，内容是有关转让邻里乐控股的补充协议。根据股份转让协议，买方（碧桂园服务）应付的销售股份代价为33亿元，于3月28日，“彩生活”已收到23亿元。
           <br/>
           <br/>
           <ok>
            “至暗时刻”
           </ok>
           <br/>
           <br/>
           “花样年”由前中共国家副主席、前中央政治局常委
           <ok>
            曾庆红
           </ok>
           侄女

            曾宝宝
           </ok>

           <br/>
           <br/>
           众所周知，曾庆红是中共
           <ok>
            <span href="https://www.secretchina.com/news/gb/tag/江泽民" target="_blank">

           </ok>
           仰仗哥哥的权势，自90年代中期就以中国文化部特别巡视员的身份派驻香港，成为曾庆红在香港的代理人。
           <br/>
           <br/>
           正是因为如此特殊的政治背景，2009年“花样年”在香港上市时，可谓“风光无限”。据港媒报导，2009年11月，花样年公司的投资者推介会，成为一场香港名流的聚会，除了曾庆淮为其女儿助阵外，到场的还有郑裕彤、刘銮雄、张松桥、蔡志明等多位香港商界大佬。郑裕彤、张松桥等人也都有认购花样年的股份。

  <br/>
  2021年10月4日晚9点57分，曾宝宝发出一则微博，里面一个字也没有，仅配图电影《至暗时刻》海报。
  <br/>
  <br/>
  这完全可以代表曾宝宝当时的心情，因为“花样年”当天遭受连续的沉重打击。
  <br/>
  <br/>
  当晚“花样年”发布公告称，受多重因素的影响，公司流动性出现阶段性紧张，未能在10月4日如期支付2021年票据的剩余本金约2.05亿美元。
  <br/>
  <br/>
  此前几个小时，评级机构惠誉将“花样年”的评级大幅下调了四个级距，从"B"调降至"CCC-"，并警告称，该公司财务状况存在不确定性。惠誉估计，从当时到2022年年底，花样年将需偿付近20亿美元的国际债券，以及近10亿美元的本地市场债券。
