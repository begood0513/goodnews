 <p>
  【看中国2019年8月8日讯】2019年8月5日，人民币兑美元的汇率“破7”，一举打破了10年来人民币兑美元一直维持在“7”以内的历史。而因此就在当日，美国总统
  <span href="https://www.secretchina.com/news/gb/tag/川普" target="_blank">
   川普
  </span>
  (特朗普)直接发推，指出这是汇率操纵。很快，美财政部次日就正式将中国确定为“汇率操纵国”。
  <br>
   <br>
    按道理说，是否是汇率操纵国，美国有3个明确的指标，只有符合才能判定的。这三个指标包括：与美国贸易顺差超过200亿美元；经常账户顺差占GDP比重至少为3%；通过汇率干预买入外汇超过GDP的2%。
    <br>
     <br>
      很显然，这些指标应该是有明确数据的，而不是随心所欲随口指控的。至于，中国是否符合这三个指标，也应该是有足够的数据确认的，不能也不应该是汇率在某一个具体的数字之上或者之下一瞬间就能划定的。如果说汇率操纵的话，那为啥中国的人民币兑美元汇率一直维持在“7”以内，就是合理的，就不是“操纵国”呢？
      <br>
       <br>
        其实，美国指责中国汇率操纵也不是第一次。早在1992到1994年，美国就多次认定中国为汇率操纵国。认定之后，中国对美出口从1994年年度增速26.1%直接下降到1995年的15.1%，同期，中国对外的整体出口年度增速从1994年的31.9%，下降到1995年的22.9%。持续长达18个月。
        <br>
         <br>
          毕竟，一旦被美认定为“汇率操纵国”，再经过一年的缓冲期之后，就会受到“制裁”，包括有禁止该国任何项目获取美国海外私人投资公司融资；将该国排除在美国政府采购供应之外；呼吁IMF加强对该国的监管；重新评估与该国的贸易协定；进行经济制裁；等等。
          <br>
           <br>
            如果延伸到金融资本市场的制裁，甚至汇率战，很可能会出现资产被冻结，或者特定商品的世界贸易被禁止等情况。要知道，目前世界经济秩序本就是建立在以美元为交换货币的基础之上。自从1971年8月15日美国关闭黄金兑美元的窗口，宣布布雷顿森林体系破产之后，虽说35美元不在能够兑换1盎司黄金了，但是，大幅贬值的美元依然是世界各国所认可的世界交换货币。
            <br/>
            <br/>
            也因此，使得美国拥有了可掌控世界贸易支付和兑换交易的两大金融系统，即就是全球金融控制系统和金融支付系统。到此，美国的财政部就可以通过这两个系统，通过海外资产的控制，实施金融制裁。如果美国将某个国家列为制裁范围，那该国的经济，尤其是对外经济将会陷入困境。
            <br/>
            <br/>
            不过，中国是“汇率操纵国”，至少美财政部式这样认定的。中国自从“8.11”汇改以后，实行以“市场供求”为基础、并参考一篮子货币进行调节的、“有管理的”浮动汇率制度。按照国际金融协会的估计，2016年当年一次性将人民币贬值2%之后，中国资本外逃高达7300亿美元，最后逼迫央行花费了1万亿美元才使得汇率稳定。
            <br/>
            <br/>
            这次汇率“破7”，按照央行的说法，是“受单边主义和贸易保护主义措施，及对中国加征关税预期的影响”的结果。实际上，人民币兑美元的汇率从去年开始，就一直徘徊在“破7”的边缘，几乎多次都是“有管理的”浮动。当中美贸易摩擦，在2500亿美元中国出口商品加征25%关税的基础上，川普又宣布将在9月1日对中国3000亿美元出口商品加征10%的关税，此种贸易保护措施，对中国的对外贸易，尤其是对美贸易将会带来极大的影响。
            <br/>
            <br/>
            市场对此“破坏”后果的预期，通过汇率的调整显示出来了。而在笔者看来，这是一个方面，还有一个很重要的因素，就是中国今年以来多次通过积极的财政政策和“稳健”的货币政策，使得市场有“通胀”的预期。众所周知，中国自从2009年以后一直世界上最大的商品出口国。
            <br/>
            <br/>
            只是，中国货币发行的总量也是绝对的世界第一。今年1月份，全面降准释放1.5万亿基础货币，5月主要针对中小银行、民企和小微企业实施定向降准释放2800亿。同时，1月、4月、7月，央行开展中期借贷便利释放流动性，分别为2575亿、2674亿、2977亿人民币。今年上半年社会融资规模高达13.23万亿，同比增长31.35%。官方的数据，是在今年6月底，中国的M2为192万亿人民币。M2数据超过美国与欧盟的总和。
            <br/>
            <br/>
            再者，中国实施的是强制结汇政策。就是说，所有合法进入中国的外汇必须通过各商业银行上缴到央行，而出境的人民币通过各商业银行从央行兑换成外汇，所以有了在岸汇率和离岸汇率之分。前者就是通过商业银行，将外汇兑换成人民币的汇率，后者是通过商业银行，主要是在香港兑换成外汇的汇率。试想，当国内的人民币数量如此之大，本来是以外汇为“锚”，后来又改成以国债或者债务为“锚”，同时各大商业银行陆续开始发行“永续债”，如此以来，缺乏必要限制的人民币，面对有限的外汇，其汇率咋能不浮动呢？特别是，咋能不“破7”，甚至，“破8”也是一种大概率的预期。
            <br/>
            <br/>
            总之，中国本就是“有管理的”浮动汇率，而且在特殊的强制结汇政策下，汇率受到了美方贸易保护政策的“破坏预期”，也包括中国“稳定向好”的“稳（放）健(水）”的货币政策影响，“破7”或者说“破8”，都完全有可能。
