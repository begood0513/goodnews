 <p>
  【看中国2019年10月20日讯】一年多前，
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/长春长生" target="_blank">
    长春长生
   </span>
  </strong>
  公司生产的狂犬病
  <strong>
   疫苗
  </strong>
  被曝存在记录造假，再度引燃中国社会对疫苗问题的热议。周三，长春长生正式进入退市阶段，预计下个月摘牌。另外，这家企业被罚90多亿元，已经申请
  <strong>
   破产
  </strong>
  清算。但中国疫苗事件频发，只靠
  <strong>
   罚款
  </strong>
  够吗？
  <br>
   <br>
    去年被查出生产了狂犬病问题疫苗的长春长生生物科技有限公司（简称长生生物）本周进入为期一个月的退市阶段。
    <span id="hideid" name="hideid" style="color:red;display:none;">
     <span href="https://www.secretchina.com">
     </span>
    </span>
   </br>
  </br>
 </p>
 <p>
  周三，因违法经营处于停产状态的长生生物在停牌半年多后复牌，首日盘前股价就已跌停。上周，深圳证券交易所发布公告说，决定对长生生物股票终止上市，使其成为A股市场中首家因重大违法行为而被强制退市的公司。
  <br>
   <br>
    去年7月，中国国家药监局通告说，他们在飞行检查中发现长生生物生产的狂犬病
    <span href="https://www.secretchina.com/news/gb/tag/疫苗" target="_blank">
     疫苗
    </span>
    存在记录造假等严重违规行为。两周后，长春警方以涉嫌生产、销售劣药罪，对公司董事长高俊芳在内的18名犯罪嫌疑人提请批捕。
   </br>
  </br>
 </p>
 <p>
  事后，吉林食药监局开具的行政处罚决定书显示，自2014年起，长生生物违法生产销售了700多批狂犬病疫苗，决定没收其违法所得约19亿元，并处三倍的罚款，公司共需缴纳91亿元罚款。事发前，长生生物总资产约为40亿元。这笔天价罚款、加上生产许可证被吊销，击垮了长生生物。目前，公司已向当地法院申请
  <span href="https://www.secretchina.com/news/gb/tag/破产" target="_blank">
   破产
  </span>
  清算。
  <br>
   <br>
    <strong>
     受害者家长：光罚款还不够
    </strong>
    <br>
     <br>
      山西疫苗受害者家长易文龙的女儿2006年因接种了问题疫苗而患上“急性播散性脑脊髓炎”。事后，他为此上访多年，至今出行仍受到当局监控。
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  易文龙周五对本台记者表示，他并不认为单靠
  <span href="https://www.secretchina.com/news/gb/tag/罚款" target="_blank">
   罚款
  </span>
  就能解决中国疫苗乱象。他说，当事人应当付出失去自由的代价。
  <br>
   <br/>
   “单纯靠罚款（是不够的）。这些事件造成如此严重的伤害，我觉得罚款只是一方面，而另一方面当事人要付刑事责任，这样才能达到更好的效果。”
   <br/>
   <br/>
   长生生物董事长高俊芳自去年7月被刑拘之后，已于今年年初被当局以行贿受贿罪、挪用资金罪移送至当地检察院。就在本周一，长生生物宣布，公司董事会决定免除高俊芳的总经理职务。
   <br/>
   <br/>
   <strong>
    新法律存在漏洞
   </strong>
   <br/>
   <br/>
   记者注意到，中国十三届全国人大今年6月通过了《疫苗管理法》，对生产销售假药或劣药的负责人划清了刑事责任。其中，生产销售的疫苗属于假药、或生产销售的疫苗属于劣药且情节严重的，将被终身禁止从事药品生产经营活动，并处五日以上十五日以下拘留。
   <br/>
   <br/>
   但在一些法律学者看来，这样的处罚标准仍然存在明显漏洞。
   <br/>
   <br/>
   中国最高检机关报《检察日报》上个月刊登了西南政法大学法学院教授卢有学和法学硕士王禹的文章。文中说，《药品管理法》规定，假药包括假冒、被污染、变质等情形，而劣药则包括成分含量不达标、更改有效期、更改生产批号等情形。据此，目前中国的大部分问题疫苗都会被列入劣药的范畴，只有一小部分符合假药的定义。而劣药的社会危害性并不低于假药。
   <br/>
   <br/>
   作者表示，在中国，生产、销售劣药罪需要满足“对人体健康造成严重危害”的高入罪门槛。但由于疫苗起到的是预防作用且并不能做到完全免疫，而疾病发生又是概率事件，因此外界很难断定“人身危害”与接种疫苗的因果关系，这使得生产、销售劣药罪沦为了一项“空罪名”。
   <br/>
   <br/>
   <strong>
    处罚毒疫苗企业不能一视同仁  出于政治考量？
   </strong>
  </br>
 </p>
 <p>
  <span href="https://www.secretchina.com" target="_blank">
   中国
  </span>
  金融学者贺江兵认为，国内疫苗事件频发的一大原因就是处罚力度不够，而长生生物遭灭顶之灾、不少其他疫苗厂商却被“网开一面”，反映了让人难以捉摸的政治考量。
  <br/>
  <br/>
  “我感觉还是震慑力不够。我知道当局对长生生物（的处理意见）得到了很多高层领导的批示，所以才处罚这么狠。对其它一般的（涉事）疫苗公司不是并没有处罚吗？”
  <br/>
  <br/>
  几乎在同一时间被曝生产销售了40万支不合格百白破疫苗的武汉生物公司的日子，看起来就比长生生物好过得多。媒体报道，武汉生物仅被没收违法所得、并处罚款，但仍在正常生产。
  <br/>
  <br/>
  据悉，11月26日将是长生生物的最后一个交易日，随后深交所将对公司股票予以摘牌。
 </p>
