 <p>
  据
  <span href="https://www.secretchina.com/news/gb/tag/美国" target="_blank">
   美国
  </span>
  之音报导，美国政府2月4日公布的报告还显示，大幅上调了对2021年11月和12月就业增长的预期，总计上调到70万9000人。报告还说，失业率从3.9%上升到仍然较低的4%，主要是因为有更多的人开始求职，而且不是所有人都能立刻找到工作。
  <br>
   <br>
    1月份的强劲招聘增长表明，许多雇主在疫情肆虐之际仍渴望招聘。这种增幅超出了之前人们作出的只会略有增加的预期。企业看来认为奥密克戎浪潮对经济的影响最多是暂时的，并对其更长期的经济前景保持信心。
    <br>
     <br>
      专注于小时工的招聘网站Snagajob的首席执行官马修·史蒂文森（Mathieu Stevenson）对美联社说，“雇主们认为奥密克戎病毒会很痛苦，但会是短期的，所以他们没有改变招聘计划”， “雇主的需求一如既往地强劲”。
      <br>
       <br>
        1月份就业人数的增加，以及对前几个月的大幅向上修正，意味着美国的就业岗位比美国政府一个月前公布的数据多了110万。强劲的招聘和稳定的工资增长正在提振消费者支出。这种情况撞上混乱的供应链，使得通胀率飙升至40年的高位。
        <br>
         <br>
          在对物价上涨进行调整后，美国人的平均工资的涨幅不及一年前，尽管许多工人加薪了。许多家庭，尤其是低收入家庭，都在挣扎着支付汽油、食物、房租和育儿等必需品的费用。
          <br>
           <br>
            这些趋势将使美联储在提高利率方面有更多的回旋余地，甚至可能比其计划的速度更快，从而为通胀降温。美联储已经表示，将在3月份开始加息，并可能在5月份举行的下次会议上再次加息。更快的加息步伐可能会减少借贷和支出。
            <br/>
            <br/>
            由于市场预期美联储将更快地收紧信贷，股市2月4日最初下跌，随后在午后回升。但10年期美国国债的收益率跃升近0.1个百分点，至1.91%，表明投资者预计借款成本将上升。

 <p>
  从整体经济来看，大多数行业上个月雇佣了员工，包括零售商、餐厅和酒店。零售商增加了6.1万个工作岗位，餐厅和酒店增加了13.1万个工作岗位。运输和仓储公司增加了54000个
  <span href="https://www.secretchina.com/news/gb/tag/就业" target="_blank">
   就业
  </span>
  机会。经济学家说，这些行业的许多公司可能保留了一些在冬季假期聘用的员工，而不是将他们全部解雇。
  <br/>
  <br/>
  奥密克戎变种病毒确实在就业报告中留下了一些痕迹：在家办公的美国人比例从去年12月的11%上升到了15%以上。上个月，因病离岗人数飙升至360万人，高于1月份不到200万人的水平，大约是大流行病爆发前水平的三倍。这迫使从餐馆到零售商再到制造商在内的许多公司因为员工短缺而减少营业时间甚至关门。
  <br/>
  <br/>
  佩拉·赫尔南德斯（Perla Hernandez）是因病离岗的员工之一。她一家八口上个月都感染了COVID-19。赫尔南德斯和她的丈夫以及20岁的女儿都没有上班，这对家庭的财政状况是一个重大打击。
  <br/>
  <br/>
  现年42岁、住在加州圣何塞的赫尔南德斯是汉堡王（Burger King）的厨子和勤杂工。她六天没有上班。因为她没有带薪的病假，所以每两周拿到的薪水只有230美元。
  <br/>
  <br/>
  大约五分之一的美国工人没有带薪病假，而在低收入的服务行业中，这一比例要高得多。在工资水平最低的10%的员工中，只有33%的人能获得带薪病假，而在收入最高的10%的员工中，这一比例达到95%。
  <br/>
  <br/>
  赫尔南德斯通过一名翻译对美联社说：“感谢上帝，我们已经付了1月份的房租。”
  <br/>
  <br/>
  赫尔南德斯说，6个月前，她的工资涨了45美分，现在达到每小时15.45美元。但她和她的同事，包括经理们，一直在加班，因为餐厅招不到人。
 </p>
 <p>
  就业网站玻璃门（Glassdoor）的高级经济师丹尼尔·赵（Daniel Zhao）对美联社表示，健康的招聘情况——不仅是1月份，11月和12月也是如此——表明上月的增长并非昙花一现。
  <br/>
  <br/>
  丹尼尔·赵说：“这是一个真实的趋势，而且就业增长比我们意识到的要快。”
  <br/>
  <br/>
  这份就业报告显示，越来越多的美国人现在正在工作或正在找工作，这一趋势使企业更容易找到工人。
  <br/>
  <br/>
  丹尼尔·赵表示：“社会上是有员工的，只是需要时间让他们重新融入劳动力大军。”
  <br/>
  <br/>
  瑞达工程和机械公司（Reata Engineering and Machine Works）的首席执行官格雷迪·科普（Grady Cope）说，上个月他的43名员工中有九人生病了，这是他经营公司近30年来最严重的一次。
  <br/>
  <br/>
  但科普这家为飞机和医疗设备制造商制造零部件的公司也拥有它有史以来最大的订单积压。他希望至少增加8名员工，包括机械师、装配工和工程师。上个月，他将工资提高了18%，远远高于通常的3%～4%的涨幅。他的公司位于丹佛附近，那里的租金和其他成本正在快速上涨。
  <br/>
  <br/>
  科普说：“人们必须有工资才能养活自己和家庭。”
  <br/>
  <br/>
  尽管如此，科普还是提高了自己产品的价格，以抵消给他的工人们支付更高的工资带来的成本增加。他说，对员工的争夺是他所见过的最激烈的。10月份，他的四名员工辞职了。“这是28年来从未发生过的。”
  <br/>
  <br/>
  就业市场的整体前景依然光明，职位空缺接近创纪录的高位，裁员速度下降，失业率已经达到一个健康的水平。在对劳动力规模进行调整后，去年美国新增的就业
  <span href="https://www.secretchina.com/news/gb/tag/岗位" target="_blank">
   岗位
  </span>
  比1978年以来的任何一年都多。这种改善在很大程度上表明，在2020年大流行病造成的衰退导致创纪录的失业之后，就业形势大幅反弹。
