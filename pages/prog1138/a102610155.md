<hr/><div class="post_content" itemprop="articleBody">
 <p class="p1">
  <span class="s1">
   【新唐人北京时间2019年06月27日讯】“
   <span href="https://www.ntdtv.com/gb/川习会.htm">
    川习会
   </span>
   ”将在20国集团（G20）日本大阪峰会上登场。消息称，由于上万港人集会促请G20成员国向北京施压，以及多个维权团体前往大阪抗议中共，北京方面已强烈要求日本政府，彻底做好
   <span href="https://www.ntdtv.com/gb/习近平.htm">
    习近平
   </span>
   的维安工作，以维护习的政治尊严。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   日本读卖新闻26日报导，
   <span href="https://www.ntdtv.com/gb/g20峰会.htm">
    G20峰会
   </span>
   将于6月28、29日在大阪举行，中国国家主席
   <span href="https://www.ntdtv.com/gb/习近平.htm">
    习近平
   </span>
   27日将抵达大阪出席G20峰会，中共高层已经提前抵达日本，要求日本政府彻底做好习近平的维安工作，以维护习的政治尊严。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   据报导，中方认为这攸关习近平的面子，所以显得有些神经质，而日方回应，将依法处理。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   <span href="https://www.ntdtv.com/gb/g20峰会.htm">
    G20峰会
   </span>
   召开前，已经有来自多个国家的维权团体，赶赴大阪市准备举行示威抗议活动，包括新疆维吾尔族人权团体也将举行示威活动。近日，中共迫害新疆维族人证据流出，据不完全统计，至少有高达450万维族人被监控。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   目前，维吾尔族海外领袖热比娅（Rebiya Kadeer）已获日本政府核发入境签证，将在大阪召开记者会指控中共的暴行。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    港人呼吁外国领袖关注香港问题
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   与此同时，成千上万的港人26日晚穿着黑衣，向G20成员国的其他19国驻香港领馆请愿，表达反对《逃犯条例》修订的立场。他们呼吁外国领袖关注香港问题。 还有香港示威者亲自到大阪，在G20峰会举行期间发起抗议活动。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   此外，香港网友将于27日至29日，在数家知名国际媒体刊登公开信，敦请G20和各国民众关注香港“逃犯条例”修订争议，并向各国政府传达香港人的忧虑。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   27日晚，在大阪还将有一场名为“守护香港的自由与民主主义行动”集会活动，参加者包括留学或旅居日本的香港人，也有香港人前往日本参加活动。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   美方加码运用“人权牌”施压北京
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   令北京当局雪上加霜的是，103万港人9日大游行结束后，来自美国国会参众两院的两党议员再次推出《香港人权与民主法》。16日200万港人再次走上街头后，美国国务卿蓬佩奥表示，
   <span href="https://www.ntdtv.com/gb/川普.htm">
    川普
   </span>
   将在G20峰会上，向习近平提出香港人权问题。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   《中央社》引述分析说，在中美摩擦升级之际，此举凸显美方加码运用“人权牌”打贸易战，而香港问题或许已纳入
   <span href="https://www.ntdtv.com/gb/川普.htm">
    川普
   </span>
   的武器箱。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   川普在启程前往日本大阪参加G20峰会前几小时，接受霍士财经新闻网访问时说：“中国经济快完蛋了，他们希望达成协议。他们比我们更希望达成协议。”
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   川普还重申，假如在大阪的“习川会”未能达成贸易协议，他已经准备好对余下的中国商品加征额外关税，并计划削减与中共的商业往来。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   《日本产经新闻》刊发社论说，各国应了解到，目前美中对立的根本原因，在于中共的问题。是中共不尊重欧美日等先进国家的市场经济规则，逐渐经济大国化，追求霸权，中共不公正的举止正动摇著世界的安定与繁荣。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   社论还表示，G20峰会应该是让各国元首重视到中国问题的场合，其中包括人权问题。香港的《逃犯条例》修订案虽被搁置，但“一国两制”遭破坏的可能性仍存在。而且新疆、西藏等人权遭严重迫害的情况持续存在，如果日本视而不见，在标榜自由、人权、民主的亚洲国家日本召开G20的意义将遭质疑。
  </span>
 </p>
 <p class="p1">
  <strong>
   <span class="s1">
    各国抗共抗暴连成一片
   </span>
  </strong>
 </p>
 <p class="p1">
  <span class="s1">
   除了
   <span href="https://www.ntdtv.com/gb/美中贸易战.htm">
    美中贸易战
   </span>
   和百万港人反“送中”恶法外，台湾朝野也越来越坚决地抵制中共，拒绝“一国两制”，22日还发起了数万人反中共红媒渗透示威活动。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   更让中共担心的是，港台民众的抗争，可能会在大陆引起蝴蝶效应。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   《自由亚洲电台》引述学者的话分析说，习近平现在是内外交困，已经陷入空前危机。外部包括贸易战，美中关系恶化，被全球孤立，香港反恶法，台湾反红媒等；内部包括经济恶化，股市、汇市、债市都会出问题，还有是粮食问题，失业问题，以及各地风起云涌的官民冲突等。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   与此同时，还有贸易战引发的中共党内惨烈权斗，该学者认为，中共党内目前存在四大反习力量，即残存江派势力，中央到地方数以百万的贪官，贪腐经商的太子党，以及军中反习势力。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   学者分析，习近平现在是党内摆不平，外部也会相互激荡。如果选择不好，习的肉体和政治生命都会受影响，而关键是习是否要抛弃共产党，目前，他在这方面也是两边摇摆。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   评论人士认为，未来中共不堪贸易战重击、为生存被迫改变路线时，习当局就是现成的“替罪羊”。而在中共历史上，每次出现生存危机都会引发一场你死我活的路线之争，最后揪出一些当权者为其“错误路线”负责。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   大纪元系列社论《魔鬼在统治着我们的世界》中则提到：神安排了中共最后的解体。中国的执政者和其他掌握权柄的人，如果有意解体中共，神为其安排好了所有的一切，包括未来天赋神授的真正权柄。相反，如果死抱中共不放，必定会在最后的过程中遭遇中共解体所带来的一切灾祸、魔难。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   （记者李韵报导/责任编辑：剑彤）
  </span>
 </p>
