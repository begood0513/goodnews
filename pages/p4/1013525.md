 <p>
  各位朋友大家好！我是苏拾莹，今天要跟大家谈一谈中国在台湾四周封锁做军事演习，台湾人为什么都不害怕，很淡定？
  <br>
   <br>
    美国众议院议长裴洛西刚刚访问台湾，在他还没有去之前，中共就做了很多的军事威吓，在7月底就已经做很多密集的演习，最少有五个地方。最近的还在平潭岛，这个地方距离台湾最近，离台湾的新竹只有125公里。警告的意味非常浓厚。
    <br>
     <br>
      裴洛西走之后，中共又从今天开始在台湾四周围六处做军事演习三天，用包夹式的，就如同是封锁台湾。影响了650航班的飞机飞行。但是台湾也已经有准备了，和日本,菲律宾协调好替代路线，就是变更一下路线，照样飞。就像裴洛西那天从马来西亚飞台湾，一般从南海飞过来最快，但中共既然在那里军演，那就避开，绕从印尼菲律宾过来就是了。
      <br>
       <br>
        只是这次中共对台湾的恫吓，台湾人好像都很淡定，没在怕。和1996年比起来差很多。1996年那时候台湾第一次举办总统大选，中共用飞弹瞄准台湾恫吓，的确掀起台海危机，对台湾人的冲击不小，很多人就抢着搭飞机出国避难。但是这一次，台湾人好像没有太大的反应，也没抢着出国。一般表现都满有信心，很淡定并不太害怕。
        <br>
         <br>
          分析原因，有三点。
          <br>
           <br>
            第一，因为中共解放军像喊”狼来了”，喊太多次了，搞的台湾人民已经麻痹了。解放军72年来一直都说要解放台湾，但也没见到行动。最近几年
            <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
             习近平
            </span>
            动不动就说要武统台湾，但也就是口头恫吓。
            <br/>
            <br/>
            去年也说如果美军出现在台湾，那就是踩到中共的底线，解放军就要行动了。后来没多久台湾蔡英文总统在接受CNN访问的时候，就证实说有美军在台湾协助军事训练。但解放军也没有真的动作。
            <br/>
            <br/>
            再说共机每天都来扰台，从2016年就开始了。2020年统计有380架次，去年更高达950架次。今年以来当然更是密集。所谓狼来了喊久了就不灵了，大家都麻痹了。
            <br/>
            <br/>
            第二方面，是台湾人对自己的军事实力，信心提高了很多。
            <br/>
            以前台湾人不太关心军事消息，国防部也把军情当成是军事机密，很少公开。但近几年都不一样了。当然是因为习近平天天喊着要打台湾。逼得人民要去了解自己的防卫能力在哪里。立法院民意代表天天都在盯着。
            <br/>
            <br/>
            台湾的国防部有一个军事情报网，这几年做得真的很不错，各种消息及科普，都公开讨论，把台湾人的军事知识水平都拉高了。对武器 ,飞弹,军机,军舰…型号和性能，都很熟悉。台湾这种军事媒体还不只一家，各个政论新闻节目也常常请专家去做讨论，普遍拉高了全民的军事常识。
            <br/>
            <br/>
            于是大家都知道，台湾防空飞弹的密度高居世界第二，仅次于以色列。什么天弓飞弹啦，爱国者导弹啦…连小孩子都晓得。也都知道云峰飞弹甚至可以打到北京以北，射程两千公里。台湾的空中防卫系统很强，这是个事实。
