   <br>
    首先，在意大利、法国、英国和北美的许多中国人，不谙本地语言，他们看见西方文明国家医院病床有限，长期在西方生活，一双眼睛日夜盯着的中国人的百度腾讯和人民日报，认定敌人一天一天烂下去、我们一天一天好起来。

     <br>
      而且他们认为一旦在意大利西班牙这种国家感染武肺，送到医院，语言不通，医院不会优先护理。

       <br>

         <br>

          <br/>
          各大航空公司看在眼里，CEO都知道航空公司倒闭即来，就是你这个民族累的，此时怎会不将机票抬成天价，关门前也报复性狠宰一笔。遂形成武肺全球爆发之外，中国人逃难奇景。
          <br/>
          <br/>
          其实留在本国自我封关，你一点事也没有。反而挤上一架飞机，喧哗浮躁，前面咳嗽，后面几位去厕所，门把椅背，万一飞机上有一两个播毒者，一到大陆就变成七八个。
          <br/>
          <br/>
          中方不是傻瓜，看见海外华人挤压回来，心中知道真相，呼吁：确实国家的疫情是稳定下来了，正在走向胜利，但请你们不要急着一窝蜂回来。
          <br/>
          <br/>
          但即使这样，由于战狼宣传效力伟大，还是挡不住一波波天涯浪子重新寻找祖国母亲的怀抱。哪知道形势天天变，各国政府各有盘算：台湾和新加坡忽然宣布禁止转机，买好跳棋式多点线套票的豪客，中途忽然之间就出了乱子，加上行李接驳有误，就此乱作一团。
