 <p>
  <br>
   太子奶当时的风头原本蒙牛和伊利大很多，太子奶成立于1996年，由当地商人李途纯在湖南株洲创立。太子奶可以说是一战成名，当时李途纯四处筹钱以8888万元的价格成为央视快消品的年度标王。
  </br>
 </p>
 <p>
  2005年，太子奶打败乐百氏和娃哈哈，拿下70%的发酵型乳酸菌市场，还请了谢霆锋和Twins为品牌代言。2001年至2007年，太子奶的销售额从5000万元飙升至30亿元，李途纯的财富也跟着水涨船高。
  <br>
   <br>
    太子奶案当时轰动整个大陆，媒体更将其命名为太子奶事件，事件使得 “二人致死，三人致残，一人致流产”，事件主角李途纯被关超过一年，这背后的直接操刀手就是
    <span href="https://www.secretchina.com/news/gb/tag/凌娅" target="_blank">
     凌娅
    </span>
    。
    <br>
     <br>
      当年凌娅为了逼迫李途纯认罪，讲李的弟弟和妹妹羁押，羁押过程中进行刑讯逼供，导致二人一个全身瘫痪，一个偏瘫。就这样凌娅还不罢手，为了逼迫李途纯的舅舅检举揭发李途纯，最终把李途纯的舅舅在除夕夜割腕“自杀”，更有人猜测是凌娅安排人杀害了李的舅舅，再将现场伪造成自杀。
      <br/>
      <br/>
      此外，凌娅逼太子奶原总工程师李立军检举揭发李途纯遭对方拒绝，于是凌娅就给总工程师灌酒。结果，从不喝酒的工程师，生生被灌酒醉死。
      <br/>
      <br/>
      后来，李途纯虽然被判无罪，但商业瞬息万变，出狱的李途纯再也无非重振太子奶，迫于无奈他开始长期的维权活动，并检举揭发凌娅的违法犯罪行，据李途纯亲口讲，凌娅对检举揭发不屑一顾，她甚至放言，“我不怕你告，我的后台还在”。
      <br/>
      <br/>
      见过嚣张的，真是没见过像凌娅这样嚣张跋扈的。据李途纯透露，，他曾被迫陪同凌娅唱卡拉OK，实际上是去买单。凌娅喜欢喝酒吸毒，且不避讳，每次喝酒吸毒后总会有生活作风问题。
      <br/>
      <br/>
      凌娅自首后，李途纯连发数篇署名文章揭露其办案过程中的违法违纪行为。李途纯爆料称，凌娅私吞太子奶千万货款当自己的小金库，他本人曾被凌娅派人监控，还被其设计仙人跳企图陷害。
      <br/>
      <br/>
      其实，株洲高科抢走太子奶后并没有再续其辉煌，反而使太子奶一蹶不振，凌娅一伙人整人很在行，搞经营却是烂的一塌糊涂。当初眼见维权无望，李途纯带领太子奶团队又在北京成立了仙山奶业公司，并承诺把几十亿资产赠送给株州市政府，债务则由仙山奶业与李途纯个人承担。
      <br/>
      <br/>
      释放出如此大的善于也没能填满凌娅一伙人的贪欲，当时太子奶在北京成立的公司获得了六个亿的订单，凌娅主导的五人专案小组眼看经营太子奶失败，又便调动几十名公安在全国各地将李途纯家人、太子奶骨干19人全部抓回株洲关押，此次李途纯想东山再起的愿望彻底破灭。
      <br/>
      凌娅活生生演了一出吃人不吐骨头的惊悚大戏，彼时搞出这么大阵仗的凌娅也只不过是一个公安局的副局长，副局长领导的五人迫害小分队，竟然将一个知名民营企业家搞得家破人亡，惨不忍睹，难怪凌娅敢当着李途纯面说自己有后台。
      <br/>
      <br/>
      最近，大陆一部反黑题材的电视剧《狂飙》大火。然而，了解了凌娅一会人的所作所为后，起码看清两个事实：第一，中共官员真是比黑社会还黑社会；第二，现实比电视剧更为惨烈。
