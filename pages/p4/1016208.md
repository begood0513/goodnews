 <p>
  【看中国2022年9月8日讯】最近，大陆一部小众的文艺片成为影坛黑马，票房轻而易举的突破亿元。其实这部电影刚上映时并不被看好，首映票房仅35万元，最低潮时不过10余万，后来电影爆红单日票房超过了1500万元，最终实现逆袭。
  <br>
   <br>
    这部电影除了海青属于一线演员，其余人都是当地农民，导演李睿珺的父母和姨父参与了影片的拍摄，他们也都是当地的农民。这部电影就是《
    <span href="https://www.secretchina.com/news/gb/tag/隐入尘烟" target="_blank">
     隐入尘烟
    </span>
    》，其实《隐入尘烟》一度因经费紧张而难产，导演李睿珺最惨的时候兜里仅剩2000块。
    <br>
     <br>
      海清作为大陆知名演员，是什么打动了她零片酬扮演一名总尿裤子的村妇？影评人认为该电影赚足了观众的眼泪，在谎言漫天的大陆，能有这样真实的电影实属不易。
      <br>
       <br>
        未来能尽快融入角色，海清在村子里住了很长时间，她头裹纱巾，喝过脏水，铲过大粪，每日素面朝天的与当地村民打成一片，时间一久外人也很难分辨的出。
        <br>
         <br>
          当然，这部展现了西北农村愚昧与贫穷的电影被一些人认为是别有用心，其过度放大了农村的苦难，最后以悲剧结束容易让人看笑话，很多大陆网友甚至表示不相信中国还有这样的地方。
          <br>
           <br>
            电影没有搭设影棚，完全是实地取景，之所以选择实地取景，一是剧组实在没钱，这个之前提到过；二是取景地就是导演的家乡，讲家乡的故事对导演来说更加信手拈来。
            <br>
             <br>
              拍摄地是甘肃张掖的高台县，此地紧邻毛乌素沙漠，环境艰苦。甘肃省人社厅将全省收入分为四类，故事发生的高台县就属于第四类地区。高台县政府公布的数据，当地农村居民人均可支配收入17870.4元，月均不足1500元，妥妥的贫困地区。

 <p>
  说回电影，其实主题可以总结成一句话，“马善被人骑，人善被人欺”，而且最先欺负你的不是别人而是你身边最亲近的人。故事主人公马有铁，他在家中排行老四，因此也被村里人叫作马老四。
  <br/>
  <br/>
  马老四哥哥马有金、马有银都已经去世，三哥马有铜日子过得不错，但老四被没有粘上光，他只能寄居在村里被弃置的房屋里，守着一头驴过日子，而且时不时的被三哥呼来喝去。
  <br/>
  <br/>
  海清饰演的女主曹贵英，一条腿还瘸了，干不了重体力活只能寄宿在哥哥嫂嫂家，住在风吹雨淋的大棚，因为被打过，导致她患有尿路疾病，小便失禁，甚至丧失了生育能力。
  <br/>
  <br/>
  马老四和曹贵英在双方家人的商量下最终成为伴侣，他俩在家人们的眼中都属于被嫌弃的对象，与其说帮其找对象，不如说是给家中的拖油瓶找下家。
  <br/>
  <br/>
  曹贵英的嫂嫂在相亲场合还不断催促曹贵英快去尿尿，她担心一不小心尿裤子的曹贵英会搅黄了婚事；马老三担心单身汉的弟弟拖累自己，急不可耐的答应下这门亲事。
  <br/>
  <br/>
  马老四和曹贵英的家人很冷血，周围的邻居大多充当看客，时不时的要看马老四的瘸腿媳妇出丑，马老四对此从不以理据争，想必是逆来顺受惯了，每当妻子尿裤子他会脱下上衣为其“遮羞”。
  <br/>
  <br/>
  马有铁是个善心人，他不嫌弃曹贵英身体的疾病和残疾，去城里看到漂亮衣服会买给曹贵英，他干活时让曹贵英去看电视，小毛驴拉的车永远让曹贵英坐着。
  <br/>
  <br/>
  原本马老四在村子里的存在感很低，结果因为是稀有血型，他瞬间成了全村人的希望，大家希望他为有钱的老板献血，从而让老板补上拖欠的款项。
  <br/>
  <br/>
  于是马老四赶鸭子上架为老板献了血，结果老板活了命但还是不想还钱，此时村子里开始有偿拆除空屋，新婚的马老四一下子成了无家可归的人。
 </p>
 <p>
 </p>
 <p>
  马老四选择与命运抗争，他盖房子借鸡蛋孵小鸡，可是霉运始终缠着他，后来
  <span href="https://www.secretchina.com/news/gb/tag/曹贵英" target="_blank">
   曹贵英
  </span>
  在送饭去给他的路上，发晕摔进水沟里不幸去世，最后马老四又成了孤家寡人。
  <br/>
  <br/>
  失去曹贵英的马老四等于失去了一切，最后他还光了所有人情，包括前期借邻居的10枚鸡蛋，在还债方面，马老四秒杀了欠钱不还的大老板。
  <br/>
  <br/>
  马有铁之所以这么事无巨细，其实是在跟这个世道算明白账，作为老实人，马有铁并不想欠这个世界什么，从剧情判断，了却一切尘世的烦恼后，马老四大概率是要追随曹贵英而去。
  <br/>
  <br/>
  最终这对苦命夫妻都将远离这个世界，而马老三包括欠钱的老板，他们仿佛成了胜利者。真是应了一句话，在大陆这个光怪陆离的世界，好人逆来顺受，坏人颐指气使，好人不偿命，坏人活千年。
 </p>
