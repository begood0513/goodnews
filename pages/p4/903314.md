 <p>
  【看中国2019年8月11日讯】亲北京的香港《大公报》“起底”美国的驻港外交官，引发美国国务院强烈的反弹。本台获得的消息显示，中国此举使美方人员及家人处于危险的境地。美国国务院的消息表示，根据外交惯例，所在国有责任保护外交使节的个人安全、自由、尊严。国务院再次批评中方的做法是“非负责任国家”所为。
  <br>
   <br>
    根据本台由美国国务院获得的消息，美方视香港《大公报》是受到中国国家操控的媒体。在美方驻港外交官被“起底”后，中国外交部在社交媒体将事件放大，更引发美方反弹。美国国务院发言人周四（8日）以“
    <span href="https://www.secretchina.com/news/gb/tag/流氓政权" target="_blank">
     流氓政权
    </span>
    ”（thuggish regime）称呼中国；一日后，中方透过外交部驻港公署在网上发声明，反指美方的有关言论是对中方的公然诋毁和诽谤。
    <br>
     <br>
      白宫去年使用“奥威尔式胡言乱语”（Orwellian nonsense），批评中国要求美国的航空公司不得将台湾标注为国家。近日，两国外交骂战再次升级。美国国务院消息向本台指，中方长久以来有著破坏承诺的纪录，包括违背在中英联合声明、南海非军事化等作出的承诺。
      <br>
       <br>
        就“流氓政权”四字用语是否恰当？本台在YouTube频道社区发起网络调查，5个小时内收到近5000反馈。当中11%的读者认为用字“偏颇歪曲事实/公然诋毁和诽谤”；4%认为“用语过火/引发不必要争议”；13%认为“恰到好处/点到即止”；69%认为“讲法直言不讳/来得正是时候”；有4%读者表示“不懂政治/无法评论”。
        <br>
         <br>
          纽约的时政评论人士陈破空对本台说，美国国务院表态中国是“流氓政权”，说的是一个事实。中方给美国贴上“香港幕后黑手”的标签，则是扭曲香港民众的抗争。
          <br/>
          <br/>
          陈破空︰我想长期以来她（美国）有认知，只是没有说出口。那么这次因为中共做得太过分，加上美国又是特朗普政府，所以就用“流氓政权”表述中共。一方面表述的事实，打破了过去的那种客套和遮掩。在一个事实上就是把中共政权跟北朝鲜、伊朗的流氓政权相提并论，也象征著美中关系进入了一个新的对抗高度。
          <br/>
          <br/>
          评论人士夏明教授对本台说，“流氓政权”是美国对中共建政以来最为严苛的描述。之前有红色中国、共产中国、集权国家、独裁政权、专制政权等；两国关系有战略夥伴、竞争对手之称等。夏明说，89年六四事件，美国都没有将中国称之为“流氓政权”，当下则显示著美国社会正对中国重新定位。
          <br/>
          <br/>
          夏明说︰用流氓政权这个词是用来描述他，主要是在国内和国际上流氓政权，这个词基本上就用在了几个国家，也就是美国完全丧失信心以后，要集中精力把它消灭的几个国家。基本上会用一切的手段，包括各种军事、经济的制裁等等，去应对中国对国际秩序的挑战。
          <br/>
          <br/>
          大陆多家媒体、网站则是转发官媒《人民日报》海外版的回呛，指美国务院承认外交官见“港独”头目，却倒“打一耙”、“血口喷人”、“强盗逻辑”，最终搬起石头砸自己脚。
