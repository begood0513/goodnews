 <p>
  【看中国2019年12月22日讯】
  <span href="https://www.secretchina.com/news/gb/tag/北京" target="_blank">
   北京
  </span>
  领导人今次对特首林郑月娥的连番训话，其实不乏发人深省之处。
  <br>
   <br>
    例如领导人指示特区政府须以“止暴制乱”、恢复秩序为当务之急。香港眼下动乱不止不休的根源，离不开警方滥暴滥捕，加上满口谎言欲盖弥彰，煽起民愤。因此“止暴制乱”，必先追究警暴，才能让社会回复秩序，而追查警暴的必经之路，当然是成立独立调查委员会。
    <br>
     <br>
      又如中央重申要解决深层次矛盾，若所指是中港之间的深层次矛盾，不失为对症下药。特别是从长计议，建立制度以保障香港的高度自治，杜绝“一国”肆意践踏“两制”，也不容中央通过小圈子选举钦点其代理人治港，炮制丧权辱港的卖港决策，制造中港矛盾，并向资本集团输送利益，不断加深贫富悬殊。若制度保障可持之有恒，中港两地或可共存共荣。
      <br>
       <br>
        以至领导人要求制止外部势力干涉香港事务，也不无道理。自2007年中联办曹二宝提出“建立第二支管治队伍”以来，中联办、港澳办之类的机构，正不断介入本港内政。由各领域的统战到各层次的政治选战，以至立法会建制派在会内的投票取向，都有中方从中操盘的身影，因此不禁绝他们参与本港内政，又怎能实行货真价实的“港人治港”？
        <br>
         <br>
          很不幸，北京跟上述想法刚巧相反。在他们眼中，警暴不算暴力，警方成为香港的人民公敌也没所谓，有北京撑腰就可以活下去，北京亦乐于把三万警员收编为嫡系治港部队，反正是香港人付粮饷。同样，中港政治矛盾不算矛盾，傀儡特首继续留任也没问题，反正都是执行中央意图全面管治香港的肮脏任务，没有人比戴罪立功的林郑更卖力。最后，连能否守住“一国两制”也不打紧了，反正以坚定不移地维护国家主权、安全和发展利益之名，干预高度自治变成天经地义。
          <br/>
          <br/>
          当北京不见中港矛盾纠结只见港人以下犯上，不见高度自治受损只见国家利益受害，他们只会当支持抗争的六成港人如仇敌，对港人所急所思所想不但无感无解，更要火上加油，或明或暗指示特区当局须就国家安全立法，加强箝制反对声音和抗争行动。不过，政治冲昏头脑无助解决问题，因为警民激烈冲突仍未止息前，便推出国家安全法，是唯恐天下不乱的恶作剧，但若如林郑所说，等待适当时机，也即成功“止暴制乱”之后才立法，香港人又怎会轻易放弃抗争，让出空间，任由当局通过国家安全法律，然后拿恶法来对付自己呢？
          <br/>
          <br/>
          北京一直妄想香港人既能生金蛋，也会做顺民，否则受到重罚。当香港人以选票明志，坚定支持维权抗命，北京根本无法接受现实，只有抽刀断水，继续纵容警方，以暴力解决政治问题，并且眼见经济持续下滑，也不愿政治和解，再全力应付经济危机，说到底，不外是迫香港人乖乖就范，否则付出代价。不过，民不畏死，北京偏偏挖空心思，用暴力或经济手段胁迫港人，其粗暴强横，不寻根究底，不以理服人，正好说明香港人半年来抗争无罪，逆权有理，若不坚持下去，只会面临万丈深渊。
          <br/>
          <br/>
          可怜但又不值得可怜的是建制政客，他们已成政治植物人。在北京眼中，林郑作孽深重，对内促成反政府运动，掀起香港人重新审视一国两制，对外则育成美国的《香港人权与民主法案》，也让台湾总统蔡英文竞选连任扭转大局，林郑早成政治废物，只待秋后算账。其次是政治问责官员也抬不起头，以警方马首是瞻，甚至北京提拔的劳动模范警员亦可以罔顾公务员规章以下犯上，开腔炮轰高层官员，仿佛特区政府已经一分为二。
          <br/>
          <br/>
          再到建制派政党，区议会选举大败，他们对林郑可以有微言，也要含泪支持，对中央盲撑警队而令警暴变本加厉，反警情绪遍地开花，导致今天一败涂地，更不敢有半句怨言。从政者可以沦落如此，不顾市民福祉，也不顾自己利益，只求听命于中央，报答朝廷养育之恩，那么他们的溃败崩坏，要怨就怨自己没脑袋没原则也没羞耻，一味紧跟党国机器乱搞政治好了。
          <br/>
          <br/>
          一场述职秀，揭示北京对香港只会愈搞愈糟，不过强权镇压只会提升港人的抗争意志，真正摧毁的首先是建制人物和组织的政治生命。
