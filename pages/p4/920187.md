 <p>
  【看中国2020年1月19日讯】美中签订贸易协议，中国会不会真的执行，成为国际关注焦点。
  <br>
   <br>
    首先，任何国际机构，由联合国、巴黎气候协定、WTO，在世界上越来越没有权威。
    <br>
     <br>
      理由是会员国太多，其中充斥大量狡诈、贪污、缺乏诚信的第三世界独裁政权。以WTO为例，会员国超过一百五十个，总部设在瑞士。总部虽然可以将一个国家定为发展中国家或发达国家，但其本国的贸易行为，皆是政治，一翻脸就可以“主权”任意阳奉阴违，或其本国官员“上有政策、下有对策”不予执行。
      <br>
       <br>
        中国此一国家则更为奇特。首先，其本质是一党专政的独裁强权，但签署任何国际协议，此“一党专政”的机器，在国内控制什么、蓄意不控制什么、口头上声称想控制约束什么而无法执行什么，此“控制”的权力阴阳弛张的权术加幻术，共三者混合为：“中国特色”。随时移形换影、变化万千，令总部设在瑞士日内瓦或维也纳的种种由西方“精英”草议规条的所谓国际机构，完全鞭长莫及，无法监管。
        <br>
         <br>
          以中国为例：虽然一个习老大在北京摆出“一切我说了算的样子”，但即使内部党政，习老大却时时不满意，发出各层级党员不准“懒政怠政”的警告，三申五令就要强烈提醒：一切要以
          <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
           习近平
          </span>
          主席为核心。
          <br/>
          <br/>
          既然连老大做核心、八千万各层级的党员也个个是政治精、学得口头敷衍一套，背后做另一套的本领，那么本国各地政府部门机关，又岂会以世贸组织一项协议或美中贸易的某项重大规条来做“核心”？
          <br/>
          <br/>
          西方对中国的幻想或误解，是企图以为这个号称一党专政的国家，签了国际协定，可以由中南海一条鞭地层层指令下来，有效执行。
          <br/>
          <br/>
          但是西方天真之处在于：中共，或者中国人（这两者绝对是合一的），将何者可以独裁强力控制、何者可以两手一摊、“下面的干部我管束不住”这套中国行政管理欺诈艺术发挥得炉火纯青，将在哈佛学MBA、日内瓦维也纳平时上班，尝尽美食和公费旅行的西方精英，骗得长期两眼翻白、口吐白沫。
          <br/>
          <br/>
          在日内瓦这种地方，一个英国或法国意大利的代表天天上班，享受美食和商务舱机位加五星酒店的旅行，表面上东谈判这样、西核实那样，也沦为懒惰的官员，而且慢慢学会由第三世界官员手上拿贿赂。国际奥运的头头萨马兰奇，你不信，叫中情局去查查这个老头收了几多钱？
          <br/>
          <br/>
          最后只有等到美国大选换届时，冒出一个川普，才会拍桌子不耐烦，将国际机构这些所谓精英的书桌和餐桌一张张掀翻，自己行事。
          <br/>
          <br/>
          这就令耍太极催眠洋人二十年的中国，终于傻了眼。所谓被迫签署马关条约，因为遇到了一个不再废话胡混的美国版伊藤博文。
