 <p>
  【看中国2021年12月7日讯】五年前，美国的未来学家、社会思想家艾尔文.托夫勒（Alvin.Toffer）在洛杉矶去世，享年88岁。他毕业于纽约大学，当过工人、记者、研究员、康奈尔大学教授。我在中国大陆工作时，在上个世纪八十年代，读过他的著作《第三次潮》，这本书是1980年出版的。在著作中，他把人类历史上的农业文明，称为第一次浪潮；把工业文明，称为第二次浪潮；把人类进入信息社会，称为第三次浪潮。他曾经预言，人类社会在世纪之交，将会发生
  <ok>
   <span href="https://www.secretchina.com/news/gb/tag/垃圾" target="_blank">
    垃圾
   </span>
   革命
  </ok>
  。
  <br>
   <br>
    大家都知道，美国很重视防止生存环境被污染，重视垃圾分类和回收。现在人们生活产生的
    <ok>
     污水
    </ok>
    和工业
    <ok>
     废
    </ok>
    水，经过净化处理，可以变为可回收再利的农业灌溉水。中国城市也实行了回收垃圾，和废水回收处理再利用。这些把回收的垃圾经过加工处理，变废为宝的做法，就是垃圾革命。这次我要告诉读者的，就是在近几年
    <ok>
     垃圾革命
    </ok>
    中，在中国发生的一个故事。
    <br>
     <br>
      在中国山东省章丘市有父女俩，父亲叫李延荣，女孩叫李鸿怡，是个大学毕业生。通常人们把喂养和精心照料狗、猫、鸟等动物，叫做养宠物，如果说，有人把
      <ok>
       蟑螂
      </ok>
      当作“宠物”养，大家一定会感到奇怪。是的，当初，李延荣在家试验养殖蟑螂时，有人讥讽他是“蟑螂疯子”。
      <br>
       <br>
        这个故事，还得从人们对蟑螂的传统观念说起，一说蟑螂，谁都讨厌牠们，因为蟑螂常被人们当作害虫对待。但是李鸿怡从网上看到一则新闻，说是国外研究，一般蟑螂不携带病菌，而且蟑螂提取物还可以治疗癌症。这使她感到好奇，于是她便搜集有关蟑螂的各种知识，此事得到她父亲的支持，父女俩便共同研究蟑螂。
        <br>
         <br>
          他们得知，在中国的传统中药材中，就有蟑螂，可以医治多种病。在中国南方，人们把蟑螂叫做“偷油婆”。文献记载，蟑螂是一种史前就存在的动物，在久远年代的遗传进化当中，牠们形成了自己的生存特点。经过他们父女的饲养蟑螂研究观察，发现蟑螂的特点就是:1.爱干净，除了吃、睡，蟑螂还会不断清理牠们藏身环境，搞卫生;2.喜欢群居；3.食谱广泛，属于杂食动物，甚至地沟油也吃。此外蟑螂体内，含有丰富的蛋白质。于是女孩悟到，如果把成千上万的蟑螂聚集在一起，不就是一种处理人类生活垃圾的工具吗？！她把自己的想法告诉了父亲。李延荣知道章丘地区，每天要产生60吨餐厨垃圾，李延荣进一步想到餐厨垃圾可以做养蟑螂的饲料，蟑螂经过加工可以生产优质蛋白质饲料。
          <br>
           <br/>
           于是他们父女开始在家里实验养蟑螂。从一把蟑螂卵鞘、一个玻璃养鱼缸开始养起，实验成功后，他们便在当地选一小片土地，建立了一个小型工厂。李延荣父女的理想，就是要把蟑螂变害为宝。在调查中他们得知，济南市每天产生餐厨垃圾3000吨，李延荣决心用餐厨垃圾饲养蟑螂，生产蛋白饲料，闯出一条有利于社会的，工业化处理餐厨垃圾的路子。
           <br/>
           <br/>
           世上一切新鲜事物的发展，都不是一帆风顺的。他们父女在规模养殖蟑螂后，出现了问题，饲养间温度过高时，会造成大批蟑螂死亡，蟑螂感染螨虫也会死亡，而且密集养殖，密度过大蟑螂会自相残杀。也就是在这时，人们的风言风语跟随而来。李延荣对女儿说，一定要坚持试验，一旦成功，就是一件功德无量的事。
           <br/>
           <br/>
           这时，李延荣在一间又小又潮湿的小屋子里，坚持研究实验，与他作伴的就是一万多隻蟑螂。一个夏天过去，李延荣累的又瘦又黑。他们顶着社会舆论压力，埋头苦干，想方设法克服各种困难，包括规划、设计建设新厂，一干就是七年。
           <br/>
           <br/>
           在他们之前，没有别人的成功经验可循，更不用说可以买现成的专用机械设备。最初，买来一台粉碎机器，试验加工餐厨垃圾，一开机，从餐厨垃圾中挤压出来又脏又臭的污水满地流，别人都躲得远远的，只有李延荣一人在机器边，一遍一遍的试验、观察，找出症结所在。然后李延荣就自己研究改进。他自己绘草图，自己出去找厂家加工零件，运回来组装成自己的专用设备。
           <br/>
           <br/>
           经过多年实验和改进，现在他们已建成了新厂，并已投产。新厂大门外挂两块牌子，一块是章丘市餐厨垃圾处理中心，一块是餐厨垃圾生物处理协会。在工厂内，建成了自动化餐厨垃圾处理系统，建成了蟑螂养殖系统，喂养了3亿隻蟑螂。高大、明亮、干净的工厂厂房面积1570平方米，工厂每天可处理110吨餐厨垃圾。
           <br/>
           <br/>
           他们的工厂建有自己的实验室，有各种检测仪器。还有为数不多的职工，在工厂穿白大褂工作。例如在动化餐厨垃圾处理系统的进料口，只需一个工人分拣出餐厨垃圾中的塑料瓶。他们还搞综合利用，在厂房上面建立种植蔬菜的温室，利用厂房散发的较高温度给温室温室加温，这样，旣使在冬季，也可以在温室里种植蔬菜，种植蔬菜的肥料，就是蟑螂粪便。他们生产的高蛋白饲料，完全符合国家检验指标。他们的工厂每月创经济效益10万元人民币。现在他们的工厂，不仅已经把章丘产生的餐厨垃圾，转化为有用的生产原料，而且，李延荣还打算把临近的高新区、历城区产生的餐厨垃圾处理掉。
           <br/>
           <br/>
           李延荣说，蟑螂浑身都是宝，烘干的蟑螂磨成粉状饲料，含蛋白质71.2%，可以喂鸡，鸡吃这种饲料产生的鸡肉含脂肪低，含蛋白质高。蟑螂卵鞘也是优质蛋白饲料，蟑螂产生的粪便，没有难闻的气味，是植物的有机肥。他们研究发现，蟑螂的繁殖能力很强，一隻母蟑螂两天产一次卵，交配一次，可以终生繁殖，甚至在没有雄性蟑螂的情况下，蟑螂也可以无性繁殖。
           <br/>
           <br/>
           有人怀疑这样大规模养殖蟑螂，一旦遇到意外事故，会不会造成蟑螂外泄。其实这个问题，他们早已经在建设厂房之前就考虑到了，他们的工厂，在厂房地面上看不到一隻蟑螂，因为是在密闭空间养殖，工厂有三层严格的防护措施。为防意外事故造成蟑螂逃逸，厂房设立了三道防线，第一道是不锈钢网和水泥做成的有玻璃窗的幕墙，第二道是在人进出的房门设有流动水帘，万一有蟑螂就会被流水冲到水池中，水池中养有鱼。第三道是在厂房外墙周边设有像护城河一样的水池，水池中养有鱼，万一有外逃的蟑螂，掉到水中，就会被鱼吃掉。
           <br/>
           <br/>
           李鸿怡说，她愿做父亲的小智囊、小帮手，从事净化環卫。她说4000年前，中国人就开始养蚕，东汉时就开始养蜜蜂。她倡议“尊重每一种生物存在的价值”。她说他们之所以能获得成功，也许是从好奇心开始，学习相关知识，坚持信念，也许是自然的安排，也许是尊重了蟑螂的生存之道。
           <br/>
           <br/>
           我用文字记录下李延荣父女，通过自己的知识和付出的艰辛劳动，开创餐厨垃圾处理产业化之路的故事，我想这个故事，或许能给人们一些启示，他们的事迹，或许能对青年人成才创业，有点参考价值。
