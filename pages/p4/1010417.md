 <p>
  【看中国2022年6月29日讯】最近大陆网络中传出三顿饭令人心情难过，一位是前广西自治区副主席徐炳松，他办了一场隆重的生日宴，在座有许多官员；另一位是孔东梅的生日宴，一份流出的酒单总价超过600万；最后是一位北京的外卖大叔，晚上9点多他边哭边吃着塑料袋内的廉价食物，三人不同的餐食折射了大陆穷奢极欲与民生多艰的百态人生。
  <br>
   <br>
    先说第一位徐炳松，视频里他正在庆祝自己的81岁生日，因为贪污腐败，徐炳松于1999年被判处无期徒刑。在座的官员中有徐的天涯沦落人俞芳林，俞是前广西钦州市委书记，他同样因涉贪在2000年被判无期，说不定两人同为狱友，徐俞二人同座共享盛宴也真是别有一番风味。
    <br>
     <br>
      据了解，徐炳松的生日宴拍摄于本月22日，徐炳松特地介绍了俞芳林，因为他们为官一方时曾是搭档，徐炳松贪污金额为55万元，俞芳林后来居上贪了390万，这些钱现在看着不多可是放在90年代末也不是一笔小数目。
      <br>
       <br>
        徐炳松隆重的感谢了广西壮族自治区副秘书长对他和他家人的“帮助”，他说，“我是时刻铭记在心，这情义，无法用金钱衡量、无法用金钱衡量” ，同时他还自我劝慰，“想开一点，对不对？睡完觉以后，记得起床，不记得起床很麻烦，希望在座各位挚爱亲朋好友跟我一样，愉快到老年……”
        <br>
         <br>
          什么叫道德的沦丧，徐炳松的生日宴就是如此，叫来同样贪污被抓的搭档，感谢自己被抓后党依旧给了他“温暖”，同时告诫还没被抓的官员自己想开了，合着徐炳松认为自己被抓是运气不好，关了18年依旧没有认识到问题，如此恬不知耻世所罕见。
          <br>
           <br>
            与徐炳松的高调相比，孔东梅则低调不少，要不是酒单外泄没人知道权贵阶层的生活原来是这么的高攀不起，孔东梅的母亲叫李敏，没错她是
            <span href="https://www.secretchina.com/news/gb/tag/毛泽东" target="_blank">
             毛泽东
            </span>
            的亲孙女，孔被认为和毛泽东长得十分相像。
            <br>
             <br>
              孔东梅非常享受自己的中共红色基因，她留学美国后创办了北京东润菊香书屋有限公司，致力传播红色文化。此前她有着非常成功的两次创业史，一个是中国本土最大的拍卖公司嘉德拍卖，另一个是总资产千亿的保险公司泰康人寿，看看基因好的人做买卖都是大手笔。
              <br/>
              <br/>
              总之，红色基因护体的孔东梅妥妥的是成功人士，不然她又怎能享用的起600万元的美酒。
              <br/>
              <br/>
              都说贫穷限制了人的想象力，孔东梅一家不愧为财富排行榜前500强的权贵家庭，她的50岁大寿花费数百万，但对于起50亿的家底来说这几百万还真算不了什么，假如孔没有红色基因，凭她的能力是否能拥有天量的财富？答案自在大家心中。
              <br/>
              <br/>
              6月23日，一位在北京送外卖的大叔一边哭诉一边吃手中的便宜食物，他说，“活了40岁，混了个啥，钱没挣到，孩子没时间照顾，现在都晚上9点多了，还得为几两碎银子奔波。好累呀，这辈子我把所有的任务都完成，下辈子人间再好我也不来了。”
              <br/>
              <br/>
              与权贵人家的盛宴相比，这位外卖大叔的饭食可谓是寒酸到底了，他说的很明白，人过中年还得如此辛苦的为了家庭奔波，结果是付出的与得到的永远无法成正比，于是乎有了下辈子人间再好我也不来了的感悟。
              <br/>
              <br/>
              活在当下，外卖大叔想要翻身谈何容易， 别说外卖大叔自己，包括其子女，奋斗一辈子都难以获得权贵手中千分之一的财富。一面是权贵财富如大风挂来轻而易举获得，一面是起早贪黑还在为一日三餐忙碌，社会的不公已经深深印在每个人的身上。
              <br/>
              <br/>
              面对这样的差距与心酸，作为升斗小民如果能选择谁会想轮回在这个群魔乱舞的世道之中。窃钩者诛，窃天下者诸侯，被中共控制的大陆不正是这样黑白跌倒的世界么。
