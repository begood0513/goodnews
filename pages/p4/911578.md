 <p>
  【看中国2019年10月25日讯】去年10月4日，美国副总统彭斯在华盛顿智库哈德逊研究所，进行了一场针对中美关系的主题演讲。彭斯在演讲中指责了北京在经济、贸易、政治、科技、军事、宣传等各方面对美国的干扰和破坏，由于此前没有哪位美国高官如此直接的抨击中共，所以当时非常有震撼力，被认为是美国政府数十年来对华政策的转折点。
  <br>
   <br>
    时隔一年，彭斯于周四10月24日，在华盛顿智库威尔逊中心就美中关系问题发表了第二次演讲，他的这次讲话同样让各方高度关注。
    <br>
     <br>
      在这次演讲中，彭斯副总统同样大篇幅对北京的一系列行为进行了严厉指责。他并说：“本着这种坦率的精神，我必须告诉你们，从我上次在哈德逊中心发表演讲的一年后，北京方面仍未采取重大行动来改善我们的经济关系。在我们提出的许多其它问题上，北京的行为甚至变得更加咄咄逼人和破坏稳定。”
      <br>
       <br>
        除了对中共指责的广度和力度不减之外，彭斯的此次讲话还多了两部分内容。
        <br>
         <br>
          <strong>
           对香港抗议者评价出乎意料的高
          </strong>
          <br>
           <br>
            彭斯说：“在今后的日子里，我可以向你们保证，美国将继续敦促中国展示克制，信守其承诺，并尊重香港人民。过去几个月来一直在和平示威以捍卫自身权利的香港人，我们与你们站在一起。你们鼓舞激励了我们（英文原文：We are inspired by you），我们敦促你们坚持非暴力抗议的道路。要知道，几百万美国人在为你们祈祷并且敬仰着你们（英文原文：you have the prayers and the admiration of millions of Americans）。”
            <br/>
            <br/>
            前白宫顾问班农先生曾经多次说，香港的抗议者们就像是1776年那一代美国的开国者们，他们为了捍卫自己的自由而抗争。
            <br/>
            <br/>
            而彭斯作为美国现任副总统，对香港的抗议者用了“鼓舞激励”和“敬仰”的词，这种评价可以说已经到了不能再高的程度。
            <br/>
            <br/>
            彭斯并毫不含糊的就香港问题指责中共，他说：“过去几年来，北京加大了对香港的干预，并且采取了遏制港人权利和自由的行动，这些权利和自由是具有约束力的一国两制国际协议所保障的。”
            <br/>
            <br/>
            对香港问题彭斯还说：“川普总统已经说得很清楚。用他的话说，‘美国将会和自由站在一起’。”
            <br/>
            <br/>
            <strong>
             价值观上美国将收复失土
            </strong>
            <br/>
            <br/>
            除了香港问题，对比上次讲话，彭斯的这次讲话，还有更重要的一部分，这从一个词的使用频率上可以看出来。这个词就是价值观（英文：Value）。
            <br/>
            <br/>
            在上次讲话中，彭斯副总统使用价值观这个词只有1次（还有1次用Value指的是物品的金钱价值，而不是精神上的价值观）。而在这次讲话中，指价值观的Value一词出现了11次，超过上次10倍，而且从始至终，遍布整篇演讲。
            <br/>
            <br/>
            彭斯在演讲的开篇部分就说：“没有人，特别是国家，能够通过放弃他们的价值观来捍卫自己的利益。”
            <br/>
            <br/>
            下面是彭斯在演讲中谈到价值观（Value）的其它各处：
            <br/>
            <br/>
            “去年的这个月，我谈到了北京许多对美国利益和
            <u>
             价值观
            </u>
            最有伤害力的政策。”
            <br/>
            <br/>
            “为了维护所有热爱自由的人们的
            <u>
             价值观
            </u>
            ，我们还公开批评了中国共产党压制中国人民的宗教自由。”
            <br/>
            <br/>
            “对于美国的行动和决心，一些跨国公司说，我们的经济政策过于强硬，坚持推进我们的利益和
            <u>
             价值观
            </u>
            与改善对中关系背道而驰。”
            <br/>
            <br/>
            “太多美国的跨国企业向中国金钱和市场的诱惑叩头，他们不仅不（敢）批评中国共产党，也不（敢）坚持美国的
            <u>
             价值观
            </u>
            。”
            <br/>
            <br/>
            “当美国公司、职业体育运动、职业运动员拥抱同意被审查的时候，这不仅是错误的，也违反美国精神。美国公司应当在国内和世界各地挺身捍卫美国
            <u>
             价值观
            </u>
            。”
            <br/>
            <br/>
            “话虽如此，总统已经明确表示，美国不寻求与中国对抗。我们寻求的是公平竞争的环境、开放的市场、公平的贸易以及尊重我们的
            <u>
             价值观
            </u>
            。”
            <br/>
            <br/>
            “中国应当知道，美国的
            <u>
             价值观
            </u>
            根深蒂固，我们对这些
            <u>
             价值观
            </u>
            的坚持与我们的开国国父们一样牢固，美国明亮的民主自由之光永远也不会有熄灭的那一天。”
            <br/>
            <br/>
            “我们将继续相信民主的
            <u>
             价值观
            </u>
            ---个人自由、宗教和良心自由、法治，符合美国和全球利益，因为它们是、而且永远将是释放人类的潜能抱负并引导全世界各国和人民之间关系的最佳政府模式。”
            <br/>
            <br/>
            特别是在演讲的最后部分，彭斯又说：
            <br/>
            <br/>
            “我们将捍卫我们的利益。我们将捍卫我们的
            <u>
             价值观
            </u>
            。我们将本着对所有人的慈善和善意的精神这样做。”
            <br/>
            <br/>
            <strong>
             美国的初心和中共的初心
            </strong>
            <br/>
            <br/>
            彭斯在演讲的结尾部分，还特别回顾了美国的初心。他说：
            <br/>
            <br/>
            “美国诞生于对压迫和暴政的反抗。那些具有超凡的勇气、顽强的决心、信念和强烈的独立性、与钢铁般意志的男男女女创建了我们的国家，并在此定居和开拓。几个世纪过去了，我们并没有改变。
            <br/>
            <br/>
            美国人相信，所有人生而平等，造物主赋予我们若干不可剥夺的权利：生命、自由和追求幸福的权利。没有什么将会改变美国人对此的信仰。这就是我们的特性，我们将永远保持这样的特性。”
            <br/>
            <br/>
            有评论认为彭斯的这篇讲话对北京偏软，但笔者并不这样认为，因为彭斯在讲话中对中共的指责不但丝毫不减，同时他非常明确的表明了对香港民众的坚定支持，更要紧的是，他指向了中共最关键的生存基础，那就是价值观问题。
            <br/>
            <br/>
            中共是一个马克思主义政党，它的灵魂就是马克思主义价值观。它当初之所以能发家夺权，就是因为它钻了中国在近代社会大转型中，价值观出现混乱的空子。
            <br/>
            <br/>
            北京近年来强调“中共的初心”，去年高调宣传马克思是人类“千年第一思想家”等，就是在中共意识形态已接近死亡的情况下，为中共招魂的举动。
            <br/>
            <br/>
            当前，北京对舆论的管控已经到了荒谬的地步，也是因为中共意识形态毫无生命力，只能靠高压续命，活一天算一天。
            <br/>
            <br/>
            笔者在此前的
            <span href="https://www.secretchina.com/news/gb/2019/05/27/894931.html" target="_blank">
             《看到川普拥抱她就知道北京遇到大问题》
            </span>
            一文中，详细分析了为什么“美国的初心”和“中共的初心”是相反的。

   </br>
  </br>
 </p>
 <p>
  彭斯的演讲表明，在今后两种初心的碰撞中，美国将丝毫不让。如果说
  <span href="https://www.secretchina.com/news/gb/tag/贸易战" target="_blank">
   贸易战
  </span>
