### 中国监狱成疫情重灾区 官方披露情况恐是冰山一角

 <p>
  【看中国2020年2月22日讯】中国官方周五（21日）首次公布了部分
  <strong>
   监狱
  </strong>
  中的
  <strong>
   武汉肺
  </strong>
  炎疫情，一天之内就有超过500例。但维权人士认为，大陆监狱独立的报告体系非常神秘，对外界有严格的
  <strong>
   保密
  </strong>
  规定，所以，现在披露的感染资料，恐怕只是真实状况的冰山一角。
  <br>
   <br>
    近一个星期，中国监狱爆发新冠肺炎的消息不迳而走。山东政府周五（21日）突然宣布，任城监狱多达200名在押人员和7名狱警感染武汉新冠肺炎。在新冠肺炎的发源地湖北，政府亦公布，该省监狱出现感染271例，仅湖北女子监狱就达230例。与此同时，浙江方面披露，位于衢州市的十里丰监狱，也出现了27例
    <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
     武汉肺炎
    </span>
    感染病例，而当天浙江全省新发感染病例只有28例。
    <br>
     <br>
      山东省司法厅厅长解维俊对任城监狱疫情防控不力，被免去省司法厅党委书记、厅长、省监狱管理局党委书记、第一政委职务。而浙江十里丰监狱和湖北女子监狱的负责人亦被免职。

       <span href="https://www.secretchina.com">
       </span>
      </span>
     </br>
    </br>
   </br>
  </br>
 </p>

 <p>
  有知情人透露，当地的医疗机构和社区早前就在要求鲁西
  <span href="https://www.secretchina.com/news/gb/tag/监狱" target="_blank">
   监狱
  </span>
  、任城监狱的负责人报备，并自我隔离，反映现时公布任城监狱武汉肺炎感染的情况可能只是冰山一角。他又分析，监狱在押人员的感染渠道不外乎新来的囚犯和狱警们，但新来犯人本身是在看守所呆了很长时间，如果他们也是带病者，后果就非常严重。
  <br>
   <br>
    他说：前几天就是当地很多微信工作群啊都被解散了，然后就有人传，说是监狱里有甚么事了，然后有社区发的那种通报啊，说甚么和这个监狱有关的人都要观察。济宁市还在辟谣，说没有解散微信群。你现在看来，那就是这个事了。你要注意到一点啦，监狱里面来新的犯人，他一定是经过判决了之后，从看守所进去（监狱）了。通过公检法这个程序，至少得大半年吧。但是说实话，我不能多作评论，我跟你讲嘛，昨天还有人过来警告我。
    <br>
     <br>
      大陆监狱集体爆发武汉肺炎似乎有迹可寻。访民王一斐的父亲和弟弟因为反抗强拆，现在分别被关在济宁的看守所和监狱里。王一斐说，农历除夕当晚，家属接到监狱方的电话，称因为疫情，所有监狱和看守所取消会见。而最新曝光的监狱感染状况，让她十分担心自己亲人的安全。
      <br>
       <br/>
       王一斐说：见不到，就是出现这个疫情之后，看守所，还有这个监狱，都是停止会见了，除夕夜那天晚上就下了通知了。就是任城区这个监狱出现这个问题是他们干警应该去武汉了，然后他隐瞒了这个旅游史。这个监狱前几天就已经出现了这个问题了，但是他们还是封锁消息。
       <br/>
       <br/>
       资深媒体人邓先生认为，目前官方只是公布了三个省个别监狱的资料，明白人一看就能知道仅仅是一小部分。因为此前监狱的报备体系和地方相互独立，加上监狱、看守所内在押人员生活条件恶劣，除非出现重症患者需要送医，否则他们也没有能力第一时间查明以及上报在押人员是否感染。而这次公开部分监狱资料，估计表明情况已经失控。邓先生又认为，在看守所监舍严重超员的情况下，任何人的感染，都可能在监区带来灾难性的后果。
       <br/>
       <br/>
       邓先生说：就是那个事搞大了，瞒不住了。你也清楚，现在监狱、看守所那都是封闭式的管理啊，与社会隔绝的。山东济宁那个监狱，1月份就发生了，一直瞒报，他13号山东省委书记都知道了，批示他们司法厅要重视。监狱为甚么瞒报？本身行业敏感，一半呢它应该是单独报的。它拿到那个当地的感染人数去报，本身就是避重就轻的事嘛。
       <br/>
       <br/>
       本台记者就监狱感染武汉肺炎确切人数向中国司法部查询，工作人员回应，以公开的数字为准，他称数字真实，但又拒绝回应重灾区湖北各监狱疫情的状况。
       <br/>
       <br/>
       司法部：我们没有，都是以那个新闻公布的为准。那都是最近的资料。这个你要是有疑问，你可以跟当地的监狱管理局联系。这个其它的东西，我们不了解。
       <br/>
       <br/>
       中国的监狱和看守所，被称为中国最不透明的领域之一，其虐待在押人员以及恶劣的基本卫生和生活条件备受诟病。但在中国武汉肺炎爆发后，关于军队和监狱的疫情状况都一直被高度保密。而此次官方公开承认监狱内发生聚集性群体感染，被指可能问题已十分严重。
