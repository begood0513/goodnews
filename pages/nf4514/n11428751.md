<div><p>
 <ok href="http://www.epochtimes.com/gb/tag/%E8%A5%BF%E6%B8%B8%E8%AE%B0.html">

 </ok>
 的开始就是一首诗：
</p>
<p style="padding-left: 30px;">
 混沌未分天地乱，茫茫渺渺无人见。
 <br/>
 自从盘古破鸿蒙，开辟从兹清浊辨。
 <br/>
 覆载群生仰至仁，发明万物皆成善。
 <br/>
 欲知造化会元功，须看西游释厄转。
</p>
<p>
 西游文化基本上就是伴随着我的成长，
 <br/>
 但是因为从小看86版
 <ok href="http://www.epochtimes.com/gb/tag/%E8%A5%BF%E6%B8%B8%E8%AE%B0.html">
  西游记
 </ok>
 ，
 <br/>
 以为西游记所有的就是这些了，
 <br/>
 但其实远远不止。
</p>
<p>
 做这个频道的初心也是因为最近在机缘巧合下，
 <br/>
 拜读了一下《西游记》的原本。
 <br/>
 这一下真是一发不可收拾啊！
 <br/>
 真心觉得我们这一代对古文化知道的太少了，
 <br/>
 老祖宗都把什么都说清了，结果呢，我们却在背道而驰中不断的坚持啊！
 <br/>
 所以这个频道就是跟大家来聊一聊西游记里的那些事儿
 <br/>
 只有读了原本你才会发现，影视作品里说的，其实是另外一个西游记！
</p>
<p>
 我们今天聊的话题是我从很久以前就想知道的，
 <br/>
 <ok href="http://www.epochtimes.com/gb/tag/%E5%9C%B0%E7%90%83.html">
  地球
 </ok>
 这个说法呢，虽然在明朝就出现了，但是那时候是西学东渐时期，
 <br/>
 就是近代西方学术思想向中国传播的历史过程，
 <br/>
 所以接近于科学术语。
 <br/>
 但是明朝以前呢？
 <br/>
 古代的科学可不比现在差，那时候不叫
 <ok href="http://www.epochtimes.com/gb/tag/%E5%9C%B0%E7%90%83.html">
  地球
 </ok>
 ，他们叫什么呢？
 <br/>
 西游记就给了我们这个答案！
</p>
<p>
 这一切还要从开天辟地说起，
 <br/>
 世界被分为了四大部洲，
 <br/>
 那每一个洲是什么样的呢？
 <br/>
 我们又生活在哪个洲上呢？
 <br/>
 看视频我就告诉你！
</p>
