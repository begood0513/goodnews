<h4 class="p1">
 4.25亲历者：后来得知中共策划了天津事件
</h4>
<p class="p1">
 周女士是
 <span class="s1">
  1999
 </span>
 年
 <span class="s1">
  4.25
 </span>
 事件的亲历者之一，她说自己在迫害之前，因为修炼法轮功而身心健康，但中共政府却有预谋地要对法轮功进行迫害。
</p>
<p class="p1">
 <span class="s1">
  “
 </span>
 在
 <span class="s1">
  1999
 </span>
 年
 <span class="s1">
  4.25
 </span>
 发生的前几天，天津很多大法弟子无端被抓被打，天津方面将责任推到了北京中南海，临近地区的大法弟子在得知这一消息后，纷纷前往北京国务院信访办上访。
 <span class="s1">
  ”她说。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “
 </span>
 北京公安警察好像早已知道此事，他们将学员们带到了中南海外围，但上访的学员都站在人行道上，连盲道也空了出来，根本没有影响交通，但警察挡在马路中间，不准行人和车辆通行。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 <span class="s1">
  周女士还说，“
 </span>
 后来我们才得知，是中共有意策划了天津抓捕事件。江泽民一伙为了达到镇压法轮功的目的，有意颠倒黑白，混淆是非，将
 <span class="s1">
  4.25
 </span>
 万名法轮功学员的和平上访，污蔑成围攻中南海。三个月之后，就发生了
 <span class="s1">
  7.20
 </span>
 对法轮功史无前例的大迫害，这场迫害到今天，已经整整持续了
 <span class="s1">
  23
 </span>
 年。
 <span class="s1">
  ”
 </span>
</p>
<h4 class="p1">
 英国工程师：希望世人了解真相
</h4>
<p class="p1">
 此次活动中的另一名法轮功修炼者、目前正在英国担任水利工程师的李梅表示，包括自己外祖母在内的很多家人都在
 <span class="s1">
  4.25
 </span>
 当天，到北京的信访办上访。
</p>
<p class="p1">
 <span class="s1">
  “
 </span>
 因为知道大法蒙受不白之冤，我姥姥以前有很严重的冠心病和脑血栓，去医院都不给做手术的，就是因为修炼了大法，一粒药也没吃过，这些病就全都没有了，医生也觉得很神奇，所以当初（中共）诬陷大法的时候，他们觉得不对，所以才想要去（信访办）反映情况。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 <span class="s1">
  李梅说，“
 </span>
 这场迫害在国内还在发生着，所以我们想要让世人了解到这一点（真相），关注这个事情，如果所有人都能发声的话，我希望这场迫害能尽快停止，也想要正告中国国内正在施行迫害的恶警们，不要再继续跟着做坏事，因为善恶有报。
 <span class="s1">
  ”
 </span>
</p>
<h4 class="p1">
 “过去23年，我每年都参加4.25活动”
</h4>
<p class="p1">
 英国法轮功学员李先生说，他自从
 <span class="s1">
  1996
 </span>
 年开始修炼法轮功之后，身心康健，家庭和睦，这都是修炼法轮功带来的益处，他觉得修炼大法的好处他说一天都说不完。他认为，“真善忍”不仅仅是对法轮功修炼者，对于全人类都是有好处的，过去
 <span class="s1">
  23
 </span>
 年来，每年的
 <span class="s1">
  4.25
 </span>
 活动他都参加了。
</p>
<p class="p1">
 <span class="s1">
  “‘
 </span>
 真善忍’对人类是非常好的，每个人都需要，对个人益处好多。这样对人类有好处的东西，突然在中南海发生
 <span class="s1">
  4.25
 </span>
 ，因为迫害很厉害，人们才反对，这样好的东西，你怎么可以压迫人家，所以每次
 <span class="s1">
  4.25
 </span>
 活动我都会参加，这个纪念活动很重要。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 早已年过古稀的李先生还笑说，每次来参加活动，都会被人家说自己越来越年轻，
 <span class="s1">
  “
 </span>
 这些都是修炼得来的，我得到了好处，所以我要将这些好处告诉他人，这才是善嘛
 <span class="s1">
  ”。
 </span>
</p>
