</p>
<p class="p1">
 <span class="s1">
  据彭博社报导，马里兰州民主党参议员克里斯‧范霍伦（
 </span>
 <span class="s2">

 </span>
 <span class="s1">

  <ok href="https://www.epochtimes.com/gb/tag/%E4%BA%8C%E7%BA%A7%E5%88%B6%E8%A3%81.html">
   二级制裁
  </ok>
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “我们还没有有效切断对普京的战争机器的资助，这可以通过减少俄罗斯的能源销售收入来实现。”同为参议院银行委员会成员的范霍伦和图米在一份声明中说，“为了成功执行（对俄罗斯原油在国际市场的）价格上限，政府显然需要国会给予新的授权。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  根据两位参议员的最新提议，美国及其盟国将被要求在
 </span>
 <span class="s2">
  2023
 </span>
 <span class="s1">
  年
 </span>
 <span class="s2">
  3
 </span>
 <span class="s1">
  月前对俄罗斯海运石油的价格设定上限。随后再逐年减少三分之一，直到三年内达到收支平衡的价格，使普京不能获得任何高于生产价格的收入。同时，法案还留下一个可校正的机制，如果白宫认为这样做会导致全球油价飙升的话，那么总统可以放弃限价。
 </span>
</p>
<p class="p1">
 <span class="s1">
  法案计划将通过对任何参与俄罗斯石油销售或运输的公司，包括银行、保险和再保险公司以及经纪公司实施二级制裁来执行。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这项尚未出台的立法还将对被发现进口俄罗斯石油、石油产品、天然气和煤炭超过战前水平的国家进行处罚。
 </span>
</p>
<p class="p1">
 <span class="s1">
  范霍伦和图米说，二级制裁将为白宫提供它所需要的工具，以“追究那些对在战争期间参与俄罗斯出口交易并从中牟取暴利的国家提供支持的金融机构
 </span>
 <span class="s1">
  的责任”。
 </span>
</p>
<h4 class="p1">
 白宫反对对俄罗斯石油进行二级制裁
</h4>
<p class="p1">
 <span class="s1">
  但是这项立法与拜登政府当前的做法可能存在冲突。拜登政府此前拒绝将二级制裁作为执行石油价格上限的一种方式。
 </span>
</p>
<p class="p1">
 <span class="s1">
  七国集团国家（英国、德国、意大利、加拿大、美国、法国和日本）领导人在
 </span>
 <span class="s2">
  6
 </span>
 <span class="s1">
  月的峰会上确认了拟减少对俄罗斯的能源依赖，并初步商定开始限制俄罗斯的石油和
  <span class="s3">
   天然气
  </span>
  价格。
 </span>
 <span class="s2">
  7
 </span>
 <span class="s1">
  月初，有人提议将限制设置为当前价格的一半。据彭博社当时报道，正在讨论的价格数字是
 </span>
 <span class="s2">
  40-60
 </span>
 <span class="s1">
  美元。
 </span>
</p>
<p class="p1">
 <span class="s1">
  白宫团队认为，价格上限的经济激励措施足以引起各方合作、共同抵制对俄罗斯的贸易，而二级制裁会造成与中、印等国家的紧张关系，这些国家现在在继续大量购买俄罗斯石油。
 </span>
</p>
<p class="p1">
 <span class="s1">
  财政部副部长瓦利‧阿德耶莫（
 </span>
 <span class="s2">
  Wally Adeyemo
 </span>
 <span class="s1">
  ）
 </span>
 <span class="s2">
  9
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  6
 </span>
 <span class="s1">
  日在接受彭博社采访时说：“我认为不需要二级制裁来实现。买家的动机与设置价格上限的国家的动机是一致的。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  彭博社说，财政部的一位发言人拒绝发表评论。一位不愿意透露姓名的熟悉此事的人士说，财政部已经听取了国会二级制裁框架的汇报。
 </span>
</p>
<p class="p1">
 <span class="s1">
  自
 </span>
 <span class="s2">
  2
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  24
 </span>
 <span class="s1">

 </span>
 <span class="s2">
  SWIFT
 </span>
 <span class="s1">
  ）的联系。
 </span>
</p>
<h4 class="p1">
 立法将直接影响中印的俄罗斯石油进口
</h4>
<p class="p1">
 <span class="s1">
  如果对俄石油进行二级制裁的立法获得通过，可能会引发美国与中、印等国大打出手。这两个国家已经加大了对俄罗斯石油的采购，并对美国的价格上限的提议反应冷淡。
 </span>
</p>
<p class="p1">
 <span class="s1">
  中国一直是俄罗斯的重要贸易伙伴，中方对俄的石油采购有助于抵消俄罗斯对欧洲出口的下降，不过，北京也一直在小心翼翼地避免触犯西方制裁。
 </span>
</p>
<p class="p1">
 <span class="s1">
  美国在与印度就俄罗斯石油价格上限进行互动时一直很谨慎，没有威胁要对不加入该计划的国家进行处罚。
 </span>
</p>
<p class="p2">
 <span class="s1">
  俄罗斯外交部经济合作司司长德米特里
 </span>
 <span class="s2">
  ∙
 </span>
 <span class="s1">
  比里切夫斯基稍早在接受卫星通讯社采访时表示，G7对俄罗斯石油和天然气实行价格上限是有风险的，只会加剧能源危机，如果西方采取此类举措，俄罗斯企业将基于经济合理性采取行动。
 </span>
</p>
<p class="p4">
 <span class="s1">
  在
  <ok href="https://www.epochtimes.com/gb/tag/%E4%BF%84%E4%B9%8C%E6%88%98%E4%BA%89.html">
   俄乌战争
  </ok>
  之后，面对西方国家的经济制裁，俄罗斯经济更加向东转。总统普京9月出席上海合作组织峰会时与中共国家主席习近平会晤时表示，当今世界正在发生许多变化，唯一不变的是俄中友谊与互信，俄中全面战略协作关系稳固如山。
 </span>
</p>
<p class="p4">
 <span class="s1">
  普京还提到，俄中两国的双边贸易在2022年前7个月增长了25%。他说，相信在年底之前将达到新的纪录水平。
 </span>
</p>
<p class="p4">
 <span class="s1">
  “在不久的将来，正如我们所同意的那样，我们将把我们的贸易额提高到2000亿美元或更多。”2021年，俄中两国贸易量是超过1400亿美元。
 </span>
</p>
<p class="p5">
 <span class="s1">
  根据中共官方电视台央视新闻发出的通告，习近平强调，中方愿同俄方在涉及彼此核心利益问题上相互有力支持，深化贸易、农业、互联互通等领域务实合作。但他没有透露任何细节。
 </span>
</p>
