</div><hr/><div><p>
 【大纪元2019年12月19日讯】自1999到2011年，河北省蠡县的鲍墟乡有
 <ok href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E4%BB%BB%E6%B4%BE%E5%87%BA%E6%89%80%E6%89%80%E9%95%BF.html">
  三任派出所所长
 </ok>
 ，外加一个副乡长
 <ok href="http://www.epochtimes.com/gb/tag/%E5%A3%AE%E5%B9%B4%E6%9A%B4%E6%AD%BB.html">
  壮年暴死
 </ok>
 。
</p>
<p class="p2">
 <span class="s1">
  明慧网报导，从1999年7月20日中共迫害法轮功开始，鲍墟乡在蠡县的迫害法轮功学员运动中首当其冲。
 </span>
</p>
<h3 class="p2">
 <span class="s1">
  彭小五，蠡县刘村人，原鲍墟乡派出所副所长
 </span>
</h3>
<p class="p2">
 <span class="s1">
  他因积极参与迫害法轮功，绑架法轮功学员，中共给他“荣立”三等功，给予奖金一万元。
 </span>
</p>
<p class="p2">
 <span class="s1">
  2003年5月8日晚十点，彭小五被大车撞死在蠡县永盛大街奔垒德村的十字路口，死时才40岁左右。
 </span>
</p>
<h3 class="p2">
 <span class="s1">
  齐广平，蠡县鲍墟乡人，时任鲍墟乡派出所所长
 </span>
</h3>
<p class="p2">
 <span class="s1">
  村里人说，按理讲本乡本土的，对家乡的人，应该手下留有余地。可他不但没有那样做，还和鲍墟乡副乡长刘建柱，把进京为法轮功鸣冤的法轮功学员绑在乡政府大院里抱着树挨冻。
 </span>
</p>
<p class="p2">
 <span class="s1">
  当时正是三九天，下着大雪，齐广平穿着皮鞋踢打法轮功学员，把学员的眼镜打碎。齐广平还带人到法轮功学员家中没收法轮功书籍。
 </span>
</p>
<p class="p2">
 <span class="s1">
  过后不长时间，齐广平便感身体不适，后死于癌症。
 </span>
</p>
<h3 class="p2">
 <span class="s1">
  王新斋，50多岁，曾任蠡县公安局纪检副书记、看守所所长、鲍墟乡派出所所长
 </span>
</h3>
<p class="p2">
 <span class="s1">
  他唆使警察毒打法轮功学员，并对法轮功学员进行长期超期非法关押。
 </span>
</p>
<p class="p2">
 <span class="s1">
  2002年8月，他因奸污妇女被免去所长职务。
 </span>
</p>
<p class="p2">
 <span class="s1">
  2011年正月初四，王新斋开车和朋友去山东，在高速公路上将护栏撞开，车子滚下，车上四人，只有坐在后座的王新斋一人死亡。
 </span>
</p>
<h3 class="p2">
 <span class="s1">
  刘建柱，蠡县留史镇大王村人，原蠡县鲍墟乡副乡长
 </span>
</h3>
<p class="p2">
 <span class="s1">
  1999年7月以后，他紧跟中共江泽民集团残酷迫害法轮功学员。
 </span>
</p>
<p class="p2">
 <span class="s1">
  2009年6月11日，在蠡县滑村板厂附近的留史道上，刘建柱开的面包车与一辆公共汽车相撞，面包车的方向盘挤压在他的胸部，造成他当场死亡。
 </span>
</p>
<p class="p2">
 <span class="s1">
  当时刘建柱的车上有好几个人，只有他一人死亡。
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  <b>
   其他蠡县遭厄运人员事例：
  </b>
 </span>
</h4>
<h3 class="p5">
 <span class="s1">
  李文彦，男，蠡县城关派出所所长
 </span>
</h3>
<p class="p5">
 <span class="s1">
  他打昏一名法轮功女学员，将其衣服扒光，在雪地里冻，使人死去活来。
 </span>
</p>
<p class="p5">
 <span class="s1">
  李文彦还时常闯进法轮功学员家，翻箱倒柜，见钱就装。晚上蹲坑抓人，白天撕真相标语。全乡法轮功学员几乎都遭其毒打和谩骂。
 </span>
</p>
<p class="p5">
 <span class="s2">
  2003
 </span>
 <span class="s1">
  年秋，李文彦因欠账被起诉，因殴打河北电台记者，被全省通报，撤销职务。
 </span>
</p>
<h3 class="p5">
 <span class="s1">
  李国昌，男，蠡县看守所警察
 </span>
</h3>
<p class="p5">
 <span class="s1">
  他在寒冬腊月扒光法轮功学员的衣服，逼他们趴在水泥地上冷冻。酒后闯进女监牢进行调戏，无耻去摸一未婚女学员的下身。
 </span>
</p>
<p class="p5">
 <span class="s2">
  2000
 </span>
 <span class="s1">
  年冬，李国昌突然瘫痪。
 </span>
</p>
<h3 class="p5">
 <span class="s1">
  王海发
 </span>
 <span class="s1">
  ，蠡县城关派出所警察
 </span>
</h3>
<p class="p5">
 <span class="s2">
  2001
 </span>
 <span class="s1">
  年，王海发领一帮人到法轮功学员崔小先家抄家，抢走私人财物，并用脚踩烂供香，砸烂香炉。
 </span>
</p>
<p class="p5">
 <span class="s1">
  这次抄家不久，王海发的儿子被车轧死。
 </span>
</p>
<h3 class="p5">
 <strong>
  <span class="s1">
   张五进，男，蠡县留史人，保定市公安局副局长；赵玉锁，男，蠡县人，保定市公安局法制处劳教科科长；张应华，男，司机
  </span>
 </strong>
</h3>
<p class="p5">
 <span class="s2">
  2010
 </span>
 <span class="s1">
  年
 </span>
 <span class="s2">
  8
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  11
 </span>
 <span class="s1">
  日，三人驱车赶
 </span>
 <span class="s1">
  赴阜平，要与河北省国保总队人员共谋绑架法轮功学员事宜，车至中途，冲出高速公路，赵玉锁、张应华当场死亡，张五进被甩出车外，多处骨折造成重伤。
 </span>
</p>
<p class="p2">
 <span class="s1">
  人们在怜惜，叹惋这些人的遭遇时，可能会想：他们为什么会有如此厄运。
 </span>
</p>
<p class="p2">
 <span class="s1">
  明慧网评论说，法轮功是佛法修炼，常言道：宁搅千江水，不扰道人心。法轮功学员遵循“真、善、忍”原则行事，不顾自身安危，在大劫来前告诉人们真相，让人们远离灾难，却遭无故迫害。
 </span>
</p>
<p class="p2">
 <span class="s1">
  如不迫害佛法，那些人本可生活得很好，尽到做为儿子、丈夫、父亲的职责，然而不幸的是他们选择了助纣为虐。
 </span>
</p>
<p class="p2">
 <span class="s1">
  自古迫害信仰者的人，都不得善终。罗马帝国那么强大，只因迫害基督徒，而遭四次大瘟疫降临，最终导致罗马帝国在世间消失。#
 </span>
</p>
