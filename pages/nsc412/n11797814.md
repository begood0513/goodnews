</div><hr/><div><p>
 【大纪元2020年01月16日讯】
 <span class="s1">
  （大纪元记者陈霆综合报导）您上次检查
  <ok href="http://www.epochtimes.com/gb/tag/%E9%80%80%E4%BC%91%E9%87%91%E8%B4%A6%E6%88%B7.html">
   退休金账户
  </ok>
  余额，是多久以前呢？随着金融机构网路安全的加强，破解银行防火墙越来越困难。许多
  <ok href="http://www.epochtimes.com/gb/tag/%E9%AA%87%E5%AE%A2.html">
   骇客
  </ok>
  开始将目标转向个人，借着
  <ok href="http://www.epochtimes.com/gb/tag/%E9%92%93%E9%B1%BC%E9%82%AE%E4%BB%B6.html">
   钓鱼邮件
  </ok>
  窃取个人资料，来冒领退休金。
 </span>
</p>
<p class="p1">
 <span class="s1">
  根据美国计划顾问协会（
 </span>
 <span class="s2">
  National Association of Plan Advisors
 </span>
 <span class="s1">
  ，为美国退休协会的相关组织）的报告，即使
  <ok href="http://www.epochtimes.com/gb/tag/%E7%BD%91%E8%B7%AF%E8%AF%88%E6%AC%BA.html">
   网路诈欺
  </ok>
  已日益减少，然而，与
  <ok href="http://www.epochtimes.com/gb/tag/%E9%80%80%E4%BC%91%E9%87%91%E8%B4%A6%E6%88%B7.html">
   退休金账户
  </ok>
  相关的诈欺案却日渐增加。
 </span>
</p>
<p class="p1">
 <span class="s1">
  更重要的是，退休金账户若遭到
  <ok href="http://www.epochtimes.com/gb/tag/%E9%AA%87%E5%AE%A2.html">
   骇客
  </ok>
  冒领，储户所遭受的损失，并不受联邦法律的保障。尽管大型共同基金公司，都承诺会赔偿损失，但依法却未必有赔偿责任。
 </span>
</p>
<p class="p1">
 <span class="s1">
  专家们表示，网路犯罪份子会精心制作假的电子邮件，伪装成用户信任的发信人或网站，挟带具感染力的附件或连结。一旦人们开启档案或连结，就会执行恶意程式，窃取用户的个资。某些连结，甚至会诱使人们输入机密的财务资料。
 </span>
</p>
<p class="p1">
 <span class="s1">
  电子邮件账户密码，也是骇客的重要目标。接管电子邮件后，骇客就可透过“重设密码”的程序，进入银行或退休金账户。
 </span>
 <span class="s2">
  <br/>
 </span>
</p>
<p>
 <ok href="http://www.epochtimes.com/gb/19/12/27/n11748489.htm" rel="noopener noreferrer" target="_blank">
  从401k退休金账户借款好吗？专家：弊多于利
 </ok>
</p>
<p class="p1">
 <span class="s1">
  专家们呼吁，所有拥有退休金账户的用户，都需要保持警惕。透过网路查询退休金账户时，务必遵照以下的建议，确保账户安全。
 </span>
</p>
<p class="p1">
 <span class="s2">
  1.
 </span>
 <span class="s1">
  确保电脑与电子设备，都受到防火墙的保护，并且具有最新的防毒软件。
 </span>
</p>
<p class="p1">
 <span class="s2">
  2.
 </span>
 <span class="s1">
  谨慎对待电子邮件，回复信件、打开附件或点击连结前，务必小心是否为
  <ok href="http://www.epochtimes.com/gb/tag/%E9%92%93%E9%B1%BC%E9%82%AE%E4%BB%B6.html">
   钓鱼邮件
  </ok>
  。
 </span>
 <span class="s2">
  <br/>
 </span>
</p>
<p class="p1">
 <span class="s2">
  3.
 </span>
 <span class="s1">
  仔细阅读有关退休金账户所有来信，如果未经授权的活动，必须马上联络共同基金公司。
 </span>
</p>
<p class="p1">
 <span class="s2">
  4.
 </span>
 <span class="s1">
  所有金融账户，都必须使用独立的高强度密码。建议结合几个好记的单字与数字，如：
 </span>
 <span class="s2">
  MoonSunLight476
 </span>
 <span class="s1">
  。
 </span>
</p>
<p class="p1">
 <span class="s2">
  5.
 </span>
 <span class="s1">
  定期检查退休金账户中的金额。
 </span>
 <span class="s2">
  <br/>
 </span>
</p>
