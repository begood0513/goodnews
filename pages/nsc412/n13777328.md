<div><p>
 【大纪元2022年07月10日讯】（大纪元记者林燕综合报导）亿万富豪
 <ok href="https://www.epochtimes.com/gb/tag/%E9%A9%AC%E6%96%AF%E5%85%8B.html">
  马斯克
 </ok>
 （Elon Musk）终止并购推特后，下一步双方法庭见，法学专家表示，很可能双方会商量一个更低的价格完成并购。
</p>
<p class="p1">
 <span class="s1">
  周五（7月8日），
  <ok href="https://www.epochtimes.com/gb/tag/%E9%A9%AC%E6%96%AF%E5%85%8B.html">
   马斯克
  </ok>
  正式提交给美国证券交易委员会（SEC）的给推特的一封信中说，他叫停并购推特的协议，理由是推特严重违反了协议中的多项条款，以及
 </span>
 <span class="s1">
  拒绝向马斯克提供关于平台上虚假和垃圾账户的关键数据。
 </span>
</p>
<p class="p1">
 <span class="s1">
  马斯克说，推特对垃圾邮件账户数量的错误陈述可能对马斯克是一个“重大不利影响”（MAE），因此，根据交易合同的条款，他决定终止并购。
 </span>
</p>
<p class="p1">
 推特稍早表示，垃圾邮件或虚假账户占其活跃用户的比例不到5%，但它也承认，这些账户的数量有可能更高。
</p>
<p>
 根据合并协议，如果马斯克单方面终止交易，将支付10亿美元的分手费。但是推特不想要分手费，想要交易达成。
</p>
<p class="p3">
 <span class="s2">
  推特董事长
 </span>
 <span class="s3">
  布雷特‧泰勒（
 </span>
 <span class="s4">
  Bret Taylor
 </span>
 <span class="s3">
  ）回复并购叫停一事，他表示
 </span>
 <span class="s1">
  将按每股
 </span>
 <span class="s5">
  54.20
 </span>
 <span class="s1">
  美元的价格和现有协议条款推动马斯克完成收购推特的交易。他们计划在
 </span>
 <span class="s6">
  特拉华州商事法院对
 </span>
 <span class="s1">
  马斯克此举采取法律行动，以“具体执行”这宗并购交易。
 </span>
</p>
<h4 class="p3">
 特拉华州法院鲜少允许放弃并购
</h4>
<p class="p1">
 <span class="s1">
  特拉华州法院对收购者被允许放弃其交易设定了很高的标准。2020年，特拉华州法院允许韩国未来资产资本公司（Mirae Asset Capital Co）退出一项价值58亿美元的豪华酒店交易，因为大流行病导致卖方中国安邦保险集团改变了其正常的酒店经营。但是，这种情况很罕见。
 </span>
</p>
<p class="p1">
 <span class="s1">
  大多数情况下，法院会对目标公司有利，并命令收购者完成他们的交易——这种法律补救措施被称为“具体执行”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  然而，许多公司会选择与收购方和解，以结束对其未来的不确定性，因为这种不确定性可能会对其员工、客户和供应商造成压力。
 </span>
</p>
<p class="p1">
 已有多位法律专家表示，推特对马斯克放弃440亿美元收购案有充分的法律依据，但双方可能选择重新谈判或和解，而不是长期的法庭斗争。
</p>
<h4 class="p1">
 法学专家：谈判更低的价格 不值得打官司
</h4>
<p class="p1">
 <span class="s1">
  路透社周六（9日）援引三位公司法学教授的话说，往往目标公司会选择以较低的价格或财务补偿进行重新谈判交易，而不是持续数月的混乱法庭纠纷。
 </span>
</p>
<p class="p1">
 <span class="s1">
  加州大学伯克利分校的法律教授亚当·巴达维（Adam Badawi）表示，马斯克和推特或以较低的价格达成和解的一个理由是，诉讼费用太高，以及“这件事太混乱了，可能不值得（打官司）”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  CNBC报导说，
 </span>
 <span class="s1">
  杜兰大学法学院公司治理教授安·利普顿（Ann Lipton）说，推特董事会现在处于非常尴尬的境地。
 </span>
</p>
<p class="p1">
 因为董事会以及其它主要股东最初不看好这次收购，但在与马斯克进一步接触后，他们已经一致同意将推特卖给马斯克，并表示，不可能找到比马斯克更好的报价了。现在马斯克说不买了，当然不能一走了之。
</p>
<p class="p1">
 <span class="s1">
  利普顿表示，如果双方走上法庭，推特似乎占上风，因为合并协议中包括一个“具体执行条款”，即只要马斯克仍有债务融资，推特就有权起诉他，迫使他完成交易。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但是，马斯克和推特也可能达成和解。
 </span>
 <span class="s1">
  利普顿说，为了避免诉讼，推特可能会同意对最早的每股54.20美元的交易价格稍作修改，虽然这可能会让推特股东不满意。
 </span>
</p>
<h4 class="p1">
 推特股票暴跌 员工对前景感到受挫
</h4>
<p class="p1">
 <span class="s1">
  在周五马斯克终止并购案的消息传出后，推特周五的股价暴跌至36.81美元。现在，
 </span>
 <span class="s1">
  推特的股价低于九年前的水平，它刚上市时是以45.10美元开始交易的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  多年来，推特一直在努力实现与同行相同的业务增长水平。数字广告占推特收入的90%左右，最近数字广告市场因通货膨胀飙升而陷入动荡，乌克兰的战争也对广告支出产生了影响，这些都影响到了推特的营收。
 </span>
</p>
<p class="p1">
 <span class="s1">
  今年5月，推特首席执行官帕拉格·阿格拉瓦尔（Parag Agrawal）表示，该公司正暂停招聘，并寻求削减成本。他当时还说，有两名高管将离开公司。推特上周表示，它已经裁掉了其招聘团队的三分之一。
 </span>
</p>
<p class="p1">
 <span class="s1">
  《华尔街日报》报导说，周五马斯克终止并购的消息传出后，推特员工在内部沟通渠道上对公司的下一步发展提出了系列问题，并表达了他们对马斯克此举感到的挫败感。
 </span>
</p>
<p class="p1">
 <span class="s1">
  现任和前任员工说，在过去几个月中，许多员工对推特董事会当初同意马斯克收购的决定持批评态度，也有一些员工则对马斯克潜在的并购前景表示受到鼓舞。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这些员工表示，终止并购的最新消息将使推特陷入混乱状态，这将损害挽留员工，并分散对长期产品规划的注意力。
 </span>
</p>
