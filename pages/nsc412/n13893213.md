<div><p>
 【大纪元2022年12月28日讯】（大纪元记者殷瑞娜编译报导）由于股票下滑，许多公司放缓了长达十年的持续增长，今年，
 <ok href="https://www.epochtimes.com/gb/tag/%E7%A7%91%E6%8A%80.html">
  科技
 </ok>
 界的亿万富翁们
 <ok href="https://www.epochtimes.com/gb/tag/%E6%8D%9F%E5%A4%B1.html">
  损失
 </ok>
 了数千亿美元。
</p>
<p>
 对
 <ok href="https://www.epochtimes.com/gb/tag/%E7%A7%91%E6%8A%80.html">
  科技
 </ok>
 股来说，这是沉闷的一年，对那些净资产与公司命运密切相关的创始人和高管的巨大财富桶来说，也是如此。
 <ok href="https://www.epochtimes.com/gb/tag/%E8%84%B8%E4%B9%A6.html">
  脸书
 </ok>
 创始人扎克伯格（Mark Zuckerberg），曾位列世界十大富豪之列，今年净资产
 <ok href="https://www.epochtimes.com/gb/tag/%E6%8D%9F%E5%A4%B1.html">
  损失
 </ok>
 近810亿美元。
</p>
<p>
 科技股在2022年大幅下跌，原因是提振许多科技股的销售热潮消退了。在过去十年中，以极快的速度增长的科技巨头们，开始冻结招聘，并进行大规模裁员。
</p>
<p>
 由于投资者和公司要应对海外战争、通货膨胀和利率上升，以及预算日益紧张的消费者，今年更广泛的市场也出现了下滑。非科技行业的亿万富翁们，也从他们的净资产中，减少了很多钱。
</p>
<p>
 但一些科技大亨的净资产跌幅最显着，看看今年几位科技界领袖损失了多少钱。
</p>
<p>
 <strong>
  埃隆．马斯克（Elon Musk）
 </strong>
 <br/>
 总损失：1320亿美元。目前净资产：1,390亿美元
 <br/>
 马斯克今年损失了1,320亿美元的净资产。
 <br/>
 这位特斯拉、SpaceX和Twitter的首席执行官，在今年大部分时间里，是世界上最富有的人。但是，马斯克的大部分财富都在特斯拉股票，随着特斯拉股票直线下降，全年暴跌近70%，他失去了这一皇冠。
 <br/>
 特斯拉股票首当其冲地感受到了对电动汽车需求的担忧，中国是该公司的一个主要市场，其需求正在减弱。特斯拉还面临着与制造汽车的供应成本上升有关的挑战。
 <br/>
 一些投资者还对马斯克专注于推特的做法表示不安。马斯克出售特斯拉股票，以资助对Twitter的440亿美元的收购，投资者们敦促马斯克再次专注于他的汽车公司。
</p>
<p>
 <strong>
  杰夫．贝佐斯（Jeff Bezos）
 </strong>
 <br/>
 总损失：841亿美元。目前净资产：1,080亿美元
 <br/>
 贝佐斯今年损失了841亿美元，因为科技股下滑，亚马逊股价暴跌近50%，成为其有史以来市场上最糟糕的年份之一。
 <br/>
 去年，这位亚马逊的创始人兼董事长，辞去了首席执行官的职务，但他仍然是亚马逊的最大股东。
 <br/>
 贝佐斯也是太空旅行公司蓝色起源（Blue Origin）的创始人，并拥有《华盛顿邮报》，他在2013年以2.5亿美元买下了该报。
</p>
<p>
 <strong>
  马克．扎克伯格（Mark Zuckerberg）
 </strong>
 <br/>
 总损失：807亿美元。目前净资产：448亿美元
 <br/>
 扎克伯格的净资产今年暴跌近810亿美元。根据彭博亿万富翁指数，这位Facebook创始人，从去年年底的第六大富豪，下滑到第25位。
 <br/>
 对Facebook来说，这是艰难的一年，去年它宣布将其母公司的名字，改为Meta。扎克伯格仍然领导着他在大学时代创立的公司，他策划了公司的转变，将重点更多地放在元宇宙（metaverse）上。
 <br/>
 Meta公司报告了今年头两个季度的收入下降，因为该公司面临着来自TikTok和其它较小的社交媒体应用程序的竞争加剧，以及更严峻的数字广告环境。该公司警告说，随着它建立起元空间，它预计明年的损失会更大。
</p>
<p>
 <strong>
  拉里．佩奇（Larry Page）
 </strong>
 <br/>
 总损失：446亿美元。目前净资产：838亿美元
</p>
<p>
 <strong>
  谢尔盖．布林（Sergey Brin）
 </strong>
 <br/>
 总损失：434亿美元。目前净资产：802亿美元
 <br/>
 布林和佩奇今年共损失了880亿美元。作为
 <ok href="https://www.epochtimes.com/gb/tag/%E8%B0%B7%E6%AD%8C.html">
  谷歌
 </ok>
 的创始人和母公司Alphabet的现任董事会成员，他们在2019年退出了高管职位，但仍然通过一类特殊的有投票权的股份，对企业拥有重大影响力。佩奇曾担任首席执行官，而布林曾是Alphabet的总裁。
 <br/>
 在数字广告受到惩罚的一年里，即使是Alphabet这样的行业巨头也未能幸免。 Alphabet已经放慢了招聘速度，并关闭了其视频游戏的云流媒体服务Stadia。
</p>
<p>
 <strong>
  比尔．盖茨（Bill Gates）
 </strong>
 <br/>
 总损失：287亿美元。目前净资产：1,090亿美元
 <br/>
 这位微软的联合创始人，在2022年损失了近290亿美元。盖茨大约20%的跌幅，几乎与今年标准普尔500指数的跌幅相当，这意味着这位亿万富翁的损失，在很大程度上与整体市场的广泛晴雨表和经济的衡量标准相吻合。
 <br/>
 盖茨的投资是多元化的，他持有多家公司的大量股份，拥有房地产和大片的农田。但他最有价值的仍然是微软公司的股票，该公司今年缩减了近30%。虽然微软的云服务收入，在该公司最近一个季度增长了24%，但其Windows操作系统的销售额却下降了15%。微软表示，它预计明年其云计算收入的增长将放缓。
</p>
<p>
 <strong>
  史蒂夫．鲍尔默（Steve Ballmer）
 </strong>
 <br/>
 总损失：202亿美元。目前净资产：855亿美元
 <br/>
 这位微软前首席执行官，今年损失了200多亿美元，因为这家科技巨头的下滑，侵蚀了他的净资产。
 <br/>
 鲍尔默的大部分财富都被捆绑在微软的股票上，他在2000年至2014年期间，担任微软的首席执行官。虽然该公司比其它更依赖广告和消费者支出的科技巨头表现得更好，但这家Windows制造商，今年仍损失了超过四分之一的价值。除了预测云计算增长放缓之外，微软还警告投资者，它预计个人电脑销售疲软将延续到明年，而广告商缩减支出，将减少LinkedIn和搜索广告的收入。◇
</p>
