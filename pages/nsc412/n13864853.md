<div><p>
 【大纪元2022年11月13日讯】（大纪元记者林燕报导）前国务院中国政策首席顾问余茂春近日撰文说，北京从不相信美国在中共军队入侵台湾时会坐视不管，所以仅凭这一点就足以表明，谈论美对台
 <ok href="https://www.epochtimes.com/gb/tag/%E6%88%98%E7%95%A5%E6%A8%A1%E7%B3%8A.html">
  战略模糊
 </ok>
 是一种一厢情愿的自我斟酌。
</p>
<p class="p3">
 <span class="s1">
  “实际上，很难在中华人民共和国（中共）政府内部中找得到有影响力的人物相信美国的
  <ok href="https://www.epochtimes.com/gb/tag/%E6%88%98%E7%95%A5%E6%A8%A1%E7%B3%8A.html">
   战略模糊
  </ok>
  。”他在英文的《台湾时报》上刊文说。
 </span>
</p>
<p class="p4">
 <span class="s2">
  余茂春列举说，从毛泽东到邓小平的所有中共领导人都坚信，美国会在中共军队入侵台湾时出兵干预，
 </span>
 <span class="s1">
  美国的全球霸权大战略需要台湾成为美国的“永不沉没的航空母舰”
 </span>
 <span class="s1">
  ，美国永远不会让中国这种共产主义国家占领台湾，如果中共军方试图入侵台湾，美国将毫不含糊地动用武力进行干预。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他说，中共也是在认定美国的清晰战略基础上，才展开大规模扩军计划，它们所有入侵台湾的战情设想中，都将美军锁定为其侵台军事行动的最大威胁。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “中共军方的致胜理论——决定其行动和战术——将击败美军视为夺取台湾的先决条件。”余茂春说，“在这一点上，中共也是毫不含糊的。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  他表示，中共是一个修正主义政权，与周边的许多国家都有过激烈的领土领海冲突，北京还准备在印度
 </span>
 <span class="s5">
  —
 </span>
 <span class="s1">
  太平洋地区发动一连串侵略，而台湾是这一连串侵略行动的第一个环节。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “我们对这种侵略并不陌生，纳粹德国的侵略系列行动起始于
 </span>
 吞并
 <span class="s1">
  捷克斯洛伐克的苏台德地区（
 </span>
 <span class="s5">
  Sudetenland
 </span>
 <span class="s1">
  ），日本帝国旷日持久的扩张始于侵占满洲（九一八事变），历史的殷鉴其实并不遥远。”余茂春写道。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “台湾就是中国的苏台德区。中共对此毫不含糊。而我们也不应该一厢情愿。”他补充说，“让我们避免一九三八年慕尼黑式那种危险而灾难性的‘战略模糊’得以重演。”
 </span>
</p>
<h4 class="p3">
 战略模糊自相矛盾 混淆了战略意图和战术行动
</h4>
<p class="p3">
 余茂春表示，因对台“战略模糊”未编入美国的战略原则，现在又被华府的政策制定者和权威人士经常提及，就跟真有这回事一样。
</p>
<p class="p3">
 <span class="s1">
  “然而，不真实的东西说得再多也不会变成真的。美国对于保卫台湾的
  <ok href="https://www.epochtimes.com/gb/tag/%E6%88%98%E7%95%A5%E6%B8%85%E6%99%B0.html">
   战略清晰
  </ok>
  ，已经延续了七十年之久。”他写道。
 </span>
 “战略模糊往往成为美国对中政策怠惰的便宜借口。”
</p>
<p class="p1">
 <span class="s1">
  他介绍说，“战略模糊”的说法始于20世界90年代中期，持这种观点的人认为，战略模糊可以作为安抚北京的一种手段，也是华盛顿与北京交往的一种方式，不会激怒中共进而破坏“世界上最重要的双边关系”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  余茂春表示，“
 </span>
 <span class="s1">
  战略模糊”在概念上是自相矛盾的，它混淆了“战略意图”和“战术行动”之间的关键区别。
 </span>
</p>
<p class="p1">
 他说，
 <span class="s1">
  在对台的“战略意图”上，美国始终维持
  <ok href="https://www.epochtimes.com/gb/tag/%E6%88%98%E7%95%A5%E6%B8%85%E6%99%B0.html">
   战略清晰
  </ok>
  的政策和作为。
 </span>
 <span class="s1">
  自杜鲁门（
 </span>
 <span class="s2">
  Harry
 </span>
 <span class="s3">
 </span>
 <span class="s2">
  Truman
 </span>
 <span class="s1">
  ）以来，历任美国总统一贯含蓄或明确地宣示在中国侵略台湾时加以干预的意图。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但“战术行动”就像军事计划一样，是唯一而且应该暧昧不明的。所以说，现在美国对
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E5%85%A5%E4%BE%B5%E5%8F%B0%E6%B9%BE.html">
   中共入侵台湾
  </ok>
  的应对考量是如何应对军事卷入的问题，而不是要不要军事卷入的问题。
 </span>
</p>
<h4 class="p1">
 真的假不了 美对台战略清晰是一贯而明确的
</h4>
<p class="p3">
 <span class="s1">
  他在文中详细列举了美国的对台战略清晰政策，认为美国在动用武力保卫台湾的问题上的立场是一贯而明确的。
 </span>
</p>
<p class="p3">
 <span class="s1">
  1950年6月25日，共产中国支持的朝鲜人民军在当天发动韩战。同一天，杜鲁门总统派遣美国第七舰队进入台湾海峡，保护台湾免于中共可能的入侵，并将该地区中立化。1955年，美国和中华民国（台湾）签署了《中美共同防御条约》。美国对台防务战略清晰的成文化持续了近30年之久。
 </span>
</p>
<p class="p3">
 <span class="s1">
  1980年1月1日，卡特（
 </span>
 <span class="s5">
  Jimmy Carter
 </span>
 <span class="s1">
  ）总统单方终止了《中美共同防御条约》。即使发生这种变化，美国也没有放弃保卫台湾的战略意图。
 </span>
</p>
<p class="p3">
 <span class="s1">
  20世纪80年代以来，卡特
 </span>
 <span class="s5">
  –
 </span>
 <span class="s1">
  里根（
 </span>
 <span class="s5">
  Carter-Reagan
 </span>
 <span class="s1">
  ）时代开始了一段“战略准透明”（
 </span>
 <span class="s5">
  strategic translucency
 </span>
 <span class="s1">
  ）的逐步演变期，包括通过了具有里程碑意义的1979年《台湾关系法》和里根总统的“六项保证”。接着，老布什（
 </span>
 <span class="s5">
  George H. W. Bush
 </span>
 <span class="s1">
  ）总统为美国长期以来的对台战略清晰赋予了新动向，批准向台湾出售数量空前的精锐
 </span>
 <span class="s5">
  F-16
 </span>
 <span class="s1">
  战斗机。
 </span>
</p>
<p class="p3">
 <span class="s1">
  最为人所知的是，在1995-1996年的第三次台海危机期间，克林顿（
 </span>
 <span class="s5">
  Bill Clinton
 </span>
 <span class="s1">
  ）总统派遣两支美国航空母舰战斗群，进驻台湾海峡附近水域，应对中共军方在该海域发射导弹恐吓台湾选民的挑衅行为。
 </span>
</p>
<p class="p3">
 <span class="s1">
  2001年，布什（
 </span>
 <span class="s5">
  George W.
 </span>
 <span class="s6">
 </span>
 <span class="s5">
  Bush
 </span>
 <span class="s1">
  ）总统明确表示，他将“不惜一切保卫台湾”。
 </span>
 <span class="s1">
  后来，在川普政府时期，美国与台湾之间的关系变得越发强健，包括大幅增加对台出售关键性武器，更频繁地派遣高层级官员访问台湾。
 </span>
</p>
<h4 class="p3">
 美用行动回应中共的每一次挑衅
</h4>
<p class="p3">
 <span class="s1">
  “在过去的六年里，不论是共和党还是民主党主政，美国一直不断强化对台战略清晰。”余茂春说，“美国的战略清晰贯穿每一次中国针对台湾发动的危机和挑衅。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  比如：美国战舰一再突破北京的所谓“红线”，在台湾海峡展开自由航行行动。这些行动的频率比过去增长了十倍以上，使台湾海峡——这条对台湾防御至关重要的水道——实质国际化。
 </span>
</p>
<p class="p3">
 <span class="s1">
  美国国会还就保卫台湾的重要性，达成历史性的跨党派共识。
 </span>
</p>
<p class="p3">
 <span class="s1">
  美国的军事领袖也重申美国对台战略清晰的立场。海军上将帕帕罗（
 </span>
 <span class="s5">
  Samuel J Paparo
 </span>
 <span class="s1">
  ）是美国太平洋舰队司令，负责在中国一旦攻击台湾时执行美国的军事任务。2022年10月19日，帕帕罗上将表明，美国将“阻止任何对台湾的入侵，以及任何以武力解决问题的尝试，在这一点上，没有任何模糊空间。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  白宫的态度也同样明确。自2021年10月以来，拜登（
 </span>
 <span class="s5">
  Joe Biden
 </span>
 <span class="s1">
  ）总统先后在四个不同场合明确表示，如果北京入侵台湾，美国将进行军事干预。
 </span>
</p>
<p class="p3">
 <span class="s1">
  余茂春解释说，白宫重申美国的“一个中国政策”不变，与美国总统在保卫台湾问题上的战略清晰并不矛盾。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他认为，正好相反，这些声明进一步肯定了战略清晰。重申美国的“一个中国政策”，巩固了美国的战略清晰立场，因为“一个中国政策”的基本要件之一，就是美国长久以来一贯反对台海两岸任何一方使用武力解决台湾议题的立场。正如自尼克松时代以来历届美国政府的多次声明，这个立场一点也不模糊。
 </span>
</p>
