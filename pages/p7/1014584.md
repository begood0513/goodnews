 <p>
  有一天，一个
  <span href="https://www.secretchina.com/news/gb/tag/乞丐" target="_blank">
   乞丐
  </span>
  慢慢悠悠的走进寺庙，跪地向佛祖拜述衷肠：“佛祖啊，我最羡慕的就是您了。只要您整天往宽大的佛堂上一坐，就有万千信徒给你供奉香火与食物，而我不但每天受人唾弃，而且还要忍饥挨饿，这也太不公平了吧。”乞丐才刚说完，佛祖就显灵了！猜猜，慈悲的佛祖看着眼前牢骚满腹，不断抱怨的乞丐，会开口对他说什么呢？
  <br>
   <br>
    只见佛祖对乞丐说道：“世间的一切皆有因果，你既然羡慕我的一天，那咱俩就互换身份一天，你做一天的佛祖，我做一天的乞丐。”
    <br>
     <br>
      听见这般好事居然降临在自己头上，乞丐连忙点头答应。谁知，佛祖却劝他先别急着答应，因为佛祖还要提出一个要求。这个要求就是：在乞丐做佛祖的这一天里，不管遇上任何信徒到访，看到或听到了什么都千万不要插手。
      <br>
       <br>
        眼见供桌上的贡品，乞丐早就按捺不住了，迫不及待的示意佛祖赶紧离开。佛祖前脚刚走，乞丐后脚便扎在供桌上吃了个饱，刚想活动一下身体，这时走进来一个身着华丽服饰的员外。
        <br>
         <br>
          员外摆上贡品，上了香，开始跪在地上祈祷：“我如今都快50岁了，膝下无子，求佛祖开恩赐我一个儿子吧。”乞丐刚想开口，想起佛祖临走前嘱咐他不要插手任何事，便忍住了。
          <br/>
          <br/>
          员外祈祷完离去时，随身携带的钱袋不小心掉落在供桌旁。就在员外出门之际，又走进来一个衣着朴素的书生，一看就是个穷书生。
          <br/>
          <br/>
          书生跪在地上拜了拜说：“佛祖，还有一个月我就要进京赶考了，十年寒窗苦读，希望佛祖保佑我金榜题名。”
          <br/>
          <br/>
          最后叩头时，书生看到了员外掉落在供桌旁的钱袋，以为是佛祖显灵，知道其没有进京的盘缠，特意赐给他的。于是连忙叩谢佛祖后，赶紧拿着钱袋匆匆离去。
          <br/>
          <br/>
          第三个进来祈福的是个渔夫，因为要出海打鱼，所以祈求佛祖保佑他平安归来。而恰好此时员外返回寻找钱袋，看见寺庙内只有渔夫一人，就一口咬定是渔夫拿走钱袋的。
          <br/>
          <br/>
          二人争执半天，员外非要拉着渔夫去见官。这时乞丐再也坐不住了，于是显灵对员外说：“你的钱袋是被书生拿走了，而不是渔夫。”
          <br/>
          <br/>
          结果，员外确实在书生那里寻得了钱袋，放了渔夫。
          <span id="hideid" name="hideid" style="color:red;display:none;">
           <span href="https://www.secretchina.com">
           </span>
          </span>
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  当乞丐正在暗暗为自己做了件好事而自鸣得意时，归来的
  <span href="https://www.secretchina.com/news/gb/tag/佛祖" target="_blank">
   佛祖
  </span>
  却反倒训斥乞丐违反了当初的约定，插手了人事，甚至要惩罚乞丐世世做牛做马，不能够转生为人。
  <br/>
  <br/>
  乞丐十分不服气，自觉没做错事情，为何还要被惩处。佛祖为了让他心服口服，遂带着乞丐前往未来，瞧瞧到底发生了什么事情。
 </p>
 <p>
  在未来，书生虽然将钱袋归还给了员外，却背上了小偷的骂名，前途尽毁。
  <br/>
  <br/>
  至于跟渔夫解开了误会的员外呢？他居然也不好过。因为他毁掉了书生的前途，反而为自己积了祸，导致求子的愿望再也无法实现了，只能孤独终老，家业衰败而无人继承。
  <br/>
  <br/>
  在这三人中，最惨的居然是渔夫。因为乞丐当时插手证明了其清白，让渔夫顺利地在第二天照常出海。谁知，渔夫却因此遭遇风暴，最终不幸葬身大海，尸骨无存。
 </p>
 <p>
  倘若最初没有乞丐的多管闲事，渔夫会被抓去见官，进而躲过风暴这一大劫。
  <br/>
  <br/>
  目睹三人的未来之后，乞丐自责的低下了头。他耳边响起了佛祖的声音：“世间的一切皆有因果，牵一发而动全身，有的时候顺其自然反而是最好的。”
