 <p>
  这一篇文章就针对薛蟠喜欢林黛玉这事儿再描述的丰富一点。很多人觉得
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
  </strong>
  那个德行怎么可能喜欢
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林妹妹
   </span>
  </strong>
  ，其实恰恰是林妹妹的风华绝代，让呆霸王这样的混人都心生爱慕难得生出一丝纯情。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
   对黛玉一见倾心
  </strong>
 </p>
 <p>
  第二十五回，贾宝玉被马道婆五鬼魇了，生命垂危，贾家大乱。所有人都忙乱中自然顾不得避嫌什么的。乱糟糟一团的氛围，
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
  </strong>
  不小心见了一眼林黛玉立刻“酥倒了”。曹雪芹用“酥倒”来形容见到
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  的薛蟠，实在太过贴切。一个酥倒不同于色受魂消，那是一种被击中内心的情感冲击，甚至不涉及淫邪的思维。这个酥倒，已经证明薛蟠对
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  一见倾心，薛蟠今生第一次心动了。
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
  </strong>
  这人性格直接，喜欢就一定会求。按照后文薛宝钗和薛姨妈的话，薛蟠是求过薛姨妈要娶
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  的。只是薛姨妈拒绝了。林黛玉可是贾母的心头肉，怎么可能给呆霸王
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
  </strong>
  糟蹋。这点薛姨妈比谁都清楚。薛姨妈说“连邢姑娘我还怕你哥哥糟踏了她，所以给你兄弟说了。别说这孩子，我也断不肯给他。”薛姨妈知道事不可为，绝不会讨这个没意思。
 </p>
 <p>
  薛蟠娶林黛玉没戏，却依然心向往之。后面
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
  </strong>
  甚至亲力亲为的做了几件事，虽然作者没写，却显然有一份心思是为了林黛玉。细读《红楼梦》往往会掩卷慨叹：情之一字万不能禁止者！
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
   挂怀黛玉故行事
  </strong>
 </p>
 <p>
  <strong>
   第一，
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
   配药
  </strong>
  。有一天贾宝玉
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  给王夫人请安。说到天王补心丹那个药，贾宝玉就说有一个药方比天王补心丹还有效，
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  吃了保准就好了。这个药方就来自薛蟠。贾宝玉不过说说就罢了，王熙凤却证实薛蟠已经着手配药了。贾宝玉既然将这个药和
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  的身体联系到一起，薛蟠他家也没吃这个药的病人，因此薛蟠配这个药的初心就很值得追究了。
 </p>
 <p>
  <strong>
   第二，
   <span href="https://zh.wikipedia.org/wiki/%E9%A6%99%E8%8F%B1" target="_blank">
    香菱
   </span>
   学诗
  </strong>
  。薛蟠走了，香菱住到大观园突然要学诗。固然
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E9%A6%99%E8%8F%B1" target="_blank">
    香菱
   </span>
  </strong>
  会倾慕大观园诸位小姐会作诗。可她一个妾，论地位，论身份都不该主动要求学这些。连迎春、惜春都不作，她实在没理由学。而且真要学，怎不见她早学？我认为
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E9%A6%99%E8%8F%B1" target="_blank">
    香菱
   </span>
  </strong>
  学诗表面是羡慕小姐们风雅，实则受到薛蟠的影响。薛蟠爱慕林黛玉，必然常挂在嘴边，
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E9%A6%99%E8%8F%B1" target="_blank">
    香菱
   </span>
  </strong>
  作为薛蟠屋里人，薛蟠一定说过你有空也学学人家。宝钗深知哥哥深意，才故意自己不教香菱，让
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E9%A6%99%E8%8F%B1" target="_blank">
    香菱
   </span>
  </strong>
  和林黛玉多亲近，学习一二分神韵气质好讨好薛蟠。
 </p>
 <p>
  <strong>
   第三，
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
   带回来的
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
   家乡土货
  </strong>
  。
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
  </strong>
  巴巴跑到
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  家乡，在虎丘捏了一个泥人，带回来一堆林黛玉家乡货。要知道如果不是薛蟠故意，他那个性格真要买东西，天下那么大，哪里不可以买，他的品味会买那些笔墨纸砚一类的东西么？显然不会。薛蟠带回来的土货绝不是无心之举。他知道带回来的东西薛宝钗一定会给
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  送一份，这算变向的亲近佳人为佳人尽一份心。宝钗明白薛蟠心意，还送了黛玉两份。而
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  果然说借薛蟠的光，一睹家乡之物。
 </p>
 <p>
  <strong>
   第四，
   <span href="https://zh.wikipedia.org/wiki/%E8%96%9B%E8%9F%A0" target="_blank">
    薛蟠
   </span>
   娶
   <span href="https://zh.wikipedia.org/wiki/%E5%A4%8F%E9%87%91%E6%A1%82" target="_blank">
    夏金桂
   </span>
  </strong>
  。薛蟠娶
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E5%A4%8F%E9%87%91%E6%A1%82" target="_blank">
    夏金桂
   </span>
  </strong>
  也是一见就相中了。
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E5%A4%8F%E9%87%91%E6%A1%82" target="_blank">
    夏金桂
   </span>
  </strong>
  长得好固然重要。可香菱当时交代这一段还说了一点就是夏金桂会读书作诗。要说夏金桂的长相让薛蟠一见倾心，似乎也没有那么明显。但再加上读书作诗就不一样了。薛蟠这个执念显然与林黛玉也有关。
 </p>
 <p>
  综上，薛蟠虽然娶不到林黛玉，但倾慕林黛玉的心思从没断过。他是个行动派，想到就会去做。求娶林黛玉，配药，去林黛玉家乡带回来土产，觉得作诗的女人都美，故娶了夏金桂，这些丝丝缕缕都与
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  有关。
 </p>
 <p>
  比起
  <span href="https://www.secretchina.com/news/gb/tag/薛蟠" target="_blank">
   薛蟠
  </span>
  默默的为林黛玉做了一些事，贾宝玉对林黛玉除了感情，似乎并没有任何行动。在配药、土产后面都有贾宝玉的夸夸其谈，愿意为林黛玉配药，买土产回来，却再无下一步行动。也应该是作者有意为之。你认为呢？
