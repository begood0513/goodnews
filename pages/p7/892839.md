 <p>
  <span style="color:#000000">
   被称为“湘江第一才子”的作家
   <strong>
    金庸
   </strong>
   写了不少可以称作为“经典”的武侠作品，被拍成电视剧的作品如《倚天屠龙记》，《碧血剑》，《射雕英雄传》等，可谓是一代80、90后的童年啊。即使是现在这些作品也是被不断翻拍，其热度不减当年。那么大家一定知道
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E9%87%91%E5%BA%B8" target="_blank">
    <span style="color:#000000">
     金庸
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   先生曾经写过两部作品
   <strong>
    《连城诀》
   </strong>
   和
   <strong>
    《笑傲江湖》
   </strong>
   吧。这两部作品的内容真的是极其相似，曾引不少“金庸迷”热烈讨论过，下面就让我们看一下有哪些相似处吧！
  </span>
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  <span style="color:#000000">
   <strong>
    一、都受到过别人的冤枉
   </strong>
  </span>
 </p>
 <p>
  <span style="color:#000000">
   在
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%BF%9E%E5%9F%8E%E8%AF%80" target="_blank">
    <span style="color:#000000">
     《连城诀》
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   中，性格单纯直率被师妹戚芳称作为“空心菜”的
   <strong>
    狄云
   </strong>
   被奸人万珪所陷害冤枉非礼妇女而关入大牢，并且被削断了四指还被刺穿了琵琶骨。
  </span>
 </p>
 <p>
  <span style="color:#000000">
   在
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%AC%91%E5%82%B2%E6%B1%9F%E6%B9%96" target="_blank">
    <span style="color:#000000">
     《笑傲江湖》
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   中，其主人公
   <strong>
    令狐冲
   </strong>
   父母双亡，被华山派掌门
   <strong>
    岳不群
   </strong>
   夫妇收为首徒，并抚养他长大，自幼生长在华山的
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">
    <span style="color:#000000">
     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   ，早就把华山当作自己的家，把华山众人当做自己的家人。但是当小师妹的心上人林平之家传剑谱被偷了之后，他被所有的华山“家人”们冤枉抛弃，他们说是他偷了林家的家传剑谱。
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">
    <span style="color:#000000">
     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   被自己最信任最亲近的人冤枉，其中辛酸滋味哪是常人可以体会。
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">
    <span style="color:#000000">
     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   的武功《独孤九剑》乃是风清扬所传授的，
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%8B%84%E4%BA%91" target="_blank">
    <span style="color:#000000">
     狄云
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   的武功是言达平传授，他们都是学习了其他武功之后被人陷害冤枉，然后被迫脱离师门。也正是因为发生的这些事情使他们的人生轨迹发生了偏转。
  </span>
 </p>
 <p>
  <span style="color:#000000">
   <strong>
    二、师父都是卑鄙无耻的小人
   </strong>
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%8B%84%E4%BA%91" target="_blank">
    <span style="color:#000000">
     狄云
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   的师父是铁锁横江戚长发，此人心机深层、阴险，其绰号“铁锁横江”，表面意思是他武功了得，善于守御，其实事实上的意思是他会挖空心思报复惹了他的人，让人如船在江心乱转，上下不得。他看似是一位老实巴交的乡下人，其实他善于高度伪装自己，其自私给徒弟
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%8B%84%E4%BA%91" target="_blank">
    <span style="color:#000000">
     狄云
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   和女儿戚芳也带了不少伤害。可谓是狠起来六亲不认。
  </span>
 </p>
 <p>
  <span style="color:#000000">
   令狐冲的师父是华山的老大
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E5%B2%B3%E4%B8%8D%E7%BE%A3" target="_blank">
    <span style="color:#000000">
     岳不群
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   ，他是“君子剑”，但是在正邪两派举足轻重的人都称他是“伪君子”。而
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">
    <span style="color:#000000">
     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   也被这心机深层的“伪君子”师父伤害和利用。
  </span>
 </p>
 <p>
  <span style="color:#000000">
   <strong>
    三、都经历了牢狱之灾
   </strong>
  </span>
 </p>
 <p>
  <span style="color:#000000">
   <span href="https://www.secretchina.com/news/gb/tag/狄云" target="_blank">
    狄云
   </span>
   是被官府逮捕入狱。令狐冲是被人陷害替代任我行成为了囚犯。
  </span>
 </p>
 <p>
  <span style="color:#000000">
   <strong>
    四、因祸得福都在牢中学到了绝顶武功
   </strong>
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%8B%84%E4%BA%91" target="_blank">
    <span style="color:#000000">
     狄云
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   在遭受了丁典虐待毒打之后，狄云得到了丁典的绝学《神照经》。
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">
    <span style="color:#000000">
     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   也是一样。他虽然进了牢房，但是阴差阳错学会了任我行留下来的《吸星大法》，学会了至上的武功。
  </span>
 </p>

   <strong>
    五、都与邪派中人结交
   </strong>
  </span>
 </p>
 <p>
  <span style="color:#000000">
   血刀老祖以为穿了宝象的衣服的狄云真的是自己的徒孙，所以
  </span>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%8B%84%E4%BA%91" target="_blank">
    <span style="color:#000000">
     狄云
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   与邪派中人往来是有点身不由己的。
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">
    <span style="color:#000000">
     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   与狄云相反，他觉得与反派人物如任我行、向问天、任盈盈等人相处，比较聊得来。
  </span>
 </p>
 <p>
  <span style="color:#000000">
   <strong>
    六、最爱的师妹都嫁他人
   </strong>
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E7%8B%84%E4%BA%91" target="_blank">
    <span style="color:#000000">
     狄云
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   一生都对师妹戚芳单相思，但是戚芳却被自己的仇人娶走，眼睁睁看着自己最爱的女人嫁做他人妇。
  </span>
 </p>
 <p>
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E4%BB%A4%E7%8B%90%E5%86%B2" target="_blank">

     令狐冲
    </span>
   </span>
  </strong>
  <span style="color:#000000">
   最爱的是小师妹岳灵珊，只可惜郎有情妾无意。小师妹嫁给了林平之。

 </p>
 <p>

    七、师妹都被丈夫杀死

  </span>
 </p>

  <span style="color:#000000">
   戚芳被嫉妒心很重的丈夫万珪杀死。岳灵珊也是被自己钟情的如意郎君林平之亲手杀死。

 </p>
