 <p>
  我想形容黛玉的诗句很多，其中有一句“质本洁来还洁去”最为贴切。如果说贾宝玉是曹雪芹的理想化身，那么
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  就是
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9B%B9%E9%9B%AA%E8%8A%B9" target="_blank">
    曹雪芹
   </span>
  </strong>
  心中一尘不染的仙子。在曹雪芹的笔下，他借
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%B3%88%E5%AF%B6%E7%8E%89" target="_blank">
    宝玉
   </span>
  </strong>
  之口说男子都是“泥作的骨肉”，认为男子“浊臭逼人”，所以我想在潜意识里
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9B%B9%E9%9B%AA%E8%8A%B9" target="_blank">
    曹雪芹
   </span>
  </strong>
  是不希望这些男子来污染我们林妹妹的眼球的。
 </p>
 <p>
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/黛玉" target="_blank">
    黛玉
   </span>
   未曾接触过多男子
  </strong>
 </p>
 <p>
  在
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  活着的这些年里，除了自己的父亲及
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%B3%88%E5%AF%B6%E7%8E%89" target="_blank">
    贾宝玉
   </span>
  </strong>
  ，还真没有接触过其他男子。在黛玉第一次进贾府时，本来应该有机会见到她的两位舅舅的，但作者巧妙的安排让她没见到。如果说贾政因为斋戒去了没见到还可以理解，那么贾赦推说自己身上不好，怕见了黛玉伤心而不见，这个理由就有些莫名其妙了吧？按照常理，自己的亲外甥女大老远来，怎么能连个面都不见？何况当时他是在书房里，所谓身上不好只是个借口而已。想想黛玉的母亲贾敏也是贾赦贾政的亲妹妹，从小应该是一起长大的，所以我觉得贾赦不见黛玉真有些奇怪，除非是
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9B%B9%E9%9B%AA%E8%8A%B9" target="_blank">
    曹雪芹
   </span>
  </strong>
  根本不愿意让
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  看到这两个人……
 </p>
 <p>
  另有一个情节，就是黛玉奔丧回来之后，
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%B3%88%E5%AF%B6%E7%8E%89" target="_blank">
    宝玉
   </span>
  </strong>
  将北静王所赠一串念珠送给黛玉，结果黛玉以“什么臭男人拿过的，我不要它！”为由扔给了
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%B3%88%E5%AF%B6%E7%8E%89" target="_blank">
    宝玉
   </span>
  </strong>
  ，虽然这表明的是
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  对其他男子的态度，更应该是
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9B%B9%E9%9B%AA%E8%8A%B9" target="_blank">
    曹雪芹
   </span>
  </strong>
  本人的态度吧？虽然这个男子是个王爷，可是在
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  ，或者在曹雪芹眼里，他仍然是一个“浊物”，一个“臭男人”。
 </p>
 <p>
  不过我倒想起来，去苏州接黛玉回来以及送黛玉回家奔丧的人都是贾琏，在这一点上，虽然我也有些疑惑，但想想在当时，除了贾琏，好像也没有其他合适的人选了。看过几篇文章，说贾琏之所以对这件差事这么热衷是因为看上了林家的财产，但这推论并非本文讨论重点，故暂且略过。
 </p>
 <p>
  <strong>
   女子对黛玉的评价
  </strong>
 </p>
 <p>
  不过话又说回来，大观园里都是女子，女孩子对女孩子的评价总是没有男子的评价更让人信服，书中安排了两个细节来让外人来对黛玉做出评价，一个是薛蟠，一个是兴儿。现在想起描写薛蟠的那一段还觉得好笑。第二十五回，
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E8%B3%88%E5%AF%B6%E7%8E%89" target="_blank">
    宝玉
   </span>
  </strong>
  和凤姐因为被施了法术而中邪那一段中，大观园里乱作一团，不知为何此时薛蟠也会跑进园子里来？而且“独有薛蟠比诸人忙到十分去：又恐薛姨妈被人挤倒，又恐薛宝钗被人瞧见，又恐香菱被人臊皮，－－知道贾珍等是在女人身上做功夫的，因此忙的不堪。忽一眼瞥见了
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    林黛玉
   </span>
  </strong>
  的风流婉转，已酥倒在那里。”每每想到这一段，我都忍俊不禁，不过曹公为何会选择这样一个粗俗的人来对
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  作出评价呢？我倒是有些迷惑了。
 </p>
 <p>
  在六十五回里兴儿给尤二姐讲述贾府的几个女孩子时，讲到黛玉，说
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  “一肚子文章”，而且这些下人还给黛玉取名“多病西施”，有时出门或上车，或在院子里瞥见一眼，“不敢出气儿”。虽然都是顽话，但也足以说明黛玉的仙气了。
 </p>
 <p>
  说真的，现在回头想想，再看
  <strong>
   <span href="https://zh.wikipedia.org/wiki/%E6%9E%97%E9%BB%9B%E7%8E%89" target="_blank">
    黛玉
   </span>
  </strong>
  之死，就不会像以前那样难过，更不会为了她和宝玉的爱情而伤感了，这一切，早在曹公写《红楼梦》之前就想好了吧？同样的妙玉，曹公都用了“欲洁何曾洁，云空未必空”来形容，而独有对黛玉，用了“质本洁来还洁去”，看来，曹公只想让我们的林妹妹就这样纤尘不染的离开尘世吧。
