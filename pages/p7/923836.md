 <p>
  明朝
  <span href="https://www.secretchina.com/news/gb/tag/刘伯温" target="_blank">
   刘伯温
  </span>
  不仅仅是朱元璋的辅佐，而且更是一位道人，是一位身负特殊使命、为后人留下警世预言的修道人。刘伯温影响比较大的预言有《烧饼歌》、《金陵塔碑文》和《陕西太白山刘伯温碑文》。可以看出，这三部预言文笔一致、用语互通、内容互补，三部预言相互呼应，共同、完整的预言了中华民国成立后的中国国运，及已经发生和将要发生的大事。本文尝试解读《陕西太白山刘伯温碑文》部份内容，因为此前已经有对《陕西太白山刘伯温碑文》的详细解读，而且解读的非常到位，所以这里只是有选择的、谈谈个人对碑文其中一部份内容的理解。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  <strong>
   《
   <span href="https://www.secretchina.com/news/gb/tag/陕西太白山" target="_blank">
    陕西太白山
   </span>
   刘伯温碑文》原文：
  </strong>
 </p>
 <p>
  天有眼，地有眼，人人都有一双眼，
  <br>
   天也翻，地也翻，逍遥自在乐无边。
   <br>
    贫者一万留一千，富者一万留二三，
    <br>
     贫富若不回心转，看看死期在眼前。
     <br>
      平地无有五谷种，谨防四野绝人烟，
      <br>
       若问瘟疫何时现，但看九冬十月间。
       <br>
        行善之人得一见，作恶之人不得观，
        <br>
         世上有人行大善，免遭此劫不上算。
         <br>
          还有十愁在眼前：
          <br/>
          一愁天下乱纷纷，二愁东西饿死人，
          <br/>
          三愁湖广遭大难，四愁各省起狼烟，
          <br/>
          五愁人民不安然，六愁九冬十月间，
          <br/>
          七愁有饭无人食，八愁有人无衣穿，
          <br/>
          九愁尸体无人捡，十愁难过猪鼠年。
          <br/>
          若得过了大劫年，才算世间不老仙，
          <br/>
          就是铜打铁罗汉，难过七月初一十三，
          <br/>
          任你金刚铁罗汉，除非善乃能保全，
          <br/>
          谨防人人艰难过，关过天番龙蛇年。
          <br/>
          幼儿好似朱洪武，四川更比汉中苦,
          <br/>
          大狮吼如雷，胜过悼百虎，
          <br/>
          犀牛现出尾，平地遇猛若，
          <br/>
          若问大平年，架桥迎新主，
          <br/>
          上元甲子到，人人哈哈笑，
          <br/>
          问他笑什么？迎接新地主，
          <br/>
          上管三尺日，夜无盗贼难，
          <br/>
          虽是谋为主，主坐中央土，
          <br/>
          人民喊真主：
          <br/>
          银钱是个宝，看破用不了，
          <br/>
          果然是个宝，地下裂不倒，
          <br/>
          七人一路走，引诱进了口，
          <br/>
          三点加一勾，八王二十口，
          <br/>
          人人喜笑，个个平安。
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  <strong>
   解读如下：
  </strong>
 </p>
 <p>
  “天有眼，地有眼，人人都有一双眼，
  <br/>
  天也翻，地也翻，逍遥自在乐无边。”
 </p>
 <p>
  是说：“人在做，天在看；三尺头上有神灵”；人世间即将发生真正的天翻地覆的巨大变化------人类将走入新的纪元。这个巨大变化其中的表现就是：
 </p>
 <p>
  “贫者一万留一千，富者一万留二三，
  <br/>
  贫富若不回心转，看看死期在眼前。”
 </p>
 <p>
  此时人世有大难，而且可能淘汰的人数非常巨大；但是，只要能“回心转”的人就能度过大劫。这个大劫的具体表现是：
 </p>
 <p>
  “平地无有五谷种，谨防四野绝人烟，
  <br/>
  若问瘟疫何时现，但看九冬十月间。
  <br/>
  行善之人得一见，作恶之人不得观，
  <br/>
  世上有人行大善，免遭此劫不上算。”
 </p>
 <p>
  是从一场大瘟疫开始的，而且是从“九冬十月间”开始爆发的一场瘟疫开始的。对于这场灾难后“四野绝人烟”的惨景，及“天也翻，地也翻”的新纪元，“作恶之人不得观”，即作恶之人已经被淘汰，只有被“行大善”挽救的人才能得见。
 </p>
 <p>
  上述两段是该预言的整体概述，详细的是后面“十愁在眼前”等：
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">
   </div>
  </div>
  <div>
   <ins class="adsbygoogle" data-ad-client="ca-pub-1276641434651360" data-ad-format="fluid" data-ad-layout="in-article" data-ad-slot="5164544770" style="display:block; text-align:center;">
   </ins>
  </div>
 </center>
 <p>
  “一愁天下乱纷纷，二愁东西饿死人，
  <br/>
  三愁湖广遭大难，四愁各省起狼烟，
  <br/>
  五愁人民不安然，六愁九冬十月间，
  <br/>
  七愁有饭无人食，八愁有人无衣穿，
  <br/>
  九愁尸体无人捡，十愁难过猪鼠年。”
 </p>
 <p>
  “十愁”中除了描写大灾难的社会惨状，其中的重点是“湖广遭大难”和时间“九冬十月间”、“猪鼠年”。对于时间“九冬十月间”、“猪鼠年”，已经明确的是发生在2019猪年和2020鼠年的
  <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
   武汉肺炎
  </span>
  。“湖广遭大难”无疑是指湖北、湖南、广东一带有大难。如果“湖”是指湖北武汉肺炎瘟疫；那么“广”就是指广东尚未出现的灾难。对于广东出现的这个灾难，刘伯温在《金陵塔碑文》中描写到：“一气杀人千千万，大羊残暴过豺狼”，“繁华市，变汪洋，高楼阁，变泥岗”。无疑这是指由于一场震动而引起的水灾，并且“繁华市”应该是指广州。因为广州是羊城，当然也就是“大羊”。那么“湖广遭大难”随之出现的“各省起狼烟”，就应当是指各省为自保而纷纷出现各自为政、中共失去对各省控制的社会状态。
 </p>
 <p>
  “若得过了大劫年，才算世间不老仙，
  <br/>
  就是铜打铁罗汉，难过七月初一十三，
  <br/>
  任你金刚铁罗汉，除非善乃能保全，
  <br/>
  谨防人人艰难过，关过天番龙蛇年。”
 </p>
 <p>
  这段的重点在于“七月初一十三”和“天番龙蛇年”。“七月初”指中共的建党日“七·一”；“一十三”是指中共一大的十三个代表。那么“难过七月初一十三”就是指：不管是什么样的人，只要是加入中共而不声明退出的就过不了中共这一难，就会在大灾难中成为中共的陪葬而被淘汰。
 </p>
 <p>
  “天番龙蛇年”之“天番”呼应下文的“上元甲子到”，是指改天换地、历史的新纪元到来，因为“番”是番号，“天番”无疑是指纪元。“龙蛇年”在中国古文化中是指凶年、灾年，所以这里的“龙蛇年”应该是指“猪鼠年”；当然也有一种理解是指2024龙年、2025蛇年。如果我们把“大劫年”理解为“猪鼠年”，那么中共百年红朝解体灭亡的时间点就应当是在2020年的“七月初”，此时正是中共自1921年7 月成立的“百年”时间点。那么中共解体灭亡后，中国社会的情势会怎样？
 </p>
 <p>
  “幼儿好似朱洪武，四川更比汉中苦,
  <br/>
  大狮吼如雷，胜过悼百虎，
  <br/>
  犀牛现出尾，平地遇猛若，”
 </p>
 <p>
  本文对“幼儿”的理解是指中华民国。之所以用“幼儿”喻指中华民国，是因为中华民国成立后，就像“幼儿”一样还没来的及坐大、坐稳，就被中共压制到了台湾岛；而从台湾岛和大陆的实力对比来看，那么避居台湾的中华民国也就像“幼儿”。《朱洪武》是台湾拍摄的一部影片。“朱”是
  <span href="https://www.secretchina.com/news/gb/tag/朱元璋" target="_blank">
   朱元璋
  </span>
  ；“洪武”是朱元璋建立明朝的年号。那么“幼儿好似朱洪武”隐含的是中共灭亡、“中华民国”复兴之意，就是说“幼儿”般的中华民国，就像朱洪武代替元王朝一样取代中共。
 </p>
 <p>
  那么“幼儿好似朱洪武”紧接的下句“四川更比汉中苦”，则是指导致中共倒台的原因------大陆人民陷于极大的苦难之中。因为连天府之国的四川成都，和富饶之地的汉中都陷于苦难之中，所以整个中国的惨状也就不言而喻。惨到什么程度？“平地遇猛若”------平地就能遇到应该在山上才有的凶猛野兽，原因是“平地无有五谷种，谨防四野绝人烟”，已经到了“千山鸟飞绝，万径人踪灭”的地步。当然这种大灾难场景的出现，是基于“贫者一万留一千，富者一万留二三”的情况下所发生的。实际上由于“世上有人行大善”，有很多人已经得救，会度过大劫，惨烈程度已经发生极大的变化。就是说刘伯温虽然把“贫者一万留一千，富者一万留二三”最坏的结局讲给了世人，但同时也告诉世人：结局有变数，并且已经在变化之中，这就是“世上有人行大善，免遭此劫不上算”。
 </p>
 <p>
  刘伯温碑文后半部份着重预言的是世上“行大善”救人的“真、善、忍”佛法，因为已经有人对此做了详细的解读，这里就不再赘述。
 </p>
 <p>
  刘伯温为什么要留下预言？显然是在警示、告诫世人：找“行大善”之人得救。刘伯温在碑文最后说出的其中一个字谜是“三点加一勾”，也就是汉字“心”。就是说“天有眼，地有眼”，大劫得救之法的根本在于“心”，只有真心、只有诚心、只有“回心转”改变人心才能大劫得救。这个“回心转”就是：“过七月初一十三”这一关----退出中共的党团队组织。
