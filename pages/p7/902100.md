    <strong>
     第32话：愤怒的村民
    </strong>
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    虽然家园已经显得残破不堪，但前院聚集了那么多喧嚣的村民，而且都面向屋子，这还是首次见到。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    给人的感觉也不像是上门慰问的，毕竟大家的遭遇半斤八两；要说是跑来给医算师治疗，倒还有几分说的过去，因为现场的确许多人都身上都能看见包扎处理后的痕迹，但又明显让人感受到空气中弥漫着异漾的气氛。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    随着角狼渐近的奔跑声，以及回过头来看着阿修的村民，原本的喧嚣声也嘎然而止，取而代之的是望着自己的憎恨表情。敏感的阿修根据过往经验，也不难猜想发生啥事，八成是上门兴师问罪来了。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    只是村民憎恨的表情，随着目光焦点的改变，很快就转为讶异，然后再化为恐惧，毕竟阿修骑乘的可是令村民长年以来闻风丧胆、束手无策的角狼，而且体型还比一般的角狼大，估计应该是少有人见过的狼王。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    平时还会在意村民感受的阿修，这时也无遐多想，乘着角狼长驱直入，慌张的村民也自动让出一条路来，就怕自己被角狼盯上，会遭遇不测。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    原本充斥愤恨不平的气氛，此时反而被恐惧所取代，有些较胆小的村民眼泪都快掉下来了，而站在路两边的村民，颤抖的双腿清晰可辨，有的裤档还湿了。他们想隐蔽到人群中，却又被后面的村民死死顶住，形成一幅奇特的景象。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    阿修看着他们的眼神则充满不屑，毕竟这些人平时在他面前总是一脸轻蔑，极尽嘲讽之能事。而到了这时候，却又不敢正眼瞧他。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    不过阿修也不想在这些琐事上费心思，于是迳自乘着角狼进入庭院，然后下地的时候拍了角狼的背几下，角狼便服贴的趴在屋子与农舍之间的空地休憩。说是休憩，锐利的眼神却还是不停歇的来回扫视着村民。似乎在守护着阿修的同时，也让村民了解这里谁说话算数。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    阿修走向站在门口的实艳与实丽，只见母女两个人紧抱在一起，全身瑟瑟发抖。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “婶婶别紧张，这头角狼很有灵性，不会随意伤人，你们不用害怕。”见实艳母女一时紧张到无法言语，阿修只得出声安慰她们，然后趋前向蹲在一旁，正整理医疗器材的医算师及
   </span>
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    菫芨问道：“前辈、姑娘好，请问我叔叔的伤势……”
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “他已经在休息了。不过是脊椎受了点伤，暂时活动不便，没什么大碍，休养一段时间就会恢复了。”菫芨抢在医算师的前面答道，医算师则是附和菫芨的话，朝阿修点了点头，示意要他放心。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    听见阿修说话，知道角狼不会带来危险的村民们，忐忑不安的心也踏实许多，但这样一来，就有人躲在人群中壮起胆气放话了。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “那奇怪的火就是这臭小子引起的，我当时就在旁边的田里，看得一清二楚。”一阵声音从人群中传了出来，而这句话也再度在人群里挑起了骚动，令人不难理解这些村民为何在被医算师包扎治疗后，迟迟不肯散去的主因，原来是等着要兴师问罪。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “这可奇了，姑娘我当时也在现场，怎么看的就跟你不一样，你确定火是他放的？”听不下去的菫芨，此刻挺身而出主持公道，这举动倒是令阿修感到讶异。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    受到医算师与菫芨救治的村民，面对菫芨的质疑，音量明显小了许多，也心虚许多，已没有方才的气势：“呃……我刚说漏了，是这臭小子的朋友引起的。”
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “你怎么知道他们是朋友？证据在哪？”菫芨的伶俐，感觉一点也不输刑娜，只是这么一想，阿修的心顿时又失落起来。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “还需要什么证据！放怪火的那家伙，整天黏着这臭小子在那说话，他们当然是朋友。”不服输的村民，继续硬拗着。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “我现在也在跟你说话啊，难道我们就是朋友吗？ ”充满正义感的菫芨继续说道，而异于往常的是，医算师只是在旁静静看着一切，并没有阻止菫芨的打算。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “……”哑口无言的村民，面对其他想了解真相的村民质疑的眼神，再也说不出话来。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “姑娘我在此对天发誓，放怪火的另有其人，但把火灭掉的，却是眼前你们想怪罪的这个人！”菫芨铿锵有力的言语，再次在村民之间引起骚动。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    毕竟这怪火来得莫名，去得也突然，而眼前这素昧平生的姑娘居然就这样站出来为阿修背书发誓，令人不得不信。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “我不管火是不是这臭小子灭的，整件事绝对因他而起。这臭小子，从以前就是衰神，有他在的地方，就是霉运不断，把他赶出去……”先前理亏的村民，这段话倒是得到许多村民的回应，要求阿修离开的声音此起彼落。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    原本义正词严的菫芨，没想到村民一句话居然让她先前的仗义直言付诸流水，不由得感到十分沮丧。在旁的医算师见状，则是不舍的拍拍她的肩膀，似是为她方才的言行感到骄傲，眼神充满了嘉许。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    看见这情况，心情本来就低落的阿修也懒得理睬这些村民，走进破烂的农舍角落，然后抱了一团稻草杆摆在身上当棉被后，就自顾自的躺下来休息。因为真的很累，心很累，所以也懒得理睬这些村民，只想好好休息。或许一觉醒来，会发现只是做了个恶梦。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    虽然村民普遍对阿修不抱持好感，却也没人知道他平常是怎么过日子的，现在透过破烂的农舍，闻着飘出来的沉闷味道，窥见阿修令人难以想像的生活方式后，嘈杂的声音顿时安静下来。
   </span>
  </span>
 </p>

    无言的村民渐次离去。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    医算师爷俩在安慰实艳母子，告知药若用尽，可到惜古镇找他们拿，并提醒她们毋需畏惧角狼后，也随即离开这个已经残破不堪的家。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    再次睁开眼的阿修，发现天空已是一片昏黄，感到腹中传来的饥饿感，阿修跑到农舍外，倚着角狼，打开木餐盒。或许是放了快一天的缘故，餐盒充满一股淡淡的酸味。餐盒的菜，则被刑娜摆出了极具创意的笑脸图案。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    感到心酸的阿修，就着淡淡的酸味，缓慢的吃着。每咬一口，总觉得牙齿也跟着酸了起来；每吞一口，喉咙及胸中也充满一股酸涩，仿佛食物不是进到胃中，而是跑到胸中。不时还有泪水滴下，在食物表面化开。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    照理说，应该变得酸酸咸咸的食物，味蕾却像是消失一般，尝不出任何味道。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    这可能是从阿修有记忆以来，吃过最难吃的一餐了，难咬、难吞、无味；却也是有记忆以来，最不想放弃的一餐，虽然冷了、酸了，每一口却又洋溢着无尽的温暖，令人百感交集。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    就连打算叫阿修进屋用餐的实艳与实丽，看见这一幕都自动闪避。她们虽然不明所以，但心中却也明白，不该去打扰阿修这一餐。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    吃完之后，阿修又跑回农舍自顾自的睡了起来。然后被实艳跟实丽叫去用餐时，就拿着餐盒，尽可能把饭菜装成刑娜当时的微笑图案，再跑去倚着角狼，用完餐就又自顾跑去农舍，躺在稻草堆中，望着天空发呆。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    狼王的食物，则有同族前来接应。村人虽然害怕，但一来角狼也不伤害他们及牲畜，再则它们只是单纯来找狼王，所以也不敢轻举妄动，深怕一个不小心结下梁子，就可能让近几年平安无事的村子，再次面临角狼的威胁。
   </span>
  </span>
 </p>
 <center>
  <ins class="adsbygoogle" data-ad-client="ca-pub-1276641434651360" data-ad-format="fluid" data-ad-layout="in-article" data-ad-slot="3646767294" style="display:block; text-align:center;">
  </ins>
 </center>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    至于实善，从平时与实艳及实丽的对话中，阿修可以听出没什么大碍。由于实善似乎不想看到阿修，加上两人在平时也没什么良性互动，所以阿修也没有主动去问候，反正病人需要的是休息，能避则避。至少阿修是这样想的。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    而实善的表现，也丝毫没有想理睬阿修的意思，虽然实艳曾试探性的询问，是否要让阿修来看望他，都遭到严词拒绝，听起来似乎也跟村民一样，怪罪阿修，认为自己现在的情况都是阿修造成的。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    整体看下来，实家一如以往，实善对阿修的态度，依然十分不友善；但与以往不同的是，实善也不叫阿修下田了，也许两个人都需要时间恢复吧，无论身心。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    实艳对阿修态度则始终如一，对阿修十分照顾。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    改变最大的，要算是实丽了，或许是被阿修救了一命的缘故，原本轻蔑的神色，早已被感激之情取代，不但在用餐时会把自己喜欢吃的菜多夹一些给阿修，来叫阿修时，也会以“哥”来尊称他了。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    只是这些也只限于生活琐事，毕竟两人零互动的时间已久，加上阿修大多时间都自己独处，实丽一下也不知该如何弭平代沟，但至少她在尽力拉近两个人的差距，而且效果也出来了。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    同上次村庄遭到暴风雨摧残一样，这次实善虽然卧病在床，依然出资帮许多村人度过难关，重建家园。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    虽然如此，忌惮角狼的村民，也不敢跑来实家。就重建实家的工匠，要适应角狼的存在也花了几天的时间。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    或许正因为角狼让人避之唯恐不及的存在，实家的重建非常快速，就算没有之前的气派，至少生活机能一应俱全。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    可惜全村的生活机能虽快速恢复正常，但村民对阿修的怨恨却未曾消减，反而还更加深化，只是碍于实家的贡献，以及角狼的存在，隐忍不发。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    一天早上，实艳亲自出门去镇上，打算帮实善拿新药的同时，顺便采买一些家用。刚出发不久，只见实丽从屋里跑过来找阿修：“哥！爹请你赶快过去一趟。”
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    觉得意外的阿修，只得跟着实丽进屋，刚进实善养病的房间里，实善就叮咛实丽：“你去屋外玩，如果看见娘回来了，记得赶快进来告诉爹哦！”
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “孩儿知道了。”实丽点了点头，就跑出去屋外玩了。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    “叔！你看来恢复得挺好，气色不错。”这应该是自那晚受伤后，两人第一次正式见面，所以阿修也不知该说什么，只得简单寒喧问候几句。
   </span>
  </span>
 </p>
 <p dir="ltr">
  <span style="font-size:18px">
   <span data-darkreader-inline-bgcolor="" data-darkreader-inline-color="" style="--darkreader-inline-bgcolor:transparent; --darkreader-inline-color:#e8e6e3; background-color:transparent; color:#000000; font-family:Arial">
    但是令阿修感到意外的是，实善看着他，脸上第一次出现笑容，虽然这笑容勉强中夹杂着尴尬，却不会令人感到不舒服……
   </span>
  </span>
 </p>
 <p dir="ltr">
