 <p>
  【看中国2022年9月30日讯】
  <ok>
   飓风“菲奥娜”
  </ok>
  上周末肆虐
  <span href="https://www.secretchina.com/news/gb/tag/加拿大" target="_blank">
   加拿大
  </span>
  的大西洋沿岸四省和魁省东部，造成严重损失。总理特鲁多本周先后访问爱德华王子岛和纽省灾区，国防部派出数百名军人协助当地救灾，这次飓风的
  <ok>
   保险
  </ok>
  索赔可能达到7亿加元，但风暴潮造成的洪水损失将无法获得赔偿。
  <br>
   <br>
    在纽省巴斯克港，一名73岁女性被海浪卷走后丧生，新省也有一名81岁男子溺亡。在爱德华王子岛，“数千”棵树倒在电线上，电力抢修恢复较慢，周二下午仍有6万多爱德华王子岛居民断电。在新省，仍有数千户断电，很多居民仍不能打手机。截至周三上午，新省仍有超过10万户家庭和企业停电。
    <br>
     <br>
      爱德华王子岛的很多奶牛场和马铃薯储存设施被飓风摧毁，特鲁多与该省长金（Dennis King）讨论了灾后恢复面临的挑战，金要求联邦立即为遭受重大损失的当地渔业农业提供援助，特鲁多承诺联邦政府将提供一切支持。纽省政府周三宣布提供3000万加元的援助，每个在9月底前仍无法返家的家庭可获1万加元。
      <br>
       <br>
        国防部长阿南德（Anita Anand）说,如当地需要，已准备好增派军人救灾。
        <br>
         <br>
          联邦政府间事务部长勒布朗（Dominic LeBlanc）正在和加东地区省政府沟通保险问题。在极端天气频繁的今天，保险公司越来越不愿为易受灾地区提供服务，或保费过高让普通人望而却步。
          <br>
           <br/>
           加拿大应急准备部长比尔．布莱尔（Bill Blair）9月25日在新闻会上表示，联邦政府将与各省协调，向大西洋沿岸地区没有保险的人提供财政援助。专家表示，联邦为洪水保险提供资金是当务之急，这将有效地支持保险提供商。
           <br/>
           <br/>
           加拿大保险局副总裁克雷格．斯图尔特（Craig Stewart）告诉《环球邮报》，该局8月估计，80万房主居住在风险非常高的地区，而私人保险公司，要么不提供海洋洪灾保险，要么这类保险非常昂贵，房主难以承担。
           <br/>
           <br/>
           特鲁多说，联邦政府承诺匹配加拿大红十字会筹集的1000万加元捐款。该红十字会一直在支付临时住宿费用，并为人们提供杂货和家庭用品。
           <br/>
           <br/>
           捐款者可发送简讯，向20222输入FIONA，可捐赠10元，或在红十字会网站或致电1-800-418-1111。
           <br/>
           <br/>
           不过，其它慈善组织认为，政府不该只与一家慈善机构匹配。
           <br/>
           <br/>
           除红十字会外，人们还可以通过其它慈善机构捐款。
