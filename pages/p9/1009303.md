 <p>
  【看中国2022年6月16日讯】（看中国记者肖然综合报导）近日，“反
  <ok>
   奴工法案
  </ok>
  ”在
  <ok>
   加拿大
  </ok>
  联邦
  <ok>
   众议院
  </ok>
  通过二读。还有议员提出
  <ok>
   国际人权法案
  </ok>
  ，向人权迫害者追责。
  <br>
   <br>
    《打击供应链中强迫劳动和童工法案》（Bill S-211）规定，政府机构及私营公司有责任报告其供应链涉及奴工或童工生产的行为。法案还设置了检查制度，并禁止进口这类商品。
    <br>
     <br>
      加拿大魁北克政团（Bloc Québécois）国会议员亚历克西斯．布鲁内尔-杜斯佩（Alexis Brunelle-Duceppe）表示，“我们要确保在加拿大的商店货架上，没有奴工产品。”
      <br>
       <br>
        自由党国会议员约翰．麦卡（John McKay）说：“你和我都不希望成为不知情的奴工支持者。”
        <br>
         <br>
          加拿大保守党众议员吉尼斯（
          <span href="https://www.theepochtimes.com/t-garnett-genuis" target="_blank">
           Garnett Genuis
          </span>
          ）呼吁，加拿大使用《马格尼茨基法案》制裁在中国境内发生的侵犯人权行为。
          <br>
           <br>
            据加拿大世界宣明会（World Vision Canada）2016年的统计数据，至少有1200家加拿大公司进口的价值340亿美元的商品，涉嫌奴工或童工制造。调查资料显示，中国的奴工产业至少包括681家监狱企业，奴工受害者包括新疆维吾尔人、维权律师和访民、法轮功学员等。
            <br/>
            <br/>
            追查迫害法轮功国际组织主席汪志远表示，“（奴工）每天工作12到19个小时，有时几天几夜不能合眼，很多情况下还遭受酷刑。”
            <br/>
            <br/>
            多伦多法轮功学员潘学峰透露，她曾被关押的山西新店劳教所与商人勾结，把法轮功学员当作免费获取暴利的工具。90多人坐在80平方的车间里，空气中弥漫着塑料味、纸屑味等刺鼻的味道。很多人出现过敏、头晕、恶心、呕吐等现象。劳教所经常强迫所有人连夜赶工，生病也不例外，从早6点一直到晚10点半以后，有时每天只许和衣睡两小时。
            <br/>
            <br/>
            华尔街日报报导，美国海关已准备好于6月21日起执行《防止强迫维吾尔人劳动法》，禁止新疆的任何强迫劳动的产品进入美国。届时美国进口商需要出示产品不涉及新疆奴工的证书。
            <br/>
            <br/>
            6月9日，欧洲议会以压倒性票数通过了类似决议。
            <br/>
            <br/>
            另外，保守党国会议员劳伦斯（ Philip Lawrence）于6月13日还提出国际人权法案，规定外长将至少每年一次公布外国政府释放的良心犯名单，并要求对涉嫌人权侵犯者使用马格尼茨基式的制裁；禁止向参与种族灭绝行为的具有外国影响的实体续签广播许可证。

 <p>
  保守党国会议员吉尼斯指出，根据该
  <span href="https://www.secretchina.com/news/gb/tag/法案" target="_blank">
   法案
  </span>
  ，外长将被要求向议会提交一份年度报告，说明该部为促进这一事业所做的工作。当议会委员会要求对个人实施马格尼茨基制裁时，外长还必须在40天内向议会作出回应。
  <br/>
  <br/>
  “马格尼茨基法案是保守党提出的制裁工具，它已通过议会，但自由党一直非常不愿意使用它，”吉尼斯说，“政府在这方面保留了自由裁量权。这项法案为议会创造了重要的问责机制，以便能推动政府在这些问题上取得进展。”
