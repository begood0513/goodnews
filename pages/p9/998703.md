  <br>
   <br>
    根据美国之音报导，2月22日，中国外长王毅在与美国国务卿布林肯通电话之时，呼吁各方应该保持克制，且通过对话谈判来缓和事态，以化解分歧。可是他同时又重申，任何国家的合理安全关切皆应该得到尊重，不指名地对俄罗斯向美国及其北约寻求安全保障之要求提供背书。

      <br>
       <br>
        尽管王毅强调了中国在乌克兰问题之立场是一贯的，可是他后续表达的意思却有相互矛盾：任何国家的合理安全关切皆应得到尊重，并暗示普京出兵乌克兰有理，那是因美国和北约忽略了俄国的安全关切；不过联合国宪章的宗旨与原则应当得到维护，却又再强调依据联合国宪章的宗旨与原则，乌克兰的主权及领土完整都应该得到维持。

 <p>
  21日晚，中国常驻联合国代表张军在安理会所召开的乌克兰局势紧急会议上，他呼吁各方应保持克制，以避免任何可能会加剧紧张局势之行动，且继续开展对话协商，并在平等与相互尊重之基础上，来寻求能够解决彼此关切的合理方案。
  <br>
   <br>
    张军说，乌克兰局势发展至目前的状况，是一系列复杂的因素共同作用结果。北京一贯按照事情本身之是非曲直来决定自身立场，且主张各国应根据联合国宪章之宗旨及原则，以和平解决国际争端。
    <br>
     <br/>
     《华盛顿邮报》表示，中国两位高级外交官针对乌克兰局势的发言，已充分反映了在中国试图和俄罗斯进一步加强关系的大背景之下，现今乌克兰危机已让北京左右为难了，甚至还出现立场相互矛盾之尴尬状况。
     <br/>
     <br/>
     对北京来说，不干涉内政及领土主权完整是该国外交政策的核心内容。此亦为北京在试图强化对香港、西藏与新疆之管控或者声称拥有台湾主权而被批评时经常据以用作回应之国际法原则。
     <br/>
     <br/>
     如果中国支持俄国入侵乌克兰，那么必将进一步伤害其与西方国家已恶化的关系；然而北京又希望能够维护它与莫斯科之间越来越密切的关系，以利在反制美国与西方的所谓“打压”之时可以背靠背抱团取暖。
     <br/>
     <br/>
     中国希望维护其与俄罗斯之间的关系，同时又坚持它的原则，以避免伤害和美国与欧盟的关系。对此，美国德国马歇尔基金会的亚洲部主任葛来仪（Bonnie Glaser）对华盛顿邮报说。处理此次危机可能将是习近平所面对的最艰难之外交挑战。
     <br/>
     <br/>
     北京冬奥会在开幕之时，
     <span href="https://zh.wikipedia.org/wiki/%E4%B9%A0%E8%BF%91%E5%B9%B3">
      习近平
     </span>
     于北京会晤了
     <span href="https://zh.wikipedia.org/wiki/%E5%BC%97%E6%8B%89%E5%9F%BA%E7%B1%B3%E5%B0%94%C2%B7%E6%99%AE%E4%BA%AC">
      普京
     </span>
     ，双方签署一项联合声明，其强调两国之间的战略伙伴关系合作并没有禁区。随着乌克兰紧张情势之升级，北京就一再附和莫斯科，还指责美国与北约故意制造出紧张气氛，而忽略了莫斯科合理之安全关切。
     <br/>
     <br/>
     19日，中国外长王毅在慕尼克安全会议上用视频致辞之时虽然曾经不指名地反对有人一味地推高紧张，来制造恐慌，甚至更渲染战争，可是他也表示，各国之主权、独立与领土完整，都应该得到尊重及维护。
     <br/>
     <br/>
     王毅说，这是国际关系之基本准则，其体现的是联合国宪章的宗旨，亦为中方一贯秉持之原则立场，而对乌克兰也不例外。
     <br/>
     <br/>
     中国官媒于报导乌克兰最新局势之时，仍然会把局势的恶化，归咎于美国之渲染与挑衅。环球时报前总编辑胡锡进在星期二于一篇贴文上，将普京承认了乌克兰东部的两国俄语区为独立国家，描述成打破僵局的举措。
     <br/>
     <br/>
     胡锡进说，通过这些具体行动，他展现了俄罗斯的战略决心。
    </br>
   </br>
  </br>
 </p>
 <p>
  22日，新华社记者明金维在其微信公众号“明叔杂谈”上贴文称，中国于
  <span href="https://www.secretchina.com/news/gb/tag/乌克兰" target="_blank">
   乌克兰
  </span>
  危机上应冷静观察，并妥善处理好和当事各方之关系，以免惹祸上身，而且乌克兰危机是考验中俄之间关系的一个重要时刻。
  <br/>
  <br/>
  明金维说，俄罗斯最大的担忧是，北京因为不敢得罪美国，当在俄方最需要中国之时，中国却不能对俄国表示理解，且提供一定的支援，甚至采取“骑墙”、“摇摆不定”之做法，在乌克兰危机问题上，北京应对莫斯科的合理安全关切表示理解。同时在此基础上，中国应继续采取劝和促谈之姿态。
 </p>
 <p>
  明金维建议北京于在公开场合上，说一些中性略偏向
  <span href="https://www.secretchina.com/news/gb/tag/俄罗斯" target="_blank">
   俄罗斯
  </span>
  的话，能够让俄罗斯多得一些“面子”；且时又尽量不要直接与美国及欧盟的立场对撞，甚至不要直接采取实质行动，来伤害美国与欧盟的“里子”。我们应清楚，未来，在中国彻底解决台湾问题之时，在中国和美国进行各种战略博弈时，我们亦需要来自于俄罗斯的理解，以及一定程度的支持。
  <br/>
  <br/>
  当年美国与中国一起联手抗击在全球咄咄逼人之苏联；如今却是中俄互相抱团取暖，以抗衡美欧。世事的变化总是出人预料之外。
