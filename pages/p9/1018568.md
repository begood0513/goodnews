 <p>
  【看中国2022年10月8日讯】
  <span href="https://www.secretchina.com/news/gb/tag/加拿大" target="_blank">
   加拿大
  </span>
  连锁超市Loblaw由五辆无人驾驶卡车组成的车队成为加拿大第一批自动驾驶商用送货车。
  <br>
   <br>
    加媒《国家邮报》报导，8月的一个晚上，一辆紫白相间的卡车从郊区的一家超市开出，行驶7公里到达位于多伦多以西的安省宾顿市的Loblaw总部。Loblaw称这是历史性的旅程，因为第一次没有人坐在驾驶座上。
    <br>
     <br>
      约两年半以来，Loblaw一直在与硅谷的自动驾驶技术公司Gatik合作，运营由五辆由摄像头、激光传感器和声纳引导的无人驾驶卡车组成的车队。这些卡车已经完成了15万次行程，没有出现安全问题或事故。测试期间，总有一位安全驾驶员坐在客座上，旁边是一个红色的橘子大小的按钮，称为紧急停止或“终止按钮”。Loblaw说这是加拿大首创的自动商业送货车。
      <br>
       <br>
        Gatik执行长高塔姆纳朗（Gautam Narang）表示，客座上有人不是必需的，而只是对当地紧急服务部门的礼貌。他说：“他们希望有人与之互动，以防车辆被拦下”。
        <br>
         <br>
          但卡车为此配备了扬声器，以备发生事故时，Gatik的工作人员可与警察或其他司机进行远程通讯。在安省，法律并不要求卡车上必须有人，省政府正在开展一项试点计划，允许某些自动驾驶汽车在没有司机的情况下行驶。
          <br>
           <br/>
           Gatik：取代司机是我们行业的宗旨
           <br/>
           <br/>
           Gatik执行长高塔姆纳朗表示，“我们能够在不需要司机的情况下进行商业活动。”
           <br/>
           <br/>
           Gatik根据卡车的数量和每小时的使用量向Loblaw收取年费，他相信自动驾驶卡车在五年内可达到比普通卡车便宜30%。
           <br/>
           <br/>
           新车队专注于物流中所谓的“中间一英里”：将在线订单从履行中心运送到商店和取货地点，而不是长途旅行将产品从制造商运送到Loblaw更大的配送仓库或客户家中。
           <br/>
           <br/>
           Loblaw和Gatik选择了中间一英里，因为路线短且可预测，因此卡车可以反复行驶在相同的道路上，收集交通模式和天气数据，以帮助指导未来的行程。
           <br/>
           <br/>
           Loblaw首席技术和分析官马克威尔（David Markwell）表示，自动驾驶卡车的主要好处是可在必要时连续24小时频繁运行，无需停下来让司机休息。
