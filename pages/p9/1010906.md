  【看中国2022年7月5日讯】（看中国记者杨天资编译/综合报导）一桩诉讼曝光科技巨擘Google Developers Studio（简称GDS）部门被

 <h4>
  一桩诉讼曝光
  <span href="https://www.secretchina.com/news/gb/tag/Google" target="_blank">
   Google
  </span>
  <ok>
   遭邪教渗透!
  </ok>
 </h4>
 <p>
  Google Developers Studio部门的一位总监，同时也是邪教“朋友联谊会（Fellowship of Friends）”的资深成员之一，将自己的教友安插进团队，给他们开启升职加薪快车道。不仅如此，这位总监还利用自己职权，用Google的钱来支付给自己的妻子和两个儿子。甚至在对外的各种活动，他还安插教友作为供应商，明明存在严重的利益冲突，却从未向Google如实汇报。
  <br>
   <br>
    一切本来都在暗中进行，直到该部门一位外包约聘工凯文·洛依德 (Kevin Lloyd) 发现了其中端倪，他刚准备上报 ，就突然被所属的外包公司开除。 一怒之下，这位前员工以职场歧视、非法终止合同、职场报复等为由，将Google 和外包公司告上了法庭。因此整个事件才曝光。
   </br>
  </br>
 </p>
 <p>
  美国《纽约时报》报导称，34岁凯文指控，他于2017年受雇为“Google Developer Studio”部门员工，主管彼得（Peter Lubbers）长期作为该
  <span href="https://www.secretchina.com/news/gb/tag/邪教" target="_blank">
   邪教
  </span>
  成员，他开始发现彼得快速升迁指定的邪教成员，且部门25人当中就有12人信教。
 </p>
 <p>
  不仅如此，彼得更引荐两位儿子大赚Google财，一位作为DJ出席公司活动、一位则是以自由工作者接部门的专案，甚至让妻子作为工作人员出席Google活动赚取工资。
 </p>
 <p>
  凯文·洛依德 (Kevin Lloyd)先生声称他去年被解雇是因为他抱怨宗教教派的影响。他的诉讼中还提到了将凯文·洛依德 (Kevin Lloyd)作为承包商派遣到谷歌的公司 Advanced Systems Group(简称 ASG)。大多数 Google Developer Studio 都通过 ASG 作为承包商加入团队，其中包括 “
  <span href="https://www.secretchina.com/news/gb/tag/朋友联谊会" target="_blank">
   朋友联谊会
  </span>
  （Fellowship of Friends）”的许多成员。
 </p>
 <p>
  凯文·洛依德 (Kevin Lloyd)于 2021年8月向加州高等法院提起诉讼，指控谷歌和 ASG 违反了加州保护工人免受歧视的就业法。
 </p>

 <h4>
  <ok>
   教友升职开快车道
  </ok>
 </h4>
 <p>
  综合《纽约时报》报导：1990年代，荷兰人彼得 Peter Lubbers 移民来了美国。为了拓展工作机会，当时只会做些木工、修理院墙的他，做了一个重要的决定：自学程式。
  <br>
   <br>
    2012年，他成功通过了Google面试，加入了这家公司。根据他的 LinkedIn 页面，他于2014年在Google内部参与创办了 Google Developers Studio，一个专门协助公司其它团队制作产品宣传片的团队。
    <br>
     <br>
      随着Google业务的扩张，各种内外部通路对于影音的需求与日俱增，GDS 团队也不断扩大，彼得也升到了总监 (Director) 级别。一切似乎都进展得很顺利。
      <br>
       <br>
        然而，根据GDS 的一位前员工，在2021年2月被开除的制片凯文·洛依德 (Kevin Lloyd)的说法，彼得其实是一个邪教组织渗透进Google总部、已知的最高级别员工。
        <br>
         <br>
          揭发者 凯文·洛依德 (Kevin Lloyd)在2017年透过外包公司 Advanced Systems Group（简称ASG）来到Google，在 GDS 部门上班。 他的工作是制作影片，服务的团队包括Google的 Firebase DevRel、Flutter、TensorFlow 等。
          <br/>
          <br/>
          在一次闲聊中，他的直属上司 Gabe Pannell 告诉凯文，自己和团队老大彼得是一起长大的朋友。
          <br/>
          <br/>
          在那之后，凯文和很多新来团队的同事打交道的时候，都惊讶地发现，这帮人的老家怎么全都在同一个地方：加州的 Oregon House 地区。
          <br/>
          <br/>
          凯文开始感觉到异样，觉得这绝对不是巧合。更别提有一位同事告诉他，Oregon House 是某个“邪教”的聚集地，这更加让他感觉到事态不对劲。
          <br/>
          <br/>
          他开始搜寻和 Oregon House 有关的资料，果不其然就找到了“朋友联谊会”(the Fellowship of Friends)，一个号称崇拜高雅艺术，希望开启全新能力，以期在地球毁灭之时能够存活下来，成为人类火种的诡异教派。
          <br/>
          <br/>
          凯文发现，在自己任职期间的25位长期同事当中，有至少12人可以确定是朋友联谊会的成员。并且，很多教友最初都是与他一样，透过 ASG 的外包途径进入到Google的。
          <br/>
          <br/>
          这些进入GDS的同事，有些甚至与这些职位所需的条件完全不符。比如，对于本地化相关的业务，职位通常会要求候选人了解多种语言，然而 GDS 本地化负责人 Hillary Pierce却只会说英语；另一位教友 Sean Searle 之前是搞园艺的，却因为跟彼得熟络，被招募到 GDS 担任计划经理，尽管没有任何计划管理经验。
          <br/>
          <br/>
          凯文感觉事情不对，必须要跟公司举报，但是自己又对法律不了解，有点担心如果站出来质疑的话，会被当成宗教歧视，所以就先忍了下来。
          <br/>
          <br/>
          但是，接下来的事情，让他完全出离愤怒了：这些教友同事没用多久，都成功转职为Google全职员工了，自己的雇佣关系却一直留在外包公司。
          <br/>
          <br/>
          凯文在诉状中表示，自己工作勤勤恳恳，也收到了长期合作的 Firebase、Flutter 等团队协作者的表扬，在各种公开场合也经常被称为"Googler"。
          <br/>
          <br/>
          自己在公司工作三年多时间，却只升了一次职位（加了两次薪），而且雇佣关系一直在外包公司 ASG；与此同时，某些其它同事却能不到一两年就转职成功。
          <br/>
          <br/>
          凯文刚加入公司的时候向刚才提到的 Gabe Pannell 汇报——这个关系本身就比较奇怪，因为据凯文观察，自己和 Pannell 做的工作是完全一模一样的。
          <br/>
          <br/>
          离谱的还在后面：Pannell 来Google上班之后一路开挂，只用了六个月的时间就从“制片人”、“高级制片人”一路升职为“执行制片人”，并且成功转职为Google全职员工。然后没过多久，到了2019年，他又再次被提拔为“执行制片总经理”，在 GDS 内部一人之下，万人之上。
          <br/>
          <br/>
          起诉书中还提到， Pannell 升职如此之快，各种糟糕行为还被老板容忍？原来在私底下，Pannell 和彼得其实是关系亲密的教友。他们俩和朋友联谊会的教主 Burton 曾经出现在同一张照片里。
