 <p>
  【看中国2022年5月8日讯】4月25日是
  <ok>
   朝鲜
  </ok>
  人民革命军建军日，该国举行了历届最大的阅兵式。在
  <ok>
   <span href="https://www.secretchina.com/news/gb/tag/金正恩" target="_blank">
    金正恩
   </span>
  </ok>
  执政10周年期间举行的第2次
  <ok>
   阅兵
  </ok>
  式上，其重点放在金委员长的偶像化与拥核国家的
  <ok>
   核武
  </ok>
  力。这次出席阅兵式的平壤市民及参与组织阅兵式的党干部们会怎么评价此次的阅兵式呢？让人震惊的是，金正恩夫人
  <ok>
   李雪主
  </ok>
  坐在比元首更高之位置上。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  根据韩媒DailyNK的报导，在4月25日深夜所举行的阅兵式上，平壤市民们最为关注的就是
  <span href="https://www.secretchina.com/news/gb/tag/李雪主" target="_blank">
   李雪主
  </span>
  。平壤市民们指称，让人相当震惊的是，李雪主竟然坐在比元首更高的位置上。
 </p>
 <p>
  而最高人民委员会常任委员长的崔龙海、总理金德勋、党组织书记赵勇远等也都与李雪主一起站于主席台上，但所拍到的镜头是李雪主好像站在金委员长的头上。平壤市民们就说，李女士坐在想像不到之位置上。
  <br>
   <br>
    当时李雪主穿了与金委员长一样的白色套装，而照明也集中在她身上，因此百姓们感叹李雪主之地位提高了。
    <br>
     <br>
      综合
      <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
       朝鲜
      </span>
      内部的采访资料显示，最近朝鲜政府正式开展了李雪主之偶像化工作。
      <br>
       <br>
        据DailyNK朝鲜内部的高层消息人士表示，党中央委员会下发关于李雪主的思想教育资料，已经开展了李雪主偶像化工作。
        <br>
         <br>
          该资料编写在2月16日的金正日诞辰（也就是光明星节）80周年之际，其高度称赞李雪主传承了主体革命，且体现出着金正淑（金日成的夫人）的母亲形象。
          <br>
           <br/>
           据消息人士称，说李雪主传承了主体革命，主要是因为她生了儿子。
           <br/>
           <br/>
           春节的时候，在平壤万寿台艺术剧场所举行春节演之时，李雪主穿着朝鲜传统服装，所以也有人分析这是为了展开“金正淑化”。因为在朝鲜这个国度，红色上衣与黑色裙子是令人联想起了金正淑的象征性服装。
           <br/>
           <br/>
           韩国国家安保战略研究院的主任研究员金仁泰对DailyNK表示，可以说通过这次阅兵式来完成金正恩偶像化之第一步，有必要进行辅助性人物李雪主的偶像化了。
           <br/>
           <br/>
           金研究员指出，金正恩偶像化中最为明显的动向，就是开始宣称金正恩为父亲元首等开始了父亲化。而在将金正恩塑造为父亲之过程中，必须要让李雪主起到了母亲的象征性作用。亦即朝鲜最高领导人夫人的作用，就是为了加快领袖偶像化而起到的辅助作用。
           <br/>
           <br/>
           穿元着首服的金正恩旁边的朴正天与李炳哲戴着白色礼帽，而金委员长的服装亦受了人们的关注。在这次阅兵式上，金正恩穿着让人会联想起他爷爷金日成之白色元首服。
           <br/>
           <br/>
           这次金委员长穿白色元首服出席活动还是第一次。朝鲜为了炫耀金委员长“加强了核武装”之10年执政业绩，而特意让他穿上了白色元首服。
           <br/>
           <br/>
           朝鲜部队内部消息人士称，站于金委员长两旁的朴正天与李炳哲戴的白色军帽，是正规军服中没有的礼帽。也就是为了与金委员长的白色元首服进行搭配，而特意制作的礼帽。
           <br/>
           <br/>
           也有人认为金委员长所穿的元首服肩章，非常像“大元首”军衔，但并没有过拥戴金委员长为“大元首”之举措。
