    原订在昆明举行的联合国《生物多样性公约》缔约方大会第十五次会议(COP15)已被主席国中国推迟了四次。中共的防控措施已对其在国际舞台上展示外交和文化影响力形成阻碍。
    <br>
     <br>
      原定于2020年10月召开的这场会议预计就保护动植物和生态系统达成一项全球协议，在推迟三次后，代表们于2021年10月举行了虚拟形式为主的第一阶段会议，近200个国家承诺，到2030年扭转当前生物多样性丧失的趋势。
      <br>
       <br>
        5月，中方通知与会者，原定8月举行的第二次线下会议由于受疫情影响将推迟。
        <br>
         <br>
          同时，当上海等城市实施清零政策之际，中方推迟了9月杭州亚运会的时间，并放弃了2023年夏亚足联亚洲杯的举办权。
          <br>
           <br/>
           国际环境与发展研究所(International Institute for Environment and Development)高级研究员荷兰德（Ebony Holland）告诉华尔街日报：“在经历了长达数月、甚至数年的不确定性之后，确定这些重要会谈的时间地址是好消息。”
           <br/>
           <br/>
           虽然会议改在枫叶之国举行，但中国仍是COP15主席国，两国在邀请嘉宾出席高级别会谈方面可能有分歧。
           <br/>
           <br/>
           今年稍晚，在中共全国代表大会上，
           <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
            习近平
           </span>

           <br/>
           <br/>
           2月初，习近平在北京与俄罗斯总统普京(Vladimir Putin)会晤，称中俄友谊“没有上限”。
           <br/>
           <br/>
           外界预计中国严格的边境防疫措施直到2023年春天才会放宽。不过北京近日已开始放宽部分入境旅行要求，缩短入境旅客隔离期，并放宽工作和探亲签证方面的要求。
