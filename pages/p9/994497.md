 <p>
  【看中国2022年1月8日讯】（法广RFI）中国外交部发言人汪文斌今天针对北
  <ok>
   朝鲜
  </ok>
  以疫情等因素为由
  <ok>
   不参加北京冬奥
  </ok>
  说，北朝鲜方面由于“一些特殊原因”无法参加北京冬奥，中方表示“充分理解”并感谢北韩支持北京冬奥。朝鲜官方说，因“敌对势力的动作”和全球疫情大流行之故，北朝鲜无法参加这次冬季奥运，但会全力支持“中国同志们筹办精彩纷呈的奥运盛典”。
  <br>
   <br>
    据中央社今天报道说，针对朝鲜不参加
    <span href="https://zh.wikipedia.org/zh-hans/2022%E5%B9%B4%E5%86%AC%E5%AD%A3%E5%A5%A5%E6%9E%97%E5%8C%B9%E5%85%8B%E8%BF%90%E5%8A%A8%E4%BC%9A" target="_blank">
     北京冬奥
    </span>
    会议题，中国外交部下午举行例行记者会。发言人汪文斌在回答媒体对此问题提问时，回答说充分理解。
    <br>
     <br>
      据汪文斌说，这次朝鲜方面由于“一些特殊原因”无法参加北京冬奥，中方表示“充分理解”。中方愿与朝鲜方面继续加强包括体育在内的各领域交流合作，推动双方传统友好合作关系与时俱进，不断取得新的更大发展。
      <br>
       <br>
        据朝鲜官方的中央通信社（KCNA）今天称，朝鲜以“敌对势力”和全球疫情大流行为由表示无法参加下个月的北京冬奥会。北朝鲜奥委会和体育部已致函北京冬奥组委会等中方对口单位，表达尽管缺席却支持这场奥运之意。
        <br/>
        <br/>
        北朝鲜奥委会在信函中声称，因“敌对势力的动作”和全球疫情大流行之故，北朝鲜无法参加这次冬季奥运，但会全力支持“中国同志们筹办精彩纷呈的奥运盛典”。
        <br/>
        <br/>
        据法新社今天报道说，朝鲜国家媒体周五说，美国及其一些盟国对北京冬奥会的外交抵制是对奥林匹克精神的 "侮辱"。据官方的朝中社透露，朝鲜体育官员已向他们的中国同行发出一封信，谴责外交抵制，根据该信，美国、加拿大、澳大利亚和英国将不派官方代表前往北京，尽管他们的运动员将参加比赛。华盛顿以中国的人权状况作为这一抵制措施的理由。
        <br/>
        <br/>
        朝中社援引的这封信说：美国及其附庸势力在阻止奥运会成功开幕的努力中变得越来越不隐蔽。信中说，这些行动是对 "国际奥林匹克宪章精神的侮辱"。
        <br/>
        <br/>
        朝鲜是唯一一个自愿不参加去年东京夏季奥运会的国家，正式原因是担心新冠肺炎病毒疫情。作为回应，国际奥林匹克委员会（IOC）在去年9月暂停了朝鲜的活动，包括禁止朝鲜参加北京的冬季奥运会。国际奥委会当时解释说，平壤拒绝了它的所有建议，包括向朝鲜运动员提供疫苗，以安全参加东京奥运会。
        <br/>
        <br/>
        朝鲜官方的信中称朝鲜的缺席是新冠肺炎病毒大流行病和 "敌对势力 "行动的结果，但没有指明是哪些敌对势力，也没有提及国际奥委会实施的制裁。
        <br/>
        <br/>
        法新社说，朝鲜缺席北京奥运会将与它在2018年韩国平昌举行的上一届冬季奥运会上的高调表现形成鲜明对比。朝鲜领导人金正恩的妹妹出席了一些活动，韩国总统文在寅利用这个机会试图推动平壤和华盛顿之间的和解。
        <br/>
        <br/>
        中国官方对朝鲜的缺席称表示理解，并对平壤谴责外交抵制表示感谢。
