  <ok>

   2023-2025

  年移民水平计划，打算未来3年共接收约150万移民。
  <br>
   <br>
    具体目标是2023年欢迎46.5万名新移民；2024年48.5万名新移民；2025年上升到50万。
    <br>
     <br>
      2021年，枫叶之国迎来了超过40.5万移民，创下历史记录，今年计划接收43.2万移民。
      <br>
       <br>
        <ok>
         快速通道和省提名配额将上升
        </ok>
        <br>
         <br>
          大多数新的永久居民通过经济类计划移民，例如快速入境（EE）或省提名(PNPs)计划。
          <br>
           <br/>
           快速入境配额（主要申请人、配偶和受抚养人）在2023年为82,880，2024年为109,020人；2025年11.4万。
           <br/>
           <br/>
           PNP从2023-2025年配额分别升为105,500、11万和117,500人。
           <br/>
           <br/>
           <ok>
            父母祖父母移民配额更高
           </ok>
           <br/>
           <br/>
           家庭团聚是第二大移民类别。申请人由配偶、伴侣、子女或其他家庭成员担保获得永久居留权。
           <br/>
           <br/>
           加拿大将继续在配偶、伴侣和儿童计划下每年接收约8万名新移民。
           <br/>
           <br/>
           父母和祖父母（PGP）额度将在2023年增至28,500人；2024年达34,000人；2025年达36,000人。
           <br/>
           <br/>
           <ok>
            难民和人道主义类别配额减少
           </ok>
           <br/>
           <br/>
           加拿大正努力完成接收约4万阿富汗难民。
           <br/>
           <br/>
           2023年和2024年，难民类别额度将超过76,000人，2025年将降至72,750人。
           <br/>
           <br/>
           人道主义类别将从2023年近16,000人降至2025年的8,000人。
           <br/>
           <br/>
           <ok>
            加拿大移民策略
           </ok>
           <br/>
           <br/>
           现任自由党政府2015年上台前，加拿大每年迎来约26万移民，后来增至30万。
           <br/>
           <br/>
           在2020年新冠疫情爆发前达到34万。尽管受疫情影响，加拿大还是实现了2021年的移民目标，并打破记录，40.5万移民全部通过加拿大经验类和省提名计划（PNP）来实现。
           <br/>
           <br/>
           加拿大每名妇女生育1.4个孩子，是全球最低生育率国家之一。目前面临劳动力严重短缺问题，有近百万个职位空缺。到2030年，约900万人或近1/4的加拿大人将达到退休年龄，到时所有经济部门的劳动力都将告急。
           <br/>
           <br/>
           移民很快将成为加拿大人口和劳动力增长的唯一途径。
