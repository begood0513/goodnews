 <p>
  【看中国2022年9月14日讯】经过1000多天的“居家”抗疫，中国国家主席
  <ok>
   习近平
  </ok>
  终于“出关”，前往中亚地区访问。他将在周四（9月14日）访问哈萨克斯坦，随后前往
  <ok>
   乌兹别克斯坦
  </ok>
  参加
  <ok>
   上海合作组织
  </ok>
  成员国元首理事会会议。
  <br>
   <br>
    目前，中国疫情反扑，全国许多地方都处于不同等次的封控状态。各地政府告诫民众尽量避免出国旅行。
    <span href="https://zh.m.wikipedia.org/zh-hans/%E4%B9%A0%E8%BF%91%E5%B9%B3" target="_blank">
     习近平
    </span>
    在这样的情况下还坚持这次出访，观察人士认为，这显示出在中共当局眼中，控制中亚地区对中共来说有着非同一般的重要性。
    <br>
     <br>
      美国财经媒体彭博社在周三（9月13日）的报道中指出，经过这1000多天的居家抗疫，习近平可能发现中国在美国主导的世界秩序中已经陷入了越来越严重的孤立状态。他需要出去巩固和加强自己小的不能再小的朋友圈之内的团结。
      <br>
       <br>
        中亚地区是习近平推动的一带一路倡议中的关键地区，但经过病毒大流疫和俄乌战争，以及不断蔓延的债务问题，一带一路项目遭受了一连串的打击。习近平选择这个地区作为他出关后的第一个访问对象无疑是要为他的一带一路项目寻找新的出路。
        <br>
         <br>
          习近平与普京的会晤也是世界关注的一个焦点。一般预计，他们两人将在上海合作组织成员国元首理事会会议期间会面。这次峰会计划在9月15日至16日在乌兹别克斯坦古城撒马尔罕举行。
          <br>
           <br/>
           上合组织成立于2001年6月15日，其成员包括中国、俄罗斯、吉尔吉斯斯坦、塔吉克斯坦、哈萨克斯坦、乌兹别克斯坦、印度和巴基斯坦。
           <br/>
           <br/>
           曾几何时，习近平还在自豪地向世界表示，中俄友谊无上限，中俄合作无禁区。可一场俄乌战争让习近平和普京遭遇了始料未及的困境。俄罗斯因这场战争遭遇了西方的全面制裁，成为国际社会千夫所指的弃儿。中国也因拒绝谴责侵略者而饱受国际社会的指责，陷入空前孤立。
           <br/>
           <br/>
           近日，乌克兰在乌东战场上发起大反攻，以迅雷不及掩耳之势夺回了6000平方公里的国土，无疑给普京和习近平的会面平添了尴尬的气氛。
           <br/>
           <br/>
           两人这次的会面被媒体视为互有需要、各取所需的抱团取暖动作，同时也表明了习近平将北京与莫斯科之间的准盟友关系视为对抗美国的一个重要工具。
           <br/>
           <br/>
           习近平此次出访的另外一个看点是他要亲自推动他在今年四月提出的“全球安全倡议”。这个倡议针对美国与西方盟国强化安全联盟遏制中国威胁而提出“安全不可分割原则”，以及“反对把本国安全建立在他国不安全的基础之上”的做法。
           <br/>
           <br/>
           美国国务院发言人普赖斯指出，这个倡议似乎在模仿“我们从克里姆林宫听到的东西”，要为侵略乌克兰制造借口。
           <br/>
           <br/>
           美联社引用印度智库“观察者研究基金会”专家拉杰什瓦里·皮莱·拉贾戈帕兰（Rajeswari Pillai Rajagopalan）的话说，“这是中国赤裸裸地谋求亚洲霸权”，其目的“就是在与美国的大国竞争中扩大自己的利益。”
