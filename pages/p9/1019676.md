 <p>
  【看中国2022年10月21日讯】《环球邮报》披露，由于大流行期间
  <span href="https://www.secretchina.com/news/gb/tag/加拿大" target="_blank">
   加拿大
  </span>
  移民部大幅增加移民额度，导致积压严重，很多留学生等高技能人才因工签到期前无法获得永居身份而被迫离境。
  <br>
   <br>
    留学生普罗海特（Gaurav Purohit）就是其中一位，疫情期间他在多伦多一家全球知名金融服务公司工作了15个月，迄今都没等到移民部发出的移民申请邀请。因10月工签到期，他只能离开加拿大另谋工作。
    <br>
     <br>
      和普罗海特一样面临困境的还有上万的留学生。加拿大政府在大流行初期看到移民数量大幅减少，急忙追加移民配额吸引大批申请，最后因不能及时处理，去年9月暂停了工签持有人的移民申请，直到今年7月才恢复。
      <br>
       <br>
        温哥华移民律师拉龙德（Meika Lalonde）告诉《环球邮报》，疫情外加移民政策的变化，让许多原本是加拿大最需要的人才本可以在这里安居乐业，如今却不得不离开，这不是加拿大希望看到的局面。
        <br>
         <br>
          <ok>
           为完成配额造成积压
          </ok>
          <br>
           <br>
            2020年政府设定移民目标为34.1万，但因疫情只吸收了18万，2020年底，联邦将未来3年移民配额追加到120多万。
            <br/>
            <br/>
            为实现配额，去年2月移民部一反常规，向快速通道人才库的所有2.7万名候选人发出邀请，并将最低分从450分降至75分。同时，对临时外劳推出限时临时身份转永久身份（TR to PR）的新通道。两个临时措施导致移民申请积压严重超负荷，普罗海特的移民申请就是因这一政策变动而被耽误。
            <br/>
            <br/>
            移民部在回复《环球邮报》的质询时说，会根据人才库积压情况随时减少或暂停邀请。目前新的申请处理周期已恢复到半年。
            <br/>
            <br/>
            <ok>
             政府补漏 专家支招
            </ok>
            <br/>
            <br/>
            移民部去年1月也推出临时政策，允许毕业留学生工签到期后延期18个月，以帮助解决毕业留学生因疫情封锁找工难。
            <br/>
            <br/>
            据移民部数据，近5.2万名毕业留学生的工签获得延长。普罗海特去年4月工签得以延长18个月，期间成功找到工作，到今年10月全职工作时间满1年，原本在延长工签到期前应该可以拿到移民申请的邀请，但今年7月恢复人才库抽签时因积压严重，最低平均分数高达500分以上，导致他无法获邀。
            <br/>
            <br/>
            今年8月，移民部允许毕业留学生工签再延18个，但只针对初始工签于去年9月至今年12月期间到期者。普罗海特不符合这个条件而被迫回国。
            <br/>
            <br/>
            自2021年初以来，快速通道人才库的候选人已增至24万，专家估计其中不少人都面临类似窘境。
            <br/>
            <br/>
            拉龙德建议移民部为像普罗海特这样的人才延长工签，方便他们移民。并且政府在处理严重积压的问题上应提高透明度。
            <br/>
            <br/>
            9月，移民部宣布缩短移民申请处理时间，新招员工处理申请，免除加拿大境内申请人的体检要求等。但这些都帮不到工签即将到期的人士。
