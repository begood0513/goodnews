 <p>
  【看中国2022年7月12日讯】在
  <ok>
   澳大利亚
  </ok>
  新政府谋求与
  <ok>
   北京
  </ok>
  改善
  <ok>
   关系
  </ok>
  之际，阿尔巴尼斯总理星期一（7月11日）表示，不会回应澳中外长近日印尼会晤后中国对澳大利亚提出的所谓修复关系的四项要求，强调只对澳大利亚的国家利益做出回应。
  <br>
   <br>
    据澳大利亚媒体报道，
    <span href="https://zh.wikipedia.org/wiki/%E6%BE%B3%E5%A4%A7%E5%88%A9%E4%BA%9A" target="_blank">
     澳大利亚
    </span>
    总理阿尔巴尼斯还表示，澳大利亚将在可能的情况下与中国合作，澳大利亚希望与所有国家建立良好关系，但是“在必要时我们将捍卫澳大利亚的利益”。
    <br>
     <br>
      阿尔巴尼斯针对中国提出的四项要求做出评论之前，澳大利亚外长黄英贤（Penny Wong）和中国外交部长王毅7月8日在印尼巴厘岛参加二十国集团外长会议期间举行双方近三年来的首次会晤。
      <br>
       <br>
        阿尔巴尼斯称澳中外长会晤是向前迈进的富有建设性的一步。他说，澳大利亚没有改变在任何问题上的立场，将继续保持富有建设性。
        <span id="hideid" name="hideid" style="color:red;display:none;">
         <span href="https://www.secretchina.com">
         </span>
        </span>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  黄英贤早些时候将此次会晤描述为“稳定”关系的“重要第一步”。中国外交部长王毅则要求阿尔巴尼斯政府必须重塑对华的“正确认识”，坚持视中国为伙伴而不是对手，抓住当前契机，“减少负资产，积累正能量”。
  <br>
   <br/>
   王毅还指责
   <span href="https://www.secretchina.com/news/gb/tag/澳大利亚" target="_blank">
    澳大利亚
   </span>
   上届的莫里森政府是近年来澳中外交关系恶化的罪魁祸首，“执意把中国视为对手，甚至威胁”。
  </br>
 </p>
 <p>
  澳中外长会晤结束后，中国政府没有立即发布公告，在等待超过24小时后，才公布中方对此次会晤的表态声明。
  <br/>
  <br/>
  中国外交部的公告表示，王毅将近年来外交关系的异常恶化归咎于前澳大利亚政府。王毅还对新一届澳大利亚政府提出了四点基本要求，“坚持视中国为伙伴而不是对手”、“坚持求同存异的相处之道”、“坚持不针对也不受制于第三方”，以及“坚持构建积极务实社会民意基础”。
  <br/>
  <br/>
  有分析表示，第四点可能是要求澳大利亚领导人发表公开讲话要更加谨慎。最近的民意调查显示澳大利亚民众对
  <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
   习近平
  </span>
  治下的中国政府的看法急转直下。
  <br/>
  <br/>
  澳大利亚广播公司ABC近日报道，澳大利亚国立大学国家安全学院院长罗里·梅德卡夫（Rory Medcalf）说，“如果王毅确实为进一步接触设定了四个条件，那他可能会失望，除非中国自己改弦更张。”
  <br/>
  <br/>
  梅德卡夫教授还说，“中国从来都不是我们的竞争对手—从权力角度来看，几乎都不是对等关系—但它是一个主要的风险来源，我们政府对这一点的认识几乎不会改变，哪怕表达上会更为明智审慎。”
  <br/>
  <br/>
  澳大利亚外交部长黄英贤在与王毅会晤后表示，澳大利亚在保护其国家利益方面不会让步，但她称稳定与中国的关系“符合我们两国的利益”。
  <br/>
  <br/>
  黄英贤表示，在会谈中她谈到两名被关押在中国的澳大利亚人，包括作家兼商人杨恒均和记者成蕾，并要求中国解除对澳大利亚实施的额外关税。
  <br/>
  <br/>
  澳中两国外长上次举行会谈是在2019年9月。
  <br/>
  <br/>
  澳中关系在过去几年呈断崖式下降，两国在技术、贸易和军事等问题上进行了激烈交锋。专家分析说，两国外长的会谈不太可能解决澳中之间存在的实质性分歧。
  <br/>
  <br/>
  自澳大利亚工党政府5月上台后，澳中两国逐渐开启了高层之间的沟通渠道。今年6月，澳大利亚副总理兼国防部长马勒斯（Richard Marles）在新加坡参加香格里拉对话期间与中国国务委员兼国防部长魏凤和举行了会谈。
  <br/>
  <br/>
  在工党胜选后，中国总理李克强曾向阿尔巴尼斯发去贺电，称赞“上世纪七十年代，澳大利亚工党作出与中国建交的正确抉择”，并表示澳中关系的稳定发展“符合两国人民根本利益”。
  <br/>
  <br/>
  上述表态一改中国官方在过去几年对澳大利亚的严苛态度。在澳大利亚前总理莫里森呼吁国际社会对新冠病毒溯源进行独立调查后，北京感到愤怒，随即展开报复，对澳大利亚的葡萄酒、大麦、煤炭、牛肉、木材、糖、棉花和海鲜等十多项价值几百亿美元的关键产品课征高额关税或贸易禁令，并冻结了双边会谈，以回击澳大利亚在国内和国际社会上对中国影响力不断扩张而发出的预警。
