 <p>
  【看中国2022年2月22日讯】
  <ok>
   澳大利亚
  </ok>
  总理莫里森表示，上星期四用
  <ok>
   激光照射
  </ok>
  澳大利亚
  <ok>
   军机
  </ok>
  的中国海军舰艇当时十分靠近澳大利亚的海岸，在岸上就可能看到。他敦促中国对此展开全面调查。
  <br>
   <br>
    莫里森星期一（2月21日）表示，
    <span href="https://zh.wikipedia.org/zh/%E6%BE%B3%E5%A4%A7%E5%88%A9%E4%BA%9A" target="_blank">
     澳大利亚
    </span>
    政府认为这起事件“危险又轻率”，迄今没有得到中方的解释。
    <br>
     <br>
      澳大利亚政府星期六说，一艘中国海军舰艇在澳大利亚北方沿海用军事级别的激光照射了一架澳大利亚P-8A澳大利亚反潜巡逻机，对机上人员构成危险。
      <br>
       <br>
        中国说，澳大利亚队对事件的描述与事实不符，澳大利亚当时在中国舰艇附近投放了一枚能够探测潜艇的声纳浮标。
        <br/>
        <br/>
        澳大利亚国防部说，中国一艘导弹驱逐舰和一艘两栖船坞运输舰当时正穿越新几内亚和澳大利亚之间的阿拉弗拉海，随后通过了狭窄的托雷斯海峡（Torres Strait）。
        <br/>
        <br/>
        莫里森星期一对记者说，“民众可能从我们的大陆上就可以看见这艘舰艇，”澳大利亚已经通过外交和国防管道要求“对此事件展开全面调查。”
        <br/>
        <br/>
        莫里森假设说，如果一艘澳大利亚的护卫舰在台湾海峡用激光照射中国巡逻机，“你能想象北京对他们的反应吗？”
        <br/>
        <br/>
        中国国防部为其舰艇的行动辩护。中国国防部发言人谭克非星期一在国防部官方微薄贴文中说，“澳P-8反潜巡逻机抵我舰艇编队周边空域活动，最近距我舰仅4公里。”
        <br/>
        <br/>
        谭并说， “从我方舰艇拍摄的照片可以看出，澳机与我舰距离很近，还在我舰周围投放声呐浮标。这种恶意挑衅行为极易引发误解误判，给双方舰机及人员安全造成威胁。”
        <br/>
        <br/>
        谭称，“我们要求澳方立即停止类似挑衅性危险行动，停止对中方无端指责抹黑，以免影响两国两军关系大局。”
