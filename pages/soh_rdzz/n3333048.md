 <p class="content-info-datetype">
  <span class="content-info-date">
   【希望之声2019年11月13日】

  <span class="content-info-type">

 </p>
 <p class="p1">
  <span class="s1">
   近日，川普政府提议加大庇护申请者在收到庇护决定之前在美国得到工作许可的难度。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   据美联社的报道，周三，美国公民和移民服务部表示，拟议的规则将让庇护申请者等待工作许可的时间增至一年，并禁止所有非法越境者申请工作许可。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   该机构的代理主任肯
  </span>
  <span class="s2">
   ·
  </span>
  <span class="s1">
   库奇内利（
  </span>
  <span class="s2">
   Ken Cuccinelli
  </span>
  <span class="s1">
   ）在声明中说，这项新规则旨在阻拦那些没有资格获得庇护的移民申请庇护，并
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   重新恢复庇护系统的完整性，减少以获得工作许可为主要目的而提出的庇护申请
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   该提议是川普政府拟定的一系列政策中的最新措施，旨在阻止移民沿美墨边境寻求庇护，并限制进入美国的移民数量。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   美国政府机关和移民法院目前有数十万份移民庇护需要处理。有些申请是进入美国后提交，有些是到达机场后提交，还有的是在美墨边境提交的。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   目前，庇护申请者一旦案件待审超过六个月，便可申请在美国的工作许可。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   移民倡议者表示，这个拟议的规则可能对最贫穷和最脆弱的移民造成最大影响，并可能给他们的社区带来额外的负担。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   公众可以在明年
  </span>
  <span class="s2">
   1
  </span>
  <span class="s1">
   月
  </span>
  <span class="s2">
   13
  </span>
  <span class="s1">
   日之前对该拟议规则发表意见。
  </span>
 </p>
