<div class="content">
 <p>
  <span class="content-info-date">
   【希望之声2019年7月11日】
  </span>
  <span class="content-info-type">
   （本台记者季云综合编译）
  </span>
  7
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    月
   </span>
  </span>
  11
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    日下午，川普总统在白宫玫瑰园宣布，将立即发布行政令，利用联邦政府所有的数据库，彻底查清美国居民中的公民和非公民数量，川普称，此举将获得比
   </span>
  </span>
  2020
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    年人口普查更准确的数据。
   </span>
  </span>
 </p>
 <div class="widget ad-300x250 ad-ecf">
  <!-- ZW30 Post Embed 300x250 1 -->
  <ins class="adsbygoogle" data-ad-client="ca-pub-1519518652909441" data-ad-slot="9768754376" style="display:inline-block;width:300px;height:250px">
  </ins>
 </div>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    川普在抨击了民主党极左派企图“掩盖非法外国人的数字”后表示：“今天，我在此宣布，我们不会在确定美国人口的国籍状况这个问题上让步”。“我们一定会翻遍每块石头”，川普说。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    川普说，反对在
   </span>
  </span>
  2020
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    年人口普查问卷上增加国籍问题毫无意义，而目前在几个联邦法庭进行的案件令他们没有办法在
   </span>
  </span>
  2020年
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    人口普查问卷印刷之前解决这个问题。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    司法部长巴尔（
   </span>
  </span>
  Bill Barr
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    ）随后表示，总统行政令要求收集的资讯在确定选举人团的组成和国会议员的分摊中将会非常有用。“（收集到的）资讯可以用于数不清的目的。比如，目前有一个关于是否非法外国人应该计入分摊的争论
   </span>
  </span>
  …
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    ，我们会研究这个问题。”巴尔说。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   根据法律，
   <span lang="zh-CN">
    计入非法移民数量的人口普查人数据会用于确定未来
   </span>
  </span>
  10
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    年国会众议员的分配、各州在选举人团中的选举人数量，以及
   </span>
  </span>
  6750
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    亿元联邦预算的分配。人口普查局的专家曾表示，在普查问卷中加入国籍问题会打击非法移民的参与积极性，因而导致数据的失实，从而导致资金和政治力量的再分配离开移民聚居、民主党领先的许多城市。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    巴尔表示认同川普的观点：最高法院上个月的决定为在普查问卷上加入国籍问题提出了一个不可逾越的“后勤”障碍，而非法律障碍。政府已经开始了冗长、费钱的普查问卷印刷过程，这个问卷不含国籍问题。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    巴尔还抨击了媒体关于白宫将发布行政令、强行把国籍问题加入问卷的传言。说媒体“歇斯底里般地”猜测川普将单方面决定把国籍问题硬塞进问卷。“这根本没在考虑之内”，巴尔说。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    民主党及左派对川普的决定反应不一，有人认为他在撤退，也有人威胁要到法庭挑战行政令。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    参议院少数党领秀、民主党籍参议员舒默（
   </span>
  </span>
  Chuck Schumer
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    ）发表声明说，川普总统如此想恐吓有色人种社区，即便在法律和法庭阻止他的情况下，他还是要坚持他笨拙的办法”，“总统在加入误导性的国籍问题上的撤退早就该发生，这是民主和公平代表的胜利。每个人都应该计入，而且不应该有人受到总统和他的任性的行为的恐吓。”
   </span>
  </span>
 </p>
 <div>
 </div>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    川普当天已经在白宫表示了自己对该事的愤怒。“我们花了
   </span>
  </span>
  200
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    亿做这个普查，他们（普查员）走遍了每个家庭，走上走下，问左问右，几个马桶啊？几张床啊，几张桌子呀？房顶是用什么做的呀？唯一不能问的问题却是：你是美国公民吗？这是不是最疯狂的事？”川普说。
    <br/>
   </span>
  </span>
  <br/>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    司法部新闻发言人库佩克（
   </span>
  </span>
  Kerri Kupec
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    ）告诉福克斯新闻，川普的替代办法终结了国籍问题的法律争议。司法部并不同意最高法院的决定。而这个行政令所收集的数据对制定政策以及其它用途很关键。因此司法部将尽快告知法院：国籍问题将不会包括在
   </span>
  </span>
  2020
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    年的普查问卷中。
    <br/>
   </span>
  </span>
  <br/>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    其实，上个月最高法院的决定并没说川普政府不可以问这个问题，他们只是明确要求川普政府提供一个合理的原因。因为法庭不认同司法部当时给出的原因：“更好执行投票权法案”。
    <br/>
   </span>
  </span>
  <br/>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    曾在法庭挑战过这个国籍问题的民权组织“律师民权委员会”的总裁克拉克（
   </span>
  </span>
  Kristen Clarke
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    ）告诉福克斯新闻，她“严重关切本届政府对国籍相关的资讯的明显不恰当的、歧视性及违宪的追求”，“根本没有法律基础允许川普总统绕过最高法院不许加入国籍问题的决定。”克拉克表示将继续抗争。
   </span>
  </span>
 </p>
 <p>
  <span style="font-family: Lucida Sans Unicode;">
   <span lang="zh-CN">
    在最高法院做出决定后，川普曾多次表示国籍问题的重要性。上周他告诉记者：“你在国会，在分（选区）上需要它，你在拨款上需要它，资金去了哪里？有多少人在那儿？他们是公民吗？他们不是公民吗？有很多原因需要它。”
   </span>
  </span>
 </p>
