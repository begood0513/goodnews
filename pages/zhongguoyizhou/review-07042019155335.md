 <p>
  1    7月1日，中共建党和香港回归纪念日当天，香港继续爆发大规模示威，多处政府行政大楼和立法会遭围堵，原定在金紫荆广场举行的升旗仪式也被当局宣布执行雨天方案，改为室内观看。在龙和道、夏慤道等地，警方多次施放胡椒喷雾并以警棍驱散示威者，示威民众高喊“沒有暴动，只有暴政”等口号，双方发生对峙。
  <br/>
  <br/>
  本台报道：示威者和警方对峙的局面在周一傍晚8点过后被打破。示威者撬开香港立法会大楼其中一个进出口的铁栅，成功打开缺口，防暴警察退守政府总部。在全无阻挡情况下，示威者从其他进出口一起闯入。有人以铁笼车撞毁图书馆的玻璃，以喷漆在墙上写上诉求。电子显示屏和监控探头被砸烂。立法会的标志被喷污和扔鸡蛋。部分人攻入办公室，文件、电脑和灭火筒成为他们损毁的目标。在会议厅外，挂有历任立法会主席的照片被喷污…示威者破门占据会议厅…会议厅中央的特区区徽上“中华人民共和国”的字样被喷漆。座位上的《基本法》被人逐页撕毁……
  <br/>
  <br/>
  立法会主席梁君彦周二表示，大楼损毁严重，未来2个星期都不能召开会议。
  <br/>
  <br/>
  这是香港立法会大楼启用近8年来，首次被示威者占据，历时3个多小时。整个过程大楼内没有警员和保安在场。
  <br/>
  <br/>
  连日来备受攻击的香港行政长官林郑月娥强烈谴责示威者以极暴力方式冲击立法会，批评他们目无法纪，无助社会寻求共识。
  <br/>
  <br/>
  有“香港良心”之称的前政务司司长陈方安生批评政府没有回应民间诉求，使部分年轻人以暴力方式表达意见，呼吁示威者回归和平理性。
  <br/>
  <br/>
  2     G20 “特习会”后美中贸易战意外休兵，但分歧犹存。美国总统特朗普允诺放松华为制裁。
  <br/>
  <br/>
  纽约时报报道，G20大阪峰会“特习会”共识不代表美中在化解根本冲突上取得重大突破，顶多只是暂时停火，再给双方谈判团队一次达成永久贸易协议的机会。
 </p>

 <p>
  3    6月29日与特朗普大阪握手言和后，心情大好的中国国家主席习近平签署发布特赦令，指示特赦九类正在服刑的罪犯。但备受外界关注的“良心犯”并不在这次特赦的范围之内。
  <br/>
  <br/>
  特赦范围，包括曾参加过抗日战争、“解放战争”和建国后历次对外战争的，曾是国家级或省部级“劳模”或“先进工作者”的，因防卫过当或者避险过当而服刑的，年满七十五岁、身体严重残疾且生活不能自理等等。特赦令规定，犯有严重罪行的罪犯，如贪污受贿、危害国家安全、暴力性犯罪的，将不获特赦。
  <br/>
  <br/>
  也即，政治犯不在特赦范围之内。
  <br/>
  <br/>
  目前尚不清楚被“寻衅滋事罪”和类似罪名的在押访民是否会被特赦。
  <br/>
  <br/>
  4   在台湾，立法院三读通过《两岸条例修正案》，明确规定政务副首长或少将以上人员退休后，如赴中国大陆参与政治活动，向中共政权的国旗、国徽、国歌等行礼、唱颂，最重惩罚可被剥夺退休金。
  <br/>
  <br/>
  台湾立法院3号临时开会，三读通过《 台湾地区与大陆地区人民关系条例（两岸人民关系条例）部份条文修正案》，明定政务副首长或少将以上人员退休后，不可参与中国大陆相关政治活动。如果有妨害国家尊严行为，违者最重可被剥夺月退俸、领一次退者最高处新台币1000万元（约220万人民币）罚款。
  <br/>
  <br/>
  5    2019年百度AI开发者大会7月3日举行，百度公司CEO李彦宏进行开场演讲，演讲过程中，突然有一名观众冲上演讲台向李彦宏头部倒了一瓶水。李彦宏停顿了几秒，然后用英语问道：What's your problem？随后该不明人士被保安制服。
 </p>

  <br/>
  聯合報称，事件中男子「直男上樹」所寫的微博顯示，這是一場有預謀的行動：7月1日，從山西運城搭乘火車到北京；2日下午，抵達北京百度AI開發者大會舉辦地。3日上午10時，「直男上樹」發文稱「他（李彥宏）上來了…在該文的下面還附了一瓶礦泉水的照片」，發布了最後一條微博「我準備上去了」，隨後發生了淋水事件。

  【手机网易网】“金融八卦女”：一次攻击性的行为却收获一次网络狂欢，只因那瓶浇到李彦宏头上的水，承载的是人民群众对百度搜索的愤懑。李彦宏那句灵魂诘问，不单要问泼水的人，也该问问百度自己：“What's your problem？”
