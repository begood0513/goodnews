 <p>
  1 《华盛顿邮报》报道，指中国招商银行、交通银行和浦发银行涉嫌对朝鲜违反美国的制裁禁令，可能被终止三家银行的美元业务。 受此消息影响， 三家银行股价大跌。其中招商银行股票价格一度跌幅超过百分之八。周二下午，三家银行对外发布声明称，没有受到因涉嫌违反任何制裁法律的相关调查。
  <br/>
  <br/>
  这三家中国银行被指控在2012年10月到2015年1月之间，通过美国的往来账户，帮助朝鲜银行转移了一亿多美元资金。以支持朝鲜的核扩散计划。
  <br/>
  <br/>
  2  6.4三十年后港台在此携手联合行动。在香港两度百万人上街反送中游行后，23号在台北，约3万名台湾人在总统府前，参加了“拒绝红色媒体”集会。
  <br/>
  <br/>
  集会口号：“香港加油！红色媒体滚出去！”
  <br/>
  <br/>
  与此同时，香港“反送中”抗议示威活动仍在持续。“民间人权阵线”星期三晚在中环集会，要求当局回应有关撤回《逃犯条例》修订等诉求，呼吁国际关注。集会结束后，数以千计的民众再次包围警察总部。
  <br/>
  <br/>
  二十国集团峰会前夕，“民间人权阵线”在中环爱丁堡广场举行集会，主题为“撤回恶法，还我自由”。大批香港市民响应号召，广场空地坐满人，不少人要站在附近的停车场和马路上，人潮甚至伸延至解放军驻香港部队总部外围。
  <br/>
  <br/>
  3  湖南新晃县近日发生“校园操场埋尸”案，该具尸体已经被埋16年，舆论为之震惊。警方初步确认尸体为新晃一中的老师邓世平。邓世平的家人称，他2003年因举报学校操场工程质量问题，失踪至今。新华网星期五称，2003年，新晃一中新修了一个操场。该校后勤老师、负责工程质量监督管理的邓世平认为操场偷工减料质量有问题，拒绝在验收单上签字。与承建操场工程的校长外甥杜少平产生矛盾。没过多久，邓世平就突然失踪了。16年来邓世平的亲属一直在举报，甚至判断可能被埋尸操场，但是各方都没有作为；此事现在闹得很大，但太多地方官员想继续掩盖事件。
  <br/>
  <br/>
  湖南懷化新晃縣「操場埋屍案」引發重視，發生在30年前的河北邯鄲肥鄉「反腐英雄郭建民女兒郭桂芳失蹤案」也被媒體再度關注。綜合媒體報導，郭桂芳的家屬近日在一處枯井處挖出幾塊疑似人骨，正尋找有關部門鑑定，這可能又是一件類似「操場埋屍案」、對反腐人士殺人報復的案件。
  <br/>
  <br/>
  4  截稿日，G20峰会日本大阪举行。备受瞩目的“特习会”将在峰会期间进行。日前中国外交部官员表示，不允许在G20峰会上讨论备受全球瞩目“属中国内政”的香港问题。
  <br/>
  <br/>
  BBC: 大阪习特会似曾相识，前夕气氛与去年12月布宜诺斯艾利斯G20峰会期间的习特会类似。在布宜诺斯艾利斯，两国领导人同意设下90天停火谈判期，但在会议当天，加拿大应美国要求拘捕了华为高管孟晚舟。
  <br/>
  <br/>
  5   6月20日出刊的美国《科学》杂志日前公布一份中国公民诚信度全球排名垫底的调查结果，激起了许多中国网民反应，质疑报告“不严谨”，指控“美帝厚颜无耻”。
  <br/>
  法广报导，一个研究团队在40个国家355座城市进行了大型实验“全球公民诚实度（Civic honesty around the globe）”，观察各国人捡到“有钱的钱包”和“没钱的钱包”后的反应。
  <br/>
  <br/>
  美国与瑞士的研究人员制作了一万七千多个同样的钱包，有些钱包里没钱，有些装有钱。
  <br/>
  <br/>
  之后研究助理走进这些城市的邮局、饭店、警局、银行、博物馆等处，将钱包交给柜台人员，称在街头捡到钱包，因赶时间想麻烦对方处理，然后离开。
  <br/>
  <br/>
  结果显示，即便钱包里面没有钱，中国人肯归还钱包的比率不到10%，为调查中最后一名。
  <br/>
  <br/>
  法新社说，这是至今针对经济和心理连结点所进行的最严谨研究之一。
