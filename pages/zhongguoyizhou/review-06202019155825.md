 <p>
  1 6月16号，香港两百万人上街反对修改《逃犯条例》，要求特首林郑月娥下台。
  <br/>
  <br/>
  三天之后，在巨大的民众舆论压力之下香港行政长官林郑月娥在6.16大游行后于星期二首次公开露面，就条例引起的争议向香港公众致歉，却没有承诺撤回修例，也没有直接回应公众要求她下台的诉求，示威者对她的回应感到失望，相信抗争行动将会持续下去。
  <br/>
  <br/>
  2 香港市民连续举行大规模抗议示威，维护自身权益，在全球范围内引来广泛同情和尊敬。有不少网友在推特上表示，香港人应当集体获得诺贝尔和平奖。
 </p>

 <p>
  美籍华人记者陈嘉韵也在推特上说，香港人应该申请诺贝尔奖：这200万的示威者，遵守秩序、回收垃圾，并且在拥挤的人潮中为救护车开道，又为外国记者戴上安全帽，2019年没有比这更好的和平示威的例子。
  <br/>
  <br/>
  旅居美国的中国著名媒体人长平在《德国之声》上发表文章认为，香港人获奖将是诺贝尔奖的光荣，因为他们坚韧不拔地对抗中共极权、成功阻止包藏祸心的《逃犯条例》修订，是对全人类自由与和平的贡献。
  <br/>
  <br/>
  3  四川自贡一名网名为「燕归来a」的女子16日发送定时微博，称已自杀离开世界，警方事后在这名女子服务单位的厕所里发现她的遗体。警方调查，这名女子是28岁自贡市农牧业局的工作人员郭女，她自杀前在微博发文「我已经受不了这种毫无希望的人生了...千万别进体制，那是毁灭梦想的地方。不是开玩笑。」
  <br/>
  <br/>
  红星新闻报导，据自贡市农牧业局负责人对此表示，郭女未婚，于2014年参加工作...主要工作是写「内部数据」。
  <br/>
  <br/>
  郭女于16日16时30分发布微博贴文：「这是一条定时发送的微博，所以，当大家看到的时候，我已经离开这个世界了，有恨有遗憾有牵挂，但是我已经受不了这种毫无希望的人生了。给弟弟妹妹们一点实话吧，但凡有其它可能，千万别进体制，那是毁灭梦想的地方。不是开玩笑。」
  <br/>
  <br/>
  郭女毕业于中央民族大学，应为少数民族。
  <br/>
  <br/>
  4  中共中央对外联络部6月17日宣布，中共中央总书记、国家主席习近平将于6月20日到21日对朝鲜进行国事访问。
  <br/>
  <br/>
  据新华社报道，当地时间20日12时许，习近平乘坐的专机抵达平壤顺安机场。机场上中朝两国国旗迎风飘扬，"热烈欢迎中国共产党中央委员会总书记、中华人民共和国主席习近平同志""鲜血凝成的朝中两国人民之间牢不可破的友谊团结万岁"大幅标语高高悬挂。由近万名群众整齐排列的欢迎队伍呼喊口号，迎接习近平来访。习近平乘车前往平壤锦绣山迎宾馆的道路两旁也站满了欢迎群众。车队经过时，这些手舞鲜花和国旗的群众热情欢呼。
  <br/>
  <br/>
  5  美国总统特朗普周二表示，他将和中国国家领导人习近平在下周日本举行的G20峰会上，就美中贸易问题进行详谈。
  <br/>
  <br/>
  据中国官媒《人民日报》报道，习近平和特朗普在周二通了电话。特朗普在通话中说，希望双方通过沟通，尽快找到解决分歧的办法。习近平则强调，双方应当平等对话；中美合则两利、斗则俱伤。
  <br/>
  <br/>
  在过去一段时间，中国一直对习近平是否会在G20峰会上与特朗普面对面会谈，保持沉默。而特朗普也曾对媒体强硬表态，习近平来与不来都不是问题。如果习近平不来，他会立即对中国额外的3000亿美元进口产品增加关税。
  <br/>
