 <p>
  1 中国异议网站先驱“六四天网”负责人黄琦，于7月29日被以“故意泄露国家秘密罪”及“为境外非法提供国家秘密罪”判处有期徒刑12年。
  <br/>
  <br/>
  2  根据联合国机构和人权组织统计，在新疆已有超过一百万维吾尔等穆斯林被关押在多个集中营里接受政治洗脑教育。
  <br/>
  <br/>
  但周二由国务院新闻办公室举办的发布会上，新疆维吾尔自治区副主席艾尔肯·吐尼亚孜宣称，“大多数接受教育培训的人已经重返社会，回到了他们自己的家中。”但同样，这位新疆官员没有提供确切的数字。
  <br/>
  <br/>
  3   由反送中引发连串抗争运动持续一个多月，有网民号召下周一（8月5日）发起全民大罢工，又倡议全港7区包括沙田、荃湾、屯门、黄大仙、大埔、旺角及金钟同时集会。由网民自发组织的「8.5三罢及七区集会文宣组」指，第一阶段拟向全体港铁及巴士车长发信，呼吁他们响应行动，并正与职工盟代表联络，稍后正式向工会发信，希望号召更多人罢工，施压逼使港府响应民间5大要求。
  <br/>
  <br/>
  在北京高调表扬香港警察在镇压示威的表现后，香港警方随即控告44名示威者暴动罪。数百名市民周二（30日）晚上包围一家警署，声援在另一冲突中被捕的人，其后演变成警民冲突。
  <br/>
  <br/>
  另一方面，彭博信息31日报导，一名美国高级官员表示，中国军力正在香港边界集结，白宫正密切监视。
  <br/>
  <br/>
  连续数周的「反送中」运动，已使香港警力不堪负荷，警方与抗议群众发生暴力冲突，开始居于下风。
  <br/>
  <br/>
  中国军力在香港边界集结的性质不清楚。这名不愿透露身分的美国官员向记者简报表示，中国解放军或武警在香港边界集结。
  <br/>
  <br/>
  4   新京报快讯：据文化和旅游部政府网站消息，鉴于当前两岸关系，决定自2019年8月1日起暂停47个城市大陆居民赴台个人游试点。
  <br/>
  <br/>
  分析指出，台湾总统蔡英文上台后两岸关系虽然恶化，但中国大陆从没有停止过个人自由行旅游，这次的暂停决定显得非同寻常。预计，明年台湾总统选举之前的不到一年里，大陆赴台游客将会大缩水。
  <br/>
  <br/>
  5    美中贸易谈判在上海结束，未获任何突破。白宫发表声明，特朗普总统对中国发出警告。白宫7月31日的这则声明说，“双方就强制技术转让，知识产权，服务，非关税壁垒和农业等话题进行了讨论。中方确认了增加购买美国出口农产品的承诺。 ”声明还说，“会议是建设性的”，并指出美国“希望9月初在华盛顿特区针对可执行的贸易协议继续进行谈判。”据报道：周三的这次美中第十二轮贸易谈判只持续了半天就宣告结束。
  <br/>
  <br/>
  而在美国贸易代表团周二抵达上海时，特朗普总统连续发推，指出北京有可能放弃购买美国农产品的承诺，指责北京期待总统大选他人当选而拖延谈判，并警告说，对北京而言，如此的后果将会更糟，他直言不讳说：“假如我赢得选举，他们获得的协议将比现在的更不利，也许他们根本得不到任何协议。”
  <br/>
