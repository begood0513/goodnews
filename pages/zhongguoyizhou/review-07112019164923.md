 <p>
  1   成都秋雨圣约归正教会基督徒廖强一家六口，带着简单的行囊，持中华人民共和国护照，和“医美健检”15天停留台湾的签证，以自由行方式，1号先到泰国，4号转抵台湾后。他们决定逃亡，不回中国大陆。
  <br/>
  <br/>
  廖强告诉本台记者，他们应该是秋雨圣约教会去年遭迫害后，第一个逃出的家庭。他们知道台湾没有难民法，不想为难台湾政府，希望透过美国基督教组织协助，辗转到美国等第三地，寻求政治庇护，企盼台湾政府能给他们多一点停留的时间，等相关手续办好他们就会离开台湾。
  <br/>
  <br/>
  2 纽约时报报道：日内瓦——22个国家发表声明，敦促中国停止在西部新疆地区大规模拘禁维吾尔族和其他穆斯林，这是中国在联合国大力捍卫的一项政策首次受到国际社会的一致挑战。
  <br/>
  <br/>
  在写给联合国人权事务高级专员米歇尔·巴切莱特(Michelle Bachelet)的一封信中，这些国家要求中国遵守自己的法律和国际义务，停止任意监禁维吾尔族和其他穆斯林及少数民族群体，并允许宗教自由。这封信于周一寄出，于周三公布。
 </p>

 <p>
  包括英国、法国和德国在内的18个欧洲国家，以及日本、澳大利亚、加拿大和新西兰，都提请关注有关大规模任意拘禁的报道，并要求巴切莱特定期向人权理事会通报最新进展。
  <br/>
  <br/>
  美国在一年前退出了人权理事会，没有签署这封信。
  <br/>
  <br/>
  本周到来的新疆七五事件十周年之际，在台北举行的一场各界声援维吾尔人权的记者会上与会者质疑，新疆维吾尔人处境比台湾五零年代白色恐怖更恐怖。
  <br/>
  <br/>
  十年前七月五日，中国称为“新疆”、维吾尔人称“东突厥斯坦”的首府乌鲁木齐，爆发“新疆维吾尔自治区”1955年建立以来最严重的维汉冲突，中国武警随之对维吾尔人进行血洗式镇压。依中国官方说法，这次事件中有197人死亡、1721人受伤。而根据海外世界维吾尔大会说法，维吾尔人死亡人数在600至800人之间。
  <br/>
  <br/>
  【纽约时报】7.5当日刊出长文《新疆7·5事件十周年，我为真相作证》作者署名PECK ZANG，文章可读性很高，主题内容大致为7.5事件“是一次（维吾尔人）对汉族的成功的'集体复仇'”。
  <br/>
  <br/>
  3  香港特首林郑月娥星期二宣布，《逃犯条例》的修订工作完全失败，草案已“寿终正寝”。但其坚持留任、拒绝问责官员和特赦被捕示威民众等姿态，引发民间反弹。
  <br/>
  <br/>
  林郑月娥周二再度就《逃犯条例》修订引起的连串风波作出交待。她承认，香港社会近期的矛盾和纷争都是修例导致。

 <p>
  林郑月娥：“我们这次的修订工作是完全失败，我早前已为此致歉。失败的原因是因为我们的工作做得不好，我们对社会的脉搏掌握得不够，我们的政治敏感度很有偏差。我今天再清晰讲明，《逃犯条例》草案已经寿终正寝，草案已死。”
  <br/>
  <br/>
  4   白宫经济顾问库德洛9号对记者说，美中贸易战最高层谈判代表今天通电话，对话非常具有建设性。
  <br/>
  <br/>
  库德洛此前表示，通话后很快就会坐下来协商。
  <br/>
  <br/>
  这是美国总统川普和中国国家主席习近平6月底在20国集团（G20）大阪峰会同意重启贸易谈判以来，双方首度正式接触。
  <br/>
  <br/>
  5  本台报道：近日，一张“三峡大坝变形”的卫星图片引发广泛关注。而官媒引述专家有关大坝“处于弹性状态”，以及三峡景区关闭的消息，更增添了舆论的担忧。
  <br/>
  <br/>
  长江三峡大坝“发生变形并可能溃坝”的消息，7月1日开始在互联网疯传。华裔独立经济学者冷山在推特上发布消息指，三峡大坝发生变形并有可能溃坝，并附上两幅Google卫星照片作比较，一幅显示大坝呈直线，另一幅则可见大坝中段扭曲变形。这两张图片在推特、微信及微博迅速流传，引发民众恐慌。
  <br/>
  <br/>
  上周五，《环球时报》引述一名“不愿透露姓名的专家”的话说，谷歌地图所使用的遥感卫星影像出现了“技术问题”。同时，官媒还援引中国航天科技集团发布的一张卫星图像对上述消息加以澄清。但是，隶属于中国国务院的长江三峡集团公司6日发布消息，承认三峡大坝存在坝基移位问题，称垂直位移和水平移位“符合重力坝变形规律，坝体变形处于弹性状态”。至于中国的百度地图和高德地图，整个坝区的卫星图片也没有了放大功能。
  <br/>
  <br/>
  湖北宜昌三峡大瀑布景区上周五（5日）发出的通知显示，因上游建设施工进入截流阶段导致瀑布断流，景区从7月6日起停业至7月13日，期间将暂停接待游客，恢复营业时间另行通知。在暂停营业期间，景区将暂停接待年卡游客。
 </p>
