 <p>
  1 因推动中国民主、发起《零八宪章》，被中国当局判处「煽动颠覆国家政权」罪成入狱的诺贝尔和平奖得主刘晓波，在2017年7月13日服刑期间罹患肝癌离世。
  <br/>
  <br/>
  本周是刘晓波逝世两周年的日子，由刘晓波生前好友组成的「晓波助澜会」发出公告，哀悼刘晓波及所有为反抗中共极权统治而牺牲的民主先烈。他们指出：我们坚信刘晓波的思想和梦想将继续生存、传播乃至终将实现，我们坚信无数仁人志士们前赴后继、为之流血牺牲的理想终将实现，我们坚信，希望在于民间从未止息的抗争。只要人心不死，一个自由、平等的民主中国终将实现。
  <br/>
  <br/>
  2 央视《新闻联播》本周二（16日）晚间报道，第7次全国对口支持新疆工作会议在新疆和田召开。报道首次以“中央新疆工作协调小组组长”介绍汪洋的身分。有评论认为，北京高调公布以务实见称的汪洋处理新疆事务，就是要应对国际舆论压力。
  <br/>
  <br/>
  3   社交媒体上最活跃的中国外交官之一、中国驻巴基斯坦大使馆临时代办赵立坚13日在推特上转发了日前37国对中国新疆政策的联合表态，称这是对“美国及其西方同伙的一记响亮耳光”。赵立坚回避前述22国对新疆再教育营的批评，并说这些国家无权替新疆穆斯林发言，而那些“职业教育培训中心”是打击宗教极端主义的一部分。赵立坚的这一说法，日前已被美国国务院反恐部门协调人内森·赛尔斯大使驳斥，称中国在新疆大规模拘禁维吾尔人等穆斯林的做法和反恐“没有任何关系”。
 </p>

 <p>
  赵立坚接着把矛头一转，指向了美国。他周六在推特上表示，华盛顿特区的白人从来不去特区的东南部，因为那里是黑人聚居区，这种隔离说明美国的种族歧视广泛存在于各个层面。
  <br/>
  <br/>
  赵立坚的这番言论引起了美国前国家安全顾问苏珊·赖斯的回击。赖斯周日在推特上表示，赵立坚是一个“可耻的种族主义者”，且“惊人地无知”。赖斯还敦促中国的外交部门尽快采取正确举措，将赵立坚革职回国。其后，赵立坚很快删除了自己的上述推文。
  <br/>
  <br/>
  4  台湾高雄市长韩国瑜在党内初选中胜出，将正式代表国民党与民进党的蔡英文角逐下一任台湾总统。现在，人们关注的焦点之一是两位候选人的大陆政策以及大陆的台湾政策将如何影响选情。
  <br/>
  <br/>
  韩国瑜以17%的较大民调优势超过另一位国民党参选人郭台铭，而出任国民党的总统候选人，结束了长达几个月的党内群雄之争。
  <br/>
  <br/>
  本台特约评论员王丹对此评论道：“对台湾的考验开始了！”
  <br/>
  <br/>
  5   中国国家统计局周一公布第二季度GDP增速为6.2%，为过去近30年来最低增长率。今年第一季度的GDP增长率为6.4%。
  <br/>
  <br/>
  分析认为，经济增速持续走低，显示中美贸易战僵局对中国经济的负面影响。
  <br/>
