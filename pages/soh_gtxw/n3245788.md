 <p class="content-info-datetype">
  <span class="content-info-date">
   【希望之声2019年10月9日】

  <span class="content-info-type">
   （本台记者高健雯综合编译）
  </span>
 </p>
 <p class="">
  是巧合还是天意，第一次听到《阿拉丁》插曲《Speechless》，直觉这简直是为香港而写。遂译出一版，敬致香港为自由民权而战的反共勇士。关于标题，看到网络上已有的几个中译版，通常会译为，沉默，无言，无言以对之类，我想，无论是这首歌本身所要表达的，或是联想到当今香港，标题都不应是无言以对，而应是「大音希声」！
 </p>
 <p class="">
  一个浪头打来，
  <br/>
  将我抛向不知何地
  <br/>
  一片暗潮汹涌，
  <br/>
  将我拖进深海之底
  <br/>
  默默吞声，无有只言片语
  <br/>
  我之呼告，早为雷音吞没，铺天盖地
  <br/>
  我却不能哭泣
  <br/>
  更不能一溃千里
  <br/>
  无论如何
  <br/>
  封堵我口，磨灭我心，千方百计
  <br/>
  我都不会沉默不语
  <br/>
  你终无法让我失语！
  <br/>
  努力坚守不要颤栗
  <br/>
  我唯知我不会在沉默中放弃
  <br/>
  面对窒息，我要呼吸，拼尽全力
  <br/>
  你视我渺若蝼蚁
  <br/>
  我却不会在沉默中放弃
 </p>
 <p>
  堂皇规则刻于石上
  <br/>
  字句陈腐不可违背如铁律
  <br/>
  「你要安分守己」
  <br/>
  「你要循规蹈矩」
  <br/>
  然而，此时，这场闹剧正告终局！
  <br/>
  我知我必须坚守不能溃堤
  <br/>
  任你封堵我口，磨灭我心，千方百计
  <br/>
  我都不会沉默不语
  <br/>
  你终无法让我失语！
  <br/>
  努力前行不要颤栗
  <br/>
  我唯知不会在沉默中放弃
  <br/>
  迎战狂风暴雨
  <br/>
  我昂首直面寸步不移
  <br/>
  岂能瘖哑偷生
  <br/>
  我知我不会在沉默中放弃
  <br/>
  牢笼无法将我囚禁
  <br/>
  我又岂能束手待毙！
  <br/>
  我将振作受伤的翅羽
  <br/>
  而你将看我化身火焰燃烧在天际
  <br/>
  我之大音在天地间久久回荡：
  <br/>
  「我绝不会沉默不语……」
 </p>
 <p class="">
  你想见我恐惧颤栗
  <br/>
  我只知我不会在沉默中放弃
  <br/>
  面对窒息，我要呼吸，拼尽全力
  <br/>
  你视我渺若蝼蚁
  <br/>
  我却不会在沉默中放弃
  <br/>
  我唯知永远不会在沉默中放弃
 </p>
