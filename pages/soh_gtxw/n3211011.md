<div class="content">
 <p>
  <span class="content-info-date">
   【希望之声2019年9月25日】
  </span>
  <span class="content-info-type">
   （本台记者高健雯综合报导）
  </span>
  「反送中」运动3个多月来，「连登仔」一直秉持「不分化 不割席」、「兄弟爬山，各自努力」原则，无论手足行为，「不谴责、不割席、不笃灰」，勇武与
  <span class="st">
   「
  </span>
  和理非
  <span class="st">
   」
  </span>
  （
  <span class="st">
   和平、理性、非暴力
  </span>
  ）互相配合，将矛头一致指向政府。但由于近期一直有乔装人士抹黑示威者，「连登仔」24日宣布与之割席。
 </p>
 <p>
  由连登网站组成的俗称「连登仔」的抗议人士24日晚发表《割席宣言》表示，有鉴于港共与警方对香港人既打压无效，极有可能出现伪装示威者以抺黑示威者之行为，现公布割席宣言。
 </p>
 <p>
  宣言同时声明：
  <br/>
  一，示威者不会恶意伤害任何无辜人士之性命财产。
  <br/>
  二，示威者不强奸、不抢劫。
  <br/>
  三，示威者对报导事实的记者，医护，外国朋友等一律表示尊重。
  <br/>
  四，示威者作出的一切行为皆对准港共极权或任何助长港共极权之人士、团体和机构。
  <br/>
  五，不会持有锁匙开警车车尾箱。
  <br/>
  六，不会有警车车匙揸警车。
  <br/>
  七，不会持有同警慌同款的伸缩警棍。
  <br/>
  八，不会抢枪如此不智让警慌为开实弹而舖路。
 </p>
 <p>
  注：
  <br/>
  无辜人士例子：路过市民，游客。
 </p>
