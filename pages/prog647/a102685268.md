 <p>
  最近有一部韩国电影突然被吹爆啊，被称之为“年度第一”
  <br/>
  现在各个YouTuber也在争先恐后的做各种影片分析
  <br/>
  关于《寄生上流》这部电影
  <br/>
  把穷人和富人的的阶层感以非常之直观的方式表达出来
  <br/>
  只所以火，我觉得应该是每个人都可以在这部电影里找到自己的影子吧
  <br/>
  一直以来，穷人和富人好像都是很尖锐的话题
  <br/>
  那我就很纳闷，在古代，人们是这么看待穷富的呢？
 </p>
 <p>
  要说寄生上流，我可能还没有这个能力
  <br/>
  但是我每天都寄生在手机上
  <br/>
  真不敢想，我哪一天要是没有手机
  <br/>
  就感觉跟世界隔绝开了一样
  <br/>
  但是你说，古代的人也没有手机电脑，
  <br/>
  他们的生活有什么意思呢？
 </p>
