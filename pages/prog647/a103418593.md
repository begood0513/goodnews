<div><div class="post_content" itemprop="articleBody">
 <p>
  这是一部由飞天大学的学生创作的视频作品，讲述了一位
  <ok href="https://www.ntdtv.com/gb/神韵.htm">
   神韵
  </ok>
  演员在幕后的离奇经历。故事中的小林是位年轻的神韵舞蹈演员，练舞认真刻苦，但就是不太爱惜演出的道具，接连弄坏、弄丢了多把扇子、警棍等。
 </p>
 <p>
  突然，有一天小林身边的道具全都“活”了，抱怨他不珍惜。小林脾气上来，根本听不进去。这些道具就用歌唱出了他们的心声，让小林回忆起了自己参与
  <ok href="https://www.ntdtv.com/gb/神韵.htm">
   神韵
  </ok>
  演出的初衷和当时的所思所为，最终在惭愧与感动中规正了自己的言行。
  <br/>
 </p>
 <p>
  <br/>
  很喜欢这个创作中的用心和用意，可以说是：小创作，大智慧。这个小品剧让我明白，万物皆有灵，道具如此，我们身边的人、事、物也是如此。人生大戏台，不管身边的人扮演着善恶美丑的任何角色，不管发生在自身的是福是祸，都要珍惜。也许正是这幕戏背后的因素促成角色于遭遇的不同，也正是这幕戏背后的因素促成这样还愿了缘的际遇。受其中歌曲的启发，很有感触，也写出一段与大家共勉：
 </p>
 <p>
  <strong>
   惜缘
  </strong>
  <br/>
  在轮回中苦苦等你，
  <br/>
  无明中长夜漫漫 我堕入痴迷
  <br/>
  而你的誓约
  <br/>
  却从耳畔响起
  <br/>
  开锁了凡尘封印的记忆：
 </p>
 <p>
  未曾忘记啊我依然记得你
  <br/>
  浊清淤泥啊重返故里
  <br/>
  那是生命的期许啊生存的意义
  <br/>
  然人海茫茫，
  <br/>
  纵天涯路断 何处觅君迹？
 </p>
 <p>
  未曾想，
  <br/>
  你找到了我
  <br/>
  你将歌声化作汩汩清泉
  <br/>
  润泽我久旱的心田
  <br/>
  你将舞蹈化作习习暖风
  <br/>
  扣启我紧闭的柴扉 送来春的信息
 </p>
 <p>
  《
  <ok href="https://www.ntdtv.com/gb/神韵作品.htm">
   神韵作品
  </ok>
  》中的歌舞、故事和课程
  <br/>
  都如此的似曾相识
  <br/>
  有如你谆谆的教诲
  <br/>
  又像久别重逢的促膝而谈
  <br/>
  将我的希望重新拾起
  <br/>
  让我看到了生命的真谛
 </p>
 <p>

  <br/>
