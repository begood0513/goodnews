 【大纪元2022年04月28日讯】

<p>
 “他们把爸爸放进了黄色的裹尸袋里，
 <br/>
 然后开始拉拉链，我很害怕但又无法阻止，
 <br/>
 这一拉我们这一世的父女缘就真的，尽了……
</p>
<p>
 母亲三点多叫的
 <ok href="https://www.epochtimes.com/gb/tag/%E6%95%91%E6%8A%A4%E8%BD%A6.html">
  救护车
 </ok>
 ，
 <br/>
 对方回复说前面有200多人。
 <br/>
 等八点120赶到的时候，
 <br/>
 妈妈说爸爸已经吐出了白沫。
 <br/>
 是的，我没见到他最后一面，
 <br/>
 这个时候我正在延安路高架下，
 <br/>
 努力地骑着共享单车，
 <br/>
 没有车啊！一辆车都没有，
 <br/>
 120都迟到那么久怎么会有出租车呢？
</p>
<p>
 我想跑到市政府问问他们，
 <br/>
 新冠死了多少人？他们都多少岁？
 <br/>
 别人的‘
 <ok href="https://www.epochtimes.com/gb/tag/%E8%B0%A3%E8%A8%80.html">
  谣言
 </ok>
 ’我不传，可我爸爸才65，
 <br/>
 因为
 <ok href="https://www.epochtimes.com/gb/tag/%E6%95%91%E6%8A%A4%E8%BD%A6.html">
  救护车
 </ok>
 的姗姗来迟他没了……
 <br/>
 他死前问了妈妈三遍–120什么时候到？
 <br/>
 都说封闭是为了老人，是为了防止
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8C%BB%E7%96%97%E6%8C%A4%E5%85%91.html">
  医疗挤兑
 </ok>
 ，
 <br/>
 今天这不正是‘
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8C%BB%E7%96%97%E6%8C%A4%E5%85%91.html">
  医疗挤兑
 </ok>
 ’吗？”
</p>
<p>
 这是一位上海市民发的一条朋友圈。她还说：
 <br/>
 “没有葬礼，我也没法为爸爸宣读悼词，
 <br/>
 我只能在朋友圈记录下这些本该不会发生的事情。
 <br/>
 他是一个好人，一个老好人，死得匆匆又潦草。
 <br/>
 今天阳光正好，南昌路上美极了，花儿开得也好。
 <br/>
 从外看这个城市好像暂停了，
 <br/>
 时间却没有为他暂停过一分一秒…….
 <br/>
 感谢我们的楼组长一家、
 <br/>
 感谢120三个急救的医生、
 <br/>
 每一位关心我爸爸的亲朋好友…
 <br/>
 爸爸，走好！ ”
</p>
