 <br/>

 </ok>

 <br/>
 充斥着阴毒，粗鄙，丑陋，邪义。

 真中国是人类的光彩。

 假中国乱世界之魔怪。

<p>
 五千年的中国是真中国——
 <br/>
 <ok href="http://www.epochtimes.com/gb/tag/%E7%9B%98%E5%8F%A4%E5%BC%80%E5%A4%A9.html">

 <br/>
 老子传道，孔子教人，光辉圣贤；
 <br/>
 秦皇汉武，盛唐大清，人君奉天；
 <br/>
 五岳四渎，唐诗宋词，壮观千年。
 <br/>
 中国历史是一部神曲，历史的中国是真中国。
 <br/>
 七十年的中国是假中国——
 <br/>
 马列装神，魔鬼登台，万里妖氛；
 <br/>
 黑毛作主，红书洗脑，欺天骂神；
 <br/>
 嗜血杀人，夺财害命，千万冤魂；
 <br/>
 颠倒黑白，偷换善恶，山河泪奔。
 <br/>
 中国当代是一座监狱，当代的中国是假中国。
 <br/>
 中共用暴力摧毁了一个真中国。
 <br/>
 中共籍谎言建起了一个假中国。
 <br/>
 中共高喊它代表中国，它是个鬼，它也配？
 <br/>
 中共嚎叫它代表人民，它是个秽，它怎会？
 <br/>
 中共目地就一个：
 <ok href="http://www.epochtimes.com/gb/tag/%E6%AF%81%E7%81%AD.html">
  毁灭
 </ok>
 人类。
 <br/>
 真中国是人类的大美。
 <br/>
 假中国祸世界之大罪。
</p>

  三
 </strong>
</p>
<p>
 走进真中国，看清假中国：
 <br/>

 <br/>
 拥抱真中国，唾弃假中国：
 <br/>
 红尘的过客，作一个世上人，多么紧迫。
 <br/>
 五千年历史是一台大戏，有你，也有我。
 <br/>
 中国是大戏的中心之地，有福，也有祸。

 你可知道这台大戏是为何事？
 <br/>
 你可明白这台大戏中心之义？

 神韵打开了中国的历史，
 <br/>
 神韵展现了中华的风致，
 <br/>
 神韵唤醒了世人的忆记，
 <br/>
