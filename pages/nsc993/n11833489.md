 【大纪元2020年01月31日讯】

<p>
 拜年忌惮面对面，

  肺炎
 </ok>
 话热门窗寒。
 <br/>
 城乡街巷空荡荡，影剧院闭游园关。
 <br/>
 庙观不见香火旺，飞机趴窝车船眠。
 <br/>
 医院人满医亦病，崛起最高火葬烟。
 <br/>
 朝鲜蒙古封边卡，撤侨卅八国封馆。
 <br/>
 <ok href="https://www.epochtimes.com/gb/tag/%E7%BA%A2%E6%9C%9D.html">
  红朝
 </ok>
 大年鬼域相，乌鸦过冬蚊翩翩。
 <br/>
 <ok href="https://www.epochtimes.com/gb/tag/%E7%BA%A2%E6%9C%9D.html">
  红朝
 </ok>
 庚子临鬼门，中共鼠年若寒蝉。
 <br/>
 七捂八捂捂不住，越捂越传越蒸怨。
 <br/>
 岁末党魁略松口，犹打侥幸马虎眼。
 <br/>
 团拜联欢装没事，疫区照办万人宴。
 <br/>
 八位医生露疫情，无形口罩公安粘。
 <br/>
 只是人命当儿戏，自我暴露何心肝。
 <br/>
 拖到年关拖不动，怒吼雷滚化针毡。
 <br/>
 春晚只好放录像，联欢报导文闪删。
 <br/>
 大年初一常委会，骂声载道耳强掩。
 <br/>
 红色处处患
 <ok href="https://www.epochtimes.com/gb/tag/%E8%82%BA%E7%82%8E.html">
  肺炎
 </ok>
 ，萨斯升级不敢谈。
 <br/>
 应对领导小组长，党魁不干总理担。
 <br/>
 成员缺少卫生官，屠刀骗笔两手全。
 <br/>
 一推再推难再推，初三总理奔武汉。
 <br/>
 秀问困难有没有，托答没有对脑残。
 <br/>
 欢迎党妈涮性命，小区开窗红歌喧。
 <br/>
 而今重演伟光正，大惹众怒彰混蛋。
 <br/>
 各种患者不满千，忙坏医务十多万。
 <br/>
 如此愚民论导向，自曝蠢二变笑谈。
 <br/>
 今日当初怎不想？殷鉴怎忘又不远？
 <br/>
 不是不想也没忘，一提萨斯腿晃颤。
 <br/>
 本想避讳称非典，反倒自缩包围圈。
 <br/>
 典字脱框共字显，框去套非共匪颜。
 <br/>
 当年攻进中南海，几个常委中了弹。
 <br/>
 这回谁知倒几位，弄不好会全玩完。
 <br/>
 市长已知罪难戴，坦言无授权才瞒。
 <br/>
 公布人走五百万，明明承认封城晚。
 <br/>
 见其口罩倒着戴，有人言其鼓造反。
 <br/>
 造不造反往下看，反正离心已显然。
 <br/>
 书记揭底也甩锅，下面没有确诊权。
 <br/>
 封城传说旨断指，大拇指头能心甘？
 <br/>
 世卫组织头见习，习称病疫亲自管。
 <br/>
 党媒都怕再惹臊，报道此事删此言。
 <br/>
 其实党妈谁都害，党魁用完亦烂蒜。
 <br/>
 不过党妈也没好，正临末日
 <ok href="https://www.epochtimes.com/gb/tag/%E9%81%AD%E5%A4%A9%E8%B0%B4.html">
  遭天谴
 </ok>
 。
 <br/>
 <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </ok>
 ①已超三亿半，人心失尽无可挽。
 <br/>
 中共年关好难熬，熬过年关却更难。
 <br/>
 天灭红魔眼前事，宇宙净化筛恶善。
 <br/>
 生留死去在一念，漂红弃党明天灿。
 <br/>
 <ok href="https://www.epochtimes.com/gb/tag/%E4%B9%9D%E5%AD%97%E7%9C%9F%E8%A8%80.html">
  九字真言
 </ok>
 ②诚心诵，天佑逃难保平安。
 <br/>
 良知快醒勿再迷，时机错过落深渊。
