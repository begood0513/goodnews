### 南岛：“这个世界真的是回不去了”

<div><p>
 【大纪元2022年05月25日讯】
</p>
<p>
 聊一件小事吧。
 <br/>
 2020年初
 <ok href="https://www.epochtimes.com/gb/tag/%E7%96%AB%E6%83%85.html">
  疫情
 </ok>
 控制住后，
 <br/>
 就有一个年轻母亲背着
 <br/>
 一个不到1岁的娃娃
 <br/>
 开始给我公司送邮件。
</p>
<p>
 第一次见他们母子时
 <br/>
 天气还有些寒冷，她进门的时候，
 <br/>
 后面背着一个厚实的宝宝背带，
 <br/>
 还围着一个像小棉被的东西在外面。
 <br/>
 我开始以为是个什么大邮件的
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8C%85%E8%A3%B9.html">
  包裹
 </ok>
 ，
 <br/>
 结果仔细一看，里面是个
 <br/>
 带着儿童
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8F%A3%E7%BD%A9.html">
  口罩
 </ok>
 的宝宝，
 <br/>
 那个
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8F%A3%E7%BD%A9.html">
  口罩
 </ok>
 明显还是有点大，
 <br/>
 几乎快把孩子的眼给遮住了。
 <br/>
 当时心里不觉一阵酸楚，
 <br/>
 也就这样，我们几乎是看着
 <br/>
 这孩子从襁褓到走路的。
</p>
<p>
 直到今天下午，她又带着
 <br/>
 颠颠跑的小男孩来送件，
 <br/>
 这一送就2年多。
</p>
<p>
 我从未和这个母亲说过话，
 <br/>
 也不知道这对母子为何会这样，
 <br/>
 不知道孩子的父亲在做什么，
 <br/>
 他们一定是有自己的难处吧。
 <br/>
 看着满大街越来越多的
 <br/>
 外卖小姐姐和中年跑腿的男女们，
 <br/>
 心中滋味说不出的难受
 <br/>
 背脊冒出阵阵寒意。
</p>
<p>
 看来，这个
 <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%96%E7%95%8C.html">
  世界
 </ok>
 真的是回不去了。
 <br/>
 那个初春寒冷的早上，
 <br/>
 那双从襁褓中露出的眼睛，
 <br/>
 让我至今也忘不了。
</p>

</p>
