<p>
 注：【大纪元2019年08月12日讯】在美国川普政府宣布加征关税、确认中共国是汇率操纵国，香港“反送中”运动依旧如火如荼之际，海外有爆料称，正在召开的北戴河会议中共高层博弈激烈，其中一个中共前任老常委以“明年是否还能在北戴河相见”为标题，抛出
 <ok href="http://www.epochtimes.com/gb/tag/%E5%85%AB%E5%A4%A7%E8%B4%A8%E9%97%AE.html">
  八大质问
 </ok>
 ，分发给身在北戴河的高官们，引发强烈震动。
 <br/>
 而且
 <ok href="http://www.epochtimes.com/gb/tag/%E5%85%AB%E5%A4%A7%E8%B4%A8%E9%97%AE.html">
  八大质问
 </ok>
 可谓是个个直捅中共要害，可以视为是在将重大问题摊牌。无疑，明年能否还在北戴河相见，暗示中共未必能撑到明年，一旦垮台，大家是树倒猢狲散，各自命运都难料。至于八大质问则进一步对标题背后的原因予以解释，它们分别是：
 <br/>
 一、香港问题最终怎么解决？
 <br/>
 二、中国的经济这样持续下去，还有明年吗？
 <br/>
 三、高压下的中国社会能撑过明年吗？
 <br/>
 四、中美关系（紧张）到这个程度，共产党和中国政府能撑到明年吗？中国现在多方面的问题，不可抗拒的，无法改变的，如新疆西藏问题。党内（现在）人人自危，（如果）发生内部动乱和暴乱，怎么解决？！
 <br/>
 五、在这个时代共产党能控制住互联网和社交媒体吗？
 <br/>
 六、如果中国的财政赤字和外债发生双爆发，结果会什么样？
 <br/>
 七、以美国为首的西方（国家），如果对（中共）所有的在海外的资产都列为非法资产，进行查封，（中共）怎么办？
 <br/>
 八、（现在的）国家安全委员会下的国家运行制度，事实上废除了常委政治局，这个模式是否能继续下去？
 <br/>
 说其是质问，是因为提出的每个问题的语气都相当强硬，而且颇有逼宫的味道，尤其是最后一条更是直指中南海最高层。
