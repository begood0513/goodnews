 【大纪元2020年02月29日讯】

<p>
 厉害了，我的国，

  贸易战
 </ok>

 <br/>
 新冠状，
 <ok href="https://www.epochtimes.com/gb/tag/%E7%97%85%E6%AF%92.html">
  病毒
 </ok>
 烈。
 <ok href="https://www.epochtimes.com/gb/tag/%E7%96%AB%E6%83%85.html">
  疫情
 </ok>
 冒，瞒两月。捂洋溢，成大疫。
 <br/>
 非大意，乃故意。生武库，为利器。当初泄，或不意。
 <br/>
 一泄露，或窃喜。试制久，验效力。魔盒开，难控抑。
 <br/>
 慌忙捂，捂不及。红江山，
 <ok href="https://www.epochtimes.com/gb/tag/%E7%97%85%E6%AF%92.html">
  病毒
 </ok>
 弥。散四海，罪恶极。
 <br/>
 此猜测，不无据。毁人类，终目的①。罪化功，恶作绩。
 <br/>
 丧事摆，喜事席。庆功酒，血海取。大国战疫书出急。
 <br/>
 伟光正，金粉泥。将党魁，捧云际。但有指，戏中戏。
 <br/>
 而幕后，深无底。有蛛丝，有马迹。病毒所，江手里。
 <br/>
 用疫病，施重击。举高高，再撤梯。党罪债，全推习。
 <br/>
 拔驴橛，该承袭。
 <ok href="https://www.epochtimes.com/gb/tag/%E8%B0%8E%E8%A8%80.html">
  谎言
 </ok>
 书，恶作剧。一页纸，一证据。
 <br/>
 书山变，火山起。果若此，惨烈异。而于其，非为奇。
 <br/>
 嗜血党，虐杀狂。恶斗术，超想像。但愿这，是空想。
 <br/>
 说空想，即愿望。好愿望，常反向。早晚会，露真相。
 <br/>
 大真相，系天象。天除魔，其必亡。世棋局，神执掌。
 <br/>
 灭邪党，救善良。再挣扎，也白忙。纸火山，烧光光。
 <br/>
 谎灰飞，党尘扬。谁保党，谁陪葬。天网开，一面祥。
 <br/>
 <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </ok>
 ②留，生路广。人快醒，莫彷徨。时机到，天大亮。
