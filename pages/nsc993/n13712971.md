 【大纪元2022年04月16日讯】

<p>
 4月14日，
 <br/>
 刚过完71岁生日的
 <br/>
 上海小提琴家陈顺平，
 <br/>
 因腹痛无法就医，
 <br/>
 在经过了一整夜剧痛煎熬后，
 <br/>
 绝望至极，
 <br/>
 从顶楼跳下身亡！
</p>
<p>
 谁能告诉我，
 <br/>
 没有死于
 <ok href="https://www.epochtimes.com/gb/tag/%E6%96%B0%E5%86%A0.html">
  新冠
 </ok>
 ，
 <br/>
 却因
 <ok href="https://www.epochtimes.com/gb/tag/%E6%96%B0%E5%86%A0.html">
  新冠
 </ok>
 而死的上海人，
 <br/>
 究竟有多少？
</p>
<p>
 仅仅我知道的就有:
 <br/>
 东方医院护士周盛妮，
 <br/>
 虹口区卫健委官员钱文雄，
 <br/>
 网信证券副总裁韦桂国，
 <br/>
 经济学家郎咸平的母亲，
 <br/>
 宝山区市民蔡云涛
 <br/>
 55岁的市民陈伟……
 <br/>
 这个名单还在不断加长。
</p>
<p>
 他们有的是因为有病
 <br/>
 无法及时就医而离世的，
 <br/>
 有的是因为无法承受工作压力
 <br/>
 而撒手人寰的，
 <br/>
 也有的是因为工作过劳走的，
 <br/>
 还有活活饿死的。
</p>
<p>
 一位死者的女儿在微博上控诉道：
 <br/>
 “我爸就这样活活被拖死了！
 <br/>
 四点多的时候人还能说话走路，
 <br/>
 你们不送医院，人死了有地方抢救了？！
 <br/>
 难道非要把人逼死，医院才收吗？！”
</p>
