<div><p>
 【大纪元2019年11月03日讯】
</p>
<p>
 我觉得自己是一个贪婪的人
 <br/>
 每天妻子要花很多钱去买肉和菜
 <br/>
 每天妻子都抱怨不断疯长的
 <ok href="http://www.epochtimes.com/gb/tag/%E7%89%A9%E4%BB%B7.html">
  物价
 </ok>
</p>
<p>
 诅咒着前段贱如纸今天贵如金的猪
 <br/>
 我窃喜她已无暇诅咒
 <ok href="http://www.epochtimes.com/gb/tag/%E6%88%BF%E4%BB%B7.html">
  房价
 </ok>
 和药价
</p>
<p>
 <ok href="http://www.epochtimes.com/gb/tag/%E7%89%A9%E4%BB%B7.html">
  物价
 </ok>
 再疯狂，我的胃口却无法拒绝肉和菜的贿赂
 <br/>
 我实在是一个贪婪的人啊
 <br/>
 物价这么高，却不体谅盛世的难处
</p>
<p>
 盛世的歌舞这么好看，却常常忘记喝彩
 <br/>
 它昡目的舞台足以粉碎人们对黎明的想法
 <br/>
 在肉身都填不饱的仓惶中年
 <br/>
 我还这么不知耻地
 <br/>
 每天淘旧书买旧书
 <br/>
 因为便宜啊因为我饿啊
 <br/>
 我还想着喂养灵魂的胃口
 <br/>
 真是一个贪婪的人啊我
 <br/>
 涏着那么一张浮肿的时代脸庞
</p>
<p>
 蹒跚在猪排骨嗤笑穷人小饭桌的黑夜
 <br/>
 竟然还奢望在一盏微弱的灯下
 <br/>
 夯实灵魂的高度
</p>
<p>
 在空空口袋和孩子苍白脸色的逼视下
 <br/>
 我躲藏在盛世神话的阴影里
 <br/>
 今夜，我手握旧卷
 <br/>
 万籁俱寂中偷擦去读书人的泪
 <br/>
 我不知道这是否愧对
 <ok href="http://www.epochtimes.com/gb/tag/%E5%9C%A3%E4%BA%BA.html">
  圣人
 </ok>
 的教诲
 <br/>
 我确实是一个因贪婪而
 <br/>
 向伟大盛世低头的人！
</p>
