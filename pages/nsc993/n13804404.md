<p>
 有一个中国，
 <br/>
 她滋养于灿烂的文明古国。
 <br/>
 有孔孟的温柔敦厚，
 <br/>
 佛家的慈悲与道家的求真洒脱，
 <br/>
 有现代的普世价值和自由民主的生活。
 <br/>
 有一个“中国”，
 <br/>
 它崇拜
 <ok href="https://www.epochtimes.com/gb/tag/%E6%92%92%E6%97%A6.html">
  撒旦
 </ok>
 与鬼怪妖魔。
 <br/>
 挥舞着假恶斗的屠刀，
 <br/>
 叫嚣着专制独裁的
 <ok href="https://www.epochtimes.com/gb/tag/%E9%82%AA%E8%AF%B4.html">
  邪说
 </ok>
 ；
 <br/>
 并擅长杀戮洗脑贪腐和
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8D%96%E5%9B%BD.html">
  卖国
 </ok>
 。
</p>
<p>
 有一个中共，
 <br/>
 它披着羊皮卖着狗肉，
 <br/>
 享受着特权特供和
 <ok href="https://www.epochtimes.com/gb/tag/%E5%85%B1%E4%BA%A7%E4%B8%BB%E4%B9%89.html">
  共产主义
 </ok>
 生活。
 <br/>
 它的吸血管道密如网络，
 <br/>
 对韭菜们的苦难乐呵呵。
 <br/>
 有一个中共，
 <br/>
 它标榜着为人民服务的承诺，
 <br/>
 信誓旦旦的要廉洁爱国，
 <br/>
 鼓吹要复兴中华民族，
 <br/>
 并构建人类命运共同窝。
</p>
<p>
 朋友，你
 <br/>
 相信的是哪一个中共？
 <br/>
 自豪的是哪一个中国？
</p>
