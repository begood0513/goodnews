<p>
 【大纪元2019年06月24日讯】
</p>
<p>
 一八九八年：
 <br/>
 阴雨天
</p>
<p>
 她的妆容弄脏了。
 <br/>
 像小孩的笔画，潦潦草草的泪水河流打湿了她的脸庞。
 <br/>
 他说“没办法”。
 <br/>
 他说“这不是我的选择”。
 <br/>
 他说“九十九年真不久”。
 <br/>
 她无语，只点头。
 <br/>
 他一关门，一笑着大声说“如果你爱你的爸去，为了帮他哦。别忘记他哦”。门关了，她等了。
</p>
<p>
 一九九七年：
 <br/>
 <span href="http://www.epochtimes.com/gb/tag/%E6%99%B4%E6%9C%97%E5%A4%A9.html">
  晴朗天
 </span>
</p>
<p>
 在她右耳边有一朵花。
 <br/>
 在她左耳边有一把金色的钥匙。
 <br/>
 像一个被遗忘的军人，没人迎接，她回来了。
 <br/>
 像星期天的太阳光线她无限无限制用她的美丽淹没那个小房子。
 <br/>
 呛着她的金子，他起来了。
 <br/>
 “宝贝”“亲爱的”“你终于回来了”抱着她不见的爸说“你长大了变成这么美可是，我知道你现在真的是很累。没事儿，你终于自由了谁都不管，这生活现在就是你的”。
 <br/>
 摸着她的头她的花就起皱。
</p>
<p>
 二零一四年：
 <br/>
 雷雨天
</p>
<p>
 带着
 <span href="http://www.epochtimes.com/gb/tag/%E9%9B%A8%E4%BC%9E.html">
  雨伞
 </span>
 她从超市回家了。
 <br/>
 想给他买一些花儿。
 <br/>
 丢了钥匙，她叫他的帮助。
 <br/>
 他叫她一只猪。
 <br/>
 “不安全，不聪明，你在这个天气为什么敢出去？”她没看，没说。
 <br/>
 他说，“我是跟你说。你必要看我，听我。”像九月份的台风，他把她的花儿，她的
 <span href="http://www.epochtimes.com/gb/tag/%E9%9B%A8%E4%BC%9E.html">
  雨伞
 </span>
 ，扔去她钥匙的地方。
</p>
<p>
 二零一九年：
 <br/>
 多云天
</p>
<p>
 有人说时间会把垃圾变成宝物。五年过，他们的垃圾桶开了一棵荆树。有一天，树终于开花儿了。
</p>
<p>
 一千朵花儿，像雨伞，飘飘悠悠地从窗户落下到路上。
 <br/>
 在肥沃的手机屏幕和伸出的手里，花儿的种子变成黑体恤衫和白口罩的香港人。她带着雨伞飘飘悠悠地从窗户落下到路上陪着他们。
 <br/>
 他看见的地上不算路上，算一个海，一个一起唱歌，拉手，无敌的海。
</p>
<p>
 二零二零年：
 <br/>
 晴天他看见，他无语。
</p>
