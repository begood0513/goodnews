<div><p>
 【大纪元4月15日讯】
</p>
<p>
 人民的天空
 <br/>
 被黑暗吞噬了
 <br/>
 到处是吃人的魔鬼
 <br/>
 谁来拯救我们？
</p>
<p>
 我的国家
 <br/>
 被坏人窃据了
 <br/>
 到处是豺狼虎豹
 <br/>
 日子真苦啊！
</p>
<p>
 我的家乡
 <br/>
 被伪政权霸占了
 <br/>
 到处是压迫与剥削
 <br/>
 人民实在是活不下去了！
</p>
<p>
 我的企业单位
 <br/>
 被狠心的领导转卖了
 <br/>
 到处是特权分子
 <br/>
 我成为下岗的工人
</p>
<p>
 我家的房子
 <br/>
 被共产党拆迁了
 <br/>
 到处是恶贼与强盗
 <br/>
 我成为无壳的蜗牛
</p>
<p>
 我的庄稼
 <br/>
 被硕鼠吃光了
 <br/>
 家里没有米
 <br/>
 我成为流浪的农民工
</p>
<p>
 我的言论自由
 <br/>
 被共产党剥夺了
 <br/>
 到处是民怨的声音
 <br/>
 我成为监狱里的政治犯
</p>
<p>
 我的父母
 <br/>
 被超时的工作压垮了
 <br/>
 家里没有钱
 <br/>
 去医院看病
</p>
<p>
 我的妹妹
 <br/>
 被穷苦所逼迫
 <br/>
 为了生活
 <br/>
 她到城里当妓女
</p>
<p>
 我的儿子
 <br/>
 考上了重点大学
 <br/>
 我的妻子付不出学费
 <br/>
 她上吊了
</p>
<p>
 我的田地
 <br/>
 今年又欠收
 <br/>
 我交不起杂税
 <br/>
 只好喝农药自尽
</p>
<p>
 西伯利亚的寒流
 <br/>
 吹遍了东北
 <br/>
 美丽的丹顶鹤
 <br/>
 找不到鱼虾吃了
</p>
<p>
 塔克拉玛干沙漠的风
 <br/>
 吹遍了西北
 <br/>
 罗布泊里的胡杨木
 <br/>
 在干渴中哭泣
</p>
<p>
 青康藏高原的雪
 <br/>
 布满了西南
 <br/>
 拉萨的月光
 <br/>
 在黑暗中流泪
</p>
<p>
 内蒙古的大草原
 <br/>
 变成了白色的沙漠
 <br/>
 马头琴的曲调
 <br/>
 在苦难中呻吟
</p>
<p>
 东海的波涛
 <br/>
 想要追逐自由的阳光
 <br/>
 人蛇船里的偷渡客
 <br/>
 被蛇头狠心推入大海
</p>
<p>
 黑色的云层飞呀飞
 <br/>
 肮脏的空气飘呀飘
 <br/>
 污染的江河流呀流
 <br/>
 有毒的食物卖呀卖！
</p>
<p>
 十五的月亮圆又圆
 <br/>
 河边的花儿朵朵艳
 <br/>
 农家的妹妹呀
 <br/>
 你为什么哭呀哭？
</p>
<p>
 何处飞来的沙尘暴
 <br/>
 挡住了我的太阳？
 <br/>
 到底是什么样的狂风
 <br/>
 一直在我的心中呐喊？@
</p>
