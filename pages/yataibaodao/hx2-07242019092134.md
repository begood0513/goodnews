 <p>
  美国国会议员计划推动《加强西藏政策法》，对干涉西藏流亡精神领袖达赖喇嘛继承人的确认或转世过程的中国官员，进行金融、经济和与签证相关的制裁。达赖喇嘛驻台代表达瓦才仁认为此举意义重大。
  <br/>
  <br/>
  美媒报导，美国国会及行政当局中国委员会主席、民主党众议员麦高文（Jim McGovern）日前参加纽约流亡藏人社区的一个集会时宣布，下周将在国会提出新的跨党派《加强西藏政策法》。法案将明确规定，凡是干涉达赖喇嘛继承人的确认或转世过程的中国官员“都将受到《全球马格尼茨基法案》金融、经济和与签证相关的制裁”。
  <br/>
  <br/>
  报导称，麦高文说，“该法案将制定一项美国政策，即包括第十四世达赖喇嘛在内的西藏佛教领袖的继承或转世是完全的宗教问题，只应由西藏佛教界来完成。”
  <br/>
  <br/>
 </p>
 <p>
 </p>
 <p>
  中国当局近日重申，达赖喇嘛不可按个人意愿转世，转世须由中国认证，也必须在中国转世。达赖喇嘛本月初过84岁生日时，明确指出，这一世达赖喇嘛的转世或重生的决定权，绝对不会落入中国手中。
  <br/>
  <br/>
  藏人行政中央和达赖喇嘛驻台代表达瓦才仁24日接受自由亚洲电台采访表示，西藏人决定自己的宗教事务，本来是天经地义，却遭受中国政府的干预。如果美国国会立法成功，将引发国际关注，促使中共不要肆无忌惮侵犯藏传佛教，能起到非常大的作用。
  <br/>
  <br/>
  达瓦才仁说：“它（麦高文拟提的法案）不是泛泛而谈地说不能允许（控制达赖喇嘛转世），而是对当时这些相关的人，直接提出追责、追究的措施，就可以具有实际操作上的便利性。”
  <br/>
  <br/>
  那么，有人会问美国订定这项法律，能对北京强行控制达赖喇嘛转世，有何实质约束力？
  <br/>
  <br/>
  <b>
   达瓦才仁：立法可揭穿“西藏的门为全世界打开”的谎言
   <br/>
  </b>
  <br/>
  达瓦才仁指出，中国政府向来不希望这些事情被外界讨论、关注，就像不让外国人去西藏，好把西藏问题锁起来、封闭起来，这样他就可以在里面为所欲为。
  <br/>
  <br/>
  达瓦才仁说：“（美国国会）对这个事情本身的讨论、关注，以及对此的动作，这些事情本身就否定了中国政府所宣扬的，中央政府可以决定西藏宗教事务的说法。中国政府本身的行为是强盗，那强盗他会做他自己的事。但是，大家都知道，那是强盗的行为，而这个是非常重要的。”
  <br/>
  <br/>
  对于美国议员麦高文提到，《加强西藏政策法》法案还有一项内容是说，“除非美国能设立驻拉萨领事馆，否则不应该允许中国在美国增加新的领事馆。”
  <br/>
  <br/>
  达瓦才仁认为，这项立法更可以揭穿中国政府总是厚颜无耻对外宣称“西藏的门永远是向全世界敞开”的这个谎言。而《加强西藏政策法》展现美国对西藏的关心，以及美国希望在拉萨建立领事馆，而中国不愿意，这更会向世人证明，中国对西藏采取铁幕、封锁政策，是不人道、不具正当性的作法。
  <br/>
  <br/>
  而对于中国官员多次重申达赖喇嘛转世须由中国认证，也必须在中国转世，达瓦才仁强调，达赖喇嘛转世完全是佛教事务。
  <br/>
  <br/>
  “中国共产党根本不是宗教信徒、不是佛教徒，而是无神论者。更主要的是，西藏在历史上是一个独立国家，跟中国一点关系都没有，只有一些宗教上的信仰、信徒这样的关系而已。所以说，中国政府这样的说法，就像痴人说梦一样。就像他在南海随便划一条线就说是他的领土一样。”
 </p>

 <p>
  <b>
   台援藏团体：中共不相信宗教却要决定宗教事务相当荒谬
  </b>
  <br/>
  <br/>
  西藏台湾人权连线理事林欣怡接受自由亚洲电台采访也说，中共不相信宗教却打压各种宗教，还强行决定宗教事务，在国际看来是非常荒谬的事。中共过去在西藏也做过相同的事。
  <br/>
  <br/>
  林欣怡：“针对班禅喇嘛，他就是找了一个假的班禅喇嘛，我们当然相信他们会用更恶劣的手段去控制第十四世达赖喇嘛转世。现在美国国会愿意用一个比较强硬、有力量的方法警告中国不应该做这样的事，我觉得这是很好的事情。”
  <br/>
  <br/>
  林欣怡呼吁其它国家包括台湾，应该在中国干预控制达赖喇嘛转世问题上，提出具体反制措施声援西藏。
  <br/>
  <br/>
  自由亚洲电台记者夏小华台北报道  责编：陈美华、申铧  网编：洪伟
