 <p>
  过去一个多月以来，香港市民反送中运动规模巨大，震惊世界。有亲北京的香港媒体报道说，中国正在全面重新考量治理香港的新策略，以解决当今的政治危机。有分析人士认为，明年一月台湾总统大选之前，北京不会有新的大动作。
  <br/>
  <br/>
  香港英文《南华早报》星期四报道说，香港局势的发展超出了北京的预期，使中共高层大为震惊。他们对在香港的情报系统未能及时准确地掌握社会民情感到诧异和不满。报道引述不透露姓名的知情人士说，目前国务院港澳办正在搜集意见，拟提出一个所谓新的治港策略，以解决目前的政治危机。报道强调，直接派出解放军，不会在这个新策略当中。

  香港科技大学社会学副教授成名分析说，北京在香港的情报系统和舆情搜集单位，确实低估了香港民间的力量。
  <br/>
  <br/>
  “过去一个多月，不同的媒体对这个的报道都是一致的，就是北京对香港局势发展到现在，完全超乎预料之外的。就是指情报部门过于乐观，看不到香港波涛汹涌，很多不满。”
  <br/>
  <br/>
  他认为，《南华早报》所报道的“不动用解放军”，其实早已有所报道，这次只是再次确认而已。
  <br/>
  <br/>
  不过，香港政经评论人士桑普认为，在危机时刻动用军队，是中共不可能放弃的选项。所谓不动用军队，只是北京认为香港的警察部队已经足够。他也认为，在习近平的统治下，中共对港政策恐怕不会大改变，最多只是作法上的调整而已。

  “我不认为中国共产党会对香港有一个怀柔的政策，有些人认为，他们会改弦更张，回到原来一国两制的方式，但我认为这不太可能。”
  <br/>
  <br/>
  《南华早报》的报道还引述“了解内情的人士”提出了所谓“引蛇出洞”的说法，就是说北京正在等待对手暴露企图和战略。报道分析说，北京认为发生在香港的事件并非孤立事件，背后有外国势力操纵。
  <br/>
  <br/>
  桑普分析说，这只是显示北京在处理香港问题时比较顾忌国际社会影响而已。他也认为，明年台湾 大选之前，北京可能不会采取强硬措施。
  <br/>
  <br/>
  “台湾明年一月有选举，如果打香港太厉害，等于帮蔡英文助选。因为一国两制被唾弃得很厉害，这个影响会很大。所以我相信，在明年1月11日之前，（北京）不会动大招。”
  <br/>
  <br/>
  有关报道认为，香港特首林郑月娥短期内不会下台，以免导致局势失控。但两位评论人士都认为，林郑月娥已被北京视为导致问题恶化的关键人物，迟早会被追责落马。
  <br/>
  <br/>
