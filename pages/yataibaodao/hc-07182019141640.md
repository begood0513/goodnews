 <p>
  近日，多位中共领导人陆续到中国各地调研，被认为是今年北戴河会议即将举行的征兆。受海内外政治和经济方面的多重压力影响，这场闭门会议将会讨论的议题引发了不少揣测。
  <br/>
  <br/>
  近一段时间，多位中共领导人陆续离京进行地方基层调研。周一，中共中央总书记习近平来到内蒙古赤峰考察民情，了解当地经济社会发展和生态文明建设。此外，中国国务院总理李克强、国家副主席王岐山、全国政协主席汪洋、中纪委书记赵乐际、副总理韩正等人分别前往了辽宁、黑龙江、新疆、上海等地调研。
  <br/>
  <br/>
  虽然中共中央政治局常委成员在入夏时节纷纷走入基层还未形成惯例，但舆论普遍认为，这预示着中共党内正在为近期即将召开的北戴河会议作最后准备。
 </p>

 <p>
  值得注意的是，就在两周前，习近平出席了在京召开的深化党和国家机构改革总结会议。他2013年曾在中共第十八届三中全会上，把全面深化改革的总目标定为“推进国家治理体系和治理能力现代化”，也被外媒称为继“工业、农业、国防、科技现代化”之后中国的“第五个现代化”。
  <br/>
  <br/>
  新华社本月引述广东省社科院研究员丁力说，这项改革提升了党的统筹决策能力，也让党的意志在党政工作上体现得更加充分。
 </p>
 <p>
  北京独立媒体人高瑜表示，习近平前些年提出的“第五个现代化”颇具讽刺意味，其目的无非是维护一党专政。他在北戴河会议前夕为党政机构近一年半来的改革成就点赞，也是为了强调他“定于一尊”的地位。
  <br/>
  <br/>
  “（主持会议的中共中央书记处书记）王沪宁提出，机构改革的目标就是要维护习核心和党中央。我认为中共目前不光没有在推动民主，而且在经历一场逆向大转折。”
 </p>

 <p>
  高瑜强调，今年北戴河会议的重心是贸易战和香港问题，而中共领导人会千方百计地拖延这些棘手问题。
  <br/>
  <br/>
  自从五月初，中国政府推翻了与华盛顿就贸易协议达成的一系列承诺后，两国重陷关税泥沼。
  <br/>
  <br/>
  中国国家统计局周一公布的数据显示，今年第二季度，中国经济同比增长6.2%。这个数字创下了1992年政府记录季度增速以来的最低点。
 </p>
 <p>
  高瑜认为，双方达成贸易协议无疑会对美中两国经济起到提振作用。但她说，中方仍然存在侥幸心理，认为他们能够挨到明年美国大选，但眼观目前国内经济形势，他们的想法未免太过天真了。
  <br/>
  <br/>
  “双方还在谈判。还在谈就说明中国经济的确面临下行压力。但是中方存在一个侥幸心理，那就是拖垮特朗普明年的总统大选。”
  <br/>
  <br/>
  如果说美中贸易谈判已经发展成一场打打停停的持久战的话，那么香港过去几个月来爆发的群体性抗议活动仍在轰轰烈烈地上演。
  <br/>
  <br/>
  迫于巨大的公众压力，香港行政长官林郑月娥已经宣布《逃犯法案》已经寿终正寝。但是抗议人士并不买账，抗议示威仍在持续进行。
 </p>
 <p>
  美国普林斯顿中国学社执行主席陈奎德说，香港近日的风波给北京当局出了一个大难题，这或许会成为北戴河会议的另一大焦点。
  <br/>
  <br/>
  “（香港事务）也涉及到习近平对自身地位的评估。他如果觉得这个问题已经影响到了他在党内的地位和形象，那么恐怕会（在北戴河会议上）集中来谈。但如果他感觉到谈起来反而会进一步扩大党内分歧的话，他就会想回避香港问题。”
  <br/>
  <br/>
  据悉，河北省委书记王东峰等人上周前往北戴河“调研检查”，这似乎也是北戴河会议临近的一个征兆。
  <br/>
  <br/>
