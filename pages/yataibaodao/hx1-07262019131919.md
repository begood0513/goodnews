 <p>
  台湾的民进党秘书长罗文嘉上周赴印度拜会了西藏流亡精神领袖达赖喇嘛。罗文嘉说，达赖喇嘛表示想念台湾。至于是否邀请达赖喇嘛来台，罗文嘉表示，此行为祝寿，没有邀请尊者访台，不过建立了直接的联系管道。
  <br/>
  <br/>
  台湾的民进党秘书长罗文嘉7月17至19日间率民进党公关处主任谢雨利、发言人李问前往北印度达兰萨拉，拜会西藏精神领袖达赖喇嘛，也与西藏流亡政府司政洛桑森格会面交流。
 </p>
 <p>
  <br/>
  今年5月底宗教和人权团体力邀达赖喇嘛到台湾出席国际宗教自由会议未获准。民进党此时首次率团拜会达赖喇嘛，在26号记者会上，记者询问罗文嘉是否当面邀请达赖喇嘛访问台湾？
  <br/>
  <br/>
  <b>
   罗文嘉：达赖喇嘛说他年纪大 不适合长程飞行
  </b>
  <br/>
  <br/>
  罗文嘉回答说：“没有，我们在会谈时候，尊者达赖喇嘛自己讲到，他第一次来台湾是1997年，他非常喜欢台湾，也很想念台湾。后来他也提到说，他自己现在年纪比较大了，所以他不太适合做太长程的飞行。”
  <br/>
  <br/>
  罗文嘉强调，他这次不是代表政府或蔡英文总统，只是代表民进党主席去拜会达赖喇嘛，而且民进党不是“以党领政”，也不是“以政领党”，各有主体、和现阶段要做的事。
  <br/>
  <br/>
  罗文嘉转述他与达赖喇嘛会面时的谈话，表示他没有明确提出邀请，不过他认为彼此算已有表达这方面的看法，比方说达赖喇嘛不会说“我要去”，而是说“我很怀念台湾，我很喜欢台湾”，他则提到：“台湾很多人喜欢他、很想念他。”
 </p>

 <p>
  <b>
   罗文嘉：达赖喇嘛说“我们两边都有严格上师指指点一点”
  </b>
  <br/>
  <br/>
  罗文嘉提到达赖喇嘛一见到他就说：“他一见到我，还没坐下来谈，因为他现在走得比较缓慢一点，他站好之后跟我说，我们两边都面临有一个很严格的上师，常常在那边指指点点，说这个可以、这个不可以。后来他说，讲到来不来？如何地来？他也希望说不要给太多的困扰。”
  <br/>
  <br/>
  罗文嘉说，这次至少建立很好且直接的管道，以便于未来若有任何达赖喇嘛访问台湾的细节方案，进行更周延的连络和讨论。
  <br/>
  <br/>
  罗文嘉说，达赖喇嘛谈到转世议题，强调1969年就发表声明，转世制度是否延续由佛教徒决定，不是由非佛教徒、共产党决定，跟共产党没有关系。
  <br/>
  <br/>
  罗文嘉转述说，达赖喇嘛甚至主动说，共产党在文革时，想彻底摧毁佛教但没有做到。毛泽东死后，也没有办法毁灭佛教，所以转世制度跟共产党一点关联没有，如果共产党真的那么关心转世问题，应该先去找毛泽东的转世。
 </p>

  <b>
   罗文嘉：是否可能转世到台湾？  达赖喇嘛：有可能
  </b>
  <br/>
  <br/>
  罗文嘉说，谈到这里大家都笑了，因为在很愉快的气氛下，他进一步问达赖喇嘛：“那如果你转世有没有考虑来台湾呢？转世到台湾？达赖喇嘛说，当然有这个可能，因为台湾有很多佛教徒。我也跟他说，如果他真的来台湾转世，台湾一定很多人期待。那后来他说，他现在84岁，他预估自己应该可以活到100岁，所以这个问题，他觉得目前还不是很着急。”
  <br/>
  <br/>
  罗文嘉说，达赖喇嘛提到，四世达赖喇嘛在蒙古出生、六世达赖喇嘛出生在印度边境。达赖喇嘛表示，他要帮助70亿人口，要做到这点，一定要在自由的国家。
  <br/>
  <br/>
  罗文嘉还说，达赖喇嘛提到，中国农村现在非常辛苦，有见过来自河南的农民提到农村腐败的情况，他很难过。曾有一名以色列总理对他说，中国现在已经是最坏的资本主义。
  <br/>
  <br/>
  对民进党秘书长罗文嘉率团拜会达赖喇嘛，以及在民进党记者会上转述达赖喇嘛的相关说法，达赖喇嘛和藏人行政中央驻台代表达瓦才仁对自由亚洲电台说，不表示意见。
  <br/>
  <br/>
  已经连续11年在台北为达赖喇嘛举行和平祈福音乐祝寿会的台北市甘丹东顶显密佛学研修协会创会会长刘金钗则说，达赖喇嘛现在身体很好，今年到印度邀请达赖喇嘛和台湾信徒在祝寿会上视讯时，达赖喇嘛也说想到台湾吃生日蛋糕。

  <b>
   台佛教徒：达赖喇嘛不能来台湾  还问他转世到台湾？
  </b>
  <br/>
  <br/>
  刘金钗说：“你如果让他来，他会有准备。不要借口说他不能来，一直推拖他身体不好，所以他不能来，因为你都没有显示出你的诚意邀请他来，你怎么可以说他身体不好。他当然会很客气，而且现在台湾在总统选举，他又不方便讲什么，他当然会说医师有建议什么，但如果是短程（旅行）的，他还是可以的。”
  <br/>
  <br/>
  刘金钗认为，重点不在于达赖喇嘛的身体，而在于蔡政府跟中共之间的问题，她要面临这个抉择。事实上达赖喇嘛2010年就交出政治的责任，他已经是宗教领袖，跟政治无关。

  对罗文嘉提到他问达赖喇嘛有没有可能转世到台湾？刘金钗说：“现在他可以来你都不让他来，你还要请他转世到台湾，那不是废话吗？”
  <br/>
  <br/>
  罗文嘉在记者会上，被问到藏人行政中央司政洛桑森格自上台后就无法入境台湾的问题时表示，民进党一向欢迎爱好自由人权的朋友到台湾，并坚守自由民主的价值。自2011年流亡政府选出民选总理（司政）后，民进党当时就发布新闻稿，欢迎流亡政府包括总理（司政）到台湾，民进党一直没有变。不过此次未触及到邀请洛桑森格到台湾的议题。
