 <p>
  美国总统特朗普周三在白宫会见了来自世界各国的受迫害宗教人士。有四名在中国受到打压的宗教人士参与了会见。会见期间，他们与特朗普总统到底谈了什么？
  <br/>
  <br/>
  特朗普总统这次会见了来自17个国家的27名受迫害宗教人士，其中有四人来自中国，他们都是来华盛顿出席国务院主办的第二届促进宗教自由部长级会议，并在会议上介绍了他们的经历。
 </p>
 <p>
 </p>
 <p>
  被中国政府判处无期徒刑的新疆维族异议人士、经济学家伊力哈木·土赫提的女儿菊尔·伊力哈木，向本台记者讲述了特朗普与她的简短对话，
  <br/>
  <br/>
  “最开始我说，有一到三百万维吾尔人被关在中国的集中营。然后他问我，在哪里？然后我说，在新疆。然后他又问在哪里？我说，在中国的西部。他说，明白了。其实他并没有过多地评论。”
 </p>
 <p>

  菊尔目前在美国印第安纳大学就读。她很遗憾这次没有更多的时间介绍新疆上百万穆斯林民众受到无辜关押的情况:
  <br/>
  <br/>
  “期待将来能有更多机会，并且能更充分、更详细地解说这个事情，并且能让美国采取行动。”
 </p>
 <p>
  法轮功学员张玉华也出席了白宫的会面，她向特朗普总统建议，对中共统治者要采取行动，仅仅说说并没有效果。
 </p>

 <p>
  贵州基督教活石教会的信徒欧阳曼萍也是带着一肚子故事去了白宫：
  <br/>
  <br/>
  “我就想说我们教会发生的事情，还有我们中国的很多教会被取缔，很多弟兄姐妹被抓捕，反正中国基督教被逼迫得很严重。”
  <br/>
  <br/>
  英语不好的欧阳虽然不能直接向特朗普总统讲述中国基督徒的遭遇，但她依然很开心。
  <br/>
  <br/>
  近年以来，美国政府对中国人权问题的强调超过以往。今年三月，美国国务院发布了《2018人权国别报告》，国务卿蓬佩奥在报告发布会上说，中国在侵犯人权的问题上“无人可比”。
  <br/>
  <br/>
  本周四，蓬佩奥又在“促进宗教自由部长级会议”上说，中国对维吾尔族人的做法是“世纪的污点”。
  <br/>
  <br/>
  在美中贸易争端久拖不决的背景下，美国对中国人权问题的强调被人质疑是用于制约中国的一张牌。中国官媒《环球时报》主编胡锡进在美国社媒Youtube上对此发表评论说：
  <br/>
  <br/>
  “打宗教自由这张牌一直是美国对外施压的传统保留项目，不过这手牌越打越烂。本届美国政府尤其显得很滑稽。他们所支持的宗教受迫害者，恰恰都是那些打着宗教旗号，从事极端政治活动的人。”
 </p>

 <p>
  但在周三与被迫害宗教人士的见面会上，特朗普并没有就中国人权问题明确表态。菊尔·伊力哈木认为，在美中贸易僵持的情况下，作为美国总统，他不会轻易表态。
  <br/>
  <br/>
  菊尔·伊力哈木本次参与“促进宗教自由部长级会议”，是她第一次在公共场合谈及维族人权问题。她为此感到很受鼓舞：
  <br/>
  <br/>
  “我们之前一直的诉求是让世界关注维吾尔人的情况，现在我们的目标达到了。现在，我的诉求就是让更多的人意识到，众多维族人关在集中营里这件事情的重要性和关键性，然后鼓动大家行动起来，而不只是嘴上说说。”
  <br/>
  <br/>
  在监狱中去世的西藏政治犯丹增德勒仁波切的外甥女尼玛拉姆也向本台记者介绍了她在这次会议上的呼吁，“今天这个会议上，很多不同国家的代表来，我就要求他们支持仁波切提出的宗教自由，还有就是支持质疑仁波切的死因。我还希望藏族里象我舅舅那样的罪犯能得到释放。”
  <br/>
  <br/>
  彭佩奥周四上午在会议上宣布，将组建一个新的国际组织用以推动世界范围内的宗教自由。这也是因应形势采取的行动。自去年第一届“促进宗教自由部长级会议”以来，英国、台湾和阿联酋等国已经召开了相关的会议。
  <br/>
