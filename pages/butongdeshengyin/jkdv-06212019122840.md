  <strong>
   【这几天又冒出了东方之珠党和东方之猪党，他们之间发生了点事：香港人感觉到自己的生活已不像从前那么自由自在，并担心未来会越来越不自在，于是不断以各种形式抗议；而一向习惯了情绪稳定的内地人，被别有用心者的手段一挑拨言语一刺激，立马情绪不稳定了，放出话说：你们这些忘恩负义的家伙，俺马上给你断水断电，猪肉供应也给你切断，看你咋活，整得自己像一家之主似的…能蠢到这个地步，脑子里的湿气必然是让海风吹拂了五千年，每一滴泪珠仿佛都是脑子里溢出来的水…换句话说，香港离了内地还能活，谁离了谁也不会死，你离了尊严和权利这么多年，不也在圈里活得膘肥体壮吗…以后咱也千万别说什么“你嫌这不好嫌那不好就滚出这个国家！”，说得自己好像翻身做主人了一样，是让你翻身了，但只是老汉让你换了个姿势推车而已——《东方之猪 整圈未眠》王五四】
   <br/>
   <br/>
   ●题评●
  </strong>
  <br/>
  <br/>
  王五四，网络大V，因微信公众号上的辛辣犀利时评文章被中国大陆网友熟知，女权研究者曾金燕将王五四比作微信时代的韩寒。“他把对政治人物和一些公共政策的批评隐藏在文章中，直指社会的强势的作恶，及对这些强势群体的调侃”——温云超   “王五四发出的是另外一种标新立异的声音，他的话语完全颠覆了以前那些大V们启蒙性的话语，也解构了体制的话语霸权。” ——野渡
  <br/>
  <br/>
  6.9及6.12三百余万港人的反共大游行，深层次撼动了一国两制治国方针的可行性和正当性；深层次震慑了北京习近平政权的极度思想控制领域和高压舆论监管空间；尤其是，深层次触犯了看似铁板一块，被鲍彤喻为“一切行动听指挥的，十四亿生龙活虎的，有力量向普世价值宣战向人类文明宣战的”，几乎全体苟且偷生的大中华民众灵魂深处的命运共同体。
  <br/>
  <br/>
  生存还是毁灭？这是个问题。

 <p>
  大陆受众莫怪本节目劈头棒喝，这周“猪流感人畜共患”，【不同的声音】“涉嫌”的其实是传播，几天前重度感染了“反送中周”的一条『今日推特金句』：
  <br/>
  <br/>
  <strong>
   “某国改个宪法，鸦雀无声。某地改个条例，地动山摇。仅隔一条河，天渊之别：一边是东方之珠，一边是东方之猪。”
  </strong>
  <br/>
  <br/>
  “东方之猪”荣登最新网络敏感词！
  <br/>
  <br/>
  “珠猪金句”也在推特中文圈带出了争论。
  <br/>
  <br/>
  “珠猪”并非呱呱坠地于反送中周。
  <br/>
  <br/>
  举世皆知，大上海的外滩有一座都会地标“东方明珠塔”。6年前，黄浦江畔突现“万头死猪氽江浮尸”，进而成为全世界侧目蔚为壮观的超级大丑闻。而上海人呢，“才饮之江水，又食黄浦鱼”，没事人似的附和着政府“黄浦江水质安全达标”的世纪谎言，以至于危机期没有发生过哪怕一次最细小的群体性抗议示威维权活动。他们的愤怒，仅止于茶余饭后的无奈调笑，仅止于！或因：2013年3月上旬，是那尊卸任上海市委书记的大位登基黄道吉日？或因，那万枚满溢着藐视的红顶猪头浦江豪华游，起始于当今圣上中南海政治生命的发祥地——钱塘之江？
  <br/>
  <br/>
  但无论如何，
  <strong>
   东方之猪，整圈未眠。
  </strong>
  <br/>
  <br/>
  那一刻的上海，还是艰难地催产了“东方名猪”。
  <br/>
  <br/>
  “谐音革命”开始频频于互联网政治笑话类调侃领域。王五四老师的上述奇文，恰恰诞生在那几年的“涉沪语境”之中。
  <br/>
  <br/>
  勿论是怒其不争的火攻激将法，还是真格对深层次冥顽不化民族劣根性的彻底绝望，反送中推特“猪珠金句”，看似贴贴切切地击中了大多数中国人的“灵魂七寸”。
  <br/>
  <br/>
  特别有意思的是，【不同的声音】这一波相关随机采访中的几位中国大陆抽访者，“珠猪金句”人尽皆知。他们言谈中不断流露出面对香港人的英勇“惭愧之极”“没脸见人”等自我贬抑，让人五味杂陈，百感交集。
  <br/>
  <br/>
  首先进入节目的是深圳的越战老兵任先生。
  <br/>
  <br/>
  社媒“珠猪”之争，上海民族党创党主席，名博【习总日记】博主何岸泉医生，在激情度升至沸点的“反送中狂热”全民性中国批判巨浪下，于6.16港岛200万人大游行翌日，在推特上泼了一盆难能可贵的冷水：
  <br/>
  <br/>
  “虽然都是维权，但大陆人下跪是安全的，香港人游行是安全的。 难道你真的不知道吗？ 批评大陆人可以，但要实事求是。 无限拔高香港人，也是对香港民族的不敬。”
  <br/>
  <br/>
  节目的中段，不同的声音与人在纽约的何大夫有一段短短的交流。
  <br/>
  <br/>
  节目的最后，来自河北的朱医生对本周的话题发表了自己的看法。
  <br/>
  <br/>
  上述两位中国大陆受访者均要求略去全名仅留姓……
