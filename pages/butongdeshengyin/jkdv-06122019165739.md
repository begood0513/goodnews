 <p>
  香港特区政府提出修订《逃犯条例》，引发市民自03年反对23条立法后最强烈的震撼。由香港“民间人权阵线（民阵)” 发起的6.9“反‘送中’大游行”，游行人数达103万，为97后示威人数新高。
  <br/>
  <br/>
  大游行翌日，香港特区行政长官林郑月娥于周一表示，她不打算撤回对此项有争议条例的修订。
  <br/>
  <br/>
  市民对此表态反应激烈，部分市民发起并实施了一百多家商户和大批学生参与的6.12罢工、罢市、罢课的“三罢”行动。
  <br/>
  <br/>
  本台报道，香港立法会本周三6月12日恢复审议《逃犯(修订)条例草案》，近万名抗议者遭到警方驱散。香港警务处处长指事件是“骚乱”。当局出动防暴警察向示威者发射催泪弹及橡胶子弹。
  <br/>
  <br/>
  当天下午三点，警方开始清场行动，4点许，警察向示威者发射大量催泪弹和橡胶子弹。

 <p>
  CNN报道：六月十二日的倾盆大雨无法阻止港人抗议。抗议民众手举着雨伞在倾盆大雨中立委会外抗议，而他们手中的雨伞又是2014年的雨伞运动的标志。有年轻人在抗议民众中分发水、食物和防止催泪喷雾袭击的眼镜、其他防暴装备以及医疗用品，这说明了这次抗议的年轻组织者们为这次抗议做了很多准备，香港人示威初见成效，《逃犯条例》立法会二读推迟。对此立法委员张超雄（Fernando Cheung ）对抗议民众表示，他们创造了一个奇迹。他说：“因为你们今天站了出来，这个立法会议才被延迟，但是我们仍然需要更多的人站出来。”
  <br/>
  <br/>
  《逃犯条例》修订草案原定6月11日立法院二读，预计下周四 (6月20日) 进行表决。当下事态犹如干柴烈火，未来几天的香港，充满一触即发的不定因素。
  <br/>
  <br/>
  【不同的声音】6.12罢市围院第一时间当日傍晚，采访了三位参与者，他们是：『爱画室』的姚先生，『MINI CON CITY』做隐形眼镜生意的赖先生，以及下半部分进入节目的金鐘行动参与者，『生活书社』的叶女士。
  <br/>
  <br/>
  由于整个下午至入夜，港岛金鐘立法院周边地带因示威和大规模出警遭局域性断网，6.12万民围院现场无法展开实时电话采访。直到中国时间晚上9点之后，参与金钟和平抗议行动的大多数香港市民安全回到家中，【不同的声音】才拨通了其中的一位6.12金鐘行动参与者，『生活书社』的女老板叶泳琳。
 </p>
 <p>
  本周节目三位受访嘉宾，均来自网上一百余家6.12反送中罢市商户的谷歌搜索结果，是一次无特定人物的随机性访谈。但节目完成之后的再搜索中发现，“随机”成了“天机”。下半部分的受访嘉宾，『生活书社』女老板叶泳琳和她的生意搭档鍾耀華,竟然是香港社运界名人，二人均为前学运领袖，是深度卷入”雨伞运动的占中灵魂人物。锺耀华在“占中九子”中排名第七，香港法院两个月前刚刚判处锺耀华八个月入囚，缓刑两年。
 </p>
