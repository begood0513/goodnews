 <p>
  <strong>
   1
  </strong>
  <strong>
   、男子和女郎
  </strong>
  <br>
   <br>
    一位年轻男子，遇见一位美丽的女郎，顿生爱慕之心，就默默而行，跟着她走。
    <br>
     <br>
      这位女郎忍了好久，问他说：你为什么跟着我？
      <br>
       <br>
        他诚恳地说：你是我所见过最美丽、有气质的女性，请你当我的女朋友，好吗？
        <br>
         <br>
          女郎回答：我妹妹更漂亮！你现在回头，就能看到她。
          <br>
           <br/>
           男子听了，马上转身向后看，但看到的只有相貌平凡的普通女子而已。
           <br/>
           <br/>
           他质问：你为什么骗我？
           <br/>
           <br/>
           女郎说：如果你真喜欢我，为何还要回头看呢？是你在骗我啊！回答完接着转身快步离去。
           <br/>
           <br/>
           启示：游移不定的
           <span href="https://www.secretchina.com/news/gb/tag/目标" target="_blank">
            目标
           </span>
           只能让你前功尽弃、一无所获！生活如此，人生也如此。
           <br/>
           <br/>
           <strong>
            2
           </strong>
           <strong>
            、狮子和狐狸
           </strong>
           <br/>
           <br/>
           一只老狮子病了，躺在洞穴中呻吟不止，附近的一些动物听到声音后，纷纷进洞探视。
           <br/>
           <br/>
           狐狸知道了这消息，也去了。走到洞穴前，听到老狮子呻吟声越来越大，可怜之极。就在准备进去的时候，狐狸忽然收回已经跨进洞穴的前脚，竖起耳朵，在洞穴周围走来走去。
           <br/>
           <br/>
           老狮子在洞里看见狐狸迟迟不进洞，忍不住问狐狸说：“狐狸啊！既然到了，你为什么不进洞来呢？”
           <br/>
           <br/>
           狐狸回答：“我只看到往里走进去的动物脚印，却看不到往外走出来的脚印，我怎么敢进去呢?！”
           <br/>
           <br/>
           启示：凡事“进易退难”,要有对环境的观察力，对社会的敏锐度，谋定而后动，才能立于不败之地；率性莽撞的作为，则很可能会处于万劫不复的险境。
           <br/>
           <br/>
           <strong>
            3
           </strong>
           <strong>
            、农妇和鸡
           </strong>
           <br/>
           <br/>
           一位农妇准备杀鸡，就在晚上喂鸡的时候，不经意地说：吃吧，吃吧，多吃点，这是你最后一顿了。
           <br/>
           <br/>
           第二天，农妇看到，鸡躺倒地上已经死掉了，还留下一封遗书：我吃老鼠药，自己死掉了。你们别想吃我了，我可不是好惹的。
           <br/>
           <br/>
           当对手知道了你的决定之后，就能做出对自己最有利的决定。——纳什均衡理论。
           <br/>
           <br/>
           启示：保密、信息安全很重要。
           <br/>
           <br/>
           <strong>
            4
           </strong>
           <strong>
            、驴和狗
           </strong>
           <br/>
           <br/>
           农村人家养了一头驴和一条狗。
           <br/>
           <br/>
           白天，驴下地干活；晚上，狗看家护院。
           <br/>
           <br/>
           话说这夜，有贼潜入家里，要偷东西，狗偷懒没叫，驴着急了，对狗说：来贼了，你快叫醒主人吧。狗却不做声，照常睡觉。驴想要惊醒主人，以免家里东西被偷，就大声地叫了起来。
           <br/>
           <br/>
           果然，主人很快拿着木棍，从屋里跑了出来，照着驴身上就是一顿乱打，骂道：你瞎叫唤什么？大半夜的，还不让人睡个安稳觉！
           <br/>
           <br/>
           贼被吓跑了，驴也挨了一顿打。
           <br/>
           <br/>
           次日，驴总结出教训：干好自己该干的活，以后少管闲事！
           <br/>
           <br/>
           启示：太精辟，太生动，太形象。应用于实践，价值无比！
