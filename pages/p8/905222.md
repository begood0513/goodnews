 <p>
  为什么要写这封信，有两个原因：首先，你正站在人生的关键点——身后是悄然溜走的童年，身前是生机盎然的青春，你得知道怎么前行。其次，你在长大，父母却在老去，陪伴不了你一辈子；有些话，父母不跟你说，也许没人会跟你提起。下面的十个嘱讬，要听好：
  <br>
   <br>
    <strong>
     1
    </strong>
    <strong>
     、关于
     <span href="https://www.secretchina.com/news/gb/tag/目标" target="_blank">
      目标
     </span>
    </strong>
    <br>
     <br>
      人，可以没有远大的理想，但生活不能没有目标。明确的目标就如同指示灯，照亮着你前行的道路。
      <br>
       <br>
        <strong>
         2
        </strong>
        <strong>
         、关于定位
        </strong>
        <br>
         <br>
          只有你才真正地了解自己，做自己就是倾听自己内心声音，找到什么才是最适合自己的事。每个人都是独一无二的，只有自己才能了解自己适合做什么、不适合做什么。
          <br>
           <br/>
           <strong>
            3
           </strong>
           <strong>
            、关于学习
           </strong>
           <span id="hideid" name="hideid" style="color:red;display:none;">
            <span href="https://www.secretchina.com">
            </span>
           </span>
          </br>
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  学习并不是唯一。
  <br/>
  <br/>
  有这样一种说法：20岁比学历、30岁比能力、40岁比阅历、50岁比财力、60岁比体力、70岁比病历、80岁翻黄历。
  <br/>
  <br/>
  人生几十载，归根到底就是比身体。学习要好，身体要锻炼、爱护。
  <br/>
  <br/>
  <strong>
   4
  </strong>
  <strong>
   、关于
   <span href="https://www.secretchina.com/news/gb/tag/未来" target="_blank">
    未来
   </span>
  </strong>
  <br/>
  <br/>
  一个人的未来由德和才决定的。德是由诚实、助人、宽容、忠诚、责任组成，才是由知识、能力、态度组成的。未来，我希望你：德才兼备，以德优先。
  <br/>
  <br/>
  <strong>
   5
  </strong>
  <strong>
   、关于自己
  </strong>
  <br/>
  <br/>
  在学校，你要尊重老师，要关心同学，才能以心换心，换得老师和同学对你好；在你的一生中，除了我和你爸，没有人有义务要对你好。
  <br/>
  <br/>
  世界少了自己，地球还会转；但也不要以为个人渺小，就放弃自我。
  <br/>
  <br/>
  <strong>
   6
  </strong>
  <strong>
   、关于朋友
  </strong>
  <br/>
  <br/>
  要找比你强大的人做良师益友，尤其记住，在朋友落难时，一定要施以援手。
  <br/>
  <br/>
  因为，雪中送炭比锦上添花更有意义。与自己一起笑过的人易忘记，与自己一起哭过的人最难忘。
  <br/>
  <br/>
  <strong>
   7
  </strong>
  <strong>
   、关于恋爱
  </strong>
  <br/>
  <br/>
  不要过分憧憬爱情的美，不要过分夸大失恋的悲。和漂亮的女人握握手，和深刻的女人谈谈心，和成功的女人多交流，和普通的女人过日子。这样，就好。
  <br/>
  <br/>
  花前月下漫步是美好的，甜言蜜语是迷人的，现实中弹奏的是锅碗瓢勺交响曲。
  <br/>
  <br/>
  <strong>
   8
  </strong>
  <strong>
   、关于距离
  </strong>
  <br/>
  <br/>
  为人父母，谁都渴望与孩子的距离短些，但随着子女的成长，你们的天空会越来越广阔，与父母的时空距离也会越来越远。
  <br/>
  <br/>
  做父母的，当然会在乎自己在孩子心中占据的位置是否重要。
  <br/>
  <br/>
  我也同样在乎，你是否常在身边。父母的心情，你能否读懂？
  <br/>
  <br/>
  <strong>
   9
  </strong>
  <strong>
   、关于得失
  </strong>
  <br/>
  <br/>
  人生在世，得失都属正常。得意时，要清醒，要记得自己的渺小，这个世界上有太多的能人；失意时，不退缩，依然故我，坚持下去：照着过去的步伐，来走现在的路。
  <br/>
  <br/>
  <strong>
   10
  </strong>
  <strong>
   、关于亲人
  </strong>
  <br/>
  <br/>
  做亲人是缘分。这一生，无法与你相伴终老，但在风雨交加的时候，会想着为你遮风挡雨。
  <br/>
  <br/>
  即使不能与你一路并肩前行，但在你倦怠的时候，家会是你永远的避风港湾。
  <br/>
  <br/>
  这辈子，无论我们会相处多久，都珍惜共聚的时光；下辈子，无论爱与不爱，相见已是陌然。
