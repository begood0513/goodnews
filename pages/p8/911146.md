 <p>
  一个中国女孩到法国留学，她在那里半工半读。
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/聪明" target="_blank">
    聪明
   </span>
  </strong>
  的她，很快就掌握了简单的法语，日常沟通完全没问题。
  <br>
   <br>
    渐渐地，她发现当地的公共交通系统有不少漏洞可钻。所有的售票处都是自助的，车站几乎都是开放式的，不设检票口，没有检票员，连随机性抽查都很少。
    <br>
     <br>
      女孩凭着自己的聪明劲，精确地估算了一个概率：逃票被查到的比例仅为万分之三。她为自己的发现沾沾自喜，从此以后，她搭车经常逃票。然而，她并不知道她的这个行为最终铸成了大错。
      <br>
       <br>
        4年后，她大学毕业了，优秀的成绩使她自信满满。她开始到各大公司去面试，踌躇满志地推销自己。
        <br>
         <br>
          这些公司一开始都很热情，但奇怪的是数日之后总是婉言相拒。一次又一次的失败使她愤怒，认为对方歧视和排斥中国人。
          <br>
           <br>
            直到最后一次面试，她才了解自己的问题在哪里。当时，经理直接告诉她不录用她的原因所在。
            <br/>
            <br/>
            经理说：“你刚来求职时，我们对你的教育背景和学术水平很感兴趣，我们相信你的工作能力。但我们查了你的信用记录，发现你有三次因乘搭公车逃票，而被处罚的记录。”
            <br/>
            <br/>
            女孩说：“我不否认这个，但你们就为了这点小事，而放弃一个人才？”
            <br/>
            <br/>
            经理说：“我们并不认为这是小事。我们注意到你第一次逃票，是在你来我国后的第一周。当时你说不熟悉自助售票系统，检查人员相信你，给你补了票。但在这之后，你又两次逃票。”
            <br/>
            <br/>
            女孩急忙解释：“那时刚好我的口袋里没有零钱。”
            <span id="hideid" name="hideid" style="color:red;display:none;">
             <span href="https://www.secretchina.com">
             </span>
            </span>
           </br>
          </br>
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  经理说：“我不接受你的这种解释，你在侮辱我的智商。我相信在被查获之前，你可能有数百次
  <span href="https://www.secretchina.com/news/gb/tag/逃票" target="_blank">
   逃票
  </span>
  的经历。”
  <br/>
  <br/>
  女孩不置可否地说：“那也罪不至死吧？干嘛那么认真？以后改不行吗？”
  <br/>
  <br/>
  经理严肃地说：“不！此事证明了两点：第一，你不尊重规则，你擅于发现规则中的漏洞，并恶意使用。第二，你不值得信任。”
 </p>
 <p>
  经理顿了顿说：“我们公司的许多工作是必须依靠诚信来进行的，如果由你负责某个地区的市场开发，公司将赋予你许多职权。我们为了节约成本，不会设置复杂的监督机构，正如公共交通系统一样，所以我们没法雇用你。同时，我可以明确地告诉你，在这个国家甚至整个欧盟，你将找不到雇用你的公司。”
  <br/>
  <br/>
  此时，女孩如梦方醒，心中懊悔难当。对方的最后一句话犹如醍醐灌顶：“
  <span href="https://www.secretchina.com/news/gb/tag/道德" target="_blank">
   道德
  </span>
  往往能够弥补聪明的不足，但聪明却弥补不了道德的缺失。”
 </p>
 <p>
  亚马逊总裁杰夫贝佐斯说：“聪明是一种天赋，而善良是一种选择。”让我们守住良知和善念，成为一个善良真诚、光明磊落、俯仰无愧于天地的人。
