  出嫁后，公姑敬，丈夫穷，莫生嗔。
  <br>
   夫子贵，莫娇矜，出仕日，劝清政。扶百姓，劝宽仁。
   <br>
    我家富，莫欺贫。借物件，就奉乘。应他急，感我情。
    <br>
     积阴德，贻子孙。夫妇和，家道成，妯娌们，要孝顺。
     <br>
      邻舍人，不可轻。亲戚来，把茶烹。尊长至，要亲敬。
      <br/>
      粗细茶，要鲜明。公婆言，莫记恨，丈夫说，莫使性。
      <br/>
      整肴馔，求丰盛。着酱醋，要调匀。用器物，洗洁净。都说好，贤惠人。
      <br/>
      夫君话，就顺应。不是处，也要禁。事公姑，如捧盈。修己身，如步冰。
      <br/>
      些小事，莫出门。坐起时，要端正，举趾时，切莫轻。冲撞我，祗在心。
      <br/>
      分尊我，固当敬。分卑我，也莫陵。守淡薄，安本分。他家富，莫眼热。
      <br/>
      行嫉妒，损了心。勤治家，过光阴。不伶俐，被人论。
      <br/>
      若行路，姐在前，妹在后。若饮洒，嫂居左，妯居右。公婆在，侧边从。
      <br/>
      慢开口，勿胡言。齐捧杯，勿先尝。即能饮，莫尽量。沉醉后，恐颠狂。一失礼，便被谈。
      <br/>
      肴面物，先奉上。骨投地，礼所严。动匙着，忌声响。出席时，随尊长。客进门，缓缓行。
      <br/>
      急趋走，恐跌倾，遇着人，就转身。洗钟盏，轻轻顿。坛和罐，紧紧缝。
      <br/>
      公姑病，当慇勤。丈夫病，要温存。爷娘病，时时问。
      <br/>
      姑儿小，莫见尽，叔儿幼，莫理论。里有言，莫外说。外有言，莫内传。
      <br/>
      勤纺帜，缝衣裳，烹五味，勿先尝。无是非，是贤良。嫂婶事，决莫言。若闻知，两参商。
      <br/>
      伯叔话，休要管。勿喞喞，道短长。孩童闹，规己子，是与非，甚勿理。
      <br/>
      略不逊，讼自起。公差到，悔则迟。些小利，莫见尽，论彼此，俗了人。
      <br/>
      学大方，人自称。晒东西，也莫轻，秽污衣，寻僻静，恐人见，起非论。
      <br/>
      他骂我，我不听。不回言，人自评。升斗上，要公平。买物件，莫亏人。
      <br/>
      夫君怒，说比论。好言劝，解愁闷。夫骂人，莫齐逞，或不是，陪小心。
      <br/>
      纵怀憾，看你情。祸自消，富自生。有女儿，不可轻。抚育大，继宗承。
      <br/>
      或耕耘，教勤谨，或读书，莫鄙吝。衣服破，缝几针，鞋袜破，被人论。是不是，自己寻。
      <br/>
      为人母，所当慎。奴婢们，也是人。饮食类，一般平。
      <br/>
      不是处，且宽忍，十分刻，异心生。若太宽，便不逊。最难养，是小人。再叮咛，更小心。
      <br/>
      妯娌多，都一心。本等话，莫生嗔。同茶饭，莫吵分。一闹嚷，四邻听。任会说，非为能。
      <br/>
      吵家的，个个论。公姑闻，不安宁。最不孝，斩先脉。夫无嗣，劝娶妾。继宗事，最为切。
      <br/>
      尊三从，行四德。习礼义，难尽说。看古人，多贤德。宜以之，为法则。
