 <p>
  <br>
   一只南美洲亚马孙河流域热带雨林中的蝴蝶，偶尔拍动了几下翅膀，两周之后在美国德克萨斯竟然引起一场龙卷风。
   <br>
    <br>
     这怎么可能呢？一只柔弱的蝴蝶怎么可能引起威力无比的龙卷风？还相距千里之遥的？
     <br>
      <br>
       但这个在科学领域是有个很著名的名词：
       <span href="https://www.secretchina.com/news/gb/tag/蝴蝶效应" target="_blank">
        蝴蝶效应
       </span>
       。
       <br>
        <br>
         “蝴蝶效应”是气象学家洛伦兹1963年提出来的，他解释其生成原因：蝴蝶翅膀的运动，导致其身边的空气系统发生变化，并引起微弱气流的产生，而微弱气流的产生又会引起它四周空气或其他系统产生相应的变化，由此引起连锁反应，最终导致其他系统的极大变化。
         <br>
          <br>
           “蝴蝶效应”在社会学界用来说明：一个坏的微小的机制，如果不加以及时地引导、调节，会给社会带来非常大的危害，被称为“龙卷风”或“风暴”；一个好的微小的机制，只要正确指引，经过一段时间的努力，也将会产生轰动效应，或称为“革命”。
           <br>
            <br/>
            在100多年前，就有一个展现出“蝴蝶效应”的故事，故事中的人物影响了彼此的
            <strong>
             命运
            </strong>
            。
            <br/>
            <br/>
            有天下午，在英国一个乡村的田野里，一位贫困的农民正在劳作。忽然，他听到远处传来了呼救的声音，原来，一名少年不幸落水了。
            <br/>
            <br/>
            农民不假思索，奋不顾身地跳入水中救人。孩子得救了。后来，大家才知道，这个获救的孩子是一个贵族公子。
            <br/>
            <br/>
            几天后，老贵族亲自带着礼物登门感谢，农民却拒绝了这份厚礼。
            <br/>
            <br/>
            在他看来，当时救人只是出于自己的良心，并不能因为对方出身高贵就贪恋别人的财物。
            <br/>
            <br/>
            故事到这儿并没有结束。
            <br/>
            <br/>
            老贵族因为敬佩农民的
            <strong>
             善良
            </strong>
            与高尚，感念他的恩德，决定资助农民的儿子到伦敦去接受高等教育。
            <br/>
            <br/>
            农民接受了这份馈赠，能让自己的孩子受到良好的教育是他多年来的梦想。
            <br/>
            <br/>
            农民很快乐，因为他的儿子终于有了走进外面世界、改变自己命运的机会；老贵族也很快乐，因为他终于为自己的恩人完成了梦想。
            <br/>
            <br/>
            多年后，农民的儿子从伦敦圣玛丽医学院毕业了，他品学兼优，后来被英国皇家授勋封爵，并获得1945年的诺贝尔医学奖。
            <br/>
            <br/>
            他就是亚历山大．弗莱明，
            <strong>
             青霉素
            </strong>
            的发明者。
            <br/>
            <br/>
            那名贵族公子也长大了，在第二次世界大战期间患上了严重的肺炎，但幸运的是，依靠青霉素，他很快就痊愈了。 这名贵族公子就是英国首相
            <strong>
             丘吉尔
            </strong>
            （邱吉尔）。
            <br/>
            <br/>
            农民与贵族，都在别人需要帮助的时候伸出了援手，却为他们自己的后代甚至国家播下了善种。
            <br/>
            <br/>
            人的一生往往会发生很多不可思议的事情，有时候，我们帮助别人或感恩别人，却可能冥冥之中有轮回。
            <br/>
            <br/>
            在第二次世界大战中，也有一个展现出“蝴蝶效应”的故事，这个故事却改变了人类的历史。
            <br/>
            <br/>
            一天，欧洲盟军最高军事统帅
            <strong>
             艾森豪威尔
            </strong>
            （Dwight David Eisenhower，1890年10月14日～1969年3月28日）在法国的某地乘车返回总部，去参加紧急军事会议。
            <br/>
            <br/>
            那天，大雪纷飞，天气寒冷，汽车一路疾驰。
            <br/>
            <br/>
            在前不着村后不着店的途中，艾森豪威尔将军看到一对老夫妇坐在路边，冻得瑟瑟发抖。他命令司机停车，让身旁的翻译官下车去询问。一位参谋急忙提醒说：“我们必须按时赶到总部开会，这种事情还是交给当地的警方处理吧！”
            <br/>
            <br/>
            艾森豪威尔将军坚持要下车去问，他说：“如果等到警方赶来，这对老夫妇可能早就冻死了！”询问之下，才知道这对老夫妇是去巴黎投奔儿子的，因为汽车在中途抛锚，他们被冻得浑身颤抖，却毫无解决的办法。
            <br/>
            <br/>
            艾森豪威尔将军听后，二话没说，请他们上车。先将老夫妇送到巴黎其儿子家里，然后才急忙赶回总部。
            <br/>
            <br/>
            没想到，这一送，艾森豪威尔将军竟然躲过了致命的伏击。
            <br/>
            <br/>
            原来，德国的军队知道他要参加这个会议，在他必经的路上已经安排好了狙击手，就等他的车过来。因为他好心帮助别人，绕了一个圈，正好逃过了一劫。这是在后来德军的狙击手被情报部门抓住了之后才知道的。
            <br/>
            <br/>
            曾有历史学家评论道：艾森豪威尔将军的一个善念，躲过了暗杀，否则第二次世界大战的历史及结局将会被改写。
            <br/>
            <br/>
            古人所云：“福在积善，祸在积恶。”以上这两个真实的故事证明了这一点，也展现了善在“蝴蝶效应”中的传播、循环。
            <br/>
            <br/>
            人有善愿，天必佑之。与人为善，于己为善；与人有路，于己有退。
            <br/>
            <br/>
            善良，是人生最宝贵的品德。种善因，得善果，种下善良，收获感动，种下美好，收获幸福。善良的心可以把岁月装点成诗，善良的行为可以把生命循环成美好的“蝴蝶效应”。
