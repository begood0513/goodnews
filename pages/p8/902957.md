 <p>
  人的一生，不可能不犯错误。但是，有些错误，在以后会有去改正的时间；有些错误，就没有改正的机会，留下的只有深深的痛悔。比如，下面的三个
  <span href="https://www.secretchina.com/news/gb/tag/关键" target="_blank">
   关键
  </span>
  点上，就一定要做好、走正，不能出错。
  <br>
   <br>
    <strong>
     1
    </strong>
    <strong>
     、失意时
    </strong>
    <strong>
     勿失信心
    </strong>
    <br>
     <br>
      失意时，不要一蹶不振，更不要结交不好的朋友。
      <br>
       <br>
        有位网友，辞掉政府部门的金饭碗，自己做生意。刚开始很平顺，做着做着，目标高了，野心大了，结果因生意扩张过度，资金链断裂，企业倒闭了。
        <br>
         <br>
          从云端到低谷，沉重的挫败感压得他透不过气来，抢地呼天，意志消沉。这个时候，那些酒肉朋友，天天邀他喝酒、打牌，借此消愁，结果越陷越深，掉到了赌博这个泥坑。半年下来，输掉了房子，还借了不少高利贷。
          <br>
           <br>
            人在困境的时候，别把信心全丢了。丢了信心，就失去了目标，茫然失措的时候，若有损友拉你入泥流，那就很容易陷进去。
            <br/>
            <br/>
            事业会出现困境，人生也会有困境，但信念决不可出现困境。
            <br/>
            <br/>
            越是逆境，越要警惕自己，越要给自己鼓舞斗志。
            <br/>
            <br/>
            <strong>
             2
            </strong>
            <strong>
             、得意时
            </strong>
            <strong>
             不可忘形
            </strong>
            <br/>
            <br/>
            得意时不可忘形，否则失意马上就到。要谨记，谦虚、谨慎，小心没大错。
            <br/>
            <br/>
            特洛伊木马讲的是：希腊联军围困特洛伊城，久攻不下，于是假装撤退，留下一具巨大的中空木马，里面藏满了希腊士兵。特洛伊守军不知是计，把木马作为战利品，运进城中，歌舞狂欢，饮酒庆贺。夜深人静之后，希腊士兵从木马腹中悄悄出来，打开城门，里应外合，攻下了特洛伊城。这就是著名“木马屠城记”。
            <br/>
            <br/>
            得意忘形，这样的例子不胜枚举，新闻上、现实中都有。奋斗了半辈子的人，成名成家，硕果累累，却曝出私生活上的问题，一下子就不可收拾，将所有的光环都染黑了。原因只有一个，就是得意忘形。
            <br/>
            <br/>
            人活着，处在哪个位置，身上有多少光环，都要头脑清醒，能够明白：作为个体，在这个世界里，是微不足道的，是渺小的。
            <br/>
            <br/>
            <strong>
             3
            </strong>
            <strong>
             、人活着
            </strong>
            <strong>
             勿昧良心
            </strong>
            <br/>
            <br/>
            良心是指善良之心、仁义之心。良心是人之根本，一个人为善为恶，都是从心地出发。有惭愧心的人，肯认错的人，懂得感恩的人，都是有良心的人。
            <br/>
            <br/>
            木匠业的祖师鲁班，有次做了一个能帮着锯木的木头人，徒弟王恩（已离师他去）看见，便悄悄的丈量木头人，也依样制作了一个，但却不会动。
            <br/>
            <br/>
            王恩只得硬着头皮回去请教鲁班，曾经的师徒一问一答：“尺寸都量对了吗？”“量对了！”“量头了没有？”“量过了！”“量脚了吗？”“量了！”
            <br/>
            <br/>
            最后，鲁班若有所悟的问道：“噢！你大概没有量（良）心吧！”王恩漫不经心的答道：“对！我就是没有量心。”鲁班沉下脸来厉声的说：“王恩，没有良心的人是不能成事的！”
            <br/>
            <br/>
            有良心的人，做事之前，必先摸摸良心，问自己这么做对不对；有良心的人，说话、做事，都能本诸良知，就不会错到哪里去。
            <br/>
            <br/>
            人，可以没有金钱、官位、权势，但不能没有良心；人，可以对不起别人，但不能对不起良心。没有良知，前途堪虑
            <br/>
            <br/>
            所以，如果在上面的三个点上犯错，或许会是一辈子的错，是毁人的错；也可能是不可弥补的错，不可救药的错。
