 <p>
  <br>
   <strong>
    一
   </strong>
   <strong>
    、不念过去
   </strong>
   <br>
    <br>
     20岁的时候，你会
     <span href="https://www.secretchina.com/news/gb/tag/怀念" target="_blank">
      怀念
     </span>
     童年的自己和知了作伴，与蝴蝶共舞，天真无邪；30岁的时候，你会怀念20岁的自己，血气方刚，精力旺盛，把生活折腾个底朝天；40岁的时候，你会怀念30岁的自己，肩膀上是甜蜜的负担……
     <span id="hideid" name="hideid" style="color:red;display:none;">
      <span href="https://www.secretchina.com">
      </span>
     </span>
    </br>
   </br>
  </br>
 </p>
 <p>
  我们会一直怀念过往生活中的种种
  <span href="https://www.secretchina.com/news/gb/tag/美好" target="_blank">
   美好
  </span>
  。过去的总是让人心生牵挂，未知的总是让人局促不安。
 </p>
 <p>
  就这样，让回忆支配生活，回不到过去，也踏不进未来，一直在原地兜圈，错过了不少风景。
  <br>
   <br>
    其实，人生的任何阶段，都应该喜欢、享受和担当，顺生而行，但不沉迷过往，无论是美好还是不堪，都要放下。快乐前行，相信未来。
    <br>
     <br>
      <strong>
       二
      </strong>
      <strong>
       、不畏将来
      </strong>
      <br>
       <br/>
       有人问农夫：“种了麦子吗？”
       <br/>
       农夫：“没，我担心天不下雨。”
       <br/>
       <br/>
       那人又问：“那你种棉花没？”
       <br/>
       农夫：“没，我担心虫子吃了棉花。”
       <br/>
       <br/>
       那人再问：“那你种了什么？”
       <br/>
       农夫：“什么也没种，我要确保安全。”
       <br/>
       <br/>
       因为害怕未知的风险，我们常会选择裹足不前。
       <br/>
       <br/>
       许多人会问：“如果这行不通呢？如果没有人喜欢呢？如果它失败了呢？”
       <br/>
       <br/>
       这些都是负面的问题。你不该只思考各种失败的可能，而也要聚焦于各种成功的可能。
       <br/>
       <br/>
       就算真的失败和错误，那也有了反思和修正的机会。每个人在成功之前，都曾经是个错误百出的新手。
       <br/>
       <br/>
       所以，我们要大胆尝试，
       <span href="https://www.secretchina.com/news/gb/tag/勇敢" target="_blank">
        勇敢
       </span>
       前行，不畏将来。
       <br/>
       <br/>
       <strong>
        三
       </strong>
       <strong>
        、输得起
       </strong>
       <br/>
       <br/>
       有时候我们总在问“为什么”：
       <br/>
       <br/>
       为什么，披星戴月地苦读，结果还是名落孙山？为什么，斗志昂扬地创业，结果还去负债累累？
       <br/>
       <br/>
       有人怀疑，自己是不幸之人，受挫之后，落落寡欢，一蹶不振。但也有人偏不服输，要一直尝试、改善、前进。
       <br/>
       <br/>
       多年之后，你会发现，前者依旧是低落消沉，碌碌无为；后者多半是逆袭人生，越挫越勇。
       <br/>
       <br/>
       输不起的人，要输一辈子；输得起的人，只输一阵子。
       <br/>
       <br/>
       <strong>
        四
       </strong>
       <strong>
        、平常心
       </strong>
       <br/>
       <br/>
       有一位禅师有三个弟子。一天，禅师问三人：“门前有两棵树，荣一棵，枯一棵，你们说是枯的好、还是荣的好”?
       <br/>
       <br/>
       大徒弟说：“荣的好。”
       <br/>
       <br/>
       二徒弟说：“枯的好。”
       <br/>
       <br/>
       三徒弟说：“枯也由它，荣也由它，都好。”
       <br/>
       <br/>
       选择前两者中的哪一种，都会有得失心，喜忧掺杂。而第三个人，枯也由它，荣也由它，世事如何变化， 皆可泰然处之。
       <br/>
       <br/>
       平常心，就是稳定的情绪，是泰山崩于前而色不变；是麋鹿兴于左而目不瞬；是大风大浪而内心宁静沉着，不为外界所扰。
       <br/>
       <br/>
       平衡好自己的情绪，面临任何事情，都能做到：想得开，看得透，顺其自然。
