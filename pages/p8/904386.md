   <strong>

   ，他画的竹子远近闻名。
   <br>
    <br>
     原来，他在自家房前屋后种了各种各样的竹子，时常观察，有创作的感受时就回到书房，铺纸研墨，把心中的印象画下来。就这样，不同季节、不同天气、不同时辰的竹子都了然于心中。作画时，各种各样的竹子就浮现在眼前，只需凝神提笔，一蹴而就，栩栩如生。面对人们的夸奖，他总是说：“我只是把心里琢磨好的竹子画下来而已。”

 <p>
  有个名叫晁补之的人，称赞他说：文同画竹，早已胸有成竹了。
 </p>
 <p>
  这就是“胸有成竹”的典故。比喻做事之前已经作好了充分准备，才会信心十足。
  <br>
   <br>
    文同观察、作画的对象是竹子，而我们做事之前，也先要确定
    <strong>
     目标
    </strong>
    和目的。目的，召唤我们拥有不屈不挠的精神，指引我们迈向正确的方向。
    <br>
     <br>
      《心灵投手》（The Rookie）是一部2002年上映的美国电影，为迪士尼改编自真人真事的题材。
      <br>
       <br>
        吉姆·莫瑞斯从小就喜欢棒球，想要进入大联盟。但因为种种原因，吉姆只能念书、结婚、生子，并且在小镇上的中学担任自然科学的教师一职。
        <br/>
        <br/>
        但是吉姆并没有放弃自己的梦想，在学校里组织了一支棒球队，并且担任指导教练的工作。为了再度尝试去大联盟，他需要放弃教职，家里的经济状况就会产生了很大的冲击。这时父亲告诉他：“在你找出天赋以前，放手做你觉得该做的事情没有关系。”结果他在三十五岁那年，以新手之姿挤进了大联盟。
        <br/>
        <br/>
        华理克（Rick Warren）在畅销书《标竿
        <strong>
         人生
        </strong>
        》（The Purpose-Driven Life）里说：“少了目的，人生就变成毫无意义的动作，毫无方向的活动，毫无道理的事件。”
        <br/>
        <br/>
        如果目的不明确，便不知道自己所应担当的角色，不知道处理事情所需要的条件和办法。没有目的的人生是盲目、杂乱的，如同大海中的一叶扁舟，没有航向，任海风吹、海浪打，却兜转无尽头。
        <br/>
        <br/>
        天道酬勤。一分耕耘一分收获，有付出就会有回报，只要不懈努力、不断地充实、完善自己，就一定会有美好的明天。
        <br/>
        <br/>
        在台北市万大路上，有一个“流行风”的店招在二楼迎风飘扬，这是陈郁洁独资开设的一家造型染烫设计发廊。
        <br/>
        <br/>
        陈郁洁，阿美族名Siko，原乡在花莲瑞穗乡，国中时和兄弟姊妹一同北上求学，喜爱美发美容，读书、工作也在这个领域中精进。之前曾在一个全国性的美发集团担任职前训练师，训练来自各地的发型设计师，累积了充足的经验和资源后，决定自行创业。
        <br/>
        <br/>
        2002年年8月，适值朋友有意顶让美发店，经过二个多月的评估与准备，郁洁就接手过来，利用原有的设施，稍作装潢，再雇用四名设计师，事业就这样展开了。半年下来，收支就达于平衡，现在做的更是有声有色。
        <br/>
        <br/>
        曾经有人问她，在经济不景气的大环境下创业，风险不大吗？郁洁自信的回答，她做事一向事前充分准备，谨慎评估，不做没有把握的事，而且一旦决定了，就会毫不犹豫地勇往直前。
        <br/>
        <br/>
        “目的”给人可以期待的发展方向，“机会”则是留给准备好的人，准备好了再开始行动，往往会一帆风顺，人生
        <span href="https://www.secretchina.com/news/gb/tag/开挂" target="_blank">
         开挂
        </span>
        。
        <br/>
        <br/>
        开挂的人生，是厚积薄发，是十年磨一剑，是水到渠成。别人开挂了，也许别人为这一天已经准备了8年，11年，22年，甚至更久。不要只羡慕别人，更应该看到努力的自己——有明确的“目标”，有万全的“准备”，下一个开挂的，可能就是你！
