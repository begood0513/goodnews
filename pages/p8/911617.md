 <p>
  人的一生，从小到老，风雨彩虹，沧海桑田，到头来用一句话就可以诠释：万丈红尘三杯酒，千秋大业一壶茶。
  <br>
   <br>
    人为什么要
    <span href="https://www.secretchina.com/news/gb/tag/喝茶" target="_blank">
     喝茶
    </span>
    ？
    <br>
     <br>
      生活就像喝茶，慢慢回味，方能品出它的滋味和醇香。
      <br>
       <br>
        有时，茶泡时间长了就苦，就像遇到不开心的事一样，很苦。有时，水温刚好，浓淡适宜，泡出的茶，会散发出淡淡的清香，入口之后齿颊留香，像遇到开心的事情一样，很幸福。
        <br>
         <br>
          人生有苦有甜，苦尽甘来是常态。人生是艰辛的，就像品茶一样，开始是带点苦涩，到中年时苦尽甘来，到老年时一切都是繁华落尽的平淡。成败也好，美丑也罢，终归一抔黄土、掩尽风流。
          <br>
           <br/>
           一壶茶，茶壶有量，茶味无量，值得品的是茶味。
           <br/>
           <br/>
           茶，收藏的是对春天记忆，在任何季节饮茶，都可以感觉到春日那明媚的阳光。好茶，不论遇到什么人，都能释放生命的本色；不论处贵处贱、顺遇或横逆，都不改自己的醇香。
           <br/>
           <br/>
           茶，是一种情调，是欲语还休的沉默；是欲笑还颦的忧伤；是“千红一杯，万艳同窑”后的落寞。喝茶，是一种心境，净化身心，滤去浮躁，沉淀深思。人生一世，伴随春夏秋冬、风花雪月、酸甜苦辣。有成功，也有失败；有欢笑，也有眼泪；有顺境，也有逆境；以何种心态而面对才是关键。
           <br/>
           <br/>
           一个人坐在房间，倒上一杯茶，看着茶叶的翻上落下，茶要沸水冲泡过才有浓香，人生也要磨练后才能坦然。
           <br/>
           <br/>
           浮生若茶，生活若品茗，但愿我们能从中体悟到人生的真谛。
           <br/>
           <br/>
           人为什么要喝酒？
           <br/>
           <br/>
           有人说，经常喝酒的人，不是爱喝酒，而是爱喝酒的感觉。
           <br/>
           <br/>
           喝酒是为了发泄和释放情感，喝醉酒是为了让疲惫的身心得到解压和安慰。
           <br/>
           <br/>
           与人相处总会有开心与不开心，有些话不能当面说，有些话不能在清醒的时候说。而在意识模糊时能说出口，在喝醉时才会说出对方的不是，也只有喝醉了才不会被人怪罪。
           <br/>
           <br/>
           一个人喝酒，喝的是无法对人倾诉的心事，喝的是郁闷，喝的是哀愁，喝的是心痛，唯有一醉方休，才不会感到难过。
           <br/>
           <br/>
           有人说，不会喝酒的人不解风情，因为喝酒是为了把埋在心中的事倒入酒中。心事融入酒，一醉愁无踪。
           <br/>
           <br/>
           与人喝酒，酒在杯中，杯在手中，话在酒里，情在心里。觥筹交错时的率真、耿介、委蛇，酒过三巡后的酒风、酒品、酒态，都折射于推杯换盏之间；谦谦君子，戚戚小人，人间冷暖，世态炎凉，都尽显于抿呷之中。
           <br/>
           <br/>
           酒场，犹如人生浓缩的舞台，有些人醉了却说自己清醒？因为想证明自己还能喝。有些人清醒，却偏要说自己已经醉了？因为不想继续喝。
           <br/>
           <br/>
           人生的起起落落，就像喝酒一样，第一杯清醒的，豪情万丈，是青年。第二杯有点醉，还想喝，似醉非醉，是中年。第三杯酒就醉了，醒来就后悔，是老年。
           <br/>
           <br/>
           万丈红尘三杯酒，千秋大业一壶茶——红尘中的纷扰，在三杯两盏淡酒里淡忘；世上的霸业雄图，在午后一壶茶里消磨。但这壶茶，该和谁喝？这杯酒，该让谁喝？明白了这事，一辈子方可过得踏实。
