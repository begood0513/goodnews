  一个装钱，就叫【钱桶】，众人喜爱。
  <br/>
  <br/>
  桶是一样的，因装的东西不同因而命运也就有所差异。人生亦如此，有什么样的
  <span href="https://www.secretchina.com/news/gb/tag/观念" target="_blank">
   观念
  </span>
  ，就有什么样的人生，有什么样的想法，就有什么样的生活。
  <br/>
  <br/>
  <strong>
   失败与成功
  </strong>
  <br/>
  <br/>
  成功的历程的确很艰辛，在心理、心态包括体能上，都需要付出很多时间和精力去训练。在这个过程中自然就淘汰80%。
  <br/>
  <br/>
  剩下来20%的人在成功道路上，中途因学习不够淘汰一些、恐惧失败淘汰一些、受消极负面影响淘汰一些、体能不支淘汰一些、自以为是中途改道淘汰一些、粮草不足淘汰一些、团队关系危机淘汰一些，又淘汰掉80%。
  <br/>
  <br/>
  因而，我们会看到，被淘汰的失败者越来越多，越来越拥挤，实际被淘汰的高达96%之多。
  <br/>
  <br/>
  在成功的道路上，同行者会越来越少。
  <br/>
  <br/>
  很多人认为成功需要：坚韧不拔、心如磐石、百折不挠、铁杵成针等等。这么多的意志成语，不如挑出一句概括所有：永不放弃！
  <br/>
  <br/>
  在现实生活中，很多成功者并没有什么过人的意志，只是他们认准目标，跟准导师，简单、听话照着做，永不放弃直到成功。
  <br/>
  <br/>
  <strong>
   牌局与人生
  </strong>
  <br/>
  <br/>
  人生就像是一场牌局，不管手中的牌好与坏，只要牌局尚未结束，每个人都还有机会；但若因手中的牌不好就半途放弃，你注定永远都是个输家。
  <br/>
  <br/>
  上天给了每个人机会去发挥自己的潜能与天赋，只要我们能不断地提升、努力完善自我，且拥有永不放弃，坚持到底的决心，让自己朝着既定目标不懈地前进，那么不管手中拿到的牌是好是坏，不管身处的环境有多么恶劣，最后都能使自己的人生显现闪亮的光芒，活出特色的人生。
  <br/>
  <br/>
  我们无法改变上天发给自己的牌，但却可以决定要如何打这副牌，只要你愿意改变，并且付出努力，即使你手中拿到的不是一副好牌，仍然倾尽全力打好这牌，在这副牌中得到最好的结果。
