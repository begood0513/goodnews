  <b>
   一、  港人忍无可忍，终于背水一战
  </b>
  <br/>
  <br/>
  北京已经把刀架在港人脖子上。香港已无退路， 背水一战是她的宿命。
  <br/>
  <br/>
  于是，曾经被称为经济动物的香港人，倾城出动，向北京说不。
  <br/>
  <br/>
  2019年6月香港接连两次百万人反送中大游行，揭开了对于北京政权离心大逃亡的序幕。大中华的多个边缘地区挑战中共统治中心，已渐成基本态势。中共草木皆兵，防范森严，但野火却无时无地不可抑制地蔓延开来。……
  <br/>
  <br/>
  <b>
   二、  香港是中国的避难所，中国人的逃城，中国人的救赎
  </b>
  <br/>
  <br/>
  戊戌变法逃亡的改良派，结束晚清的仁人志士、抗日时期之爱国者、四九变局之交的绅商、红色恐怖中的逃港人士、大饥荒时垂死逃难的灾民、六四时救援逃亡的学运民运异议人士以及维园纪念六四的烛光烛泪烛海……大凡社会动荡引发的人道灾难，香江都是他们的庇护所和新大陆。 香港有大恩于中国。

 <p>
  现在香港有难了，曾受其恩典的中国人何以为报呢？
  <br/>
  <br/>
  现在香港人奋起自救，背水一战了，中国人将何以援手呢？谁能为香港打上一把大雨伞，为未来的香港挡雨？ 香港，何处是你今日的逃城呢？
  <br/>
  <br/>
  <b>
   三、  港台唇齿相依  北京为绿营助选
  </b>
  <br/>
  <br/>
  香港“一国两制”本欲为台湾垂范，然“一国两制的”大失败，却适时为港、台离心运动添加薪火。
  <br/>
  <br/>
  港、台本土力量的崛起与互助：台湾太阳花运动期间和香港雨伞运动时光，港台两地年轻世代交流互动频繁，一损俱损，一荣俱荣，命运相依。
  <br/>
  <br/>
  日前由北京的愚蠢和野蛮决策引发的香港浩大游行，中南海恰恰是在为中共痛恨的绿营助选。台湾飘离大陆的倾向日益强化。港台年轻世代在精神上共鸣加深。均欲迅速逃离中共控制。香港这次的同仇敌忾表明，目前已经扩展至全体市民，面对野蛮专横的大陆共产政权，形成“大逃亡”潮流。
 </p>
 <p>
  这是一个边缘逃离中心，自由小邦脱离恐龙王国的大逃亡运动。
  <br/>
  <br/>
  <b>
   四、  港台藏维蒙：一条离心链条
  </b>
  <br/>
  <br/>
  台港藏维蒙……，是紧贴中国大陆边缘线的离心链条。
  <br/>
  <br/>
  台港藏维蒙各自问题异常复杂：历史不同，族群有别，路径各异，恩怨纠结，源远流长；它们在历史上与中国大陆关系有各自不同的曲折演变，似乎相互牵连甚少。
  <br/>
  <br/>
  目前情势是，除了上述港台两地的共同拒中共倾向，大中华的其他边缘地区—藏维蒙，他们挑战中共统治中心，也已成为基本态势。
  <br/>
  <br/>
  在西藏，达赖喇嘛出离家园，流亡印度已60年；为抗议北京当局压迫乃至灭绝西藏的独特文化与信仰，一百五十多位僧侣和普通百姓为争取信仰自由、为促成达赖喇嘛返还西藏而举火自焚，前赴后继；
 </p>

 <p>
  在西北，百万维吾尔族人的集中营使新疆处于军事戒严状态，维吾尔族对于北京的野蛮压迫业已忍无可忍，中共已觉草木皆兵，从而防范森严，但野火却无时无地不可抑制地在地底蔓延；
  <br/>
  <br/>
  在北陲，蒙古族也不甘于俯首帖耳，一股整合内蒙外蒙的力量正在萌生，他们试图恢复其祖先成吉思汗的荣光。……
  <br/>
  <br/>
  综上所述，虽然台港藏维蒙各自问题异常复杂，历史不同，差异极大，但是就当下大势而言，五地皆有共同指向：强烈抗拒并逃离北京的控制。这一离心倾向，是连接三地民意的枢纽。是日益野蛮极权的中共政权把它们粘合在一起了。它们的共识是：逃离中共控制。
  <br/>
  <br/>
  <b>
   五、  印太区域的多重巨型岛链
  </b>
  <br/>
  <br/>
  倘若俯瞰印太地区，从更宽阔的视野，不难发现，平衡并制约恐龙党国、环绕大陆中原的多重环形圈正在逐步成型：从台港藏维蒙逐级外推，则是印太大战略包围圈的第一、第二和第三岛链。
  <br/>
  <br/>
  它的外围，是所谓第一岛链：北起日本群岛、琉球群岛，中接台湾岛，南至菲律宾、大巽他群岛的链形岛屿，它连接的国家、地区：日本、台湾、菲律宾、印度尼西亚、印度尼西亚、马来西亚、泰国、新加坡。
  <br/>
  <br/>
  它的第二层外围是所谓第二岛链，包括日本列岛，经小笠原群岛、硫黄群岛、马里亚纳群岛、雅浦群岛、帕劳群岛，延至关岛及哈马黑拉马等岛群连接的国家：印度，缅甸，越南，南海诸国，南韩、朝鲜、日本……
  <br/>
  <br/>
  它的第三层外围是所谓第三岛链， 北起阿拉斯加，经夏威夷群岛延伸经某些美属太平洋岛屿直至美国重要盟国澳大利亚、新西兰。
  <br/>
  <br/>
  在地缘政治上，作为残存的共产大国，当今中国大陆处于相当孤立的地位，可谓四面楚歌。即便是北京当局竭力拉拢的俄国，亦是貌合神离，绝难结盟。此外，河内与北京，因南海争执，已势同水火。而朝鲜更是吃定了北京，金正恩乖张的做派，其潜在动机其实是企图摆脱北京，向华盛顿投怀送抱。
 </p>
 <p>
  因此，所有的上述四层链条各地各国，在北京政权眼中，基本上皆非友善之邦，源于北京皆视之为美国盟友也。虽然中国外交部门惨淡经营，然收效甚微。雕虫小技无法解决战略上的无能为力，盖因根本的问题出在中国的国家定位。近年来，亚太地区多个政权交替，还没有出现过一个不亲西方的政权。
 </p>

 <p>
  曾几何时，习近平甫一上台，挟“中国模式”财大气粗之势，自我膨胀，妄言“自信”。内政向毛氏祸国路线倒退，集皇权，逮律师，封网络，禁媒体、乱股市；外交抛弃邓氏“韬光养晦”姿态，剑拔弩张，咄咄逼人：震南海，胁邻国，压台湾，纵朝鲜，“履至尊而制六合，执棰拊以鞭笞天下，威振四海。于是废先王之道，燔百家之言，以愚黔首；……以弱天下之民。”自以为富甲天下，可延续红色帝国“子孙帝王万世之业也”。然自2018以来，股市成灾， 货币贬值，资本外逃，外交孤立，顿时即被打回原形，陷入楚歌四面茕茕孑立之境，何哉？——“仁义不施，而攻守之势异也。”)
  <br/>
  <br/>
  <b>
   六、  港台藏维蒙对中国人的伟大功能
  </b>
  <br/>
  <br/>
  代言者：台港藏是中国人的代言者，是中国动向的风向标，是中国人的出气口。在以言入罪之中国，大陆人不能说、不敢说、没处说的话，在言论自由的台港被说出来了，也被流亡印度和西方的达赖喇嘛及藏人行政中央说出来了，这就给世界提供了一个中国大陆真实民意的重要参考系。事实上，台港藏已经成为国际社会对封闭的中国的最大观察哨。
  <br/>
  <br/>
  中介桥：台港藏维蒙是中国人大逃亡的中介桥。从1949（逃台逃港）、1959（藏人逃印，伊犁事件维吾尔人逃苏）、1962（逃港）、1989（经港去西方）以来，它们是濒临绝境的中国人的“出埃及”之路，是通向文明世界之现代桥梁。
  <br/>
  <br/>
  导火索： 台港维藏是中国转型的导火索。当僵硬的恐龙帝国的中枢日益腐败顽固野蛮时，在其控制力薄弱的边远地带，天高皇帝远，它们往往是帝国崩解的先声和导火线。回望晚清末年，倘若没有香港（上海租界等）这类远离北京的边缘区域作为反清志士的人财以及舆论基地，没有边陲西南四川绅商及民众的保路运动，辛亥革命难尽其功。事实上，今日的台港藏，正在日益逼近当年香港和上海租界的对中原变迁的历史性角色。
  <br/>
  <br/>
  稳定器： 回顾1900庚子年间，权力中央的慈禧纵容义和团（当年五毛党）杀洋人攻使馆，并自下诏书，向列国宣战，宣称要“大加挞伐，一决雌雄”后，东南各省，时任两广总督的李鸿章，闻此讯，覆电朝廷：“此乱命也，粤不奉诏”，并迅即与两江总督刘坤一、湖广总督张之洞及李秉衡、袁世凯、袁昶等达成“东南互保章程”，以免慈禧愚蠢野蛮的内外政策殃及东南各省，稳定并保全了东南地区。
  <br/>
  <br/>
  倘若今日习近平党国为摆脱内政困局，敢冒天下之大不韪，攻打民主台湾时，中国各边缘地带的主事者，即使平日再胆怯懦弱，但面对自杀式的疯狂，也势必将如同当年李鸿章们一样，划地自保，致使其军令不出中南海，从而令政治中枢在台海铩羽而归。
  <br/>
  <br/>
  面临党国的野蛮专横及衰颓，大陆边缘地带当得风气之先，将开始离心大逃亡。恰似晚清末年，当武昌枪声划破夜空时，早已对满清王朝积怨深重的各省，纷纷通电独立，两百多年的王朝，从此一朝倾覆。
  <br/>
