  主持人：陈奎德

  <br/>

  <b>
   一、    G20峰会前夕，习近平穿梭“访友”：俄、伊、朝
  </b>
  <br/>
  <br/>
  6月6号到8号，习近平访问了莫斯科，又去了圣彼得堡参加了国际经济论坛会议。习近平希望中俄连手对抗美国。然普京在习前脚刚走，就声称他对中美争端态度是坐山观虎斗。
  <br/>
  <br/>
  6月14日，习近平在吉尔吉斯斯坦与伊朗总统鲁哈尼会面，习近平称无论国际和地区形势如何变化，中国都愿意与伊朗推动全面战略伙伴关系，并持续稳步发展，强调双方要加强战略沟通。人们注意到，之后不久，伊朗突然击落了美国一架无人驾驶机，差点酿成一场局部战争。(假如川普下令轰炸了伊朗，G20峰会的局面将会如何？)
  <br/>
  <br/>
  接着，习近平急匆匆访问朝鲜，6月20号上午去，21号下午回。中国和朝鲜双方的高级军事将领参加了会谈，据信金正恩把朝鲜核武器的情况向习近平做了通报。
  <br/>
  <br/>
  以上是习在大坂峰会前迫不及待打的三张外交牌，高调张扬，夸张而愚蠢。
  <br/>
  <br/>
  他无疑是向世界、特别是向美国显示，我（只）有三个朋友呢。
  <br/>
  <br/>
  <b>
   二、    序曲：19 : 1, 中共G20 大丢面子
  </b>
  <br/>
  <br/>
  6月中旬，在日本同时举行了两场G20集团会议，一个是贸易部长会议，另外一个是财政部长会议。在这个会议上，中共的贸易行为受到了指责。在贸易部长会议上就讲到，中共对企业的补贴问题非常严重。会议对中共提出了要求，停止对企业的补贴。20国家的贸易部长进行表决，结果是19比1，中国是1票，其它19个国家都指责你不应该这样补贴你的企业。
  <br/>
  <br/>
  之前，习政权还不太希望他这种没有盟友，只有三四个难兄难弟邪恶政权的局面太过显眼，为遮掩难堪，还要高唱“人类命运共同体”之类的调门。现在在外交日益孤立的情势下，在美国贸易战的压力下，习已经难唱高调，于是即使只有邪恶轴心做友邦也顾不得那么多面子了。
  <br/>
  <br/>
  <b>
   三、     香港大游行的心理影响和地缘政治影响
  </b>
  <br/>
  <br/>
  1）香港震撼：
  <br/>
  <br/>
  6月9日及16日的香港百万多人大游行，震荡全球。香港民意清晰呈现了出来。

 <p>
  鉴于此，新冷战的态势有了某种微妙变化，港台被清晰地列入反邪恶轴心的营垒：
  <br/>
  <br/>
  中、俄、朝 、伊   Vs.   美、日、澳、加、欧盟、英、港、台 、印…….   新冷战营垒浮现
  <br/>
  <br/>
  台湾大选前选情的反转。总统与大陆民运人士的互动。台湾十万人大游行：拒绝红色媒体，守护台湾民主。
  <br/>
  <br/>
  美国国会新版本《香港人权和民主法案》：
  <br/>
  <br/>
  美国跨党派国会议员6月重推《香港人权和民主法案》 ，设下2020年香港普选立法会的“时限”，并且明确要求普选特首。法案如获通过，若港府不回应，势必影响《香港政策法》， 最严重的可引来美方制裁，变相迫使港府启动政改。
  <br/>
  <br/>
  香港的人权组织人权阵线已经宣布，要在G20峰会前的26号举行大的游行，让G20的会议看到香港人怎么样进行反抗、中共对香港老百姓的这种剥夺自由、不给安全和不给普选的权利，要让香港问题再一次凸显到国际舞台。
  <br/>
  <br/>
  <b>
   四、    军事角力
  </b>
  <br/>
  <br/>
  习轴心的牌：俄国，朝鲜，伊朗，加上东海举行实弹射击演习：
  <br/>
  <br/>
  中共宣布，在东海进行实弹炮弹演习，组成了一个航空母舰的编队，通过宫古海峡，超过了第一条岛链，开向美国关岛的方向，对美国施加压力。同时中共空军进驻永兴岛，又飞降到三个人造岛上的飞机场上。
  <br/>
  <br/>
  西方呢，美国不断派军舰通过台湾海峡，英国、法国、加拿大也参加进来，连德国的海军也参加进来。同时美国跟日本在南海进行军事演习。
  <br/>
  <br/>
  <b>
   五、    新旧冷战：相同与相异
  </b>
  <br/>
  <br/>
  关于各阵营成员国的不同分布。新冷战轴心阵营总体力量大不如前苏联社会主义阵营，且新冷战丧失意识形态统一性，
  <br/>
  <br/>
  从相互隔离到————你中有我我中有你。但现在贸易战、科技战在开始逐渐分离的进程。
  <br/>
  <br/>
  旧冷战，曾经的两大阵营
  <br/>
  <br/>
  我们的前半生，被这样的世界地图所宰制--
  <br/>
  <br/>
  社会主义阵营：苏联、中国、东欧各国、北韩、北越、外蒙古、古巴…..
  <br/>
  <br/>
  资本主义阵营：美国、英国、法国、联邦德国、意大利、荷兰、比利时、瑞典、挪威、芬兰、日本、南韩、台湾（中华民国）、加拿大、澳大利亚、新西兰…………….
  <br/>
  <br/>
  冷战重大事件：柏林危机、韩战、1956 年反斯大林秘密报告、匈牙利起义、柏林墙起、古巴导弹危机、越战、布拉格之春、尼克松访华、波兰团结工会起、天安门事件、柏林墙倒。
  <br/>
  <br/>
  新冷战，首先载入历史的，就是中美贸易战、科技战，……港、台之战…………然后逐渐卷入其他国家。其前景扑朔迷离。
