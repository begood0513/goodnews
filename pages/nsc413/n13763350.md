<div><p>
 【大纪元2022年06月20日讯】（大纪元记者洪宁报导）著名异议人士
 <ok href="https://www.epochtimes.com/gb/tag/%E9%83%AD%E9%A3%9E%E9%9B%84.html">
  郭飞雄
 </ok>
 去年底欲前往美国照料病危妻子，被中共当局非法抓捕关押至今，当他得知妻子离世后悲痛欲绝，写下了这首怀念诗《致我永生的妻子》。
</p>
<h4>
 致我永生的妻子
</h4>
<p>
 你持续一年，
 <br/>
 悲惨的呼唤，早已把我的心撕裂，
 <br/>
 在你被步步推向万丈悬崖的危境，
 <br/>
 我无论如何奋力，都无法——
 <br/>
 挣脱锁链，
 <br/>
 无法上前给你救援。
 <br/>
 在你人生的末了，
 <br/>
 我不能拥抱你颤栗的肩膀，
 <br/>
 不能用热泪温暖，
 <br/>
 你苍白冰凉的脸。
 <br/>
 最终就这样，我和千万同胞
 <br/>
 一道眼睁睁看你，
 <br/>
 非正常掉进了，
 <br/>
 万丈深渊！
 <br/>
 我的心从此坍塌，
 <br/>
 陷入极度的悲恸、愧疚、罪感。
 <br/>
 我的生命从此被砍掉了一半。
</p>
<p>
 女人最伟大，女人最苦，
 <br/>
 母性至高无尚。
 <br/>
 我亲爱的妻子，
 <br/>
 为了我和儿女，你熬干了心血，
 <br/>
 在异国他乡，
 <br/>
 经受了无数重压苦难，
 <br/>
 我却无力在你至危至痛末了，
 <br/>
 为你拚命排除险情，
 <br/>
 无力在你千唤万呼的悲惨吁求中，
 <br/>
 回报你哪怕十天的
 <br/>
 最后慰籍！
 <br/>
 我亲爱的妻子，对你的深情重义，
 <br/>
 我实在亏欠太巨，
 <br/>
 你的悲惨，是我的重罪。
 <br/>
 现如今，我的心
 <br/>
 完全被血泪淹没，
 <br/>
 我无法拔出，也不必拔出。
 <br/>
 我已经永远告别在世的乐感，
 <br/>
 我的命运注定万劫不复。
</p>
<p>
 我亲爱的妻子，
 <br/>
 你是长江的女儿，
 <br/>
 即使临当悬崖边缘，
 <br/>
 你的眼睛依旧那样，
 <br/>
 明亮清澈，那里面有光，
 <br/>
 有生命真光！
 <br/>
 你有常人难免的各种缺点
 <br/>
 但你的心地多么干净、纯粹，
 <br/>
 你的坚强、尊严和气节，
 <br/>
 令我一直高度尊敬。
 <br/>
 我和你身为患难夫妻，
 <br/>
 为了中国的自由，
 <br/>
 承受了堪比战争年代的重重灾祸、劫难。
 <br/>
 这一次你最终掉入万丈深渊，
 <br/>
 实为自由而牺牲，
 <br/>
 但你的生命之光，
 <br/>
 永远与我同在。
 <br/>
 你明亮清澈的眼神，每每回想起，
 <br/>
 都会令我敬仰、热爱、心潮激荡，
 <br/>
 令我不断在极限苦痛中，
 <br/>
 抬头望远。
</p>
<p>
 我亲爱的妻子
 <br/>
 年年月月，
 <br/>
 我都会伏在有形无形哭墙上
 <br/>
 为你痛哭。
 <br/>
 我盼望着母亲回到母亲的土地，
 <br/>
 我期盼着那一天，
 <br/>
 我们夫妻最终团圆，
 <br/>
 在故乡高高的山岗上，
 <br/>
 我和你
 <br/>
 最终合葬在一座大冢间，
 <br/>
 我要永远向你陪罪，
 <br/>
 亿万斯年，我的灵魂，永远与你的灵魂一道飞翔。
 <br/>
 我期盼，世世代代人类同胞，
 <br/>
 永远悲歌你高贵的人性
 <br/>
 和悲惨的命运！
</p>
<p>
 2022年4月
</p>
<p>
 这首诗是2022年6月16日，律师经历半年的努力，在广州第一看守所会见
 <ok href="https://www.epochtimes.com/gb/tag/%E9%83%AD%E9%A3%9E%E9%9B%84.html">
  郭飞雄
 </ok>
 （原名杨茂东）时听他亲口所述。
</p>
<p>
 <ok href="https://www.epochtimes.com/gb/tag/%E9%83%AD%E9%A3%9E%E9%9B%84.html">
  郭飞雄
 </ok>
 是中国新公民运动和南方民主运动的重要参与者和领导者之一。因维权行动、为法轮功学员辩护、要求中共官员公示财产，他被当局关押四次，两度被判刑入狱共11年。第二次入狱6年，2019年8月才刑满出狱。
</p>
<p>
 2006年，郭飞雄的妻子张青携一对子女经泰国逃亡美国，三年后获政治庇护，后定居于马里兰州。
</p>
<p>
 2020年1月，张青不幸罹患肠癌，病情急剧恶化，需要亲人照顾。2021年1月28日，郭飞雄在上海浦东机场准备搭机赴美探望病妻时，遭当局阻止登机。11月29日，郭飞雄向中共总理李克强发出公开信，要求放他出境赴美探望病妻，但没有得到回音。
</p>
<p>
 2021年12月5日，郭飞雄在第二次试图登上赴美班机去探望已癌症晚期的妻子时，在广州白云机场被当局拘留。一个月后家属收到逮捕通知书，称郭飞雄于1月12日被广州市公安局以涉嫌“煽动颠覆国家政权罪”正式逮捕。
</p>
<p>
 2022年1月10日，张青不幸去世，终年55岁，留下一对儿女。今年3月，妻子张青病逝两个月后，郭飞雄曾写信向姐姐杨茂平交待身后事，杨茂平也曾上书高层，要求释放弟弟赴美照顾未成年的孩子。
</p>
<p>

</p>
