<h4 class="p4">
 <span class="s1">
  <b>
   吉林省视频监控系统
  </b>
 </span>
</h4>
<p class="p4">
 <span class="s1">
  视频监控是中共利用高科技监控民众，迫害上访人士、异议人士、法轮功修炼者等使用的主要手段之一。
 </span>
</p>
<p class="p4">
 <span class="s1">
  虽然中国现行的《刑事诉讼法》和《刑侦诉讼法》并没有把电子证据作为一种法定证据，但中共公安机关却把它作为法定电子证据。这种行为被指本身违背了《刑事诉讼法》和《刑侦诉讼法》。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据《吉林省视频监控联网建设情况》公开披露的信息显示，2011年底，吉林省拥有视频监控点达22万个；2012年，吉林省新增加6万6千个视频监控探头；2013年，吉林省新增加12万9千个视频监控探头；2014年，吉林省新增加16万个视频监控探头。
 </span>
</p>
<h4 class="p4">
 <span class="s1">
  <b>
   王宏凯、徐晶岩个人背景
  </b>
 </span>
</h4>
<p class="p4">
 <span class="s1">
  王宏凯，男，1970年12月生，吉林四平人。2008年9月，任中共吉林省看守所副所长；2012年4月，任中共吉林省公安厅科技处副处长；2014年4月至2018年9月，任中共吉林省公安厅视频监控处处长，警务技术一级主任；2019年8月，任中共吉林省公安厅户政管理总队总队长。
 </span>
</p>
<p class="p4">
 <span class="s1">
  徐晶岩，男，1983年1月生，吉林市人。2014年7月，任中共吉林省公安厅视频监控处应用指导科副科长；2018年9月，任中共吉林省公安厅视频监控处警务技术一级主管；2019年8月，任中共吉林省公安厅图像侦查总队警务技术一级主管。
 </span>
</p>
<p class="p4">
 明慧网报导，王宏凯和徐晶岩多年来积极执行中共迫害法轮功政策，推广视频监控，提供警务技术，培训技术警察，使大批的法轮功学员因视频监控而被绑架、非法抄家、非法关押、诬判，甚至被迫害致死、致残。
</p>
<h4 class="p4">
 <span class="s1">
  <b>
   吉林法轮功学员因监听、监控遭迫害案例
  </b>
 </span>
</h4>
<p class="p4">
 <span class="s1">
  据明慧网2018年不完全统计，吉林省共有9个地区777人次法轮功学员遭受不同程度的迫害。其中：非法判刑65人，非法庭审26人，非法起诉15人，非法批捕34人，绑架463人次，骚扰134人次，各类经济迫害31人，迫害致死1人，另有3人早前在迫害中离世，2018年才得以曝光，失踪2人，流离失所3人。
 </span>
</p>
<p class="p4">
 <span class="s1">
  以上这些遭迫害的法轮功学员，在被迫害前，多数都遭到了电话监听、视频监控。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  1. 长期网络监控  30余名长春法轮功学员遭非法抓捕
 </span>
</h3>
<p class="p4">
 <span class="s1">
  吉林省四平市公安局、国保大队、四平梨树县公安局及国保大队、梨树县康平派出所，根据长期网络与电信的监控及跟踪，于2019年8月15日，出动约几百警察，包括国保大队、巡警、交警、梨树辖区内派出所等，绑架了30多位法轮功学员及家属。现在仍然被非法关押在四平市看守所的法轮功学员至少有15人。
 </span>
</p>
<p class="p4">
 <span class="s1">
  其中，男性法轮功学员有：孟祥岐、孟凡军、王克民、王东吉、韩建平、李长坤、江涛，谭秋成。女性法轮功学员：付贵华、于健莉、王凤芝、崔桂贤、刘冬英、张绍平、周姓老太太。他们所有人被诬陷为一个案子，已全部被非法批捕。
 </span>
</p>
<p class="p4">
 <span class="s1">
  梨树县国保大队长王明山、教导员张喜昆具体策划、实施，手段是长期电话、网络监控及跟踪。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  2. 吉林法轮功学员樊云鹏被小区摄像头录像 遭诬判
 </span>
</h3>
<p class="p4">
 <span class="s1">
  2019年3月26日下午，榆树市法轮功学员樊云鹏被榆树市华昌派出所警察劫持到拘留所，4月3日转到看守所。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据悉，樊云鹏在一居民小区黏贴真相时被小区摄像头录像。警察然后锁定目标重点监控，跟踪到家实施绑架、抄家，强抢法轮功书籍等个人物品。
 </span>
</p>
<p class="p4">
 <span class="s1">
  2019年9月20日，樊云鹏被德惠市法院非法秘密冤判2年6个月，10月13日当局才通知家属。樊云鹏不服判决，已上诉到中级法院。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  3. 长春法轮功学员赵旭、赵秋悦、郑炜东被视频、音频构陷
 </span>
</h3>
<p class="p4">
 <span class="s1">
  2018年10月12日，长春市法轮功学员赵旭、赵秋悦、郑炜东、吴雅楠被中共南关区检察院非法批捕。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据悉，长春市明珠派出所绑架法轮功学员赵旭，将他非法拘留15天后，明珠派出所又以累积的视、音频构陷赵旭，还恐吓赵旭，逼迫她签字。
 </span>
</p>
<p class="p4">
 <span class="s1">
  警察将构陷赵旭的案卷递交到检察院，被检察院二次以“证据离谱”为由退回给明珠派出所。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据悉，赵旭已被再次被构陷到朝阳区检察院。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  4. 长春市杨香华、李秀凤、郭晓英被网络监控 遭非法判刑
 </span>
</h3>
<p class="p4">
 <span class="s1">
  吉林省长春市法轮功学员杨香华、李秀凤、郭晓英遭诬判，分别是5年，7年，郭晓英非法刑期不详。
 </span>
</p>
<p class="p4">
 <span class="s1">
  2017年9月7日晚，三人在西环城宅家屯的出租屋内被绿园区国保大队、绿园区迎宾路派出所十多个警察联合绑架，当晚分别被非法关押到长春市第四看守所、第二看守所。近一年来，家人打听不到任何消息。直到2018年8月份才经检察院查到，三人已遭非法判刑。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据悉，警察是通过人工监控和网络监控绑架三位法轮功学员的。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  5. 长春法轮功学员张桂香被网络监控并非法关押
 </span>
</h3>
<p class="p4">
 <span class="s1">
  2016年12月18日下午，吉林长春市春城大街派出所警察，一非法闯入法轮功学员张桂香的家里，这些警察晃了一张纸，说是检查证，就开始强行抄家，将私人物品及现金人民币两万元等抄走。并将张桂香的丈夫带到派出所，进行威胁恐吓。
 </span>
</p>
<p class="p4">
 <span class="s1">
  之后，长达数月，警察对张桂香家的亲朋好友的电话实施监听、网络监控，导致张桂香数月一直流离失所。
 </span>
</p>
<p class="p4">
 <span class="s1">
  2017年4月10日晚上，张桂香回家，结果第二天一早六点左右，被一直监控的警察非法逮捕。
 </span>
</p>
<p class="p4">
 <span class="s1">
  4月12日上午，派出所告知家人，人已经送到第四看守所，没有相关文书通知，只有口头回复。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  6. 吉林长春市盛香梅、王英君被大数据监控，遭劫持到看守所
 </span>
</h3>
<p class="p4">
 <span class="s1">
  吉林省长春市法轮功学员盛香梅、王英君，6月23日下午在向民众讲法轮功真相时被永吉派出所警察绑架。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据永吉派出所警察说是上级的指派他们出警的，是长春南关区大数据监控中心举报的。
 </span>
</p>
<h3 class="p4">
 <span class="s1">
  7. 舒兰市6名法轮功学员被监听、监控并非法关押半年
 </span>
</h3>
<p class="p4">
 <span class="s1">
  吉林舒兰市公安局国保大队调集市区和各乡镇派出所大量警力混合编组，于2018年7月18日晚上10点统一行动，绑架了12位法轮功学员与一名家属。至今还有6名法轮功学员被非法关押。
 </span>
</p>
<p class="p4">
 <span class="s1">
  这次行动是由吉林市公安局、国保负责监听、监控该地区法轮功学员已有约一年的时间，是预谋已久的一场迫害，按照名单抓捕。
 </span>
</p>
<p class="p4">
 <span class="s1">
  在法特与白旗镇，几乎每个被骚扰的法轮功学员家都去了三、四辆车的警察，前后门围堵。因为是深更半夜，人们熟睡中被吓醒；老人和孩子被吓得大声哭喊；有位女学员穿的是短裤，有的警察竟然连衣服都不让穿上就要强行带走。非法审问时，多数都是用监听到的法轮功学员讲话录音作为所谓“证据”。
 </span>
</p>
<p class="p4">
 <span class="s1">
  目前，法轮功学员徐洪玉、王志刚、孙东伟，被非法关押在舒兰南山看守所，女学员李凤娟、李凤玲、曲秀敏被非法关押在吉林市小白山看守所。
 </span>
</p>
<p class="p4">
 <span class="s1">
  据悉有五个多月不让家属接见，家属非常担忧。其中，
 </span>
 <span class="s1">
  李凤娟的公婆是已80多岁的老人，孝顺的儿媳突然被抓捕，公婆惊吓过度，公爹得了脑血栓，瘫痪在床，急需人照顾。#
 </span>
</p>
