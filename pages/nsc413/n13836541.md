 【大纪元2022年10月01日讯】

<p>
 <strong>
  声明退出共青团、少先队 摆脱独裁的中国共产党精神束缚
 </strong>
 <br/>
 我李雨顺在此严正声明退出中国共产主义青年团、中国共产主义少先队。
 <br/>
 我本没有资源加入过共青团，但上中学时被迫写上是共青团团员，否则影响升学，就这样“档案”中注明了何时何地加入共青团。我在自己小学的时候就被老师要求戴红领巾加入少先队。
 <br/>
 自从翻墙后众多的新闻媒体的报导：1989年6月4日用坦克碾压学生、中国共产党对法轮功学员的迫害、新疆的再教育营、香港的返送中、西藏的活佛转世，现在正在进行中的传染病毒、各种封城等等罪状无不证明其邪恶本质，无论其如何掩盖也无从改变事实真相。总有一天历史的真相会大白于天下，届时与中国共产党站在一起的不论是少先队员、还共青团员如何自处？
 <br/>
 我的家庭自父辈就遭受中共的各种迫害，父亲的家庭遭公私合营、之后又遭受辞退勒令回乡的悲惨遭遇，之后又是因“家庭成分”受到各种歧视与排挤，直到如今所有的都没有解决。从记事开始父母就为这件事耿耿于怀，到处找政策、上访等等各种办法都无济于事。其实，根本不能解决，一个独裁、极权的政党怎么会能被监督被纠正。中共系统性的腐败、独裁各种宣传，只要发对它就是叛国，只要有人试图批驳它的各种独裁政策就会有100种方法刑事他（提出反对意见的人）。中共自始至终就是欺骗民众，直到骗得中共自己都相信了也不知悔改，不要心存侥幸等待中共得改革。
 <br/>
 我们的父辈们怎么被骗的就会在我们这一代人身上复制、重演。我是被计划生育影响的一代人，是被房屋货币化改革的一代人，是被要求“延迟退休”的一代人……可笑的是我们还是鼓励生二胎的一代，我们即是鼓励购房又是限制卖房的一代人。 我在国企有过一年的工作经历，因为只有国企才有城市落户的名额，辞职时的各种刁难，扣着本属于个人的证件不放，这就是“党组”对自己员工做为，党组不止管理党员也包括普通员工。中共领导下的工会几时发过声？就连表演一下都不会去做。加在民众身上的“镣铐”太多了：户口、准生证、职称、档案、成分、粮食关系、政治面貌…… 中国民众有眼睛但看不到、有头脑但无法去思考，民众太在乎自己的私利，放弃了太多属于自己权力，导致无从抗争，把寄生在中国劳苦大众上的“寄生虫”、“毒瘤”养的太大了，割舍不得，直到它就是自己的一部分，这其实一个魔咒。
 <br/>
 中共最看不得民众有信仰，不论是基督教、佛教、道教还是伊斯兰教，中共都千方百计的控制，法轮大法更是中共眼中刺，容不得。根源就是不论是任何组织只要不受中共控制就有反对它的可能，他要根除所有对它有不利可能的各种社会团体，这种不是邪恶什么是邪恶。穷尽各种手段，无所不用其极，一个满嘴谎言的政府、政党根本没有辩驳的权力，我们有理由相信对中共的所有的指控。对付中共这个邪恶的政党，这个邪教，只有正义、诚信的力量才会颠覆它。
 <br/>
 所以，我真诚的希望这个邪恶的政党从这个世界上消失，早日遭到清算，换世间的太平，让世人远离这个政党、远离谎言与邪恶，我退出少不经事时加入的这个邪恶政党的附属组织——少先队和共青团，也同时退出包括但不限于工会、协会等等所有中共用于迫害、控制民众的各种组织团体。
 <br/>
