 台湾宜兰县法轮功学员在县政府行政大楼向师父拜年。（大纪元）
</div>
</div><hr/><div><p>
 【大纪元2019年12月31日讯】2019年12月29日，来自宜兰县各乡镇的部分法轮功学员，在县政府行政大楼向他们的师父
 <ok href="http://www.epochtimes.com/gb/tag/%E6%8B%9C%E5%B9%B4.html">
  拜年
 </ok>
 。值此象征除旧布新的岁末，宜兰学员由衷感念师父的谆谆教诲与慈悲救度之恩，合影恭祝师父新年好，表达内心最真挚的祝愿，并互相鼓励来年更精进。
</p>
<p class="p2">
 <span class="s1">
  法轮大法（又称法轮功）自1992年5月传出以来，已洪传一百多个国家，使全世界超过一亿人身心受益，深受各族裔人士喜爱。法轮功的学炼者普遍身强体健、心性道德提升。法轮功学员实践“真、善、忍”后所展现的精神面貌，也给社会带来清新、祥和的气象。
 </span>
</p>
<h4 class="p2">
 <span class="s1">
  <b>
   炼功喜获重生
  </b>
 </span>
 <span class="s1">
  <b>
   感念师恩浩荡
  </b>
 </span>
</h4>
<p class="p2">
 <span class="s1">
  宜兰县法轮功学员，遍及士农工商各行各业，包括大学教授、县政府公务人员、开业医师、药师、护士、工程师、中小学教师、大学生与家庭主妇等社会各阶层人士。受益于法轮大法赐给他们崭新的生命，从原本药罐子变成无病一身轻，“一人炼功，全家受益”，很多人亲身体会了心性提升、身体健康，家庭平安幸福的感受。
 </span>
</p>
<p class="p2">
 <span class="s1">
  目前住在宜兰的何来琴回忆，1994年6月，她经由中国大陆的亲戚建议，到山东济南去听李洪志师父讲法，在“法轮功济南第二期学习班”的短短几天内，困扰她长达二十多年的多种宿疾神奇般消失，有如脱胎换骨，她由衷感激师父恩赐新生。
 </span>
</p>
<p class="p2">
 <span class="s1">
  家住五结乡、就读台湾大学财经所博士班的张瑄凌表示：“我修炼法轮功后，明白了什么是真，如何善良地对待别人、为人着想，遇到矛盾退一步海阔天空，在待人处事与学业上更加得心应手。学炼五套功法让我身体健康，每天都觉得很轻松，不会有疲累的感觉。还有一种普遍现象就是法轮功学员都会显得很年轻，就算我年纪已经不小了，还是很多人会误以为我是高中生或大学生”。
 </span>
</p>
<p class="p2">
 <span class="s1">
  八十四岁的退休西装裁缝师傅林伯伯，1997年5月开始学炼法轮功，才炼没多久，原本手指头的关节痛、身体酸痛、颈椎僵硬的毛病都不见了，脸色愈来愈红润，也戒去了每天要抽一、二包烟的烟瘾，吃了四十几年中药的梦魇终于远离。
 </span>
</p>
<p class="p2">
 <span class="s1">
  厨师退休的罗东学员王先生以感恩的口吻表示，“修炼前我的脊椎骨容易受伤，虽然靠游泳健身，但一有感冒就得拖个一星期才好。炼功后，脊椎不再受伤，从此身强体健，不曾罹患感冒。”
 </span>
</p>
<h4 class="p2">
 <span class="s1">
  <b>
   幸得世间至宝
  </b>
 </span>
 <b>
 </b>
 <span class="s1">
  <b>
   无病一身轻
  </b>
 </span>
</h4>
<p class="p2">
 <span class="s1">
  郭千华二十多岁时即罹患子宫内膜异位，生理期经常疼痛甚至昏倒，以致无法工作。遍访中西医均束手无策，经人介绍去医院找名医做了手术，不料两个月后又复发，自以为此生难脱顽疾缠身。2000年7月，郭千华收听电台广播时得知法轮功，随即找到了宜兰运动公园炼功点。修炼四个月后，恼人的宿疾不翼而飞，从此不再经痛，连贫血也一并消失无踪。
 </span>
</p>
<p class="p2">
 <span class="s1">
  高中退休教师李光辉，原本喜欢打牌，经常熬夜，烟瘾很大，身体状况很差，做过各种医疗检查都查不出病因，中西医都药石罔效，甚至有轻生的念头。二零零零年底，学校同仁鼓励他炼法轮功，从此他学法炼功，恢复健康，严格要求自己守住心性，并把打牌和抽烟等不良嗜好都彻底戒掉。
 </span>
</p>
<p class="p2">
 <span class="s1">
  服务于台湾电力公司的宜兰学员王添丁多年前发生严重车祸，亲友把他送去医院急诊，
 </span>
 <span class="s4">
  X
 </span>
 <span class="s1">
  光显示腿部骨折，医生要他住院。王添丁没有住院，而是回家持续学法、炼功，三星期后就回去上班。他身体很快就恢复健康，双盘炼功也没问题。法轮大法的神奇超常，真切地印证在王添丁身上，他非常感念师父救度之恩。
 </span>
</p>
<p class="p2">
 <span class="s1">
  苏澳镇一家塑胶袋工厂的老板林先生原本长期失眠，伴随着忧郁，患有严重便秘及神经衰弱的毛病，修炼后众病全消，一家十人都是法轮功修炼者，全家和乐融融、一片祥和。
 </span>
</p>
<p class="p2">
 <span class="s1">
  每逢新年或师父生日（也是法轮大法日），
  <ok href="http://www.epochtimes.com/gb/tag/%E5%AE%9C%E5%85%B0%E6%B3%95%E8%BD%AE%E5%8A%9F%E5%AD%A6%E5%91%98.html">
   宜兰法轮功学员
  </ok>
  总是齐聚一堂，向师父
  <ok href="http://www.epochtimes.com/gb/tag/%E6%8B%9C%E5%B9%B4.html">
   拜年
  </ok>
  或贺寿，略表内心的万分感恩，一如
 </span>
 <span class="s5">
  全世界法轮功学员通过明慧网向师尊的贺卡所示，表达他们得度重生的谢忱。2020
 </span>
 <span class="s1">
  新年将届，声声“恭祝师尊新年好”、“法轮大法好、真善忍好”，是
  <ok href="http://www.epochtimes.com/gb/tag/%E5%AE%9C%E5%85%B0%E6%B3%95%E8%BD%AE%E5%8A%9F%E5%AD%A6%E5%91%98.html">
   宜兰法轮功学员
  </ok>
 </span>
 <span class="s5">
  感念
 </span>
 <span class="s1">
  师恩浩荡的诚挚心意。
 </span>
</p>
