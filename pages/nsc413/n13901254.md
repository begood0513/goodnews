 【大纪元2023年01月07日讯】
</p>
<p>
 <strong>
  申明退出少先队、共青团。还好我没有入党！
 </strong>
 <br/>
 今年使我坚定了决心，要在此申明：我要退出青少年时期曾经加入的那些组织，其实是为那些准备被洗脑的成年人所准备。
 <br/>
 从铁链女事件的不准说，到上海极端封控，再到唐山打人事件，这些我看得见但摸不着的事件已经让我很想站出来点醒老家那些麻木不仁的人们！
 <br/>
 直到年底最终动了回家的念头，我再也不能沉黙下去！因为中共肆无忌惮地向世界散播病毒，距离上次回国探亲已是三年零四个月…落地香港二度自我隔离后，终于过关来到大陆健康驿站所谓的“酒店”进行强制隔离5+3天。
 <br/>
 明明是强制自费隔离却在群里要求大家承认是“自愿”；只提供一张擦手巾而不换需要使用5天；期间规定不允许传递物品（比如收送外卖等都不允许！只能吃里面提供的食物，我看到的大块猪肉里面是紫红色，根本不敢吃全部放垃圾袋！）这种集中营似的管理让我怀疑共党要求隔离的真正动机！
 <br/>
 为什么我们在海外都生活好好的，连口罩都不用戴，一到了大陆大白们就如临大敌把入境人员强制独处一室8天，每天或隔天强制测核酸，稍不如他们的意就有可能提供问题食物弄成“阳性”？这种隔离方式让人们有苦不能说，有冤不能报。
 <br/>
 等到我刚结束隔离出来深圳时，正好遇上全城尽阳的时刻！在家人的强烈建议下，我被迫中途取消了年底的探亲计划终于回到自由国度。感觉像是逃也似地离开。。。
 <br/>
 然后，看到北京、上海、成都等地的火葬场实景新闻，我早已看清国内的权势造成的一些恶果，很庆幸在18年前就已经决定不做体制内的人，对于中国现在的种种乱象，我只能说，是普通人不醒悟、不作为的结果。
 <br/>
 希望更多的青年不要入团入党，从小树立自己真实的理想、善良的愿望、勇于忍耐，真正看清自己未来的方向！
 <br/>
 一位看清共党邪恶本质的人 张琳霞
 <br/>
 籍贯：四川成都
 <br/>
 祖上在火车北站附近经营多亩果园土改时期被共党充公，导致家业丧失。这些都是共产党欠的人民的债，早晚要还的，天在看！
