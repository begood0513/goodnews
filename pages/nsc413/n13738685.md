<div><p>
 【大纪元2022年05月17日讯】（大纪元记者林燕报导）
 <span class="s4">
  中共央视上周五（5月13日）播出“美民众对政府抗击疫情不力表示不满”的3分钟宣传视频，因
 </span>
 <span class="s5">
  严重脱离事实而引发海内外网民集体吐槽。
 </span>
</p>
<p class="p6">
 <span class="s7">
  央视镜头显示商场里
  <span class="s5">
   人来人往、人们在悠闲地逛街购物，记者许弢却在视频中表情焦虑地说：
  </span>
 </span>
 “现在当我们走在美国的街头上时，几乎已经看不到多少戴口罩的人了。
 <span class="s1">
  美国是真的走出了疫情？还是在继续走向疫情的深渊？”
 </span>
</p>
<p class="p6">
 <span class="s7">
  她还说：“
 </span>
 <span class="s1">
  更糟糕的情况恐怕还在后头——据近期20多位科学家、医生和公共卫生专家共同发表的一份报告指出，
 </span>
 <span class="s7">
  美国想要距离回归正常生活还有很长一段路要走。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  现居纽约的时事评论员王赫表示，
 </span>
 <span class="s1">
  疫情尚未结束，美国学术自由、言论自由，对美国未来疫情的走向有不同看法是太正常不过了。但是，现在美国基本上走出了对疫情的恐慌、生活正常了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “这就是美国人的自信。”他说，“比较而言，中共封锁疫情数据，没有任何第三方可以独立查证，张嘴谎言怎么说都行。
  <span class="s2">
   ”
  </span>
 </span>
</p>
<p>
 在过去一个多月以来，上海的混乱封城已经激起了民间对中共高层
 <ok href="https://www.epochtimes.com/gb/tag/%E6%B8%85%E9%9B%B6%E6%94%BF%E7%AD%96.html">
  清零政策
 </ok>
 的激烈批评，居民们一直在想尽办法自救，如何获取足够的食物和药品，而激增的病例数已使上海的医疗设施不堪重负。
</p>
<p>
 中国首都北京市也从13日起，一连三天在12个区对民众进行区域核酸检测。当局需要转移民众对极端
 <ok href="https://www.epochtimes.com/gb/tag/%E6%B8%85%E9%9B%B6%E6%94%BF%E7%AD%96.html">
  清零政策
 </ok>
 的民愤。
</p>

</h4>
<p class="p4">
 <span class="s1">
  <span class="s4">

  </span>
 </span>
</p>
<p class="p4">
 “共产党中宣部的脸皮真的比城墙拐弯角还厚。”有一位网民说。还有人说：“周围人自如坦然的表情就说明了一切。”
</p>

 </span>
 <span class="s6">

 <span class="s1">

  </span>

 </span>
</p>
<p class="p4">
 <span class="s1">
  仅从媒体角度看，该名记者的做法也欠缺公平。在商场人流中，她是唯一一个戴口罩的人，既然批评周边人不戴口罩、不重视防疫措施，她却没有给现场任何一位不戴口罩的人士一个采访机会，或者听取他们的说法。
 </span>
</p>

 <span class="s1">

 <span class="s1">

 </span>
 <span class="s6">
  good luck
 </span>
 <span class="s1">
  ）。”
 </span>
</p>

 <span class="s2">
  远超过疫情本身的巨大民生灾难，表明当局没有一丁点的自信，对疫情是恐慌的，丧失了理智，歇斯底里，这种恰恰是灭亡前的表现。”

</p>
