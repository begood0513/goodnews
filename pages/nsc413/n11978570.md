 <ok href="https://www.epochtimes.com/gb/tag/%E6%AD%A6%E6%B1%89.html">
  武汉
 </ok>
 同济
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8C%BB%E9%99%A2.html">
  医院
 </ok>
 ，医院却拒绝收治。120急救车上的两位工作人员与医院工作人员理论，问他们为什么不收治病人，医院保安出面搪塞
 <ok href="https://www.epochtimes.com/gb/tag/%E6%8B%92%E6%94%B6.html">
  拒收
 </ok>
 。
</p>

 <br/>
 120急救工作人员：我们是120的，这儿有一个糖尿病足的病人，你们收不收？
 <br/>
 <ok href="https://www.epochtimes.com/gb/tag/%E5%8C%BB%E9%99%A2.html">

 <br/>

 <br/>

 <br/>
 医院保安：你说什么呀？
 <br/>
 120急救工作人员：我什么呀，我们是120的，送病人，你们医院不收病人，我想问一下什么原因。
 <br/>
 医院保安：说话好好说，你是什么态度啊？
 <br/>
 120急救工作人员：什么是什么态度啊，好好在说呀。
 <br/>
 医院保安：有话说呀，沟通就完了。
 <br/>
 120急救工作人员：你们收不收？我问你们收不收？
 <br/>
 医院保安：你问我收不收？
 <br/>
 120急救工作人员：那你是哪一位？
 <br/>
 医院保安：我想问你干何事？
 <br/>
 120急救工作人员：我是120的，我已经表明身份了，你是哪一位？
 <br/>
 医院保安：那你这个说话的态度就不对，你晓不晓得？
 <br/>
 120急救工作人员：什么不对，就这态度，我就问你，这个已经跟社区的人沟通了很时间了，你们屁股对着这个病人，不收病人，也不看说没骨科医生，我想问一下你们是不是不收病人了？不收病人我们就拖走，你们给个明确的回复，你们收不收？
 <br/>
 医院保安：不是已经告诉你们了吗？
 <br/>
 120急救工作人员：你不能再说一遍吗？你们是不是不收病人了？不收病人我们就拖走。
 <br/>
 医院保安：他是什么事啊？
 <br/>
 120急救工作人员：糖尿病足啊，这个病人足底烂了，你们看不看？
 <br/>
 医院保安：你是找骨科医生吗？
 <br/>
 120急救工作人员2：我们是来找急诊外科，我哪晓得你们这里是什么情况。
 <br/>
 医院保安：我们这里没有急诊科医生。
 <br/>
 120急救工作人员2：那就是没得病床，没得医生，不看，要我们走，就这个意思。所以说我们要说清楚，你们就到底，人命关天，对吧，他也是个病人。
 <br/>
 医院保安：我现在告诉你，我们这里没有骨科医生。
 <br/>
 120急救工作人员2：那这里有没有个领导说了算呢，我们得找个急诊外科呀，我拖到骨科来你们不让进。
 <br/>
 医院保安：我们这里没有骨科医生，我已经说了。
 <br/>
 120急救工作人员2：那就是不收，是不是这个意思？
 <br/>
 医院保安：没有医生，谁给你们看呀？
 <br/>
 120急救工作人员：你们同济医院没有骨科医生？
 <br/>
 医院保安：医生去支援去了，光谷分院去了。
 <br/>
 120急救工作人员：整个同济医院都没骨科医生？
 <br/>
 医院保安：现在没有。
 <br/>
 120急救工作人员：是不是没得？
 <br/>
 医院保安：现在是没有。
