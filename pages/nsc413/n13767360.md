<div><p>
 【大纪元2022年06月25日讯】
 <span class="s1">
  （大纪元东京记者站采访报导）菅原潮曾是
  <ok href="https://www.epochtimes.com/gb/tag/%E6%97%A5%E6%9C%AC.html">
   日本
  </ok>
  黑社会山口组的一个组长，2015年他金盆洗手，现在是一名经济评论家，有十几本经济方面的著作。他也是一名网络名人，常常发表批评中共的言论，他在推特拥有22万粉丝。
 </span>
 <span class="s3">
  15
 </span>
 <span class="s1">
  年前，他意外地在中国知道了
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B4%BB%E6%91%98.html">
   活摘
  </ok>
  人体器官的内幕。
 </span>
</p>
<p class="p3">
 <span class="s3">
  2007
 </span>
 <span class="s4">
  年，
 </span>
 <span class="s1">
  菅原的朋友的哥哥肝病恶化，被宣告余命不长，除非接受器官移植。朋友通过中介很快在中国找到了供体，并准备接受手术。
 </span>
 <span class="s1">
  可是在手术前，医院准备的一种叫白蛋白的血液制品被发现是假货，不能用于手术。朋友请菅原在
  <ok href="https://www.epochtimes.com/gb/tag/%E6%97%A5%E6%9C%AC.html">
   日本
  </ok>
  采购白蛋白并将其送到北京。因此，菅原得以了解中共
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B4%BB%E6%91%98.html">
   活摘
  </ok>
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </ok>
  学员器官的黑幕。
 </span>
</p>
<p class="p3">
 <span class="s3">
  6
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  20
 </span>
 <span class="s1">
  日，菅原潮在东京接受大纪元的采访，讲述了事情的经过。
 </span>
</p>
<p class="p3">
 <strong>
  <span class="s1">
   记者：菅原先生，能不能先请您讲一下当时的经历？
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s5">
  菅原：
 </span>
 <span class="s3">
  2007
 </span>
 <span class="s1">
  年，我朋友的哥哥肝病恶化，并被告知他的时间已经不多了，唯一的办法就是接受肝脏移植。当时肝脏移植，也只有在美国、法国或中国才能进行。而在美国和法国，等待时间长，费用也高，而且有严格的法律限制。所以他选择到中国接受手术。
 </span>
 <span class="s3">
 </span>
</p>
<p class="p4">
 <span class="s1">
  <ok href="https://www.epochtimes.com/gb/tag/%E5%8C%97%E4%BA%AC%E6%AD%A6%E8%AD%A6%E6%80%BB%E5%8C%BB%E9%99%A2.html">
   北京武警总医院
  </ok>
  接受来自日本的患者，也有沙特阿拉伯和德国的富人来接受手术。医院
 </span>
 <span class="s5">
  说
 </span>
 <span class="s1">
  很快可以找到供体，费用为
 </span>
 <span class="s3">
  3,000
 </span>
 <span class="s1">
  万日元。
 </span>
 <span class="s3">
 </span>
</p>
<p class="p4">
 <span class="s3">
  2007
 </span>
 <span class="s1">
  年
 </span>
 <span class="s3">
  8
 </span>
 <span class="s1">
  月，中国方面
 </span>
 <span class="s5">
  说
 </span>
 <span class="s1">
  找到了供体，随时可以进行手术。手术需要白蛋白，即浓缩的人血。但在手术前发现，医院的白蛋白是假货，没法手术。所以让我在日本买，并送到北京。
 </span>
</p>
<p class="p4">
 <span class="s3">
 </span>
 <span class="s1">
  因此，我得以知道了所有的
 </span>
 <span class="s5">
  内
 </span>
 <span class="s1">
  幕。
 </span>
</p>
<p class="p4">
 <span class="s1">
  我设法在日本找到了白蛋白，但由于它属于药品，进出口需要有许可证。在日本方面，我想办法带出了海关。我按照中国那边中介方的指示，从大连入境，然后搭乘他们指定的航班去了北京。
 </span>
 <span class="s6">
  在北京，我遇到了麻烦。当时有武警高官来接我，但机场人员在托运的行李里发现了药品，说没有许可不可带出去，我被扣留了。
 </span>
</p>
<p class="p4">
 <span class="s1">
  机场警察、公安警察和武警都不属于同一系统的，所以他们互不相让，争吵了几个小时。最后还是通过政治人物的作用，他们终于让我离开了机场。
 </span>
</p>
<p class="p4">
 <span class="s1">
  到北京之后，我把药品交给医院，并在手术的前一天去看望了朋友的哥哥。
 </span>
</p>
<p class="p4">
 <span class="s1">
  在那里，一位负责接待的医生告诉我
 </span>
 <span class="s5">
  说
 </span>
 <span class="s1">
  ，供体就在隔壁，你要不要看一看？他拉开布帘，我看到供体
 </span>
 <span class="s6">
  躺在床上，是一位
 </span>
 <span class="s3">
  21
 </span>
 <span class="s5">
  岁的年轻男子
 </span>
 <span class="s1">
  。因为打了麻药，他看上去没有意识。
 </span>
</p>
<p class="p4">
 <span class="s1">
  这位医生去日本留过学，会说日语。
 </span>
</p>
<p class="p4">
 <span class="s1">
  他告诉我，供体是一个坏人，被判处死刑的罪犯，他反正最终会死的，就让他在死前再做点贡献。并说，“他很年轻，肝脏非常健康。”
 </span>
</p>
<p class="p4">
 <span class="s3">
 </span>
 <span class="s1">
  我问他，这个人是干什么的。他
 </span>
 <span class="s5">
  说
 </span>
 <span class="s1">
  “是恐怖组织的成员”。我继续追问他到底干了什么，他回答
 </span>
 <span class="s5">
  说
 </span>
 <span class="s1">
  他是“
  <ok href="https://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </ok>
  ”。
 </span>
</p>
<p class="p4">
 <span class="s1">
  但是，手术最终失败，病人在手术中死了。
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：你看到那位年轻人时，他当时是什么样子？
  </span>
 </strong>
</p>
<p class="p6">
 <span class="s5">
  菅原潮：
 </span>
 <span class="s7">
  我看到他
 </span>
 <span class="s1">
  躺在那里，手脚都缠着绷带，他在前一天被切断了双手和双脚的肌腱。医生告诉我是为了防止他逃跑，还有就是人在害怕的时候，身体会蜷缩，
 </span>
 <span class="s5">
  影响
 </span>
 <span class="s7">
  器官的质量，所以他们给他做了肌腱切断手术。
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：器官移植的时候，他还活着？
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s3">
 </span>
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  当然，我去看他的时候他还是活着的，器官摘除后这个人就会死掉，所以是活体移植。器官摘除的同时做手术效果最好。遗体后来怎么处理的，我就不知道了。
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：你的朋友花了多少时间找到了供体？
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  他先去中国做了检
 </span>
 <span class="s5">
  查
 </span>
 <span class="s1">
  ，然后回日本等待。一个月后，中国方面就找到了一个匹配的供体。
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：中介是一个什么人？
  </span>
  <span class="s3">
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  他是作医疗中介的。
 </span>
 <span class="s3">
  2007
 </span>
 <span class="s1">
  年，很多有钱的中国人开始组团来日本进行医疗旅游，他也是干这个的。他是中国人，曾在日本留学，并有相当广的人际关系，和不少日本有名的医生有来往。
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：日本医生是否知道这些器官的来源有问题？
  </span>
  <span class="s3">
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  他们当然知道，但他们故意回避这个问题，因为医生和病人的利益是一致的。而中方的理由是，因是死刑犯，反正最终会被处决，为什么不让他们再为别人做一些好事再死呢？
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：武警总医院积极开展器官移植手术？
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s3">
 </span>
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  是的。据他们介绍，欧美、俄罗斯、沙特阿拉伯和其它中东国家的有钱人都会来做手术。我看到了一些欧美人。日本人当时只有我的朋友在，但我听说很多日本人来做手术。
 </span>
</p>
<p class="p4">
 <span class="s1">
  患者们有一个专门等候的地方，我估计应该是在附近的酒店。医院经常对他们进行体检
 </span>
 <span class="s6">
  。
 </span>
 <span class="s3">
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：中共官员也参与其中？
  </span>
  <span class="s3">
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  当然。他们
 </span>
 <span class="s5">
  说了
 </span>
 <span class="s1">
  ，没有中共官员的参与，是不可能开展器官移植的，很多事没法处理。
 </span>
</p>
<p class="p4">
 <span class="s1">
  在机场，我们使用的是特别出口，是只为高官准备的通道和秘密地下隧道，我没有看到其它车辆。当时除了来接我的高官，还有
 </span>
 <span class="s8">
  4
 </span>
 <span class="s1">
  个持枪的武警。从机场去北京，是武警的车为我们开道。
 </span>
 <span class="s3">
 </span>
</p>
<p class="p4">
 <span class="s1">
  我不知道这位高官是什么级别，但能感觉他有一定的权力。他来机场接我的时候，专车直接开到飞机下面，然后海关官员再到飞机下为我盖章，之后专车从写着“贵宾口”的地方开出去。
 </span>
</p>
<p class="p4">
 来接我的车是两辆黑色凌志，车前面插着旗，走的都是专用通道和秘密通道。
</p>
<p class="p7">
 <span class="s1">
  我当时和高官交换了名片，但因为时间太久了，现在已经没有名片了。
 </span>
</p>
<p class="p4">
 <strong>
  <span class="s1">
   记者：日本主要的媒体知道此事吗？
  </span>
 </strong>
</p>
<p class="p4">
 <span class="s5">
  菅原：
 </span>
 <span class="s1">
  他们知道，但他们不报导，他们还想在中国继续开展业务。
 </span>
</p>
<p class="p4">
 <span class="s1">
  当时有一家日本的主要媒体的记者也在，他们希望采访器官移植的事，当然被院方拒绝。我和中介的人一起吃饭的时候，记者也在。
 </span>
</p>
<p class="p4">
 <span class="s1">
  这件事真是很残忍，现在谈起来，都觉得很残忍。花了3000万，并搭上两条人命，对谁都不是好事。
 </span>
</p>
<p class="p4">
 <span class="s1">
  但是中国人有冠冕堂皇的理由，他们觉得是在做正确的事情。都被洗脑了。
 </span>
</p>
<p class="p4">
 中国的医生都这么想，他们不认为自己做错了什么，认为是再处理死刑犯，都被洗脑了，他们都那么想。
</p>
<p class="p4">
 的确是很残酷的事情。
