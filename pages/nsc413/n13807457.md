<p>
 “对于安倍骤然离世的消息，直到现在仍令人难以接受。”赖清德说，他曾有很大的缘分，能为安倍送行并参与告别式，看着照片上的他，英挺依旧、笑容灿烂的模样，让人不禁想起他们过去的互动，直到现在，这份温暖真挚的情谊，仍如阳光般让人感动。​
 <br/>
 ​
 <br/>
 赖清德说，从2016年台南大地震发生时，安倍即表达对台湾的友好，不只立刻派遣团队，来台进行救援与协助，更在第一时间送来物资；2018年花莲地震，安倍也亲自写下“台湾加油”的字版，为台湾加油送暖；当台湾农民受到中共打压，他更亲自吃凤梨，帮助推销台湾农产品。​
 <br/>
 ​
 <br/>
 “尤其，在台湾面临疫苗缺乏之际，”赖清德说，安倍更促成日本政府，捐赠台湾400多万剂疫苗，帮助台湾度过难关，此后，更在国际上持续声援台湾，发表“台湾有事，等同日本有事”的坚定立场。​
 <br/>
 ​
 <br/>
 “安倍为台日关系付出的努力，至今回想起来仍历历在目”，赖清德说，“我们永远怀念他的真诚、他的温暖，并感谢他对国际社会的贡献，与维持印太区域和平稳定的战略倡议。​”
 <br/>
 ​　​
 <br/>
 他相信，台日友好的情谊关系，以及台湾社会对安倍的感谢，不会因为安倍的离世而改变，反而会更加地深化，延续他的信念与理想，持续让台日关系越加紧密。​
 <br/>
 ​
 <br/>
 赖清德表示，记得安倍曾在书中期许自己，能成为“爱国家、爱人民，虽千万人吾往矣的行动派”，藉由这场追思音乐会，他想告诉安倍，“您不只做到了，甚至做得更多、更好。敬请您在天上安息，我们会永远感念你”。“台湾会继续与日本互相扶持，一起捍卫民主自由，感谢台湾永远的挚友，感谢安倍前首相。”
