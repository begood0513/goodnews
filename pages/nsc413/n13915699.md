  声明被邪党欺骗下所说所做的一切作废
 </strong>
 <br/>
 本人由女儿代为发表此声明：在我一生中，因为邪恶共产党的逼迫、威胁和欺骗下所做的一切不符合天理的事情作废，也同时为自己在无知的情况下随和这个恶党所说过的话和做过的事情深深忏悔。
 <br/>
 我出身于一个地方的首富家庭，我知书达理的父亲自费在当地开了一所学校造福民众，父亲的兄弟也因为长期的勤劳而使得这个家族富甲一方。在土改和各种运动中，我熟识水性的父亲被“自杀”在一条水深仅过漆盖的水沟里，听说是因为当地恶党恶霸仗着土改运动逼迫我父亲“坦白”钱藏在了哪里，其实家族的钱财已被恶党抢劫一空，我父亲因为交代不出还有钱藏在哪里而被“自杀”。我的兄长是一位优秀的教师，因为承受不了每天被侮辱、毒打、批斗，吊死家中。我的姐姐被邪党蛊惑逼迫要上台批斗甚至殴打我母亲，要与地主成分的父母“决裂”。当时我年仅四岁。
 <br/>
 在毫无生路的情况下，母亲带着我逃到了另外一个城市，我改了名字，从此在一个陌生的地方过起了社会底层的生活，再也没有回过令人心碎的家乡。期间，我目睹了各种各样的运动，都是挑拨一些群众去批斗另一些群众。为了生存，我无可奈何地要顺着邪党的要求去说话和做事，被了保存自己。
 <br/>
 1992年，李洪志师父传出了法轮大法，我有幸接触到大法，非常惊喜，马上开始修炼，每天都到附近的公园和其他人一起炼功、学法。然而，1999年江泽民发起了对大法的迫害，我身边的炼功点辅导员一个个被监控，各种谣言和假的东西到处传。为了讲清真相，一天我骑着自行车去市政府要去讲真相，然而在路上，我看到和感受到了土改文革哪种风雨欲来的恐怖气氛。从小就在我心中滋生的恐惧逐渐地控制了我，加上得不到真实的资讯，我的怕心越来越重，为了逃避现实，转而走入佛教。我的女儿去了国外，她仍然在坚持修炼大法。
 <br/>
 打从邪恶共产党夺取中国的政权开始，它给每个中国人、每个家庭和这个国家带来的苦难从未停止过。更可怕的是，中国人在这种长期的洗脑和欺骗中，根本就很难识别真假，甚至善恶不分，其中包括我。
 <br/>
 我真心希望，我在这一生中随着邪党带动、欺骗和逼迫下所说的、所做的一切错事都作废，和这个邪恶的党彻底决裂。我希望大法能够早日回到中国，给这里善良的民众重塑道德和对神的信仰。也希望我女儿能够回到中国。
