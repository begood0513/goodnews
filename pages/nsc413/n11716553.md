</div><hr/><div><p>
 【大纪元2019年12月12日讯】
 <span class="s1">
  锯门声越来越大，楼道里嘈杂声越来越多，门眼看就要被锯开，妻子
  <ok href="http://www.epochtimes.com/gb/tag/%E6%9D%8E%E8%89%B3%E6%9D%B0.html">
   李艳杰
  </ok>
  第一个握着接起来的床单从六楼顺下，丈夫
  <ok href="http://www.epochtimes.com/gb/tag/%E5%AE%AB%E5%87%A4%E5%BC%BA.html">
   宫凤强
  </ok>
  也接着下来，大概在四楼处床单突然断裂
 </span>
 <span class="s2">
  ……
 </span>
</p>
<p class="p1">
 <span class="s1">
  12月的北方天寒地冻，北风呼啸！等昏迷的
  <ok href="http://www.epochtimes.com/gb/tag/%E5%AE%AB%E5%87%A4%E5%BC%BA.html">
   宫凤强
  </ok>
  苏醒过来时，大概已下半夜，警车还在楼下停着，警察还在楼上做恶，他摸摸身边的妻子已没有任何生命迹象，极度悲伤的宫凤强，此刻却无法送妻子最后一程…….
 </span>
</p>
<p class="p1">
 2019年12月10日，中国大陆知情人士投稿大纪元，揭露这对夫妻遭遇中共迫害，
 <ok href="http://www.epochtimes.com/gb/tag/%E5%AE%B6%E7%A0%B4%E4%BA%BA%E4%BA%A1.html">
  家破人亡
 </ok>
 的悲惨故事。
</p>
<p class="p1">
 <span class="s1">
  黑龙江省依兰县
  <ok href="http://www.epochtimes.com/gb/tag/%E8%BE%BE%E8%BF%9E%E6%B2%B3%E9%95%87.html">
   达连河镇
  </ok>
  的
  <ok href="http://www.epochtimes.com/gb/tag/%E6%9D%8E%E8%89%B3%E6%9D%B0.html">
   李艳杰
  </ok>
  ，现年41岁，丈夫宫凤强现年48岁，夫妻二人因修炼法轮功的缘分走到一起。在中共20年血雨腥风的迫害中，他们无数次地被非法抓捕
 </span>
 <span class="s2">
  、
 </span>
 <span class="s1">
  关押
 </span>
 <span class="s2">
  、
 </span>
 <span class="s1">
  酷刑
 </span>
 <span class="s2">
  、
 </span>
 <span class="s1">
  判刑
 </span>
 <span class="s2">
  ……
 </span>
 <span class="s1">
  丈夫宫凤强被冤判5年，在监狱被迫害致精神失常，奄奄一息。也正因为如此，他们失去了令人羡慕的工作、幸福的家庭，背井离乡、颠沛流离，经济拮据，直至李艳杰被迫害致死
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  依兰县
  <ok href="http://www.epochtimes.com/gb/tag/%E8%BE%BE%E8%BF%9E%E6%B2%B3%E9%95%87.html">
   达连河镇
  </ok>
  ，这里民风淳朴古色古香，身为小学校长的父亲李忠孝育有三个女儿，三个女儿都被培育的通达情理，李艳杰是最小的一个，由于她聪明好学，从小学到大学都是老师眼中品学兼优的好孩子。
 </span>
</p>
<p class="p1">
 <span class="s1">
  1992年，法轮大法从吉林长春传出，很快就传播到了李艳杰读书的佳木斯农校校园，聪慧的李艳杰马上认同了“真、善、忍”的法理！
 </span>
</p>
<p class="p1">
 <span class="s1">
  毕业后她回家乡依兰做了一名幼师，李艳杰的勤劳善良很快得到家长们的信赖
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  此后，李艳杰遇到了同修宫凤强，婚后小夫妻生活美满幸福，丈夫在依兰煤矿工作，薪资待遇优厚
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但是好景不长，1999年7·20，中共前党魁江泽民利用手中的权利，军、警、特务、媒体齐上，在全国范围内掀起了二次文革式运动，发动了对法轮功修炼群体灭绝式的迫害打压，李艳杰生活的依兰县小城也没能幸免
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  依兰公检法人员为了捞取政治资本，对手无寸铁的李艳杰、宫凤强夫妇不遗余力地打压，抓捕、酷刑折磨，由于依兰公安“610”（专门迫害法轮功学员的非法机构）人员株连式的迫害，工作单位不敢继续聘用宫凤强，丈夫被迫失业。
 </span>
 <span class="s1">
  李艳杰也被迫离开幼师的工作。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2000年8月12日晚9点多，李艳杰刚生下女儿10天左右，坐月子时，公安又来抄家抓人，丈夫宫凤强眼睁睁地看着虚弱的妻子和刚出生的女儿，自己却被依兰公安非法抓捕。宫母为追恶警，昏倒在冰天雪地中，李艳杰光着脚把婆婆拖回
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  就是这样，公安也不肯放过这对夫妻，抄家之后，又派人24小时住在李艳杰的家里监控她们。
 </span>
</p>
<p class="p1">
 <span class="s1">
  等宫凤强从牢狱回来后，为了维持生计，李艳杰做起了冷面厂的生意。，由于李艳杰待人诚恳，从不短斤少两，冷面厂的生意很快红火。
 </span>
</p>
<p class="p1">
 <span class="s1">
  由于中共前党魁江泽民“名誉上搞臭、经济上截断、肉体上消灭”迫害政策的逐步升级，
 </span>
 <span class="s1">
  宫凤强夫妇又在多次抓捕中受尽酷刑折磨，为躲避迫害，冷面厂被迫关门。
 </span>
 <span class="s1">
  夫妻二人从此颠沛流离，流落到延寿县自谋生路。
 </span>
</p>
<p class="p1">
 <span class="s1">
  李艳杰后来学会了开出租车。她曾在出租车里拾到现金手机等贵重物品返还失主，失主为答谢她拾金不昧，给出租公司送锦旗表扬信等，却被依兰国保发现，到延寿县追捕，夫妻俩再次过上了流浪的生活。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2013年，依兰县大抓捕法轮功学员，李艳杰虎口脱险
 </span>
 <span class="s2">
  ，宫凤强被非法冤判5年，
 </span>
 <span class="s1">
  在监狱被酷刑迫害致精神失常，生命垂危！
 </span>
</p>
<p class="p1">
 <span class="s1">
  监狱怕出人命为了推卸责任，把没有任何生还希望的宫凤强办理保外就医扔回家，但在法轮大法神奇的功效下，奄奄一息的宫凤强奇迹般地活了下
 </span>
 <span class="s3">
  来。
 </span>
</p>
<p class="p1">
 <span class="s1">
  经历多次变故后，流落到七台河市的李艳杰以给中学生补课维持生计，平时成绩平平的学生，经过李艳杰的悉心补课讲解后，很快就成为班级里成绩领先者，家长们为感谢她，在手机微信里互相转告！
 </span>
</p>
<p class="p1">
 <span class="s1">
  而丈夫宫凤强只得隐姓埋名打些临时工赚些小钱
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2019年12月7日晚上8点多，七台河市总医院东侧一栋老式家属楼内，一阵急促的敲门声把夫妻二人惊醒，并听到外面公安叫嚷
 </span>
 <span class="s1">
  ：宫凤强，我们知道你们在这里，你们不开门我们也有办法
 </span>
 <span class="s2">
  ……
 </span>
</p>
<p class="p1">
 <span class="s1">
  这时李艳杰想到要通知亲属，但电话手机信息怎么发送都出不去，她知道外边来人已经带来仪器把她的手机信号屏蔽掉了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这时只听得外边有人去敲对门，对门没有给开门，只听有人拿起宫凤强的照片对准门镜问道，“你们认识这个人吗，他们是否在这里居住？”
 </span>
 <span class="s1">
  不知对方回答了什么，接下来敲门声、脚步声一阵紧似一阵。
 </span>
</p>
<p class="p1">
 <span class="s1">
  楼下警车灯闪烁，楼道里有人说找开锁的来，他们给开锁的打通了电话，但等了半小时开锁的也没来，听到他们中又有人说，开锁的不来也有办法，找当地公安借工具把门锯开，一会儿功夫就听到金属门被锯的声音。
 </span>
</p>
<p class="p1">
 <span class="s1">
  经历数次绑架酷刑折磨的宫凤强夫妇，不愿再去面对那些种种对精神和肉体的酷刑凌辱，
 </span>
 他们快速地拽下窗帘和被罩床单等，把它们链接到一起从窗台顺下去。
</p>
<p class="p1">
 <span class="s1">
  此时锯门的声音越来越大，楼到里嘈杂声音越来越多，门眼看就要被锯开，妻子李艳杰第一个握着接起来的床单从六楼顺下，丈夫宫凤强也接着下来，大概在四楼处床单突然断裂
 </span>
 <span class="s2">
  ……
 </span>
</p>
<p class="p1">
 <span class="s1">
  12月的北方天寒地冻，北风呼啸！等昏迷的宫凤强苏醒过来时，已经大概在零点之后的下半夜了，警车还在楼下停着，依兰和七台河两地的警察还在楼上做恶，宫凤强摸摸身边的妻子没有任何生命迹象！使劲推也不动，呼唤也没有任何回应，摸摸妻子的头，发现早已没有呼吸
 </span>
 <span class="s2">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  宫凤强极度悲伤，眼看着妻子年轻的生命就这样断送在中共的手里，自己却无力回天，更无法送妻子最后一程
 </span>
 <span class="s2">
  。
 </span>
</p>
<p class="p1">
 <span class="s1">
  他望着漆黑的夜空，摇摇晃晃离开了这个令他永远都不愿意回忆的地方
 </span>
 <span class="s1">
  ，消失在茫茫的夜空
 </span>
 <span class="s3">
  。
 </span>
</p>
<p class="p1">
 第二天一早，四辆警车和一辆特警车停在李艳杰租住的小区楼下，李艳杰的遗体早已被弄走。六楼李艳杰家敞开的窗户被关上，李艳杰家被撬坏的防盗门也被换成了新的。
</p>
<p class="p1">
 <span class="s1">
  一切似乎又恢复了宁静，但人命关天的大事怎能掩盖得了？
 </span>
</p>
<p class="p1">
 <span class="s1">
  中午10分，上李艳杰补习课的学生如往常一样陆续走进这栋楼房，但再也找不到她们的老师。
 </span>
</p>
<p class="p1">
 <span class="s1">
  7日晚上，被折腾了将近一夜的邻居们都在议论纷纷
 </span>
 <span class="s1">
  ：
  <span class="s1">
   “这警察半夜逼死人！
  </span>
  <span class="s2">
   ”“
  </span>
  <span class="s1">
   这些警察逼人跳楼！
  </span>
  <span class="s2">
   ”“
  </span>
  <span class="s1">
   现在的警察不干好事。
  </span>
  <span class="s2">
   ”“
  </span>
  <span class="s1">
   别说了
  </span>
  <span class="s2">
   ……
  </span>
  <span class="s1">
   被他们听到会报复的
  </span>
  <span class="s3">
   。”
  </span>
 </span>
</p>
<p class="p1">
 <span class="s1">
  12月9日上午，依兰公安派俩警察来到李艳杰80多岁的父母家中，恐吓二位老人，想套出宫凤强的下落，却
 </span>
 只字不提老人的小女儿李艳杰已经被他们逼死的情况。
</p>
<p class="p1">
 <span class="s1">
  现在，依兰公安警察还妄图找到李艳杰的姐姐和女儿。
 </span>
</p>
<p class="p1">
 <span class="s1">
  李艳杰的家庭遭遇只是中共迫害法轮功的一个缩写，更多被虐杀者无法上明慧网，罪恶至今被掩盖
 </span>
 <span class="s2">
  。
 </span>
</p>
<p class="p1">
 目前，李艳杰的家属和她生前的朋友、律师正在凝聚成一股正义的力量，准备声讨这些参与逼死李艳杰的中共警察。#
</p>
