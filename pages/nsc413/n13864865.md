 <div>
  (一)
  <br/>
  秋尽霜降渐入冬，
  <br/>
  姹紫嫣红已朦胧。
  <br/>
  乍暖还寒今又至，
  <br/>
  连天核检病无穷。
 </div>
 <div>
 </div>
 <div>
  (二)
  <br/>
  秋风露寒入
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%A5%E5%86%AC.html">
   严冬
  </ok>
  ，
  <br/>
  阴霾茫茫雾朦朦。
  <br/>
  期待来年春意暖，
  <br/>
  日照山河乐融融。
  <br/>
  (壬寅十月)
 </div>
 <div>
 </div>
 <div>
  (三)
  <br/>
  朔风肃杀草木枯 ，
  <br/>
  淫雨霏霏星辰疏。
  <br/>
  讵料岁寒今来早，
  <br/>
  当忧隆冬有备无？
 </div>
 <div>
 </div>
 <div>
  (四)
  <br/>
  和诗
  <br/>
  《
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%A5%E5%86%AC.html">
   严冬
  </ok>
  》
  <br/>
  风雪严寒树凋枯，
  <br/>
  花残叶落败草枯。
  <br/>
  但愿瘟君随冬祛，
  <br/>
  春回大地万物苏。
 </div>
 <div>
 </div>

  <br/>
  凛然寒意朔风呼
