<div><p>
 【大纪元2019年09月11日讯】大家好，欢迎大家关注
 <ok href="http://www.epochtimes.com/gb/tag/%E6%96%B0%E9%97%BB%E7%9C%8B%E7%82%B9.html">
  新闻看点
 </ok>
 ，我是李沐阳。
</p>
<p class="p1">
 <span class="s1">
  刚刚结束台湾之行，香港众志秘书长
  <ok href="http://www.epochtimes.com/gb/tag/%E9%BB%84%E4%B9%8B%E9%94%8B.html">
   黄之锋
  </ok>
  又抵达了德国。昨天（9日），他在柏林联邦议会的屋顶餐厅出席酒会中表示，香港示威者绝不会因为港府正式撤回逃犯条例，而结束长达
 </span>
 <span class="s3">
  3
 </span>
 <span class="s1">
  个月的抗争。他说，“如果现在是新冷战时期的话，那么香港就是新的柏林”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  当晚他与
  <ok href="http://www.epochtimes.com/gb/tag/%E5%BE%B7%E5%9B%BD%E5%A4%96%E9%95%BF.html">
   德国外长
  </ok>
  马斯
 </span>
 <span class="s3">
  （Heiko Maas）
 </span>
 <span class="s1">
  进行了会面，双方谈及了香港街头的情形，以及港人争取双普选与民主的诉求。未来几天，他还将要与德国国会议员见面。（对于
  <ok href="http://www.epochtimes.com/gb/tag/%E9%BB%84%E4%B9%8B%E9%94%8B.html">
   黄之锋
  </ok>
  ）与马斯的会面，中共又怒了。刚升职不久的中共外交部发言人华春莹今天（
 </span>
 <span class="s3">
  9
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  10
 </span>
 <span class="s1">
  日）表示，“
  <ok href="http://www.epochtimes.com/gb/tag/%E5%BE%B7%E5%9B%BD%E5%A4%96%E9%95%BF.html">
   德国外长
  </ok>
  公然同这样的人接触”，从事反华分裂活动，中方“强烈不满和反对”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  香港反送中已经逼近
 </span>
 <span class="s3">
  100
 </span>
 <span class="s1">
  天，像这样的谴责、抗议、反对等等表态，中共有过很多次了，特别是对美国的指责更多。外界认为，北京无力破解香港危局，只能依靠制造“外国势力”捣乱来蒙骗国内群众，但是却留下令人耻笑的话柄。
 </span>
</p>
<p class="p1">
 <span class="s1">
  众所周知，香港爆发声势浩大的游行后，港府先是“暂缓”修例，却引发更大规模的抗议，抗议者从当初要求撤回修例变成“五大诉求、缺一不可”。林郑一个令人想不到的“忏悔”谈话被披露后，突然宣布撤回修例，但抗议者表示“太晚、太迟、太少”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  整个事件中，北京当局似乎总是慢一拍，缓慢适应事件。实际就是采取了拖延策略，等着事情发生变化后，让它自行结束。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但这种策略不但没有化解和控制危机，反而扩大了港人与港府、港警之间的鸿沟。
 </span>
</p>
<p class="p1">
 <span class="s1">
  随着香港公众的愤怒情绪不断上升，中共的反应看上去也变得“气势汹汹”，使香港问题通过政治解决的方案越来越捉摸不定，而这也可能是林郑反应迟钝的原因。
 </span>
</p>
<p class="p1">
 <span class="s1">
  其实对香港问题是让步还是加强控制，中共内部并不统一。
 </span>
</p>
<p class="p1">
 <span class="s1">
  《纽约时报》表示，在今年
 </span>
 <span class="s3">
  7
 </span>
 <span class="s1">
  月的一次闭门会议上，北京当局就讨论了香港问题。那次会议的结论是，香港当局和警察有能力最终自行恢复秩序。与会者都认为，中央政府不应该强力干预，至少目前不应该。
 </span>
</p>
<p class="p1">
 <span class="s1">
  另外，北戴河会议上也就香港问题发生了争论，有人呼吁让步，有人敦促采取行动，让香港更直接地处在大陆控制之下。
 </span>
</p>
<p class="p1">
 <span class="s1">
  香港浸会大学教授高敬文表示，“中共领导层在香港问题以及如何解决危机上存在着分歧。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  就是说，究竟如何处理香港问题，北京当局并没有完全成形的对策方案，完全是盲人过河
 </span>
 <span class="s3">
  ——
 </span>
 <span class="s1">
  走一步算一步。很像应对贸易战，像无头苍蝇一样，到处乱飞乱撞，这也凸显出北京当局的短见，也反映出朝中无人的事实。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但是中共并不承认这种情况，它要让人们相信，香港问题难解，并不是北京无能，而是因为有“国外反华势力”在捣乱。
 </span>
</p>
<p class="p1">
 <span class="s1">
  中共官媒英文电视网
 </span>
 <span class="s3">
  CGTN
 </span>
 <span class="s1">
  有一篇文章，引用中共外交部报告，说美国国家民主基金会
 </span>
 <span class="s3">
  （National Endowment for Democracy）
 </span>
 <span class="s1">
  与美国中情局有合作，资助香港的示威活动。文章说“美国不满足于口头上公开支持香港，还诉诸于提供财务支持”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  看上去
 </span>
 <span class="s3">
  CGTN
 </span>
 <span class="s1">
  说的有鼻子有眼的，但只要稍微想想，就会发现非常荒唐。
 </span>
 <span class="s3">
  200
 </span>
 <span class="s1">
  万人同时上街，如果是美国人花钱雇用人们上街，得需要多少钱？每人
 </span>
 <span class="s3">
  100
 </span>
 <span class="s1">
  不算多吧？给
 </span>
 <span class="s3">
  100
 </span>
 <span class="s1">
  会有多少人愿意冒着高温酷暑拥挤上街头？
 </span>
</p>
<p class="p1">
 <span class="s1">
  假如每人给
 </span>
 <span class="s3">
  100
 </span>
 <span class="s1">
  ，
 </span>
 <span class="s3">
  200
 </span>
 <span class="s1">
  万人那就是
 </span>
 <span class="s3">
  2
 </span>
 <span class="s1">
  亿美元，这可不是一笔小数目。而且
 </span>
 <span class="s3">
  3
 </span>
 <span class="s1">
  个月当中，大大小小的各种活动几十次，参加活动有几百万、甚至上千万人次，这得需要多少钱？
 </span>
</p>
<p class="p1">
 <span class="s1">
  大家知道美国不像中国，中共想花多少钱它说了算。美国是三权分立的国家，即使是川普花钱，也要经过国会审议通过才行。所以中共的说法，纯属子虚乌有。正如《纽约时报》所说，中共的指控“不过是粗制滥造的虚假信息”。
 </span>
</p>
<p class="p1">
 <span class="s3">
  CGTN
 </span>
 <span class="s1">
  还说香港集会上有“更多的美国面孔”，其中一个是香港传媒大亨黎智英的高级助手马克
 </span>
 <span class="s3">
  ·
 </span>
 <span class="s1">
  西蒙
 </span>
 <span class="s3">
  （Mark Simon）
 </span>
 <span class="s1">
  。并且说他安排了黎智英与美国副总统彭斯的会面。但西蒙表示，自己与美国情报部门没有任何关系，也没有在抗议活动中起任何作用，中共说“一个给中情局当智囊的大个子肥胖白人男子在操纵香港局势，这种说法太荒谬了”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  街头抗议的菲比
 </span>
 <span class="s3">
  ·
 </span>
 <span class="s1">
  陈
 </span>
 <span class="s3">
  （Phoebe Chan）
 </span>
 <span class="s1">
  嘲笑中共关于抗议者拿钱参加活动的报导，她表示“我们都是自愿来的”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  的确，中共虽然不断造假宣传，却拿不出任何具体证据。这次抗议活动没有领导人，完全自发自愿。这是因为中共对香港渗透控制越来越严重，港人的言论自由越来越被收窄，所以香港人都在起来抗争。
 </span>
</p>
<p class="p1">
 <span class="s1">
  其实中共从
 </span>
 <span class="s3">
  30
 </span>
 <span class="s1">
  年前就这么做，它制造六四天安门大屠杀，也说是“国外反华势力”干的。几十年中，每次遇到批评和无法辩解的时候，它就会抛出“国外反华势力”。它就是这么侮辱人的智商，从来没有什么创意。
 </span>
</p>
<p class="p1">
 <span class="s1">
  有这么个段子，是网友嘲讽中共“国外反华势力”论的，跟大家分享一下：

</p>
<p class="p1">
 <span class="s1">
