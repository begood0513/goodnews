 <br/>
 我叫吕永乐，一九三九年生人，六十年代大学生，五十多年邪党党龄，副局级退休。
 <br/>
 在一九九六年，家里的女儿、女婿、儿子、儿媳妇和老伴相继开始修炼法轮功，并从中身心受益，虽然我对气功不了解，并且在中共无神论的教育下，根本不相信神佛的存在，但听说炼法轮功对身体好，还教人做好人，也就没有反对。
 <br/>
 一九八九年六月初，我当时正好在北京出差，当看到北京的大学生和平请愿的场面时，根据亲身经历的土改，文化大革命经历，我深知学生们的和平请愿一定没有结果，并可能发生危险事件，我火速离开北京。事后证明正如我所料，震惊中外的六四镇压事件发生了。出于对邪党专制的恐惧，我随时提醒自己一定要在政治上和邪党中央保持一致。
 <br/>
 当一九九九年七月二十日，公安开始对法轮功动手时，我一夜没睡，坐在家门口，阻挡家人出门去当地政府上访，并开始阻止家人继续学炼法轮功。我当时认为国家的宣传和新闻的报道一定是真的，我的家人们被法轮功洗脑了，被骗了，所以根本不听家人们的一句解释，就是一味地反对。
 <br/>

 <br/>
 这几天，我重病住院治疗，大夫说我随时有生命危险，儿女们没有放弃我，在我住院期间，一直给我讲真相，告诉我默念：“法轮大法好，真善忍好”九字真言，给我放真相视频，讲大法的神奇，邪党对世人的谎言欺骗。
 <br/>

 <br/>
 我现在发自内心严正声明：退出邪恶的共产党、共青团、少先队组织，和邪恶组织一刀两断；发自肺腑地向大法和大法师父表达深深的歉意。
 <br/>
 法轮功教人向善，我的孩子们都是好人，她们坚持修炼法轮功没有错。如果世界上人人都能按真善忍做好人，这个世界该有多么美好！
