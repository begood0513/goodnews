 <strong>

 年轻的时候因为在大陆的关系被中共所蒙骗。后来自己上高中的时候和出来留学的时候分别接触了翻墙和大纪元。在初中的时候发生了一些事情，让我觉得对中共十分失望。后来看到了一些历史真相和
 <ok href="http://www.epochtimes.com/gb/tag/%E4%B9%9D%E8%AF%84.html">
  九评
 </ok>
 以及终极目的两书后，更加明白中共和共产邪说是不可能改变的，再加上最近香港发生的中共对港镇压，于是对中共彻底绝望。今天刚好在国外，决定趁这个机会，退出中共党团队，不再被这个邪恶带到沟里去。
 <br/>
 声明人: 林伯清
 <br/>
 2019-08-15 00:19
 <br/>
 <ok href="http://tuidang.epochtimes.com/index/showpost/id/24989098">

 </ok>
</p>
<p>
 <strong>
  退！
  <ok href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
   三退
  </ok>
  ！
 </strong>
 <br/>
 首先我来自大陆，中国二个字我都不愿意打字了。唉，我为中国这片土地感到可悲，它被共党挟持了。在白象塔里受教育，导致我很爱国，不可避免，没办法。从白象塔里初步入社会，工作不好找，大学生毕业没有安居乐业的前景，这是我第一次怀疑它，我爱国，可国给过我什么？！什么都没有呀！什么都是靠我自己的双手得来的，享受到什么狗屁福利，也是我自己劳动，自己工时赚来的。规矩制度指标数不还是它规定的！
 <br/>
 我现在的工作性质，是给政府打工，是最最底层的打工仔，是小屁屁，最廉价的服务人员，面对的也都是最最基层的民众，也就是老百姓。
 <br/>
 我没想到我会走政途，我也不爱走，可没法子，我给养家糊口，我也走不了，也没本事换一份工作，到哪里都一个德行，不如就赚它共党的钱，即使工作很苦逼，唉。
 <br/>
 这不，走政途，就是要入党的，唉，我真不想入，我喜欢自由，喜欢无拘无束，在大陆生活已经很艰难了，再加上一个共党的标签，心情超级差。我就没觉得共党有什么好的，我连好处都看不到，这里还以入党为荣，唉，可能党员好找工作也说不定，谁知道呢！好不容易过了28岁，自动退团了，不用交团费了，这党费又开始想从我腰包里抽取了。
 <br/>
 在这里能接触到政府，了解政策，然后报纸就刊登出来了，可事情实际八字还没一瞥呢！报纸就说，这件事已经开始实施了，这不就是骗人吗！而后过了一年半以上，这事情才落到实处，可大街小巷早就在一年半以前传开了。我第一次发现原来这里的报纸有些报导是骗人的！我第二次对这个政府和社会表示怀疑，我不怎么爱国了，没劲，虚假！哼！
 <br/>
 今年，我顺利得下载到了VPN，恩，大陆封了VPN很久了。没事就刷刷网页后，我惊呆了。我起初以为最高层是晴空万里滴，是中层在吃福利，底层劳苦力。可看看国际新闻，原来中共一直在自说自话，自编自导！不觉得此地无银三百两吗？！以前被教育灌输，让所有人都觉得做中国人很自豪，我们这些人都被骗的很彻底！我现在作为中国人，是中国国籍感到很丢脸！看电视剧里的情节，说什么某人给家族蒙羞，导致族人出门在外都感到害臊，原来家族丢人现眼是这样的感受呀！唉！我不爱国了，我没有国籍，我只承认我是外星人好了！
 <br/>
 现在工作上有考试，我背题的时候发现，恩，题目都是一波强力洗脑，唉！我一直不愿意入党，很惆怅。题目里说6个月不交党费就视为
 <ok href="http://www.epochtimes.com/gb/tag/%E9%80%80%E5%85%9A.html">
  退党
 </ok>
 ，我觉得这好事呀！我突然不害怕入党了！原来我还可以退！我可以撕标签了！我就很开心了呀！等我退休了，我就不交了，我不在乎党费那点钱，我是不喜这个国家，不愿意让它占便宜，让它玩去！
 <br/>
 和同事打趣说了可以怎样退党的这道题，我得到的反馈是，一旦退党了，后三代别想入党。我去，什么鬼！？你的自由民主呢？入党、退党个人行为，和我后代有啥关系！？逼我骂街呢！中共真是说一套做一套的行家！鄙视！我更反感它了！
 <br/>
 写了这么多，重点来了，我没辙呀，这党给入，可我不想入。我看到这个网站，第一个念头是，我可以先退党吗？我现在还是积极分子阶段呢，明年可能就跑不了了，我想先把党退了，我带着假身份入党，我心里接受得还好一些。唉！
 <br/>
 能在没入党之前就把党退了，也是一条退党的路。我希望能特事特办。我是一点也不想入党呀！唉！
 <br/>
 退少先队、共青团和还没有入的共产党。
