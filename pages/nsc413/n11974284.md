 <span class="s1">
  严重的

  </ok>
  ）疫情给
  <ok href="https://www.epochtimes.com/gb/tag/%E6%AC%A7%E6%B4%B2.html">
   欧洲
  </ok>
  带来越来越大的压力，想出新的方法来减轻瘟疫给地区造成的
  <ok href="https://www.epochtimes.com/gb/tag/%E7%BB%8F%E6%B5%8E%E5%BD%B1%E5%93%8D.html">
   经济影响
  </ok>
  成为当务之急。发行共同
  <ok href="https://www.epochtimes.com/gb/tag/%E5%80%BA%E5%88%B8.html">
   债券
  </ok>
  的提议再次浮出水面。
 </span>
</p>
<p class="p1">
 <span class="s1">
  <span class="s1">
   <ok href="https://www.epochtimes.com/gb/tag/%E6%AC%A7%E6%B4%B2.html">
    欧洲
   </ok>
   正处于危机之中，
   <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1%E7%97%85%E6%AF%92.html">
    中共病毒
   </ok>
   病例在急剧上升，有几个国家已经处于封锁状态。截至周三上午，欧洲已确认超过
  </span>
  <span class="s2">
   182,000
  </span>
  <span class="s1">
   例中共病毒病例。
  </span>
  疫情下，九个欧洲国家呼吁欧盟其它国家发行共同
  <ok href="https://www.epochtimes.com/gb/tag/%E5%80%BA%E5%88%B8.html">
   债券
  </ok>
  。这种债券是由来自不同国家的证券合并而成。
 </span>
</p>
<p class="p1">
 <span class="s1">
  发行这种共同债券引起争议。来自德国、荷兰和奥地利等国家的保守政策制定者通常对与意大利、希腊和葡萄牙等高举债国家一起发行债券的想法持谨慎态度。
 </span>
</p>
<p class="p1">
 <span class="s2">
  CNBC
 </span>
 <span class="s1">
  报导，意大利、法国、比利时、希腊、葡萄牙、西班牙、爱尔兰、斯洛文尼亚和卢森堡等国元首在周三的一份联合声明中说：“我们需要认识到形势的严峻性以及今天采取进一步行动来支撑我们经济的必要性。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  致命的病毒使欧盟主要经济体纷纷陷入停滞状态，大多数人被困在家中。饭店、电影院和其它零售场所关闭，航空公司基本上已经停飞。
 </span>
</p>
<p class="p2">
 <span class="s3">
  欧洲的病患、死亡人数在持续攀升。
 </span>
 <span class="s1">
  根据政府数据，在意大利，周二（
 </span>
 <span class="s2">
  3
 </span>
 <span class="s1">
  月
 </span>
 <span class="s2">
  24
 </span>
 <span class="s1">
  日）有
 </span>
 <span class="s2">
  743
 </span>
 <span class="s1">
  人死于该病毒，再次创下单日最高增幅。意大利共有74,386
 </span>
 <span class="s1">
  人染病，其中7,503人死亡。
 </span>
</p>
<p class="p2">
 <span class="s1">
  西班牙周二也创下单日最高死亡增幅680例，该国目前累计有47,610人确诊，3,434人死亡，成为欧洲第二重灾区；
 </span>
 同一天，法国也激增240个死亡病例，迄今确诊人数达到25,233，死亡人数为1,331。上述三国当前的单日新增病例均为四位数。
</p>
<p class="p2">
 九位国家元首在声明中说：“我们需要推出一种由欧洲机构发行的共同债务工具，本着同一基础在市场上筹集资金，并使所有成员国受益。”
</p>
<p class="p1">
 <span class="s1">
  他们的呼吁是在欧盟
 </span>
 <span class="s2">
  27
 </span>
 <span class="s1">
  国领导人举行相关电话会议的前一天。这并非该地区首次提出发行共同债券的问题。在
 </span>
 <span class="s2">
  2011
 </span>
 <span class="s1">
  年主权债务危机最严重的时候，欧洲国家已就此问题进行初步讨论，但德国、荷兰等国家认为与其它国家联合举债风险太大，因为违约风险较高。
 </span>
</p>
<p class="p1">
 <span class="s1">
  然而，中共病毒给该地区造成的金融冲击史无前例，因此欧洲共同债券的议题再次浮出水面。
 </span>
</p>
<p class="p1">
 <span class="s1">
  欧洲央行
 </span>
 行长
 <span class="s1">

 </span>

  </ok>
  。
 </span>
</p>
<p class="p1">
 <span class="s1">

 </span>
</p>
