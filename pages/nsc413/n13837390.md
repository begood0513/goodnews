 <br/>
 文革阴魂起，
 <br/>
 国运渺茫茫。
 <br/>
 医食住学难，
 <br/>
 三座大山扛。
 <br/>
 货币在贬值，
 <br/>
 物价在上涨。
 <br/>
 维稳施高压，
 <br/>
 执法靠无常。
 <br/>
 防疫行管控，
 <br/>
 清零进方舱。
 <br/>
 大白无忌惮，
 <br/>
 民众成羔羊。
 <br/>
 社会不公事，
 <br/>
 正义难伸张。
 <br/>
 校园肇车祸，
 <br/>
 “我爸是李刚”。
 <br/>
 徐州铁链女，
 <br/>
 唐山烧烤堂。
 <br/>
 黑白相勾结，
 <br/>
 世风谬荒唐。
 <br/>
 到处是星火，
 <br/>
 欲盖又弥彰。
 <br/>
 自吹称盛世，
 <br/>
 皇帝穿新装。
 <br/>
 封建残渣起，
 <br/>
 登基欲称皇。
 <br/>
 历史车轮滚，
 <br/>
 挡车的螳螂！
