  9
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  9
 </span>
 <span class="s1">

 </span>
 <span class="s3">
  2.2
 </span>
 <span class="s1">
  公里的“自由之路”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  今天其实也是中共很紧张的一天，美国国会在今天复会，可能要讨论《香港人权与民主法案》。昨天有数万民众到美国领事馆请愿，推动美国通过这个法案。而且美国多位资深议员都已经表态，促成这项法案尽快通过。
 </span>
</p>
<p class="p1">
 <span class="s1">
  从
 </span>
 <span class="s3">
  6
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  9
 </span>
 <span class="s1">
  日到今天，香港反送中运动已经整整
 </span>
 <span class="s3">
  3
 </span>
 <span class="s1">
  个月了。过去的
 </span>
 <span class="s3">
  13
 </span>
 <span class="s1">
  周，香港民众不屈不挠的抗争，终于迫使港府宣布撤回《逃犯条例》修法。虽然只回应了民众五大诉求中的一项，但外界认为，林郑的宣布撤回修例，却坏了中共挽救华为和孟晚舟的“大事”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  美国国会还没有进一步的说法，但是中共官媒却反应很强烈。在今天的评论中表示，美国无权定义香港是不是高度自治，称这是美国要把这项法案变成向北京施压的一张牌。还说香港极少数最极端的人是在“揽炒”，意思就是“玉石俱焚”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  港府也对美国国会议员再提香港人权与民主法案一事表示遗憾。发言人表示，外国议会不应该干涉香港的“内部事务”，绝不允许“跨境执法”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  在昨天的请愿当中，数以万计的示威者聚集在美国驻港领事馆前，向美国领馆递交了香港民众的连署书。有些人手持美国国旗，高唱美国国歌，要求川普“解放”香港。有的标语上也写着“川普总统请解放香港”，并且高呼口号“争自由，撑香港”等等。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据了解，在国会审议的过程中，香港众志秘书长黄之锋和歌手何韵诗还将受邀出席美国国会的听证会。目的也是促请美国通过这项法案，制裁香港侵反人权的官员。
 </span>
</p>
<p class="p1">
 <span class="s1">
  香港浸会大学副教授陈家洛表示，美国国会听证会关系着两大党派及参众两院是不是能够达成共识，“达成共识就水到渠成”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  我们之前已经说过，这项法案如果通过，对香港，对北京，影响都是巨大的，所以中共肯定紧张。
 </span>
</p>
<p class="p1">
 <span class="s1">
  不过中共紧张，可能还有一个因素。中共的“大事”，被林郑给搅黄了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  有知情人向台湾媒体透露，北京之所以要求港府推动修例，是因为华为首席财务官孟晚舟在加拿大被抓后，中共希望在香港也掌握类似的法律
  <ok href="http://www.epochtimes.com/gb/tag/%E7%AD%B9%E7%A0%81.html">
   筹码
  </ok>
  进行反制。
 </span>
</p>
<p class="p1">
 <span class="s1">
  众所周知，去年
 </span>
 <span class="s3">
  12
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  1
 </span>
 <span class="s1">
  日，加拿大应美国的引渡要求，拘押了身系中共重大机密的孟晚舟。美国打了“中共的孩子”，令中共又惊又恼。消息称，中共曾希望用购买大豆，换取美国（释放）孟晚舟。但美国官员认为，华为是国安问题，不能跟贸易问题混为一谈。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但中共并不甘心，一直伺机报复。而能配合中共的，香港是为数不多的其中一个。虽然名义上还是一国两制，但其实中共对香港已经完全控制了。中共要在这个主权已经移交的亚洲金融中心，在这个国际重要政商人士广为聚集的地方布下一张网。可以就此抓一些人，特别是美国人，这样就可以与美国做交换，甚至反制美国。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这个事未知真假，但今年
 </span>
 <span class="s3">
  2
 </span>
 <span class="s1">
  月，港府突然强推《逃犯条例》，还刻意回避一个关键程序
 </span>
 <span class="s3">
  ——
 </span>
 <span class="s1">
  征询公众意见。可以看出，林郑在努力满足北京的要求。
 </span>
</p>
<p class="p1">
 <span class="s1">
  林郑和北京可能都没想到，如意算盘引爆了港人强烈不满。有香港资深媒体人对《上报》表示，林郑启动修例程序，并没留下任何协商的回旋空间。
 </span>
</p>
<p class="p1">
 <span class="s1">
  林郑可能想，既然已经接到北京的指令，那就无论如何也要完成。所以尽管民间抗争节节升温，而且北京也通过不同渠道放话，暗示她可以放低姿态，只要达到完成修例的目的即可。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但公务体系出身的林郑，似乎养成了中共官场的“官僚性格”：奉命行事，根本不需要认错，更不应该让步。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但港人无畏无惧的大规模、长时间抗争，最终迫使林郑宣布正式撤回修例。
 </span>
</p>
<p class="p1">
 <span class="s1">
  按说撤回修例这么大的事情，北京方面应该会有说法。但到目前，北京一直没有正式的正面回应。可以想见，北京非常生气。
 </span>
</p>
<p class="p1">
 <span class="s1">
  香港政坛的确在流传一种说法，北京对林郑的耐性已经完全消失了。在我们的印象中，中共对已经采取的政治行动，从来没有撤回的先例。这次硬生生吞下撤回的败绩，是非常罕见的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  其实林郑现也非常狼狈，成了香港主权移交后民望最差的港首。香港人都在盼着她快点下台，甚至记者会上有港媒记者直接引用市民的话提问，“林郑，市民问你几时死啊？！”
 </span>
</p>
<p class="p1">
 <span class="s1">
  林郑搞的声名狼藉，除了暴露自身在官场养成的强硬僵化性格、缺乏政治判断力之外，还有就是她身边可能没有好的幕僚。律政司司长郑若骅、保安局长李家超和警务处长卢伟聪等这些人，只懂得迎合奉承林郑和北京，不知道合理规劝和体恤民情民意。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这也应了中国那句话，不怕没好事，就怕没好人。
 </span>
