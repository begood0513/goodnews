 【大纪元2019年09月05日讯】
 <span class="s1">
  大家好，欢迎大家关注
  <ok href="http://www.epochtimes.com/gb/tag/%E6%96%B0%E9%97%BB%E7%9C%8B%E7%82%B9.html">
   新闻看点
  </ok>
  ，我是李沐阳。
 </span>
</p>
<p class="p1">
 <span class="s1">
  反送中出现了新转折，今天（
 </span>
 <span class="s3">
  9
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  4
 </span>
 <span class="s1">
  日）下午，林郑长达
 </span>
 <span class="s3">
  8
 </span>
 <span class="s1">
  分钟电视讲话表示，“正式撤回”《逃犯条例》修订草案，不过她没有回应市民的其它四大要求。对这个重大事件，中共官媒并没有马上报导，但是网络先沸腾了。许多网民质疑“这是中央的意思吗？”还有许多大陆网民对林郑进行责骂。
 </span>
</p>
<p class="p1">
 <span class="s1">
  不过对
  <ok href="http://www.epochtimes.com/gb/tag/%E6%9E%97%E9%83%91%E6%92%A4%E5%9B%9E%E4%BF%AE%E4%BE%8B.html">
   林郑撤回修例
  </ok>
  ，反对派并不接受。不论是“和理非”还是“勇武抗争”，都认为港府现在的回应“太迟、太少、太假”，担心这是中共“以退为进”，为启动《紧急法》做铺垫。反对派重申，“五大诉求，缺一不可”。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  撤回修例是谁的意思？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  林郑除了宣布撤回修例，还表示邀请前教育署署长余黎青萍和大律师公会前会长林定国加入监警会。并且表示港府官员会落区与市民对话，独立研讨香港社会深层问题等等。
 </span>
</p>
<p class="p1">
 <span class="s1">
  对这个重大消息，中共外交部和港澳办似乎并不知情，到我们节目录制时还没有反应。而网民问的最多的就是“这是中央的意思吗？”是不是中央的意思，每个人都会有自己的判断。但中国有句话，自古木偶不会动，幕后必有牵线人。
 </span>
</p>
<p class="p1">
 <span class="s1">
  林郑“真假录音”当中，明确表示没有自主权，连辞职的权力都没有。等于是明白无误告诉外界，香港大小事务，都是中共在“垂帘听政”，她只是傀儡，不敢擅自作主。从这些判断，这个决定很可能来自北京，而且可能是习当局的直接命令。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  中共内部分裂明显
 </span>
</h4>
<p class="p1">
 <span class="s1">
  就是说撤回修例，并不是中共高层的统一意见。有两个现象值得注意，一个是港澳办，昨天还强硬表示，香港局势“复杂严峻”，已经到了维护“一国两制”原则底线、维护香港繁荣稳定的重要关头。
 </span>
</p>
<p class="p1">
 <span class="s1">
  另一个是中共官媒，对这么大的事，竟然外媒、港媒热炒之后才有反应，非常奇怪。显然是事发突然、不知如何应对。因为此前中共媒体一直是强硬口气，（声称）不会退让。
 </span>
</p>
<p class="p1">
 <span class="s1">
  由此判断，中共高层在香港问题上很可能存在严重分裂，现在已经公开化。
 </span>
</p>
<p class="p1">
 <span class="s1">
  大家知道，香港之前被控制在江派官员曾庆红、张德江的手上，现在分管港澳事务的韩正，也是江泽民的人马。韩正曾两次南下，近距离观察香港事态。每次（韩正南下）之后，香港的矛盾都呈现激化状态。
 </span>
</p>
<p class="p1">
 <span class="s1">
  前不久也有体制内人士对中央社透露，中共对香港局势存在“鹰鸽之争”，有
 </span>
 <span class="s3">
  3
 </span>
 <span class="s1">
  种观点，分别是“颜色革命”、“非颜色革命”和“暂且搁置按兵不动”。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  撤回修例的用意
 </span>
</h4>
<p class="p1">
 <span class="s1">
  综合分析，林郑的动作，明显在北京授意之下。这么做，可能达到两个目的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  一是给自己洗白，试图澄清没有在背后干预香港。路透社早前报导，北京向林郑建议，要“寸步不让”。那么林郑宣布撤回修例，可以给人制造是“港人自治”的假象。
 </span>
</p>
<p class="p1">
 <span class="s1">
  二是中共在分化瓦解香港的反对派，削弱反送中抗争的正当性。中共可能有这样的想法：香港人抗议的初衷就是反对修例，那现在已经撤回了，你就不能再闹了。再闹下去，那就是“颜色革命”了。这样可能会将“和理非”与勇武派分化，从而孤立勇武派。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这或许是中共用意很深的“金蝉脱壳”之计。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  港人会接受吗？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  但港人的诉求已不再是单一的撤回修例，而是“五大诉求”。林郑只回应其中一点，香港人会答应吗？
 </span>
</p>
<p class="p1">
 <span class="s1">
  民主派召集人毛孟静表示，林郑现在撤回已经“太迟了”，社会伤痕仍在流血。她说民主派不会代表在前线被打得头破血流和面对监禁的抗争者发言，不能接受林郑月娥的“政治表演”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  她同时指出，林郑新任命的两位监警会成员，都是她“自己人”，亲中共人士，根本谈不上独立。就是说，林郑是用自己人调查自己人，“监守自盗”，明显有意维护警察的违法行为。
 </span>
</p>
<p class="p1">
 <span class="s1">
  余黎青萍曾是林郑参选特首的竞选办资深顾问，早在六月她曾提出“四不”：不适合设立调查委员会、不适合特赦、不适合放弃司法审讯和不适合向寻求外国支持的人退让。简单说就是不能答应民众的任何要求。
 </span>
</p>
<p class="p1">
 <span class="s1">
  而林定国在任职大律师公会主席期间，在备受争议的“一地两检”问题上，没有对此提出任何质疑，等于是答应了中共的要求。
 </span>
</p>
<p class="p1">
 <span class="s1">
  民主党主席胡志伟认为，林郑不回应市民对警队滥权的民愤，却强调严正执法，是“一手软一手硬”，在分化港人，为启用《紧急法》铺垫。他斥责林郑是“假让步、真斗争”，受苦的是前线抗争者。
 </span>
</p>
<p class="p1">
 <span class="s1">
  公民党党魁杨岳桥则表示，林郑是“活在时差当中”。过去几个月，香港人已经活在警队的恐惧之下，现在必须有独立调查。反送中不会因此终结，港人和民主派都会坚持“五大诉求、缺一不可”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  范国威议员则认为，林郑所谓的“四大措施”是“包藏祸心”。因为警暴、警黑勾结的“潘朵拉盒子”已经打开，没有独立调查，就无法制止警方过度使用武力。
 </span>
</p>
<p class="p1">
 <span class="s1">
  民阵召集人岑子杰认为，撤回修例是好的开始，但远远不够。如果林郑真心要平息风波，至少还要成立独立调查委员，并撤回所有与运动有关的控罪，特赦被捕的人。
 </span>
</p>
<p class="p1">
 <span class="s1">
  建制派议员田北辰也表示，社会焦点已不在撤回修例，应该成立独立调查委员会。他认为这是“今时今日、百分百必须要做的事”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  而中大学生会长苏浚峯表示，大专学界的行动会持续，包括继续筹备罢课。他说如果放弃抗争，会对不起为反修例运动牺牲的人以及被捕者的付出。
 </span>
</p>
<p class="p1">
 <span class="s1">
  据了解，港警至今已经拘捕了近
 </span>
 <span class="s3">
  1200
 </span>
 <span class="s1">
  人，部分人被控暴动罪，最高可判囚
 </span>
 <span class="s3">
  10
 </span>
 <span class="s1">
  年。另有
 </span>
 <span class="s3">
  3
 </span>
 <span class="s1">
