<hr/><p>
 【大纪元2019年07月20日讯】
 <span class="s1">
  （大纪元记者穆清采访报导）7月18日，在美国第二届推进宗教自由部长级会议中，美国国务卿
  <span href="http://www.epochtimes.com/gb/tag/%E8%93%AC%E4%BD%A9%E5%A5%A5.html">
   蓬佩奥
  </span>
  列举
 </span>
 <span class="s3">
  山东
  <span href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </span>
 </span>
 <span class="s1">
  学员
  <span href="http://www.epochtimes.com/gb/tag/%E9%99%88%E6%85%A7%E9%9C%9E.html">
   陈慧霞
  </span>
  因坚持信仰、被中共投入监狱的案例，并表示，中国有信仰的人需要美国的支持。
 </span>
</p>
<p class="p5">
 <span class="s4">
  “
 </span>
 <span class="s1">
  感谢美国政府关注我母亲的案例
 </span>
 <span class="s4">
  ”，
 </span>
 <span class="s1">
  <span href="http://www.epochtimes.com/gb/tag/%E9%99%88%E6%85%A7%E9%9C%9E.html">
   陈慧霞
  </span>
  的女儿、加拿大公民
  <span href="http://www.epochtimes.com/gb/tag/%E9%B2%81%E9%B8%BF%E9%9B%81.html">
   鲁鸿雁
  </span>
  表示，
 </span>
 <span class="s4">
  “他们
 </span>
 <span class="s1">
  站在正义与良知的一方，给人带来希望。
 </span>
 <span class="s4">
  ”
 </span>
</p>
<p class="p5">
 <span class="s1">
  <span href="http://www.epochtimes.com/gb/tag/%E9%B2%81%E9%B8%BF%E9%9B%81.html">
   鲁鸿雁
  </span>
  希望川普政府对中共迫害
  <span href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </span>
  的关注可以影响到全世界，并推动其它国家共同行动。
 </span>
</p>
<p class="p5">
 <span class="s4">
  “
 </span>
 <span class="s1">
  20年的迫害已经太长久，人间的理不可姑息（这种恶行），神也不能再容忍，（应该）让这场对信神的迫害早日结束。
 </span>
 <span class="s4">
  ”
 </span>
 <span class="s1">
  她说，
 </span>
 <span class="s4">
  “
 </span>
 <span class="s1">
  抛弃邪恶的共产主义，人类将走回传统对神的信仰，才能有美好的前景。
 </span>
 <span class="s4">
  ”
 </span>
</p>
<p class="p3">
 <span class="s1">
  7月16日，美国第二届推进宗教自由部长级会议在美国国务院正式开幕。这是有史以来美国国务院召开的最大的人权活动，也是世界上最大的宗教自由活动。
 </span>
</p>
<p class="p3">
 <span class="s1">
  今年的会议有1,000多名民间代表、宗教领袖以及100多个外国代表团参加。数名法轮功学员受邀出席主场会议。而中共未获邀请。
 </span>
</p>
<p class="p3">
 明慧网数据显示，2019年上半年，大陆45位法轮功学员被迫害致死；自中共1999年镇压法轮功以来，至少4,322名法轮功学员被确认迫害死亡。
</p>
<p class="p6">
 <span class="s1">
  谈到在中国发生的迫害，
  <span href="http://www.epochtimes.com/gb/tag/%E8%93%AC%E4%BD%A9%E5%A5%A5.html">
   蓬佩奥
  </span>
  说：“中国共产党不仅控制中国人的生命，还要控制他们的思想，中共官员还阻止其它国家来参加这个会议。中国的宪法明文规定信仰自由，所以它们（中共）也是不一致（违宪）的。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他说：“在座的都知道，中国有信仰的人需要我们的支持，去年9月，一位法轮功学员陈慧霞被判刑3年半，原因只是她坚持信仰。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  “今天，我们团聚在一起，把我们的‘信仰’变为‘行动’！”蓬佩奥说，“对信仰的迫害，是我们每一个人都应该关心的问题。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  会议期间，蓬佩奥还提到美国第三届总统杰弗逊起草的“宗教自由法案”内容，“全能的上帝创造了人的自由意志”（Almighty God hath created the mind free）。
 </span>
</p>
<p class="p5">
 <span class="s1">
  鲁鸿雁说：
 </span>
 <span class="s4">
  “
 </span>
 <span class="s1">
  对神的信仰是超越国家、超越种族，信仰神的权利是神赋予全人类。蓬佩奥的话表明，川普政府选择站在正义的一方，守护良知和对神的正信。如果一个国家能有这样的对神的正信的觉醒，一定会给他们的人民带来福分。
 </span>
 <span class="s4">
  ”
 </span>
</p>
