 【大纪元2019年09月05日讯】大家好，欢迎大家关注
 <ok href="http://www.epochtimes.com/gb/tag/%E6%96%B0%E9%97%BB%E7%9C%8B%E7%82%B9.html">
  新闻看点
 </ok>
 ，我是李沐阳。
</p>
<p class="p1">
 <span class="s1">
  继昨天宣布撤回修定逃犯条例草案后，香港特首林郑月娥今天（
 </span>
 <span class="s3">
  9
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  5
 </span>
 <span class="s1">
  日）召开记者会。她强调撤回修例的决定是港府的评估，中央政府完全尊重一国两制。
 </span>
</p>
<p class="p1">
 <span class="s1">
  与此同时，中共外交部以“不是外交问题”为由，拒绝回应媒体提问。而人民日报海外版却发文：“港独”勾连“台独”终究死路一条。新华社也评论称，“政治黑手直指分裂国家，具有极度的危险性”。政法委昨夜发文表示，“留给激进份子的时间不多了”。同时五毛和小粉红们像炸了锅一样，说什么的都有。
 </span>
</p>
<p class="p1">
 <span class="s1">
  反送中发生重大转折后，中共接连出现诡异现象，其中缘由相当值得玩味。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  林郑被追问
 </span>
</h4>
<p class="p1">
 <span class="s1">
  在记者会现场，简直是“炮声隆隆”。质疑林郑反应过慢，港人
 </span>
 <span class="s3">
  3
 </span>
 <span class="s1">
  月要求撤回修例，
 </span>
 <span class="s3">
  6
 </span>
 <span class="s1">
  月才暂缓。市民
 </span>
 <span class="s3">
  6
 </span>
 <span class="s1">
  月中要她回应
 </span>
 <span class="s3">
  5
 </span>
 <span class="s1">
  大诉求，等到出现示威者伤亡，甚至千人被捕才宣布撤回，而且让步“太少太晚”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  记者还追问她作为香港至今“名望最低”的特首，是否辞职下台？质疑不成立“独立调查委员会”，却说设立“对话平台”是为港府执政“化妆”。甚至质疑仅撤回修例，是让示威失去正当性，为《紧急法》铺路。
 </span>
</p>
<p class="p1">
 <span class="s1">
  虽然没在现场，也可以想见林郑的尴尬与狼狈。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  <ok href="http://www.epochtimes.com/gb/tag/%E6%9E%97%E9%83%91%E5%A4%9A%E5%8F%98.html">
   林郑多变
  </ok>
 </span>
</h4>
<p class="p1">
 <span class="s1">
  从林郑的狼狈当中，可以看出她的多变。先是强推修例，遇阻后改口“暂缓”、“寿终正寝”到“撤回”，林郑几次变化。
 </span>
</p>
<p class="p1">
 <span class="s1">
  不断变化的还有她宣布撤回先前的“录音门”。那份录音，外界认为是她“不可思议的忏悔”。录音中林郑表示“她有两个主子，一个是北京，一个是港人”。自己无法做什么单独决定，事无巨细都要听中共当局的。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这个“私下谈话”内容并不新鲜，谈不上“私下”。但经过路透社公布，冲击力就被放大了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  著名作家龙应台对此写道，“大国之所以为大国，不在于导弹、金钱和权力”。她把香港比作“花园地上的一颗鸡蛋”，北京应该用包容的气度、长远的目光轻轻拾起，捧在手心，万万不能摔破。“如果中国把香港当敌人对付，用武力处理，那恐怕不是香港背叛祖国，而是他的祖国背叛香港”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  林郑这一手，出乎许多人的意料。从这开始，中共诡异现象就接连不断。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  中共诡异
 </span>
</h4>
<p class="p1">
 <span class="s1">
  林郑的“私下谈话”录音被路透社公开后，成了外媒港媒的报导重点，但中共官媒一点没有都反应。
 </span>
</p>
<p class="p1">
 <span class="s1">
  林郑宣布撤回后，中共又诡异了。最初全部噤声，直到昨晚
 </span>
 <span class="s3">
  7:30
 </span>
 <span class="s1">
  以后，新华社才发报导。而且刻意淡化撤回，强调
 </span>
 <span class="s3">
  4
 </span>
 <span class="s1">
  项行动，之后各大媒体才纷纷转载。
 </span>
</p>
<p class="p1">
 <span class="s1">
  发生这么大的事，港澳办直到我们发稿还在装聋作哑。中共外交部在林郑宣布决定
 </span>
 <span class="s3">
  21
 </span>
 <span class="s1">
  小时后，举行了例行记者会。但发言人耿爽竟抛出一句“很遗憾，这不是外交问题”，拒绝了记者的提问。
 </span>
</p>
<p class="p1">
 <span class="s1">
  昨天夜里，政法委微信公号发文，欢迎政治性对话，但对“政治要胁”和“颜色革命”会“追究到底”等等。
 </span>
</p>
<p class="p1">
 <span class="s1">
  更诡异的是，人民日报海外版和新华社一起发声，直批“港独”和“台独”。文章把香港众志秘书长黄之锋等人到访台湾形容是“两独”合流，目的是缔结“抗中（共）同盟”，分裂国家，具有极度危险性等等。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这等诡异，法广惊呼是不是“出大事”了？
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  分裂公开化？
 </span>
</h4>
<p class="p1">
 <span class="s1">
  林郑与外交部和官媒的表现迥异，可能说明中共内部不止一个声音。
 </span>
</p>
<p class="p1">
 <span class="s1">
  大家知道，林郑在“录音门”中已经说得很清楚。她连辞职都不能决定，就像拉磨盘的驴
 </span>
 <span class="s3">
  ——
 </span>
 <span class="s1">
  完全听喝。就是说，她撤不撤修例，完全取决于上面的命令。
 </span>
</p>
<p class="p1">
 <span class="s1">
  大家记得几天前，林郑在脸书
 </span>
 <span class="s3">
  PO
 </span>
 <span class="s1">
  文，说“大家都累了”，“可否坐下来谈一谈？”我们在
 </span>
 <span class="s3">
  8
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  26
 </span>
 <span class="s1">
  日分析，林郑可能在暗示，习当局可能要回应港人的诉求。
 </span>
</p>
<p class="p1">
 <span class="s1">
  但中共媒体始终一个口径，抹黑香港市民，称之为“暴徒”，要“以暴治乱”等等。还高调宣传驻港部队轮防，制造可能武力镇压的气氛。与此同时，港警的镇压手段在升级，动用了水炮车，还开了真枪。
 </span>
</p>
<p class="p1">
 <span class="s1">
  眼见事态越发严重，林郑的“私房话”被公开了，并随即宣布撤回修例。
 </span>
</p>
<p class="p1">
 <span class="s1">
  明眼人都看得出来，林郑有些自相矛盾，也与中共媒体的宣传存在矛盾。这些矛盾的背后，很可能是有不同的命令。
 </span>
</p>
<p class="p1">
 <span class="s1">
  从中共官媒和外交部、港澳办的反应来看，林郑宣布撤回修例，他们并不知情，所以有错愕的表现。或许是上面主管的某高层、某派系没有通知他们，或许连他们也不知情。
 </span>
</p>
<p class="p1">
 <span class="s1">
  照此推理：高层分裂已经公开化。现在直接给林郑下命令的，很可能是习当局。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  <ok href="http://www.epochtimes.com/gb/tag/%E4%BA%94%E6%AF%9B%E7%82%B8%E9%94%85.html">

 </span>
</h4>
<p class="p1">
 <span class="s1">
  中共分裂，也给五毛和小粉红们造成了困惑，不少人感觉自己被中共欺骗了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  一直引领网民舆论的环球时报以前一直喧嚷不能撤回修例，昨晚却说“撤回修例挺好”。招来了网民的斥骂：“反对撤的是你，现在支持的也是你，你还是人吗？”“你才是唯恐天下不乱之人”。“天天说香港暴徒下场如何如何，到头来人家还是胜利了，我在反思是不是一开始我们就真的有问题？”
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  各界声援
 </span>
</h4>
<p class="p1">
 <span class="s1">
  跟着中共起哄是否有问题，的确应该反思，因为整个世界都在声援香港。撤回修例后，美国国会两党两院都有议员发声明，包括众议长佩洛西、共和党参议员卢比奥、民主党参议员麦戈文等等，在表示欢迎的同时，呼吁港府拿出更多行动，“全面”满足港人的合法要求。
