<hr/><p>
 【大纪元2019年07月11日讯】7月初，瑞典民主政治周活动期间，法轮功学员揭露中共迫害真相受到当地政界和媒介关注，期间中共企图搞破坏，最终以
 <span href="http://www.epochtimes.com/gb/tag/%E5%A4%B1%E8%B4%A5.html">
  失败
 </span>
 告终。
</p>
<p class="p4">
 <span class="s3">
  七月初是瑞典一年一度的政治周（Almedalen），也叫民主周，东南部的哥特兰岛首府威斯比（Visby）举办了系列活动。
 </span>
</p>
<p class="p4">
 <span class="s3">
  明慧网报导，活动中，各党派、各大媒体、商业、教育界精英、非政府组织和民间人士，每天有四到五万人聚集在这个本身只有两万多居民的中世纪小镇，一周内举办四千多个活动，多数对外开放，社会各阶层相互直接交谈，这是瑞典民主制度下开放透明的社会体系的最大展现。
 </span>
</p>
<p class="p3">
 <span class="s3">
  6月30日至7月5日，瑞典法轮大法学会也参加了民主周活动。期间，瑞典的部分法轮功学员们介绍法轮功、举办研讨会、会见媒体、接受采访，向政要、民间团体讲法轮功真相。
 </span>
</p>
<p class="p3">
 <span class="s3">
  法轮功学员舒缓的功法演示与和平理性的活动，吸引了很多人来了解真相。
 </span>
</p>
<p class="p4">
 <span class="s3">
  政治周的第三天，中共大使馆邀请各界人士，参加其举办的“中国日”的活动，宣扬中共历来宣称的所谓人权最好时期。
 </span>
</p>
<p class="p4">
 <span class="s3">
  法轮功学员在其开幕式的时段，在活动场所的门口和街对面举牌，以展板等揭露中共实施酷刑、活摘器官罪恶，同时告诫中共大使和其随行人员勿再行恶。
 </span>
</p>
<p class="p3">
 <span class="s3">
  中共人员悄悄要求瑞典警察把法轮功学员赶到远一些的地方。
 </span>
</p>
<p class="p3">
 <span class="s3">
  瑞典警察回答说：“这是民主周，每个人都有权利发表意见。这里是瑞典国土，实行的是瑞典法律，法轮功学员有权利站在那里。”
 </span>
</p>
<p class="p3">
 <span class="s3">
  在中共宣传活动的开幕式上，还有几位法轮功学员到场提问，中共大使的回答相当尴尬。不少客人从这个开幕式一走出来，直接走到法轮功学员面前看展板和索取真相资料。
 </span>
</p>
<p class="p3">
 <span class="s3">
  中共大使馆的活动还被多家瑞典媒体批评：“极权中共到我们民主周来做外交宣传。”
 </span>
</p>
<p class="p3">
 <span class="s3">
  众多路过的民众对坚持在街对面举展板和发资料的法轮功学员说：“你们做的好！继续！加油！”
 </span>
</p>

 <span class="s3">
  还有人因参加中共的“中国日”活动而得到法轮功学员的真相传单，他们随后观看了电影，深深被《求救信》中主人公的故事打动，不少人一边看纪录片一边流泪。
 </span>
</p>
<p class="p6">
 <span class="s3">
  看完电影后和现场的中国问题专家讨论时，瑞典政要和民间人士提出：如何突破中共的防火墙、让更多的中国人能有信息自由和知情权，中共在瑞典的渗透活动等问题，深入地了解了真相。现场气氛热烈，一直讨论到电影院关门。
 </span>
</p>
<p class="p6">
 <span class="s3">
