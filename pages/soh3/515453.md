 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     中国有句老话说“人过留名，雁过留声”。把道义看得比生命重要，不只是读书人知道的道理，就算是普通人也愿意身体力行。老祖宗几千年口耳相传，流传到今天的故事，一定是被老百姓认可的，值得留给自己的后代子孙的故事。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     这话还要从大宋朝说起。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     唐朝末年军阀混战的教训，让宋朝开国天子，立下了重文轻武的国策。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     宋朝北方有个骑马放牧为生的辽国景超骚扰大宋。反正是宋朝一打，辽国就跑。转一圈回来还骚扰。后来好不容易和宋辽签下了盟约，才有了大宋百姓的安生日子过。这时，东北方又一个马背上的民族金国异军突起。对辽国造成了致命的威胁。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     金人以打下辽国，归还大宋燕云十六州为筹码，成功与大宋结盟，一起攻打辽国。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    1122
    <span style="font-family:宋体">
     年，宋金两国从南北两面向辽国发起攻击。金军很快就攻克了辽国中京和西京（今山西大同），可是宋几万宋军却让几百辽郡打的满地找牙。
    </span>
   </span>
  </span>
 </p>
 <figure class="OImage__StyledFigure-sc-1lfley0-0 hHSfVg">
  <img alt="辽与宋金的战争中 岳飞看到了金人的野心（示意图片：明代绘画局部）" src="https://img.soundofhope.org/2021-06/1623015466688.jpg"/>
  <br/><figcaption>
   辽与宋金的战争中 岳飞看到了金人的野心（示意图片：明代绘画局部）
  </figcaption>
 </figure>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     这一仗打下来，金国统治者算是看明白了，花团锦簇的大宋，是一只肥的流油的纸老虎。
    </span>
   </span>
  </span>
 </p>
 <div class="AD_Embed__Wrap-sc-1xslmin-0 igMuqX module desktop">
  <div>
  </div>
 </div>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     金国也就定下了吃掉宋朝天下的决心。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     但毕竟是瘦死的骆驼比马大，一口全吃掉不可能，但是一小口一小口的吃，还是稳赢的。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     于是灭掉辽国之后，金国开始骚扰大宋。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     金国士兵踏进宋朝国土，看什么都是好的，就开始烧杀抢夺，。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     宋朝女子肤白貌美，宋朝的金银器物精妙珍贵，实在抢不到金贵东西的小兵，就连锅碗瓢盆也得带着，这些生活用品，对于马背上的游猎民族来说，都是做工精巧难得的好东西。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     抢一次也就抢了，但是金军隔三岔五的就来抢一次，百姓是苦不堪言，人心惶惶。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     不光是百姓害怕，皇帝也害怕。这么多年重文轻武，朝中可用的武将不多，民间舞刀弄枪的青年也不多，之前辽国来骚扰都搞不定，这下把辽国打趴下的金国又来骚扰，更是束手无策。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     所以大宋的朝堂之上，议和的声音就成了主流。
    </span>
   </span>
  </span>
 </p>
 <figure class="OImage__StyledFigure-sc-1lfley0-0 hHSfVg">
  <img alt="宋高宗赵构与大多数朝臣主张议和（图片：：国立故宫博物院）" src="https://img.soundofhope.org/2020-12/660px-gaozong_of_song-1608434322359.jpg"/>
  <br/><figcaption>
   宋高宗赵构与大多数朝臣主张议和（图片：：国立故宫博物院）
  </figcaption>
 </figure>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     就在百姓生灵涂炭的时候岳飞出现了。但是岳飞的目标很明确，就是要上阵杀敌，让百姓免遭侵害。再加上求和失败，徽钦二帝，大宋的祖宗牌位都被金国掳走，让大宋子民猛抽奇耻大辱，岳飞打败金国的意志更加坚定了。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     在和金国几次的大小交战中，岳飞看懂了，金国是吃定大宋了，无论表面如何进进退退，最终的目的，就是要吃掉大宋。所以对于金国，只有一个办法，就是要打到金国没有还手之力。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     但是新上任这皇帝宋高宗，虽然是在金国死里逃生回来的，但是贪图眼前的享乐，忌吃不记打，亲爹亲妈，祖宗牌位都让人家给抢去了，还听信秦桧的鼓动，还要跟金国议和。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     宋高宗求和成功大肆封赏的时候，岳飞坚决不接受封赏，并且说
    </span>
    <span style="font-family:宋体">
     “金人不可信，和好不可恃”，并直接质问宰相秦桧，议和投降的做法是用心不良。自此就和秦桧接下了梁子。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  <span style="font-size:10.5pt">
   <span style='font-family:"times new roman"'>
    <span style="font-family:宋体">
     直到这个时候，岳飞没有站在自己的角度考虑个人得失，而是想着如何根治金国大患，如何接回自己的皇帝，一雪前耻。
    </span>
   </span>
  </span>
 </p>
 <p style="text-align:justify">
  百转千回，就在临门一脚的时候，岳飞到底是没能接回自己的皇帝。反而被构陷杀害。那么这狱卒到底做了些什么呢？
 </p>
