 <p>
  5月3日至24日《逃犯条例》经过立法程序激烈攻防后，港府决定绕过委员会在6月12日直接到大会二读，由此立法会内的反对声浪在5月底演变成社会运动。
  <br/>
  <br/>
  民间人权阵线于2019年6月9日举办首次“反送中”游行，同日海外29个城市举行集会声援。但港府宣布如期在6月12日二读，6.12当天示威者占领立法会附近街道，多处发生激烈对抗，当局指控示威者暴动，而警察武力过当引起广泛反弹。迫于民意6月15日港府宣布暂缓修例。但民间人权阵线仍然举行了6.16游行，人数超过了两百万，估计创造了历史之最。提出五大诉求：1、完全撤回《逃犯条例》修订；2、追究警察开枪镇压；3、不检控和释放反送中示威者；4、撤销定性6月12日集会为暴动；5、林郑月娥问责下台。但林郑当局对五大诉求不作回应，致使抗争绵延不断，有7.1游行，（部分年轻人占领立法会），7.7九龙游行向大陆民众传递信息，7.17银发族上街支持年轻人等等一系列抗争活动。7.21四十三万香港市民再次上街，发生冲击中联办事件和白衣人有组织暴力袭击示威者的事件。有诸多证据证明，白衣人的袭击与中央政府有关。钟建平申请7.27元朗游行，警方首次发出反对通知，并宣布参与即属犯法。但钟建平本人继续徒步走完全程，仍有28.8万市民自发参与散步。刘颖匡申请7.28游行再次被警方拒绝，仅允许集会，但集会后部分市民自发前往中联办抗议。已有越来越多的市民被拘捕。目前抗议者除了重申：撤回逃犯条例、撤回6.12暴动定义、撤回对所有抗争者控罪、成立独立调查委员会，追究警队滥权、立即实行双普选等五大诉求之外，还增加了彻查元朗暴力事件等诉求。
  <br/>
  <br/>
  香港市民从“反送中”开始的抗争已提升为反对中共专制制度全面控制香港，争取真普选的民主运动。适逢六四屠夫李鹏去世，在讣告中中共以肯定李鹏“采取果断措施制止动乱，平息反革命暴乱，稳定了国内局势”的话语透露了对香港民主运动可能采取血腥镇压的信息。
  <br/>
  <br/>
  中共已经成了民主国家乃至文明世界的公敌，在香港市民争取民主权利的重要关头，国际社会理应全力支持，可能采取的举措有：一、联合国派出调查组彻查港警滥权及元朗暴力事件真相；二、鉴于中共已否定中英联合公报的承诺，启动中华民国与英国的谈判，让香港回归中华民国；三、一旦有中共暴力镇压的迹象即派出联合国维和部队接管香港。
