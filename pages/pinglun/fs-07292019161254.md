 <p>
  <span dir="ltr" id="breadcrumbs-2">
   李鹏死了，对习近平来说是一次机会，是把六四继续当作负资产背负，还是把它转变为对自己、对国家和民族有益的正资产的一次机会。鉴于习维护中共专制的决心，我不相信他会变性。果然，他毫不犹豫的借讣告的言论表达了他成为习大帝，维护中共红朝的顽固立场，讣告说：
   <br/>
   <br/>
   “1989年春夏之交的政治风波中，在以邓小平同志为代表的老一辈无产阶级革命家坚决支持下，李鹏同志旗帜鲜明，和中央政治局大多数同志一道，采取果断措施制止动乱，平息反革命暴乱，稳定了国内局势，在这场关系党和国家前途命运的重大斗争中发挥了重要作用。”
   <br/>
   <br/>
   这段文字的逻辑是清晰明了的，就是李鹏在六四屠杀事件中发挥了“重要作用”，就是在采取果断措施制止动乱，平息反革命暴乱，稳定国内局势的六四屠杀中发挥了决策的作用，而“以邓小平同志为代表的老一辈”只是起了“支持”的作用。
   <br/>
   <br/>
   关于这段文字的表述有不同的解读，有评论者认为，这是邓和老一辈的后人担忧日后六四屠杀被追究时，为先人减轻罪责的铺垫。这一说法是以一个假设为前提：即这一大批手握大权和巨量财富的红二代，担忧六四的定论会被推翻，屠杀会被追究罪责！

   正确的解读是：习大帝借此表达了自己的态度和立场，一、赞赏李鹏的“果断”，觉得当时优柔寡断的老一辈缺少“男儿”风范；二、六四镇压正确，屠杀有理，结论一个字都不能改；三、正值香港民主运动风起云涌之际，习大帝要借此发出不惜血腥镇压的信号。

   <br/>
   对此我要说的是：一、六四屠杀是否李鹏“果断”的结果，现在认定还太早，屠杀的决策过程终有大白于天下的一天，李鹏肯定有罪，而其他人的罪责也逃不了；二、六四屠杀不是错，是罪，不需中共作平反的结论，习当局改不改结论都一样，未来必将发生追责的审判，而到了大监狱墙倒的那一刻，中共不会有什么“男儿”，习本人也不是！三、至于屠港只是恐吓，习大帝有这份心，没这份力，也就没这份胆！
