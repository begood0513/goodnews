 <p>
  朋友们还记得王默和谢文飞吗？还记得著名的《王默辩护词》吗？没错，就是那位在辩护词中写道： “我主观意愿上想颠覆的是中国共产党专制独裁政权，而不是国家政权”......大陆民运没有退路，也没有能和中共政权同谋的第三条中间道路，没有抗争，一直的抗争，非暴力和暴力同在的抗争，菜刀精神要永存，革命准备要永远。抗争才有自由，抗争才有尊严，抗争才有改变” 的勇士王默。还没读过他写的自我辩护词的朋友们，我郑重推荐你们找来读一读，读后必定会有恍然大悟、豁然开朗的感觉。
  <br/>
  <br/>
  王默对邪恶中共专制政权的认识确实十分准确而到位，而且对如何抗争有独特的见解。我想，只要中国能有百分之十的人有了这种认识，何愁民主不成？！
  <br/>
  <br/>
  不过，由于觉醒者缺乏有效的组织，不能集合智慧、形成合力，因而不断被中共独裁者分别迫害。正因如此，王默在被以“煽动颠覆国家政权罪”关押四年半后，于今年四月二日刚出狱，五月十四日就又被中共以“寻衅滋事”的口袋罪抓了起来，六月二十二日批捕，现在被关押在江苏淮安看守所。
  <br/>
  <br/>
  7月23日，律师带着《委托书》到淮安要求会见王默。看守所以“没有会见室可用”为由，要他明天再来。可当地的国保却迅速地找到王默的父亲，并威胁他“不要与外地律师签委托书，否则另外两个儿子的工作都会受到影响”。
  <br/>
  <br/>
  24日，当律师按时来到看守所时，却被带到一个房间里，当地的国保、司法局的人员，公安和律师等八人早已等在那里。他们以“探讨委托书是否真实”为由与律师见面，实际却是要求律师退出本案，并以“否则将对律师展开调查”来威胁。家属的不予配合、当局的非法打压，王默的案子已经不在法律的范畴之内，无奈之下，律师只好退出。一个所谓的“寻衅滋事”案，中共动用如此大的阵容来阻止律师会见，已经充分说明王默这次被抓是赤裸裸的政治迫害。当局的目的是让此案完全陷入中共的黑箱操作任意胡为。相信官派律师会很快登场。
  <br/>
  <br/>
  对于任何一个既认清了中共的邪恶本性，又真心反共的觉醒者，中共毫无疑问地都会将其视为它的敌人，一定会千方百计地对其进行打压迫害。再者，在下智而上愚，矛盾异常激化的当下，中共抓王默的另外一个重要目的是用此恐吓其他已觉醒的公民。因为中共非常害怕他们的思想广泛传播，唯恐他们勇敢的行动在全国蔓延成燎原之势。中共也知道，谎言已经失效，要维护其极权专制，就只有靠暴力这一根救命稻草啦。
  <br/>
  <br/>
  事实上，中国已经有很多人认清了中共政权的独裁本性，知道民主是中国必然的未来。可是为何一直未能使得中国实现民主呢？是战略战术出了问题。因为没有很好地提出系统策略，很多人只好等。还好，现在已经有了清楚有效的解决方案，那就是郭天剑写的《必由之路》。该文不仅对一直束缚民运很久的问题给出了符合逻辑的清楚结论，还对一些思想误区做出了清晰的勾划，对中共的欺骗宣传与真实目的做了精彩的剖析...... 依照《必由之路》给出的方法而进行，定能结束中国的共产专制统治，让中国实现宪政法治下的民主自由。特别提醒大家的是，由于中共政权也知道《必由之路》的威力，因此，为了不让大家很容易地找到读到此文，中共花了大力气混淆视听、增加在网上搜索此文的难度，甚至故意把搜索此文者引导到别的同名其它影视作品链接上。
  <br/>
  <br/>
  这里，我要告诉朋友们，只有同时输入“必由之路”和“郭天剑”这两个关键词才能在网上找到该文章。真心关心中国未来的朋友们，我推荐你们赶快去找来此文阅读。找到并运用其结束专治、建立民主的方式方法。
  <br/>
  <br/>
  一旦我们有了如同《王默辩护词》那样对中共的认识，真正理解并运用好郭天剑《必由之路》中的策略，中国的民主大业没有不成功的道理。
