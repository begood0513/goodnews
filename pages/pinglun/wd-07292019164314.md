 <p>
  各位听众你们好，我是王丹。台湾2020年总统大选，现在主要的两个政党的候选人---民进党的蔡英文和国民党的韩国瑜---都已经确定，选战虽然还不到白热化的程度，但是已经开始了。在香港发生大规模社会抗议的背景下，蔡英文连任的声势一直水涨船高，不过，近日爆发的私烟案，使得外界对蔡英文的领导和管治能力产生质疑，而蔡政府对这个事件的危机处理也出现错误，一度试图用“超买”代替外界认知的“走私”来界定整起案件，引起社会普遍观感不佳，导致蔡英文出面道歉。这一系列发展，毫无疑问，已经对蔡英文的选情造成伤害。那么，蔡英文的连任之路还能走下去吗？我认为这次事件对蔡英文的选情确实有冲击，但是还不至于严重威胁到她的连任之路。目前来看，明年1月的总统大选，蔡英文获胜的可能性还是比韩国瑜大。为什么这么说呢？
  <br/>
  <br/>
  首先，虽然这次私烟事件导致蔡英文民调下降，但是毕竟离投票还有半年之久，在未来的半年中，一定还会有各种各样正面或者负面的对选情会造成冲击的事件发生，私烟事件引起关注的热度不可能延续半年之久。事实上，只要蔡英文政府依法办理，让事件进入司法程序，整起事件造成的伤害，应当会逐渐降低，不至于对于蔡英文的选情造成致命伤害。
  <br/>
  <br/>
  其次，从台湾过去的选举经验来看，总统连任基本上都会过关。陈水扁执政时面对国民党的国会多数和地方执政优势，以及国民党的团结，尚能惊险过关；马英九执政时出现633跳票等饱受诟病的危机，又面对蔡英文的强力挑战，也是顺利连任。一般来讲，在政党轮替已经比较习以为常的国家，执政者掌握执政资源，就选举来说本身就是天生的优势，除非出现极为严重的危机，例如重大自然灾害或者总统本人犯法，连任的可能性都是相当大的。蔡英文上任以来，虽然在执政上出现一些问题，但是还没有严重到整个社会不能接受的地步，这个基础不会因为私烟案的发生而改变。
  <br/>
  <br/>
  第三，民进党去年年底的地方选举大败，年轻选民没有动员起来是很大的原因。当时，蔡英文政府在一些重大的社会议题，例如同婚上，让年轻人感到失望。自那以后，蔡英文政府已经认识到问题所在，这半年来致力于弥补与年轻世代的关系，再加上国民党的候选人在年轻族群中的认同度很低，因此，可以肯定的是，大部分的年轻人已经回流到对民进党的支持上。更重要的是，2020年的总统大选主轴，已经确定为是否要捍卫台湾的民主自由，反对九二共识和两岸统一，在这个议题上，民进党和蔡英文也是具备天然优势，再加上香港问题的发烧。可以肯定，明年1月的总统大选，年轻人的票会被激发出来，而且会压倒性地投给蔡英文，这对蔡英文的连任来说，是一大利多。
  <br/>
  <br/>
  最后，对于另一个大票仓中产阶级来说，他们比较希望社会稳定。蔡英文的理性沉稳的个人风格比较符合都市阶层的期待，美国已经明确表达的对蔡英文的支持，这一点也会影响到很多经济选民的判断。再加上国民党候选人韩国瑜没有什么政治历练，个人风格天马行空，一旦当选会出现什么样的时局发展，没人有把握。这些因素加在一起，使得蔡英文在都市中产阶级中的支持度也相当稳定。这个阶层平时不太进行政治表态，因此支持度不容易在民调上表现出来，所以蔡英文的民调支持度不高，并不一定准确反映事实。
  <br/>
  <br/>
  总之，蔡英文的连任之路确实已经遇到重大考验，未来还会有更多的挑战，但至少目前来看，她的连任还是会有惊无险地成功的。以上是自由亚洲电台特约评论员王丹为您做的评论。
