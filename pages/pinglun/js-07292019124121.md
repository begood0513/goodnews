 <p>
  中共前总理李鹏于今年7月22日死亡。中共当局发讣告，予以高度评价。有关李鹏的三大政治遗产，如何盖棺定论？讣告尽都涉及。
  <br/>
  <br/>
  其一，六四屠城。讣告毫无避讳地重提“制止动乱，平息反革命暴乱”,明显有历史和现实的双重含义：以中共方式，纪念六四30周年；同时警告眼下正在抗争的香港民众，“不要走得太远”。借此宣示：习近平政权将坚持继承李鹏的镇压路线和六四模式。
  <br/>
  <br/>
  对习近平来说，政治上的得失利弊就在其间。从正常的历史方向而言，六四镇压，既可能成为习近平的负资产，也可能成为他的正资产。六四镇压以杀人定性于人类史，习近平肯定之并继承之，固然可以强化一党专政，却背负这个沉重包袱终将令自己以负面形象定位于历史。如此，六四，实为其负资产。如果习近平能够重评六四，重启中国民主化进程，则可正面打造自己的历史定位。如此，六四，又可成为其正资产。古人云：天与不取，反受其咎。
  <br/>
  <br/>
  其二，讣告简单提到李鹏主导的三峡大坝：“他在三峡工程科学民主决策和建设中发挥了重要作用。”暗含的意思，为李鹏推卸硬上三峡大坝的历史责任。所谓“民主决策”，就是要把责任推到“众人”头上，而淡化李鹏作为始作俑者的决定性作用。原因大概在，李鹏死亡前不久，国内外传出三峡大坝扭曲变形、随时可能出事的新闻，习近平当局似乎及时为李鹏卸责、解套、圆场。
  <br/>
  <br/>
  其三，众所周知，李鹏家族是中国数一数二、且臭名昭著的腐败家族，但讣告却这样写道：李鹏“坚定支持党风廉政建设和反腐败斗争。”这无疑是天大的讽刺，跟13亿中国人民开了一个天大的玩笑。玩笑之余，也不排除这样一种可能：当有一天，中共当局出于政治需要，必得全部拿下李鹏家族，包括李鹏老婆朱琳、长子李小鹏、次子李小勇、女儿李小琳 – 无不以贪赃枉法而著称，那么，讣告中的这句话就成了李鹏大义灭亲的“遗嘱”。照此“遗嘱”行事，不会有错。

 <p>
  纵观讣告，关键的伏笔还是在六四。讣告描述李鹏在六四屠城中的角色，这样写道：“1989年春夏之交的政治风波中，在以邓小平同志为代表的老一辈无产阶级革命家坚决支持下，李鹏同志旗帜鲜明，和中央政治局大多数同志一道，采取果断措施制止动乱，平息反革命暴乱，稳定了国内局势，在这场关系党和国家前途命运的重大斗争中发挥了重要作用。”
  <br/>
  <br/>
  这段话，简化后的意思就是：某个总理或者某个政治局常委，可以通过取得政治老人的支持，而推翻总书记。1989年的情形就是如此：作为总理的李鹏，联手政治老人邓小平等人，推翻了党和国家最高领导人、身为总书记的赵紫阳。违反党章，也违反宪法。
  <br/>
  <br/>
  这段大赞老人政治的话，从政治逻辑上而言，对习近平不利。因为，老人政治是习近平上任以来的最大忌讳；而政治老人，本是习近平的最大噩梦。去年7月，习近平就差点因众多政治老人的拍案而起（痛斥他违反党章、大搞个人崇拜）而丢失皇冠。
  <br/>
  <br/>
  下月，众多政治老人与现任高层又将聚集于北戴河开会。不得不说，这份大概由王沪宁主持写成的讣告，暗藏玄机。而最大的玄机就在这里：对中共这个一党专政的体制而言，老人政治不仅合理合法，而且有用有效。包括正在当总书记的习近平，也必得受制于政治老人 - 而他的权力来源，原本就出于此。
  <br/>
