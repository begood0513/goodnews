 <p>
  7月29日，中国国务院港澳办新闻发言人杨光对香港连续发生数周的“反修例”示威表达了三点意见：
  <br/>
  <br/>
  第一，希望香港社会各界人士旗帜鲜明地反对和抵制暴力。第二，希望香港社会各界人士坚决守护法治。第三，希望香港社会尽快走出政治纷争，集中精力发展经济、改善民生。中央政府愿意与特区政府和香港社会各界人士一起努力，为他们的发展创造良好的条件。
  <br/>
  <br/>
  杨光还表示，由于不少香港市民对内地的情况和法律制度、司法制度了解不多，对修订条例有所疑虑。特区政府于6月15日决定暂缓修例工作，相应的立法工作也随之完全停止。对特区政府的这一决定，中央政府表示支持、尊重和理解。
  <br/>
  <br/>
  有人说，这一表态相当低调，既没提反对“港独”，也没提反对“颜色革命”。笔者认为，这一表态实际是在说，香港的事情香港人自己解决，中央不会干涉。
  <br/>
  <br/>
  有人认为，这一表态是因为上周国防部新闻发言人吴谦在回答香港记者提问时说：“部分激进示威者的行为挑战中央政府的权威，触碰‘一国两制’的原则底线，是绝对不能容忍的。东方之珠不容玷污。关于你提的具体问题，驻军法第3章第14条已有明确规定。”
  <br/>
  <br/>
  《中华人民共和国香港特别行政区驻军法》第三章第十四条内容如下：
  <br/>
  <br/>
  香港特别行政区政府根据香港特别行政区基本法的规定，在必要时可以向中央人民政府请求香港驻军协助维持社会治安和救助灾害。
  <br/>
  <br/>
  香港特别行政区政府的请求经中央人民政府批准后，香港驻军根据中央军事委员会的命令派出部队执行协助维持社会治安和救助灾害的任务，任务完成后即返回驻地。
  <br/>
  <br/>
  香港驻军协助维持社会治安和救助灾害时，在香港特别行政区政府的安排下，由香港驻军最高指挥官或者其授权的军官实施指挥。
  <br/>
  <br/>
  香港驻军人员在协助维持社会治安和救助灾害时，行使香港特别行政区法律规定的权力。
  <br/>
  <br/>
  因此，吴谦发表上述言论之后，外界普遍怀疑这是中央要干涉香港事务的信号。港澳办新闻发言人杨光的表态，大概是为了澄清中央在这一问题上的态度，暗示中央并不打算干涉香港事务。
  <br/>
  <br/>
  其实笔者从不相信驻港部队会根据《驻军法》第3章第14条出动，“协助维持社会治安”，“任务完成后即返回驻地”，原因很简单：因为这样解决不了什么问题。
  <br/>
  <br/>
  举个例子，即使驻港部队真能顺利完成“协助维持社会治安”的任务，但是如果香港反对派改变斗争策略，改成号召所有人待在家中罢工罢课罢市，驻港部队能够逼人们出门上班，或者代替人们去完成工作吗？所以中央如果想要干涉香港事务，仅派驻港部队“协助维持社会治安”，“任务完成后即返回驻地”，肯定是不够的。
  <br/>
  <br/>
  那么，如果中央想要彻底接管香港，把“一国两制”变成“一国一制”，又会怎样呢？
  <br/>
  <br/>
  第一，香港的普通法体系与内地的法律体系完全不同，要想彻底改造，会是一个极为巨大的工程。试想，如果香港的各级法院中都是一些不熟悉本地情况的大陆法官在审案，这会是一个什么情形，更不用说这样做一定会造成巨大的矛盾冲突了。
  <br/>
  <br/>
  第二，如果中国这样做，香港一定会失去自由港的地位，台湾人民也会对“一国两制”彻底失去信心。愤青们可能觉得这两点无关紧要，但是领导们可不会这么认为。因为在国际社会封锁或者制裁中国的时候，自由的香港往往充当了中国唯一的绕开封锁或者制裁的通道。如果香港失去自由港的地位，中国在中美贸易战中的处境一定会艰难百倍。
  <br/>
  <br/>
  虽然愤青常常叫嚣“香港已经不重要了”，但是中央可不像愤青那么没脑子。在香港和愤青之前，中央一定会选择香港。
  <br/>
