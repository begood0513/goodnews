 <p>
  李鹏遗体告别式，一众大佬缺席，引起外界议论纷纷。
  <br/>
  <br/>
  这次李鹏遗体告别式，新华社报道，除了现任七常委加王岐山出席外，仅提到两位前总书记，江泽民和胡锦涛，一个是到场，一个是在外地送花圈，其余退休元老，包括李瑞环、朱镕基、温家宝、曾庆红等均未提及。于是不少人解读为上层权斗，解读为这些退休元老和李鹏划清界限。我认为这是解读过度。
  <br/>
  <br/>
  查看2015年万里的遗体告别式和乔石的遗体告别式，也是除了现任国家级正职领导人即现任政治局常委外，仅提到两位前总书记江和胡（或到场或在外地送花圈），也没有提到其他退休元老，包括其他退休的政治局常委。
  <br/>
  <br/>
  早先，党国领导人遗体告别式，新华社播发的通稿，会有一个长名单，退休元老基本上都会提及。不过据了解，自十八大以来，中共当局对此类报道采用了规范化、模式化的处理。新华社在播报党国领导人遗体告别式时，除现任国家级正职领导人和前任总书记之外的领导人，其他退休元老，包括其他退休政治局常委，一律不再提及，都以“等”字代替。
  <br/>
  <br/>
  按理说，国家级正职领导人是指以总书记为首的政治局常委。国家副主席一般不是由政治局常委出任，并不算国家级正职领导人。在万里和乔石去世的时候，国家副主席是李源潮，李源潮是政治局委员但不是常委，因此他的名字并没有出现在万里和乔石的遗体告别式的报道上。这次李鹏遗体告别式，却把既不是政治局常委、也不是政治局委员、甚至连中央委员都不是的国家副主席王岐山写进了报道。这明显是一个例外，虽然并不令人意外。
  <br/>
  <br/>
  在当局发布的李鹏同志逝世讣告中，李鹏被评价为“中国共产党的优秀党员，久经考验的忠诚的共产主义战士，杰出的无产阶级革命家、政治家，党和国家的卓越领导人”。查看2015年万里的讣告和乔石的讣告，当局给予的评价也都是“中国共产党的优秀党员，久经考验的忠诚的共产主义战士，杰出的无产阶级革命家、政治家，党和国家的卓越领导人”——一字不差，连标点符号都一样，整个一替换词练习。
  <br/>
  <br/>
  由此可见，这里有统一的规格和模式，不会因人而异。因为李鹏和万里、乔石都出任过党国一级领导职务，所以对他们评价和告别式的规格都一样。中共高层当然是尔虞我诈、勾心斗角，但是在包括死后评价和告别式报道等问题上，当局的做法是一律对待，而把他们之间的矛盾统统掩盖起来——除了那些因为“犯错误”下台的。因为这些党国领导人各自的经历不同，因此，例如在讣告中，当局对他们业绩的陈述不会那么模式化，而会有所不同。
